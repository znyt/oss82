        这天中午，连星儿和顾千凝一起在食堂吃饭的时候，明司寒端着自己的餐盘突然坐了过来。
<br/>
<br/>        他在连星儿的对面落座，顾千凝见状，连忙起身，端着餐盘坐在了连星儿的身边。
<br/>
<br/>        “我发现，这两天有个男的总是在暗中盯着你。”明司寒打趣道，顺手将自己餐盘里的一个大鸡腿，夹到了连星儿的餐盘中。
<br/>
<br/>        连星儿毫不犹豫地夹起明司寒送给自己的鸡腿，长大嘴巴咬了一口后，一边咀嚼一边含糊不清地说道：“你说的是郎安吗？”
<br/>
<br/>        “你又开始了见一个喜欢一个的本性吗？”明司寒接着打趣。
<br/>
<br/>        坐在一旁的顾千凝，偷偷瞥了明司寒一眼，直觉上感觉，这位明大校草是打翻了醋坛子。
<br/>
<br/>        可偏偏连星儿却听不出来，反倒是跟明司寒斗起嘴来：“对啊！我就见一个喜欢一个怎么着？郎安他确实长得挺帅的吧！”
<br/>
<br/>        明司寒不禁扯了扯嘴角，目光落在了连星儿的身后。
<br/>
<br/>        只见她身后的那桌，坐着一个留着黑灰色斜刘海短碎，身着灰色衬衫的男生。
<br/>
<br/>        而他那双凛冽的眸子，隐隐地透着一丝令他不可小觑的寒光。
<br/>
<br/>        从直觉上来讲，明司寒觉得自己并不大喜欢这个名叫郎安的男生。
<br/>
<br/>        郎安无意间地抬眸，与明司寒四目相对。
<br/>
<br/>        隔着食堂里的聒噪声，以及各种饭菜的香气，郎安仍旧闻到了来自于明司寒身上的气味。
<br/>
<br/>        他的气味，令他似曾相识。
<br/>
<br/>        这种气味，让郎安觉得自己曾经好像与他有过一面之缘一样。
<br/>
<br/>        连星儿见明司寒看着自己这边发呆，于是抬起手来在明司寒的眼前晃了晃。
<br/>
<br/>        “喂！你看什么看呆呢？”连星儿好奇地问道。
<br/>
<br/>        明司寒回过神来，目光的焦点落回在连星儿身上时，冷不丁地回答：“你如果仍旧是这种见一个爱一个的性子，只怕将来，你是得不到真爱了。”
<br/>
<br/>        “我说你这人今天好奇怪哦！我得不得到真爱，跟你有什么关系？”连星儿不满地皱起了眉头。
<br/>
<br/>        明司寒只觉心头一揪，隐隐作痛后，却在强颜欢笑：“是吗？跟我确实没多大的关系。反正，我又不会是你的真爱。当然，你也不会是我的真爱。”
<br/>
<br/>        “是是是！我知道！你不用跟我强调！”连星儿顿时没好气地应和。
<br/>
<br/>        顾千凝忍不住偷偷地看了看连星儿，又偷偷地看了看明司寒。
<br/>
<br/>        总感觉这两个人有点像情侣之间的小吵小闹。
<br/>
<br/>        “你慢慢吃，我先走了！”明司寒还未吃几口，直接端起餐盘，起身离开。
<br/>
<br/>        连星儿继续啃着鸡腿，只是突然觉得这鸡腿的味道如同嚼蜡。
<br/>
<br/>        自从郎安来他们学校后，又惹来了一些闲言碎语。
<br/>
<br/>        而明司寒也在那天中午食堂里跟她说了那些莫名其妙的话后，连着两三天都未跟她再联系。
<br/>
<br/>        有个时候，连星儿会拿着手机，望着通讯录里明司寒的手机号码发呆。
<br/>
<br/>        久而久之，她都不知道自己能把明司寒的手机号码倒背如流。
<br/>
<br/>        “星儿，你发什么呆啊？赶紧抹啊！”耳畔忽然传来顾千凝的催促声。
<br/>
<br/>        连星儿回过神来后，才想起自己现在的任务是打扫图书馆的书架。
<br/>
<br/>        “咳——咳——”
<br/>
<br/>        书架的上层，落满了灰尘。
<br/>
<br/>        连星儿手中拿着鸡毛掸子，捂着嘴咳了起来。
<br/>
<br/>        而下一秒，原本握在手心里的鸡毛掸子突然飞走。
<br/>
<br/>        连星儿下意识地仰头望去，只见郎安手中拿着她的鸡毛掸子，面无表情地说道：“你去那边抹，这里我来弄就好。”
<br/>
<br/>        “噢！”连星儿讷讷地应了声，然后转过了身去，心里依旧在想着明司寒的事情。
<br/>
<br/>        要不，她主动给他发条短信问候一下？
<br/>
<br/>        不行不行！
<br/>
<br/>        明司寒那家伙似乎是真的不喜欢主动的女生。
<br/>
<br/>        连星儿手里拿着抹布，一边抹着书架上的板子，一边若有所思起来。
<br/>
<br/>        郎安微微侧头，看向那边的连星儿，脸上依旧没有任何表情。
<br/>
<br/>        当他们打扫的这层图书馆里四下无人的时候，郎安挪动着悄无声息的步伐，慢慢地向连星儿的身后靠近。
<br/>
<br/>        来自她身上的气息，逐渐扑鼻而来。
<br/>
<br/>        郎安缓缓抬起手来，冷冽的黑色瞳仁瞬变成了令人不寒而栗的金色。
<br/>
<br/>        连星儿莫名地感到背后发凉，下意识地转过身去。
<br/>
<br/>        郎安突然向她快速出手，连星儿吓得闭上了眼睛。
<br/>
<br/>        待她讷讷地睁开眼睛时，只见郎安的手伸向了她的头顶，缓缓落下来时，他的手掌中正托着一本厚重的字典。
<br/>
<br/>        “谢谢啊！”连星儿尴尬地笑了笑。
<br/>
<br/>        刚刚她还以为他又要攻击自己来着，没想到他只是为她接住了一本从上方书架上掉下来的字典。
