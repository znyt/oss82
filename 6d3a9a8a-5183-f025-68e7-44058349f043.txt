　　洪荒之中，在各路大神的关注下，这突然冒出的万劫魔尊力抗准提圣人，使得三界之中无人不惊，万劫魔尊之名，算是彻底响彻三界了。
<br/>　　混沌之中，女娲圣人与接引圣人在大战，一个造化之道，一个佛家寂灭之道，看上去仿佛有些相互克制，造化之道，造化万物，创造万物，寂灭之道，寂灭天地。
<br/>　　大战动静很大，甚至是惊天动地，但是除了几位圣人之外，没有几人能够看清，也没有几个敢前去观看，圣人大战，道法惊天，一个余波，便能将一些大神通者毁灭。若是一个不小心，刚好碰上这事，那还不倒霉？
<br/>　　当然，几位圣人就没那个顾及了，他们圣念伸了出去，延伸到混沌之中，看着女娲战准提。虽然三清希望女娲能够赢，但事实上，女娲还是落入了一定的下风。
<br/>　　准提不愧为准提，毕竟算是有大毅力，能够坚持斩去三尸之人，而女娲则只是斩去一尸，便以功德成圣了。是故，女娲终究是弱了一点。当然，这也有女娲之道，不擅长战斗的原因。
<br/>　　西方大地、苍穹之巅，万劫魔尊全身劫魔之气滚滚，他妖异的眼瞳，乱舞的黑发，嗜血的眼神，无上的魔身，凶悍无比，虽然一直落入下风，但也没有被准提击杀或者擒住。
<br/>　　在他的周身，一道黑色的魔域形成，在里面，魔气翻滚，煞气惊天，劫气弥漫，一道道嗜杀好战的劫魔在里面翻滚出现，他们手持兵器，与万劫魔尊一同大战。
<br/>　　“谁言圣人不可战胜？哪个敢称无敌？今日本尊定要以我魔血，溅染青天！吼！”
<br/>　　万劫魔尊面目狰狞，口出狂言，手中魔刀飞舞，带动无上魔威，斩向准提圣人。
<br/>　　“嗯？他的实力在增强？难道是……”
<br/>　　准提圣人眼中闪过一丝怒火，看向万劫魔尊，感受到他的实力，心中若有所思，想到这万劫魔尊想必是要借着他的压力来突破混元。准提有些恼火，若是没有突破还好，若是真让他突破了，那还了得，以后贫道岂不是成为别人的笑柄了吗。不行，这绝对不行。
<br/>　　他能够想到，以后别人提起万劫魔尊，便会想起他准提圣人，想到万劫魔尊是借着他准提圣人上位的。想到这里，准提心中就憋着一股怒火，手中抓起七宝妙树，周身法力狂涌，各种释道神通法术如暴雨般落下。
<br/>　　“发现了吗，可惜没用了。呵呵。”
<br/>　　万劫魔尊心中冷笑，出手狠辣，他周身环绕着魔域，覆盖苍穹，魔威尽显，手中持着魔刀，威力惊人。他越战越凶，越战越强，体中魔界已经在飞速完善，只剩下最后一点，便可破入中千世界之列，魔道之源也在飞速凝聚，只差最后一小道缺口，便可圆满。
<br/>　　“战！万道森罗，寂灭魔道！”
<br/>　　万劫魔尊在大吼，黑发乱舞，魔威震慑诸天，凶煞惊天，一刀既出，黑色刀罡布满天地，劫魔之气，铺天盖地，魔道天地，寂灭万物，神通寂灭，法力寂灭，生命寂灭，一切都将归于寂灭。
<br/>　　越是到后面，万劫魔尊便越战越强，甚至，到了如今，准提圣人也只可以稍微压制一下万劫魔尊，不可如之前那般，全面压制万劫魔尊。时间久了，万劫魔尊自然也适应了圣人的手段，不至于那般被动了。
<br/>　　“一切有为法，如梦幻泡影，如露亦如电，应作如是观！”
<br/>　　准提圣人再次祭出广阔的菩提法相，慧光照耀，使得准提圣人高高在上，光辉洒遍天下，道韵汩汩，伟岸无比。让人看上一眼，便会倒头就拜，法力提升，智慧大增。
<br/>　　他一掌拍出，看似很慢，实则迅若闪电，轻飘飘的一掌，陡然让万劫魔尊如临大敌，全身寒毛颤栗，只觉得生死危机降临。
<br/>　　所有人看到那一掌，都会愈迷愈深，不肯回头，永被轮转，如梦，如幻，如影，如泡，如露，如电，莫非虚假，一切皆空，万事皆休。
<br/>　　“叱！万道森罗，劫灭天地！”
<br/>　　万劫魔尊发出一声低吼，震散心中的异样，同样一道黑色巨掌携着覆海翻云般气势对着准提的那一掌拍出，顿时，天公大怒，雷云翻滚，劫灭之气，如惊涛骇浪般狂涌而来，充斥天地，如量劫降临，毁天灭地，覆灭苍穹。
<br/>　　“万劫吾掌，魔道独尊！混元之境，给本尊破！吼！”
<br/>　　突然，只听万劫魔尊一声大喝，魔气纵横，劫运丛生，魔势屠天，黑发乱舞，豪气直冲云霄。
<br/>　　“轰隆隆！”
<br/>　　他全身一震，体内已经扩展到极致的小千魔界也在这一刻发出轰隆隆的响声，大陆在迅速扩展，之后分裂成三千小千世界，各自演化成一方世界，黑色的太阳，横贯于空，魔日凌天，照耀魔界。蓝色的月亮，隐藏虚空，日月星辰在快速扩大，按周天规律排列起来。
<br/>　　其余各小世界也在快速演化，天地山河，日月星辰，草木虫鱼，鸟兽飞禽，各种生物都在快速衍生，但他们都有一个共同的特点，那便是全身充满劫魔之气。
<br/>　　在万劫魔尊的体内，魔道本源也在这一刻彻底凝聚出来了，它就如同一道黑色的圆盘，上面刻着一道圆形轨迹，代表着混元一重天，本源圆盘在这一刻魔光大盛，各种魔道法则全部刻录在内，道韵流转，出现了一种圆满现象。
<br/>　　混元，即是圆满。
<br/>　　“轰隆隆！”
<br/>　　万劫魔尊的气势陡然一震，一股滔天的魔威冲天而起，凌驾诸天之上，镇压苍穹。九天之上，风云色变，日月失***气滔天，遮天蔽日，万鬼嚎哭，万灵拜服，劫魔之气纵横天地，一股魔中至尊之威，铺天盖地般压迫而来，惊天动地。
<br/>　　“万劫不灭铸我身，滔天魔道炼我魂。
<br/>　　天地日月藏我心，至尊主宰我为尊！”
<br/>　　“本尊万劫魔尊，今日入混元，谁敢阻我？！吼！”
<br/>　　万劫魔尊一声大吼，惊天动地，霎时间，天地之间，狂魔乱舞，魔气滔天，一尊堪比圣人的魔神，在此刻诞生了。
<br/>　　“嘶！”
<br/>　　“什么？万劫魔尊竟然突破了混元，成就混元大罗金仙，真是厉害！”
<br/>　　“突破了，这怎么可能？”
<br/>　　“竟然是真的，真的突破了，没有鸿蒙紫气，一样可以突破混元！”
<br/>　　“这怎么可能？！万劫魔尊是怎么做到的，没有凭借功德，更没有鸿蒙紫气，仅仅凭着自己的实力，便能突破混元，真是不可思议。嘶！”
<br/>　　“……！”
<br/>　　天地间各大神通者，各位圣人，其他万物生灵，都被这股惊天魔威给惊醒了，万劫魔尊这股滔天魔威，虽然没有圣人那般惊天异象，但气势完全超越了准圣，给人一种面对圣人的感觉，没有人怀疑，万劫魔尊还在准之境圣。
<br/>　　众人纷纷倒吸一口气，脸色惊骇异常，有的甚至已经惊叫起来。就连各路圣人都惊异非常，为此女娲与接引两位圣人都停止了大战，前来观看。
<br/>　　准提圣人脸色铁青，目中怒火焚烧，怒视着正处在风暴之中的万劫魔尊。三清也是脸色变得凝重起来，之前身为圣人的高高在上，在这一刻荡然无存，因为已经有后来人赶上了他们。
<br/>　　“轰隆隆！”
<br/>　　就在众人脸色各异之时，九天之上，风云汇聚，乌云翻滚，风暴不断，雷光电鸣，轰隆作响，一场天地灾劫，即将降临。
