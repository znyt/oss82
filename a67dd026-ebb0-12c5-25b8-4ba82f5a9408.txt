　　轰！
<br/>　　叶尘这话一出口，所有人全都炸了。
<br/>　　尽管，此人之前一剑斩杀了北山赤目，实力堪称惊艳。
<br/>　　可是他说这话，未免也太嚣张了些吧？
<br/>　　罗刹城虽然处于九大主城之末位，但也好歹有十万魔兵，近二十名御空境强者！
<br/>　　这家伙竟然敢口出如此狂言，莫不是脑子坏掉了？
<br/>　　就连纳兰百川听到这话，也是怒极反笑。
<br/>　　他原本还打算，
<br/>　　先摸清楚此人的实力，然后再寻找对策。
<br/>　　却万万没有想到，
<br/>　　对方竟然如此嚣张！
<br/>　　如此直接！
<br/>　　如此霸道！
<br/>　　一张口，就要屠城！
<br/>　　便是魔皇城的那位，来到这里，也不敢如此口出如此狂言吧？
<br/>　　“小子！你知道自己在说什么吗？敢在我罗刹城，说出这种话的，你还是头一个！”
<br/>　　纳兰百川两眼死死盯着叶尘，寒声开口道，身上的杀气，已经掩盖不住。
<br/>　　可惜这对叶尘丝毫没有任何影响，
<br/>　　叶尘嘴角微微一扬，
<br/>　　“不信？那你大可以试试看！”
<br/>　　纳兰百川顿时眉头一皱，便要对叶尘出手。
<br/>　　然而就在这时，从纳兰百川身后，忽然窜出了一道人影。
<br/>　　“何必劳烦城主大人亲自出手，我来教训一下这个狂妄的小子！”
<br/>　　这是一名满头白发的老者，不过气势浑厚，一看就知道修为不弱。
<br/>　　“是大长老！”
<br/>　　“据说大长老的修为，不在城主大人之下，比北门将军还要高出一线呢！”
<br/>　　人群中，顿时有人惊呼起来。
<br/>　　不过，在见识过叶尘之前的表现后，已经没人再敢小瞧他。
<br/>　　那大长老站出来之后，直接踏空而起，然后猛然一道暴喝，
<br/>　　“天残脚！”
<br/>　　轰！
<br/>　　天空之中，顿时出现了一道巨脚虚影，犹如一座小山般，自上方空中，狠狠朝叶尘踢落下来。
<br/>　　呼！
<br/>　　众人只感觉到一股，强悍无比的气势压迫，哪怕是身在百米之外，都生出一股难以形容的窒息感，心中纷纷暗叹，
<br/>　　“这就是大长老的成名绝技‘天残脚’吗？果然可怕！”
<br/>　　就连罗刹城城主纳兰百川，也不由得暗暗点头，
<br/>　　“大长老的实力又精进，怕是要不了多久，我罗刹城，终于要出一位冲霄境强者了！”
<br/>　　然而，
<br/>　　纳兰百川的这道念头才刚刚升起，心头又忽然生出一股极度不安的感觉。
<br/>　　只见另外一边，
<br/>　　叶尘缓缓一步迈出，然后遥空一拳砸出，
<br/>　　“虚空神拳！”
<br/>　　轰！
<br/>　　一道硕大的金色拳影，竟然穿透了大长老的天残脚，狠狠砸落在他的身上。
<br/>　　砰！
<br/>　　大长老整个人，直接便倒飞了出去，足足飞出近百米，撞毁各种建筑无数，淹没在一片废墟之中。
<br/>　　眼看就算没死，恐怕也已经身受重伤。
<br/>　　“什么！”
<br/>　　所有人再次惊呆！
<br/>　　尤其是十七公主和戴安娜两人，她们忽然无比震惊的发现，
<br/>　　从北山铁男、东门统领到北山赤目，再到如今的大长老，对方始终都是一招制敌，从来没有用过第二招！
<br/>　　要知道，眼前的这位大长老，已经几乎是她们罗刹城的最强者了，竟然依旧在他的手下，走不过一招！
<br/>　　这家伙的实力，究竟强大到何种程度？
<br/>　　就连纳兰百川，也是两眼瞪得老大，望着那道年轻的身影，身体不由得微微发颤。
<br/>　　他的实力，比大长老也就略胜一筹而已，对方既然可以一招打败大长老，那就同样也能击败他！
<br/>　　‘这家伙到底是什么来头？恐怕就算比魔皇城的那位，也差不了太多了吧？’
<br/>　　‘可是那位，已经足足活了近四百岁，而眼前的这个人，却还只是一名少年！’
<br/>　　‘难道此人，是某位精通驻颜之术的老怪物？’
<br/>　　‘又或者，这幅面容本身就是假的？’
<br/>　　就在纳兰百川，正思绪飞转之时，叶尘那冰冷的声音，已经再次响起，
<br/>　　“如何？城主大人还想亲自来试试吗？”
<br/>　　听到这话，纳兰百川顿时身体微微一颤，略微犹豫了片刻之后，当即上前一步，再次恭恭敬敬的行了一礼，道：
<br/>　　“不敢！前辈修为惊人，晚辈甘拜下风，前辈有什么吩咐，尽管开口！”
<br/>　　虽然，纳兰百川有自信，
<br/>　　如果集全城之力，未必不能与此人一战。
<br/>　　可是，想要灭杀此等强者，必定要付出几位惨重的代价！
<br/>　　而叶尘，也不想节外生枝，看到纳兰百川服软，于是点了点头，淡淡道：
<br/>　　“我说了，不需要你为我做什么，只要你将把那万年玄冰的下落，告诉我即可！”
<br/>　　听叶尘这话，纳兰百川也松了一口长气，连忙道：
<br/>　　“是是！那块万年寒冰，的确是世间少有的宝物，我原本想要自己留着，可惜却被天下第一商会的少会长阿里津看中。”
<br/>　　“我罗刹城偏僻穷困，很多地方都需要阿里商会的支撑，所以也不得不给他这个面子……”
<br/>　　纳兰百川生怕叶尘生气，当即将来龙去脉和原因，全都一五一十的讲了出来。
<br/>　　没等纳兰百川说完，叶尘摆了摆手，直接打断道：
<br/>　　“立刻带我去找阿里津！”
<br/>　　纳兰百川顿时露出一抹为难之色，
<br/>　　“前辈，阿里津此时人应该在魔皇城，而我还要处理罗刹城的大小事务，实在是走不开，要不我派人随您去，您看可好？”
<br/>　　叶尘不由得两眼微微一眯，
<br/>　　他又岂会看不出纳兰百川的心思，于是冷冷一笑，道：
<br/>　　“也好！那就让她们二人陪我走一趟吧！”
<br/>　　叶尘指了指戴安娜和十七公主两人。
<br/>　　纳兰百川顿时脸色微微一变，继而连忙陪笑道：
<br/>　　“戴安娜确实是不错的人选，不过小女莽撞，不懂礼数，前辈您……”
<br/>　　没等纳兰百川说完，叶尘直接一摆手，
<br/>　　“不用多说，就这么定了！”
<br/>　　“给你们十分钟时间准备，即刻随我前往魔皇城！”
<br/>　　说完这话，叶尘直接转过身去，双手负于身后，不再说话。
<br/>　　之所以要带上那十七公主，叶尘自然也是有自己的考虑的。
<br/>　　今天他杀了三个罗刹城的重要人物，难免会被这纳兰百川记恨，暗地里找他的麻烦。
<br/>　　只要他的女儿在自己的手里，他应该就不敢轻举妄动。
<br/>　　纳兰百川见叶尘态度如此坚决，也不敢再继续反驳。
<br/>　　而那十七公主，顿时小脸煞白。
