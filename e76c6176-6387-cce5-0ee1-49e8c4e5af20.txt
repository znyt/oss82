
王惊蛰看着地上羚羊的内脏，碎骨还有几处血迹，烦躁的摇头说道：“算不了，这头藏獒连三头不化骨僵尸都能干掉，你觉得一头羊肉串有什么资本能让它受伤和流血？这地上肯定不会有它的血迹啊，我拿什么算？”<br/>　　<br/>　　古老的占卜，算卦的方式中，也不光是用八字来算的，其中还有用鲜血，骨头甚至是毛发的占卜方式，据说道行够深的算命先生，用一根人骨就能把对方祖宗往上几代给推出来，王惊蛰也懂此道，虽然达不到太精深的地步，可要给他足够的媒介他也能推出来，只是说白了那头獒王是不可能受伤的，他自然就算不了了。<br/>　　<br/>　　孔良说道：“那没办法了，我们只好顺着它的痕迹继续找了，但愿这头獒王还能留下点线索吧”<br/>　　<br/>　　獒王留下的线索暂时还没有断，它离去的方向幸好也不是继续朝向高海拔地区的，这让人欣慰了不少，如果獒王一直往雪山腹地海拔六七千甚至八千以上的地方走，那他们三个可就一点都没有往下继续追的心思了，因为海拔越高危险性越大，光是御寒一项他们就很难做得到。<br/>　　<br/>　　这头獒王的移动轨迹，是朝向喜马拉雅山脉的南麓走的，斜线方位，这个海拔高度还能维持在有绿植和生物的环境内，对他们来说这无疑还是有机会的。<br/>　　<br/>　　于此同时，三井美奈的队伍，也在朝着喜马拉雅山脉南麓的方向前进，他们是根据导航仪上定位走的，由于准备的相当充分，再加上一路始终都保持着急行军的速度，所以原本和王惊蛰三人的直线距离也就几公里远后，如今正在逐步的拉开着，此时已经差不多有七八公里左右了，处于的海拔高度也比他们低了一点。<br/>　　<br/>　　也是在这时，黄九郎和鲁靖尧也进入了山脉中，不过他的速度却略微慢了不少，反倒是像在吊着目标走的。<br/>　　<br/>　　一处山坡中稍微平坦点的区域，三井美奈这支队伍每隔一个半小时以后，就会停下来做些调整，补充水分和热量，以其能够始终都保持着充沛的体力。<br/>　　<br/>　　三井美奈喝了口矿泉水，然后吃了点压缩干粮和牛肉干，慢慢的消化了一会后，从口袋里拿出一根细长的香烟吞云吐雾了几口。<br/>　　<br/>　　三井美奈抽着烟，一手从登山包里拿出个笔记本，本子显得很破旧，翻开后纸张都已经泛黄了，外面是小牛皮的包装，封面上印着个万字旗的标志，翻开第一后就是一篇篇的德文日记。<br/>　　<br/>　　“1945年三月底，我是马丁·鲍曼，我们伟大的帝国即将就要结束了它的使命，为此尊敬的元首大人派遣我和忠诚的卫军押送大批的帝国物资离开我们的土地，向着东方前进……”<br/>　　<br/>　　“四月初，我们的卫军进入原始森林地带，为了躲避联军的追击，卫军们一路上疲于奔波，再加上运送的物资过于庞大我们接连有很多士兵倒在了路上，人数正在急剧骤减着”<br/>　　<br/>　　“四月中旬，我们进入了帕米尔高原……”<br/>　　<br/>　　“一九四五年六月，历经千里跋涉，我马丁·鲍曼和卫军的士兵们，终于抵达了喜马拉雅山脉……这里，曾经是神秘的东方国度”<br/>　　<br/>　　“这里是喜马拉雅山脉的腹地，仰头就能看见巍峨的珠穆朗玛峰，四周全是峰顶常年积雪的雪山，但我们到达的这块腹地，景色却令人惊奇的优美……”<br/>　　<br/>　　这个泛黄的笔记本是个叫马丁·鲍曼写的日记，但到了三井美奈翻的这一页后就断了，笔记本的后面有着很明显被人撕过的痕迹。<br/>　　<br/>　　这个本子，早在几十年前就已经在三井家族的手中了，一直被当做重要的资料保存着，家族中除了一些直系成员以外任何人都没有权限阅读，因为它关乎着一个至关重要的秘密，一座消失的宝藏。<br/>　　<br/>　　据说，在一九四五年春天的时候，那个战争狂人，疯狂的种族狂热头子，知道自己的帝国最终要走向了毁灭，于是绝望的阿道夫就还想着做出最后的挣扎，以求能够东山在起，他命令自己的绝对心腹马丁鲍曼把这些年来他们疯狂搜刮的黄金和艺术珍宝都给转移走了，当时一共被分为了三批，一批运往色林吉的森林里，另外一批则送到了远巴伐利亚的南部，而最后一批则是被马丁鲍曼亲自押送跋涉千里送往了遥远的东方国度，最后进入了雪域高原中。<br/>　　<br/>　　但让马丁鲍曼没想的是，帝国结束的太快了，这三批有着惊人数量的黄金和艺术品珍宝其中有两批还没有抵达指定地点就被联军发现了，数以百吨的黄金全都被带走了，只有他们这一批物资由于行进的路线非常隐秘，并且极其遥远，这才躲过了一劫，而与此同时帝国的命运也在这个时候结束了，彻底走向了毁灭。<br/>　　<br/>　　据说，后来马丁鲍曼来到了藏区，带着价值连城的艺术品和珍宝进入了喜马拉雅山脉腹地，但那个时候的马丁鲍曼由于一直在路上，和阿道夫的联系是处于中断状态的，他并不知道帝国已经被端了，于是藏好这批艺术珍宝后，就从喜马拉雅山脉返回了。<br/>　　<br/>　　万万没想到的是，在半路上他们遭遇了联军堵截的军队，眼看着就要被擒的时候，这些阿道夫最忠诚的士兵全都自尽了，而马丁鲍曼这时恰好身处一条河岸旁，最后高举xxx万岁的口号时，纵身跳下了湍急的河水中。<br/>　　<br/>　　马丁鲍曼肯定不想死啊，他的身上带着藏匿宝藏的位置路线呢，就是三井美奈手中的这个笔记本，这是他们帝国再次崛起的希望啊，他还得要留着呢。<br/>　　<br/>　　辗转过去了几十年，这个本子不知为何到了三井家族手中，而随着中华大地各处解放，接着再因为各种历史原因，虽然得到了这份笔记，但三井家族一直都没机会过来探寻。<br/>　　<br/>　　一直到进入九十年代之后，机会才渐渐的来了，这些年间，三井家族在暗中以登山的名义派遣了不少的队伍，进入喜马拉雅山脉中，来寻找马丁鲍曼藏匿的这批帝国珍宝。
