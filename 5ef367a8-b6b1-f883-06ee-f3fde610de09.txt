        “你还记得傅夜沉？”看到纸上的素描画，裴金澈诧异地看向童筱颖。
<br/>
<br/>        他没想到，这个女人忘了全世界，却没有忘记傅夜沉。
<br/>
<br/>        由此可知，傅夜沉在这个女人心里是有多重要。
<br/>
<br/>        就在裴金澈正感慨的时候，童筱颖面无表情地询问：“真有这个人的存在？”
<br/>
<br/>        很明显，她还是把他给忘了。
<br/>
<br/>        裴金澈顿时疑惑不解：“你怎么会有他的肖像画？”
<br/>
<br/>        “我做梦的时候，经常会梦到他，但是不知道他的名字。所以，就把他在梦里出现的样子画了下来。”童筱颖将纸从裴金澈手中拿了回来，小心翼翼地叠好塞进了口袋中。
<br/>
<br/>        裴金澈大概明白是怎么一回事了。
<br/>
<br/>        或许，这是她的潜意识记忆。
<br/>
<br/>        于她而言，这是好事。
<br/>
<br/>        裴金澈倏然想到了什么，接着说道：“你从实验室里掏出来，他们一定会想尽办法把你抓回去。为今之计，你得想办法找到你画上的这个男人。他会保护你。”
<br/>
<br/>        “我该怎么找到他？”童筱颖反问，然后一本正经地复述，“裴姨只说让我带着解药来找你，没说要我去找他。”
<br/>
<br/>        “是她放你逃出来的？”裴金澈顿时恍然大悟。
<br/>
<br/>        他不明白，那个生了他、赐予他姓氏，却只把他当实验品的女人为什么要这么做？
<br/>
<br/>        难道是母爱泛滥了吗？
<br/>
<br/>        呵！
<br/>
<br/>        怎么可能！
<br/>
<br/>        “既然你也不知道该怎么找到他，那就算了。”童筱颖答非所问，“我先走了。”
<br/>
<br/>        “你去哪？”裴金澈连忙握住童筱颖的手肘。
<br/>
<br/>        童筱颖微微侧身，提起手中的药箱摇了摇：“我得去把这个贵重物品藏起来。”
<br/>
<br/>        “你要是没地方去，就去找连华生，这是他的手机号码。他虽然是连叔叔的儿子，但是他不会出卖你。他是向着傅夜沉这边的。”裴金澈说完转身去了卧室，拿来了手机，直接塞到童筱颖的手中，“手机解锁密码是三个二和三个六个。”
<br/>
<br/>        裴金澈说完，又让夏玄灵去拿来了衣服和鞋子。
<br/>
<br/>        滨江城的冬天虽然不冷，但外面依旧很凉，她童筱颖就这个样子在外飘荡，身子骨再硬也会着凉。
<br/>
<br/>        “谢谢。”童筱颖握紧手机，接过衣服和鞋子，在夏玄灵的带领下去了房间里换上。
<br/>
<br/>        若不是看着她童筱颖换衣服，夏玄灵还真未发现，她童筱颖是真的很瘦很瘦，腰身细得好像可以用一只手就能握住一样。
<br/>
<br/>        而她两只手臂上还埋有采血用的暗管，像一条蚯蚓藏在她的皮肤之下微微凸起，看着有点渗人。
<br/>
<br/>        “筱颖，其实，妈妈她很想你。”看到这里时，夏玄灵也不知道自己为什么会将这句话脱口而出。
<br/>
<br/>        她在夏家长大的那十八年里，真的是被夏家的人给宠大的，特别是妈妈，只是后来，她才知道，妈妈只是把对那个亲生女儿的爱，寄托在了她这个冒牌女儿身上罢了。
<br/>
<br/>        童筱颖听着夏玄灵的话，只是淡漠地瞥了她一眼，好像她嘴里所说的那个“妈妈”跟她童筱颖一点关系也没有，所以对她童筱颖来说，“妈妈”只是一个无关紧要的人。
<br/>
<br/>        就像裴金澈和他的生母之间的关系那样疏离。
<br/>
<br/>        夏玄灵陪着裴金澈一起送走童筱颖后，讷讷地说道：“其实，她就是那个真正的夏家少小姐。”
<br/>
<br/>        “对。”裴金澈不假思索地点头。
<br/>
<br/>        “那你为什么不让她去找我爸？她是我爸的亲生女儿……”
<br/>
<br/>        “你爸和华生的父亲，早就串通了。只是这件事情，华生他还不知道。”裴金澈打断了夏玄灵的话。
<br/>
<br/>        “爸爸怎么那么残忍，拿自己的亲生女儿做实验。”夏玄灵不禁感到一阵唏嘘，在她的记忆里，夏阳枭是位慈父，却没想到他对自己的亲生女儿这般残忍。
<br/>
<br/>        “这件事情，你既然已经知道了，就当什么也不知道比较好。”裴金澈若有所思地说道。
<br/>
<br/>        夏玄灵不禁纳闷道：“你为什么不把这些事情全部告诉华生？”
<br/>
<br/>        “我已经告诉华生，他父亲在做这种研究，倘若再告诉他，夏叔叔也是这一伙的，只会给华生带来麻烦。更何况，他所说的话，华生不一定全信。与其让他左右为难，还不让他自己去发现真相。”裴金澈凝了凝眉回答道。
<br/>
<br/>        夏玄灵忽然想到了什么，连忙摇了摇裴金澈的手：“我想回夏家一趟。”
<br/>
<br/>        “回夏家做什么？”裴金澈有些纳闷。
<br/>
<br/>        “告诉我妈，让她知道，她的亲生女儿已经找到了，并且还好好地活在这个世上。”夏玄灵振振有词地回答。
