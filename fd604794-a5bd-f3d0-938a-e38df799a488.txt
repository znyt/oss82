　　黑耀十方石，青雀血，天界果。
<br/>　　这三件宝物的价值，在场的任何武者都知晓。
<br/>　　谁能够得到这三件宝物，就算是没有得到云羽天宫的传承，也足以一步登天。
<br/>　　云羽天宫的传承虽好，但是现在可还是在虚无缥缈之中。
<br/>　　这三件宝物，可是如今可以眼睁睁的看着的。
<br/>　　几乎一个瞬间，十几名武者便是同时出手，身形一动，一下子踏入到了这彩虹阶梯之中，猛然冲了上去。
<br/>　　“一群蝼蚁，也敢跟本尊争抢？”
<br/>　　突然之间，一声冷哼猛然爆发而出。
<br/>　　就在一旁的荒圣君，冷声咆哮一声。
<br/>　　在他身边的无数荒古门弟子，纷纷出手，无数武技直接冲着彩虹阶梯轰击过去。
<br/>　　那冲进入的十几名武者，便是在瞬息之间便是全部在这无数武技之下，纷纷身躯破碎，一个个当场被轰杀。
<br/>　　“该死！”
<br/>　　“那些荒古门的人！”
<br/>　　其他一些没有被波及到的武技，都是有些心有余悸，这些荒古门弟子出手太狠。
<br/>　　一招轰杀，丝毫不给机会。
<br/>　　但是这霸道的作风，正是七大宗门的行事作风。
<br/>　　“谁敢来跟本尊争夺，谁就得死！”
<br/>　　荒圣君看着周围的武者，冷哼一声，他的身躯猛然一动，直接向着那彩虹阶梯，走了出去。
<br/>　　“哦？荒圣君，别人怕你，我可不怕你！”
<br/>　　就在这荒圣君话音未落的一刻，在一旁又是一道身影突然之间便是冒了出来。
<br/>　　这道身影，正是那同为七大宗门之一，药王谷的弟子，药海。
<br/>　　他的身躯，也是几乎同时，也向着彩虹阶梯走了过去。
<br/>　　“药海？”
<br/>　　荒圣君看到这药海出现，冷笑道。
<br/>　　“就凭你，也敢来跟我争？”
<br/>　　“有何不敢，这云羽天宫的宝藏，人人可得，可不是只有你们荒古门才可以得到的！”
<br/>　　又是一道声音响起。
<br/>　　一道女子的身影，突然冲了上去。
<br/>　　天枢殿的星筠。
<br/>　　如今她也是一步上前，速度丝毫不落后与药海，荒圣君二人。
<br/>　　荒圣君的话，其他宗门的弟子，也许都会忌惮一二。
<br/>　　但是这七大宗门的弟子，却不一定会真的在意，显然这药海，星筠二人，丝毫不在乎荒圣君的威胁。
<br/>　　“星筠，你也要跟我为敌？”
<br/>　　荒圣君冷笑连连道。
<br/>　　“没想到剑祖荒，鹿战神那两个废物死了，你们倒是又跑出来跟我作对了。”
<br/>　　“为敌？云羽天宫的宝藏，大家各凭本事，荒圣君，你就不用来开口威胁了吧。”
<br/>　　又是一道身影，从古月峡谷弟子之中，脱颖而出。
<br/>　　是这古月峡谷弟子中的首领，古白。
<br/>　　这如今在场七大宗门之中的强者，纷纷尽出。
<br/>　　“各凭本事，很好，本尊不过是在善意提醒诸位而已，你们若是想要来争，也无妨，不过倒时候可不要怪本尊手狠！”
<br/>　　荒圣君扫视了一眼众人，冷哼一声。
<br/>　　他也不在废话，身躯一动，猛然向着彩虹阶梯上冲了过去。
<br/>　　另外古白，星筠，药海三人，也是毫不示弱，身躯一动，全部追了上去。
<br/>　　眼前这三件宝物，似乎是变成了这四人的争夺战。
<br/>　　其他武者，站在下面，一个个的眼神中尽是不甘，却无人主动出手。
<br/>　　在场的大部分武者，都不可能有着跟他们四人抗衡的机会，尤其是他们四人，代表的更是七大宗门。
<br/>　　这个时候，谁敢冒出来，跟他们去争抢，便是在跟七大宗门为敌。
<br/>　　在九域之中，敢跟七大宗门为敌之人，简直就是在自寻死路。
<br/>　　就在这众人，都以为这三件宝物，肯定是会落入到这四人手中一刻。
<br/>　　又是一道身影，从这人群之中冲了出去。
<br/>　　战龙宝车，咆哮一声，速度快到极致，如同一道光芒一般，直接踏入到了彩虹阶梯之上。
<br/>　　“什么人？”
<br/>　　“又有人敢出手？”
<br/>　　“是谁？”
<br/>　　无数目光，突然看了过去。
<br/>　　这一道身影，自然是楚风眠跟寒月漓，他们二人坐在战龙宝车之上，御使宝车，直接向着彩虹阶梯冲了上去。
<br/>　　这三件宝物，楚风眠是势在必得，怎么可能会这样白白让给其他人。
<br/>　　“一个生死境五重的小子？”
<br/>　　“这小子冲上去干嘛？找死？”
<br/>　　当不少人看到楚风眠，是一张陌生的面孔的一刻，一个个脸上都露出了几分嘲笑。
<br/>　　这种实力，敢跟这四人争，简直就是在找死，自寻死路。
<br/>　　“恩？”
<br/>　　就连那荒圣君四人，都是转过头啦，看到楚风眠，一个个有些惊讶，目光冰冷。
<br/>　　“一个生死境五重的小子，也敢来跟我们争夺，找死？”
<br/>　　古月峡谷的古白，更是直接冷喝一声，突然便是凝聚灵力，向着楚风眠一掌轰击过来。
<br/>　　这一掌轰击而出，周围的光亮似乎都被吞噬，铺天盖地的黑暗席卷而来，将楚风眠直接打入到了黑暗之中。
<br/>　　这一掌，将楚风眠都是给打入到了极致的黑暗之中，似乎是要永远困在黑暗中，无法脱身。
<br/>　　楚风眠坐在战龙宝车中，面对这一掌袭来，一动不动，任由这一掌轰击过来。
<br/>　　就在这无尽黑暗快要将楚风眠的身躯都给包裹起来的一刻，在他坐下的战龙宝车，突然便是咆哮一声。
<br/>　　八尊远古战龙，同时龙啸而起。
<br/>　　周围的黑暗，瞬间被冲散，而楚风眠，则是毫发无伤，甚至是坐下的战龙宝车，都是毫不停留，直接向着彩虹阶梯上冲了过去，已经是逼近了他们。
<br/>　　“这是什么灵器？宝车？九魔城的天魔宝车？这天魔宝车怎么会在你的手中，还变化为这一副样子？”
<br/>　　古白看着楚风眠，一脸吃惊。
<br/>　　他的出手，被这宝车轻易击破，能够做到这一切的，唯有九魔城的天魔宝车。
<br/>　　可现在楚风眠坐下的战龙宝车，却是跟当初的战龙宝车，力量截然不同，让他一时间都没有认得出来。
<br/>　　“你得到了天魔宝车又如何？凭借一件灵器也妄想跟我们抗衡？”
<br/>　　古白大喝一声，再度出手，无尽灵力在他的手心中，顿时化为了一道黑色弯月，顿时降临，向着楚风眠轰击而来。
<br/>　　“给我破！”
<br/>　　突然之间楚风眠从这战龙宝车上站了起来，一步之间，突然冲了过去。
<br/>　　他的右手，顿时化为了战龙之爪，一爪轰击过去，这黑色弯月直接破碎，逼得古白直接后退连连，他身上的灵力，都直接全部崩溃。
