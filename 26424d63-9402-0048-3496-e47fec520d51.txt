　　等他们走了之后，崔府君微微转头，不带任何情绪地看了摇光仙子等人一眼，看得众人心里发虚。
<br/>　　摇光仙子深吸一口气，说道：“我阐教东岳大帝，曾与阴司约定，青冥界之事，阴司不得插手。今日崔府君是要亲自撕毁约定了？”
<br/>　　一席话说出来，连身边黎山老母都有为她捏一把汗，崔府君毕竟不是钟馗，钟馗虽然名头大声望高，但毕竟自己不愿意掌权，挂职元帅，但现在也不掌兵了，只是一个散人，虽然是最牛逼的散人。
<br/>　　但崔府君不同，他是阴司真正的掌权派，地位崇高，虽然一向温文尔雅，但基本上从来没人敢当面呛他。不过他们也都知道，事情到这一步，也只能硬顶着了，不然真的是前功尽弃。
<br/>　　黎山老母也紧接着说道：“府君可知道风是转世鬼童，阴司难道放任不管，并且反而做出有利于他的事？”
<br/>　　一群人目光灼灼地望着他，等着他的回答，毕竟就算是崔府君，做事也不能不讲道理。
<br/>　　崔府君望着他们，脸上表情仍然没有一丝变化，缓缓说道：“我来，与这些事都没有关系，我只是救我一位下属，一位很重要的下属。我……很生气。”
<br/>　　“很生气”这三个字，让所有人心头一颤。
<br/>　　“我以天子之名保证，即日起，凡空界生灵，擅离空界者，一概视为孤鬼游魂，被阴司捉拿之后，即便押送轮回……”
<br/>　　崔府君说完，不看任何人，转身朝山谷外面缓缓走去。
<br/>　　摇光仙子等人却是彻底怔住。
<br/>　　他们都明白，崔府君方才那听似普普通通的一句话，意味着什么。
<br/>　　从今往后，空界的任何生灵，再也不能离开空界前往人间或者鬼域……这是一种极为霸道的约束，但是从崔府君的口中说出来，没有人会怀疑它的真实性。
<br/>　　惹他生气的只是空界的几大宗师，但整个空界的生灵都跟着受到惩罚，这就是天子之怒，没有任何道理可讲。
<br/>　　捉鬼联盟的成员们却是转悲为喜，互相望着，都露出了轻松的笑意。
<br/>　　崔府君不惜违反阴律，亲自来这一趟，把橙子的尸体带走，自然有办法救活她。不为别的，只为他是崔府君。
<br/>　　叶少阳也从绝望中缓解过来，仔细回味之前脑海中涌现的东西，就像清醒后记不住来的梦境，只有一些粗浅的印记，完全想不起具体细节，他也没有工夫去想，在芮冷玉的搀扶下勉强站起来，却立刻又被人按着坐下了。抬头一看是广宗天师。
<br/>　　“祖师你来的真是及时，差一点我就挂了。”叶少阳哭笑起来。橙子获救了，之前缠绕的心结也打开了。
<br/>　　广宗天师一笑：“福兮祸之所倚，结局谁知道呢。”
<br/>　　叶少阳怔怔问道：“什么意思？”
<br/>　　“没什么意思，事情还没了结，你没时间调息了，我助你一臂之力。”说完捏着叶少阳一条胳膊，将罡气灌入他体内，引导他体内残余的罡气运转起来，修复受损的经脉和神识。
<br/>　　叶少阳也闭上眼睛，
<br/>　　“无量天尊，若真如此，我等都会成为空界的罪人……”普法天尊忧心忡忡地叹道，他已经想到了崔府君这句话将来在空间引起的连锁反应，那绝对不是小麻烦……
<br/>　　这时候，一直没开口的苏沫站了出来，环顾众人，说道：“崔天子金口一开，不可能更改了。已经付出了这么大的代价，如果现在放弃的话，那真的就是前功尽弃了。”
<br/>　　伸手一指道风，“一定要擒杀转世鬼童！”
<br/>　　黎山老母说道：“不错，一起上，死战到底！”
<br/>　　说完自己先朝着道风扑了过去。
<br/>　　道风提着打神鞭应战，冷冷道：“你已经输了。”
<br/>　　“我大意了。”黎山老母望着他，冷然一笑，“但是……你以为这就是我全部实力了？”
<br/>　　摇光仙子在一旁笑道：“师姐，是时候了。”
<br/>　　黎山老母面对道风张开双臂，十根手指在空中摇动，幻化出无数雪白的花瓣，在空中旋转，将自己完全包裹在中间，花瓣逐渐凝聚，形成一朵花的形状，等花瓣完全打开之后，叶少阳等人顿时就惊呆了。
<br/>　　“怎么会！”有人叫了起来。
<br/>　　花瓣里出现的，不再是黎山老母，而是一个妙龄的道姑，身穿一件雪白的道袍，头发高高盘起，用一根木簪子插着，两束头发从两只耳朵边上垂下来，亭亭玉立，五官十分精致，眉心处点着一点朱砂，看上去最多只有二十出头。
<br/>　　这是……黎山老母？
<br/>　　所有人怔在当场，不可思议地看着她。
<br/>　　摇光仙子走到她身边，莞尔一笑，说道：“师姐，我可是好久没见你这么漂亮的样子了。”
<br/>　　黎山老母冲她笑笑，转头看着道风的时候，脸上立刻蒙上了一层寒霜。
<br/>　　“你真以为，我就之前那点手段了？”
<br/>　　众人哗然。
<br/>　　原来……她真的是黎山老母。
<br/>　　“卧槽，返老还童啊！”
<br/>　　四宝一句话，道出了所有人的心声，望着眼前这个美貌娴静的少女，谁也无法想象她就是之前那个鹤发鸡皮、满脸皱纹的黎山老母……
<br/>　　四宝看见广宗天师在身边，于是问道：“祖师，这是什么法术，驻颜术？还是幻化出来的假象？”
<br/>　　广宗天师手拈胡须，摇了摇头，“不是假象，这才是她的本尊。至于是什么法术，我却是不知。”
<br/>　　四宝说道：“那你该跟她学学，也好返老还童，瞬间成帅小伙啊！”
<br/>　　广宗天师瞪了他一眼，“我就现在也比你好看！”
<br/>　　黎山老母望着道风说道：“几百年来，你是唯一一个逼我现出本尊的人。转世鬼童，不愧是转世鬼童。
<br/>　　两人说话的工夫，有守山弟子不断从山上下来。这些弟子，毕竟都是星宿海和其他门派的精英，虽然以寡敌众，在牺牲了大概三分之一人数之后，总算杀尽了血海万魔幡中的恶鬼邪灵，陆续下到山谷中，堵住了道风一干人等的退路。
<br/>　　其中就有几个梨山弟子。
