　　“这南域皇室的推算，就这样被瞒过去了？”
<br/>　　一旁倒在地上的司空轩，简直是不敢相信。
<br/>　　他们这些大势力的弟子，赖以依靠的保命手段，最为重要之一，便是这推算过去。
<br/>　　一旦是他们陨落，家族强者便是会出手推算过去，查找凶手，疯狂追杀。
<br/>　　正因如此，他们背后的家族，也就相当是他们的底牌。
<br/>　　在强大的强者，都不敢去招惹这种庞然大物，所以他们才一个个有恃无恐。
<br/>　　可现在，这南域皇室的推算，就被楚风眠这样轻而易举的祸水东引。
<br/>　　这简直是可怕到了极致，楚风眠化身远古战龙，他的气息就跟真正的龙族没有任何区别。
<br/>　　不管是如今楚风眠杀了谁，都可以将这祸水，引到龙族之中去。
<br/>　　司空轩本来还想要用这一点来威胁他的，可是现在看着云公子都死在了楚风眠的手中，司空轩的心中，已经是一片恐惧。
<br/>　　“北域之中，什么时候诞生了如此妖孽，这种手段，真的是一个神力境四重的武者可以做到的吗？”
<br/>　　这种手段，令司空轩的身体，都无比颤抖起来。
<br/>　　“楚风眠，你放了我好不好，我保证不将今天的话说出去，颜古龙帝洞府中的宝藏，也全部归你，只要你放过我。”
<br/>　　司空轩直接跪在了楚风眠的面前，大声的讨饶道。
<br/>　　白萱看到司空轩如此举动，都是不由的有些叹息，在这生死面前，就算是司空轩这样的天才，也不顾尊严，下跪讨饶。
<br/>　　其实越是像是司空轩这样的天才，他们也就越是怕死，他们资质极高，将来成就无限。
<br/>　　谁也不想早早死去。
<br/>　　“放了你？可笑，今日你们四人，都要死，你也去直接给他们作伴吧。”
<br/>　　楚风眠批了司空轩一眼，直接冷笑一声，手一捏，司空轩直接死在了楚风眠的手中。
<br/>　　“砰！”
<br/>　　司空轩的一身血气，都被楚风眠，尽数吞噬。
<br/>　　这司空轩若是有些骨气，甘愿受死，也许楚风眠还有心情去留他一名，将他收服也有可能。
<br/>　　如今这司空轩为了活下去，直接跪下，不要尊严，甘愿求饶。
<br/>　　楚风眠最看不起的便是他这种人，前世楚风眠手无缚鸡之力，在七大宗门的威逼之下，也不愿意求饶，宁愿自杀。
<br/>　　这种就连尊严都没有的人，就连做楚风眠奴隶的资格都没有。
<br/>　　“司空轩死，司空家肯定不会善罢甘休的，云公子在南域皇室之中，只是一个不太受宠的皇子罢了，死了也就死了。”
<br/>　　白萱看着楚风眠杀了司空轩，出言提醒道。
<br/>　　“但是司空轩可是司空家年轻一代中的第一天才，你杀了他，司空家肯定不会善罢甘休，以司空家的手段，查清楚今天的事，并不困难，必须要小心才行。”
<br/>　　楚风眠冷哼一声说道，脸上的表情，轻松无比。
<br/>　　“无妨，司空轩此人，也是必死，司空家的人想要查清楚今天的事，也不是一朝一日的事，只要给我一些时间，这司空家，也算不得什么，谁敢来报仇，谁就得死！”
<br/>　　“只要是实力足够，司空家又算得了什么，司空家的人敢来找麻烦，那我就迟早要毁了司空家！”
<br/>　　楚风眠冷哼一声。
<br/>　　他的实力，已经是足以抗衡一些生死境四重，五重左右的强者，只要是楚风眠的实力再进一步，那些生死境高阶的强者，他也能够抗衡。
<br/>　　倒时候司空家的人，想要来对付楚风眠，也不容易。
<br/>　　生死境高阶，在北邙学院中都是地位极高的长老了，至于生死境之上的，可都是一些不问世事的老古董。
<br/>　　只要是有着生死境高阶的实力，在这个北域，也就足以立足了，无人敢小看。
<br/>　　“走，先去颜古龙帝的洞府！”
<br/>　　楚风眠直接开口道。
<br/>　　兵来将挡水来土掩，如今重中之重，还是要提升实力才行，只要是实力足够，别说是对抗司空家，毁灭了司空家都不成问题。
<br/>　　在这颜古龙帝洞府之中，肯定是有着什么对楚风眠有着大用的东西，不然不会让如今楚风眠的血脉如此渴望。
<br/>　　这件东西，楚风眠一定要得到。
<br/>　　“轰隆！”
<br/>　　突然在外面，又是传来一声响声。
<br/>　　随后出现的，则是无数气息。
<br/>　　“哈哈哈哈，这洞府之内，果然另有乾坤！居然是一方小世界！不知道这其中，到底是有着什么宝物！”
<br/>　　无数身影，突然涌入到了这一方小世界之中。
<br/>　　这些人，许多人皆是一袭红衣，一个个皆是一身魔气，魔气纵横。
<br/>　　“是地魔门的人！”
<br/>　　白萱看到这些突然出现的身影，脸色突然一变，尤其是她一个个看向那些人的时候，更是脸色大变。
<br/>　　“白骨公子！”
<br/>　　“战魔尊者！”
<br/>　　“鬼谷圣子！”
<br/>　　“嗜血魔尊！”
<br/>　　每念到一个名字的时候，白萱的脸色都是变的苍白几分。
<br/>　　“这些都是地魔门的天才弟子，一个个的实力，不比我们北邙学院的内门弟子要低，甚至其中的一些佼佼者，更是足以媲美种子弟子的实力！”
<br/>　　“这些地魔门的人，怎么找到这里来的。”
<br/>　　“看来得知这颜古龙帝洞府的人，不只是司空轩一个，地魔门的人，也有人知晓这这个消息！”
<br/>　　楚风眠目光闪烁，看向这些突然出现的地魔门弟子，这些人的实力，每一个都不比司空轩等人要差。
<br/>　　甚至是楚风眠一眼看过去，其中的几尊强者，实力更是要比起司空轩等人还要强大的多。
<br/>　　这地魔门的弟子，足足是有着十多人，一下子涌入到了其中。
<br/>　　这还不止，在那入口中，突然一阵更加滂湃许多的灵力，一下子爆发出来。
<br/>　　“生死境！”
<br/>　　看到这一股魔气的一刻，白萱的声音都微微有些颤抖。
<br/>　　这一次来的人，居然是地魔门的一尊生死境强者。
<br/>　　这是一尊鹰鼻男子，浑身上下，充斥着一股滔天魔意，可以说是一尊恐怖到极点的人。
<br/>　　“这是不死魔尊！地魔门中的大人物！生死境六重的无敌人物，这样的老古董，居然出世了！”
<br/>　　白萱看到那鹰鼻男子，大声的开口了起来，语气中尽是不可置信。
