　　这个如幽灵一般出现在詹姆斯身后，并且把刀架在詹姆斯脖子上的人正是蛰龙。
<br/>　　对于蛰龙的突然出现，就连龙隐部队这边的众人都是非常意外，如果不是蛰龙对着罗昊悄悄做了一个手势，示意他不要知声，罗昊差点就惊呼出声了。
<br/>　　旁边，海豹突击队队员立刻转身把枪口对准蛰龙，同时他们也觉得自己的脖子有些发凉，如果刚才蛰龙要下杀手，他们中间有人已经死了。
<br/>　　听到詹姆斯对自己的称呼，蛰龙轻轻皱了皱眉，嘀咕着说道：“其实我挺不喜欢‘幽灵刺客’这个称呼的，说得我像鬼一样。”
<br/>　　你丫的来无影去无踪的，可不就是跟鬼一样？龙隐部队众人听到蛰龙的话，暗暗嘀咕。
<br/>　　蛰龙根本无视那些指着自己的枪口，从迷彩裤中摸出一块塑胶炸药当着所有人的面仍在甲板上，手里握着一个遥控器，说道：“一个太平洋舰队司令加上一队精锐的海豹突击队陪我一起死，怎么算我都不亏，我们要不要来赌一把，是你们枪快，还是我按遥控器的动作快？”
<br/>　　麦克米兰把枪口放低，同时示意身边其他海豹队员也把枪口放低。对于蛰龙这个人，麦克米兰也是有些了解的，中央情报局对他的评价是“不要跟他玩命，他玩起命来连撒旦都要让步！”
<br/>　　“聪明的选择！”
<br/>　　蛰龙对着罗昊招了招手，说道：“现在把你手里的东西给我！”
<br/>　　“我凭什么把东西交给你？”罗昊看着蛰龙反问道。
<br/>　　既然是演戏，那自然是要演全套。蛰龙也很快明白了罗昊的意图，晃了晃手里的遥控器，说道：“我拿这些人的命跟你交换！一个太平洋舰队司令，一队精锐的海豹六队队员再加上一个DEA卧底，你不吃亏，这么多重量级人物在你手里，就是坐在白宫办公室里那位也无法淡定。”
<br/>　　“谁不知道你们是一伙儿的，何必在这里演戏。”
<br/>　　詹姆斯冷眼扫了蛰龙一眼，说道：“虽然你从来没有承认过，但是大家都知道，你是中国军方的人。”
<br/>　　“饭可以乱吃，话不能乱讲。”蛰龙手中军刀在詹姆斯脖子上划出一道浅浅的口子，说道：“乱说话是要死人的。”
<br/>　　“我似乎没有其他选择的余地。”
<br/>　　罗昊拎着背包走上前。他不介意把背包交给蛰龙，P病毒落在蛰龙手上跟在自己手上没有任何区别，反正最后也是送进国内相关机构的研究所进行研究的。
<br/>　　罗昊走上前把背包交给了蛰龙，然后从他手里到了甲板上那块塑胶炸药的遥控器。
<br/>　　蛰龙把背包背在身后，对詹姆斯说道：“现在，把直升机上那两个人放了，并且让那架直升机立刻降落，不然你那架直升机上三个人就完蛋了。”
<br/>　　一名蒙着脸的战略行动组队员蹲在甲板上那架直升机上，手中握着一把MPX冲锋枪，左手也拿着一个遥控器，而在他脚边也扔着一块塑胶炸药。
<br/>　　詹姆斯眼睛微眯，最后还是做出了妥协，让那架李毅峰和慕容飞所搭乘的“黑鹰”直升机降落在货轮甲板上，并且将李毅峰和慕容飞两个人释放。
<br/>　　“好了，事情结束了，我也差不多可以离开了。”
<br/>　　蛰龙把军刀从詹姆斯脖子上移开，快速的收回刀鞘，然后拔出手枪指着詹姆斯，往后退去。
<br/>　　那名蹲在直升机上的战略行动组队员也跟着跳下直升机，拿出一个智能手机一样的平板电脑，在上面按动了几下，海面下出来充气泵充气的声响，紧接着一艘特种潜航器浮出了水面。先前，蛰龙和这名战略行动组成员就是利用这艘特种潜航器接近货轮，并且爬上货轮的。
<br/>　　詹姆斯心中不甘，非常不甘，原本他占据了绝对的优势，可是因为蛰龙的出现，这些优势瞬间消失无踪，被人牵着鼻子走。
<br/>　　“蛰龙，你觉得你真的能带走P病毒吗？”詹姆斯冷声说道：“你不承认没关系，但是我告诉你，你们的人现在还在联合行动司令部，你如果离开，他们就会立刻没命。”
<br/>　　“威胁我？”蛰龙冲着詹姆斯摇晃手指，说道：“你不配！”
<br/>　　敢对着美国太平洋舰队司令摇手指，并且对他说出“你不配”这三个字，麦克米兰还不知道在蛰龙有谁有这样的胆量敢这么做，这么说！
<br/>　　“你看过圣斗士没有？”蛰龙突然对詹姆斯问道。
<br/>　　“什么！？”
<br/>　　蛰龙的问题让詹姆斯一脸懵逼，根本不明白他想要表达什么。
<br/>　　“同样的招数对圣斗士没有用的。这么老掉牙的招数，你用了一遍又一遍，累不累啊？你有个女儿在加州大学读书，你的妻子是只为政府工作的知名律师，如果你有小动作，你就见不到她们了。”蛰龙对着站在自己身边的战略行动组队员使了个眼色，说道：“而且你手中的筹码已经失效了，根本起不了任何作用。”
<br/>　　那名战略行动组队员拿出一台平板电脑，在上面按动几下，随后蹲下身把平板电脑滑到詹姆斯脚边。
<br/>　　詹姆斯低头朝脚边的平板电脑看去，上面显示的是联合行动司令部里的情况，Damien已经失去了联合行动司令部的控制权，此时联合行动司令部的控制权已经到了三个跟眼前蛰龙和那名战略行动组队员同样装扮的人手里，Damien跟一只大闸蟹一样被五花大绑躺在地上。
<br/>　　詹姆斯拳头紧握，心中无比愤怒，但是却无可奈何。
<br/>　　李毅峰、慕容飞这些刚得知龙隐部队有战略行动组存在的人心中无比骇然，这些家伙的单兵作战能力也太强了，三个人居然控制了联合行动司令部，就算是有宋萱、云熙这些人的帮助，可他们所有人加起来也才只有十来个人，而且联合行动司令部可谓是守卫森严，一般人根本不可能摸进去。
<br/>　　不过想到蛰龙和眼前这名战略行动组队员神不知鬼不觉的摸上货轮，李毅峰、慕容飞这些人心中也就释然了，这些家伙可都不是普通人，不能常规眼光去看待他们！
