　　听闻微胖青年之言，苏莫心中瞬间就明白了。
<br/>　　这所谓的九大神朝，除了天池神朝和轮回神国之外，其它七大神朝，均是太古万族榜上那些顶级种族的国度。
<br/>　　轮回神国，想必就是轮回天尊以前的势力，也是帝一魂现在最有可能在的地方。
<br/>　　当然，轮回天尊转世这么多年，轮回神国具体什么情况，不得而知。
<br/>　　至于天池神朝，苏莫无法猜测，还需要对方讲解。
<br/>　　“天池神朝有八十一个界域？”苏莫又问道。
<br/>　　“不错，天池神朝的地域，占据了神域的三成大小，共分八十一个界域，有八十一个神王殿，我大夏神王殿便是其一。”
<br/>　　微胖青年点头，继续道：“加入了我大夏神王殿，对于你日后的成就，有极大的好处，你考虑清楚了！”
<br/>　　说到最后一句，微胖青年的语气，稍稍加重，明显有警告的意味。
<br/>　　这让苏莫心中暗凛，此人是在警告他啊，若是他不加入，估计不好离开。
<br/>　　苏莫沉默不语，没有立刻回答，心中急思，到底加入与否。
<br/>　　若是不加入的话，很难离开，这是肯定的事情。
<br/>　　再者，就算能离开，自己该去何处呢？
<br/>　　他有两个去处，一是去找绫裳，二是去混沌神朝。
<br/>　　不过，这两个去处都不太妥，其一，他不清楚绫裳在何地，天池神朝貌似很大，而神域的空间如此坚实，飞行都成问题了，还怎么去找绫裳。
<br/>　　其二，去混沌神朝的话，也是不妥。
<br/>　　因为当初在神路，青铜男子就知晓他修炼混沌之力，那么帝一魂知不知道呢？
<br/>　　若是帝一魂知道，会不会在混沌神朝守株待兔呢。
<br/>　　这倒是很有可能，毕竟，他解除了诸天神灵印，帝一魂肯定能感应到，估计已经做了很多安排。
<br/>　　混沌神朝什么情况，他也不清楚，并不是说去了，就能得到混沌神朝的培养。
<br/>　　苏莫思来想去，貌似留在大夏神王殿，反而更好。
<br/>　　因为，他所要面对的帝一魂，如今不知道修为高达什么地步了。
<br/>　　而他，需要时间，需要快速的强大起来。
<br/>　　大夏神王殿，或许是个理想的安生之处，他在这里默默进步，蛰伏在此地。
<br/>　　待修为足够强大之后，再行离开。
<br/>　　最起码，到时候，他能有对抗帝一魂的资本了。
<br/>　　而在大夏神王殿的这段时间，就是他积累资本的时候。
<br/>　　“上官昊，你考虑的怎么样了？”微胖青年郑重的问道，他大夏神王殿的飞升殿，平常上万年都难有一个飞升者，如今好不容易来了一个人族，他一定要留下来。
<br/>　　“阁下，我愿意加入大夏神王殿！”苏莫微微一笑，抱拳说道。
<br/>　　“呵呵，如此最好！”
<br/>　　微胖青年见此，面上也是露出了笑容，他自我介绍道：“我名伏卓，以后我们共同效力于大夏神王。”
<br/>　　“多谢伏兄了！”苏莫感谢道，虽然口中道谢，他心中却是不以为然，因为他若是不加入，估计对方立马翻脸了。
<br/>　　“既然如此，我现在便带你前去分殿！”伏卓长身而起，笑着说道。
<br/>　　分殿？此地难道不是神王殿？
<br/>　　苏莫心中疑惑，不过，他也并未多问，立刻点头起身。
<br/>　　随即，他跟着伏卓，离开了阁楼，走了出去。
<br/>　　原来，所谓的飞升殿和阁楼，均是坐落在一个宽敞的府邸之中。
<br/>　　府邸名为接引府，只有两名护卫把守，空空荡荡，显得极为荒凉。
<br/>　　出了接引府，苏莫便来到了一条僻静的街道上。
<br/>　　“护卫都这么强！”苏莫扫了一眼接引府门前的护卫，心中暗暗惊讶。
<br/>　　这两人的修为，均是虚神境四变。
<br/>　　不过，从这两名护卫身上，倒是反应了一个问题，神域也不是全是神灵，也有低阶修为的武者。
<br/>　　苏莫跟着伏卓，在街道上大步行走，转过小道，进入了大道之上。
<br/>　　这貌似是一座城池，街道非常的宽阔，武者众多，建筑连绵，恢弘大气。
<br/>　　很多的武者，身上的气息均是深邃浩大，整个人都闪烁着无形的神光，对人的心灵造成强大的压迫感。
<br/>　　走在街道之上，苏莫感觉到自己非常的弱小，这种感觉很多年都不曾有过了。
<br/>　　呼！
<br/>　　苏莫深深的吸了口气，很快便摆正了心态，既然弱小，那就要快速的强大起来。
<br/>　　此时此刻，苏莫最郁闷的，便是神识无法延伸太远，无法大规模的查探此地的情况，对于他来说，感觉是最大的束缚。
<br/>　　“伏兄，大夏界域有多大？”行走之间，苏莫向伏卓问道。
<br/>　　“很大，方圆数百万里！”伏卓道。
<br/>　　“数百万里？”苏莫闻言一怔，想法与伏卓恰恰相反，这也太小了吧？
<br/>　　要知道，就算是古周星，其上的地域，也有方圆几千万里不止。
<br/>　　“我们大夏界域，有一百个城池，全部由大夏神王殿统领，这里乃是千渊城。”伏卓又道。
<br/>　　“原来如此！”苏莫心中恍然，原来，这并不是在神王殿所在的城池，只是一座神王殿掌控的城池。
<br/>　　没过多久，两人来到了一座恢弘的府邸之前。
<br/>　　这府邸，比接引府不知道大了多少倍，院墙高耸，延绵数十里，其内恢弘高大的建筑，如同一座座拔地而起小山峰。
<br/>　　“大夏神王殿，千渊殿。”
<br/>　　府邸大门之上，匾额上是这八个鎏金的大字，神光千丈，霸气非凡。
<br/>　　千渊殿之前，守卫森严，足有十几名护卫，个个气息深不可测。
<br/>　　“伏卓，此人是谁？”
<br/>　　为首的一名灰衣中年护卫，看到伏卓和苏莫到来，随意的问道。
<br/>　　伏卓是千渊殿的弟子，他们都认识，只不过对方一般都住在接引府，很少来千渊殿。
<br/>　　“哈哈，他叫上官昊，乃是今日从下界飞升而来的人！”伏卓轻笑着说道，虽然对苏莫他面带一丝闻言，但是对于其他人，伏卓完全没有任何的架子。
<br/>　　这些护卫，可都是千渊殿的弟子，灰衣中年人可是他的师兄。
<br/>　　“什么？”
<br/>　　“下界飞升之人？”
<br/>　　“终于有下界之人来了！”
<br/>　　一众护卫闻言，个个好奇的向苏打量了起来，如同苏莫是一个异类一般。
<br/>　　他们不得不好奇，毕竟，从下界来到他们大夏界域之人极少，有时候数千年来一人，有时候十万年，都不会有一人前来。
<br/>　　苏莫眉头微皱，被所有人另类的目光围观，让他有些无语。
<br/>　　“诸位，有理了！”苏莫向众护卫抱了抱拳。
<br/>　　“才虚神三变，这修为也太低了吧？”
<br/>　　“你是怎么来到神域的？”
<br/>　　“听说下界贫乏，你能来到神域，想必天赋很高吧？”
<br/>　　“真是稀奇，虚神三变居然能来神域！”
<br/>　　一众护卫七嘴八舌，如同好奇宝宝一般，哪里像是神灵，均是如同下界小宗门没有见过世面的普通弟子一般。
<br/>　　“这……有机会再与各位详聊！”苏莫拱了拱手，他可没有心情，一一回答这些人。
<br/>　　“上官昊，走吧！带你去见殿主！”
<br/>　　伏卓见此，也不想耽搁时间，招呼一声，便走进了千渊殿之中。
<br/>　　苏莫见此，立刻跟了上去。
<br/>　　千渊殿极大，伏卓带着苏莫，径直向一座极高的塔楼走去。
<br/>　　“必须要隐藏一番！”苏莫心中思索，这个所谓的殿主，必定实力恐怖。
<br/>　　他想要在此地蛰伏，要懂得藏拙，不然很难在此长久。
