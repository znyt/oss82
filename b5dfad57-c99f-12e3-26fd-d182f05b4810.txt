　　楚风眠变换为龙摩罗的样子，在他身上显现而出的龙威，几乎是跟真正的龙摩罗，没有任何的区别。
<br/>　　不过。
<br/>　　龙摩罗身为魔龙一族，却并非是纯粹的龙族，在他的身上还有着另外一道力量的存在，那就是魔意。
<br/>　　这身上的魔意……
<br/>　　楚风眠目光闪烁，在思考这，以他现在的手段，凝聚而出的魔意，瞒住九劫古帝，是不成问题。
<br/>　　但是这一次，那少帝都极有可能，隐藏在这山谷之中。
<br/>　　只怕是以少帝的能力，瞬息之间都足以看出楚风眠的身份来。
<br/>　　倒时候楚风眠进入这山谷之中，无异于是自寻死路。
<br/>　　“魔意……魔意，对了，心魔之王！”
<br/>　　楚风眠脑海之中闪烁着一个个办法，突然眼神一亮。
<br/>　　想要显现出魔意来，没有比起一尊真正魔族身上的魔意，要更加真实的了。
<br/>　　他心神一动。
<br/>　　心魔之王，便是出现在了楚风眠的背后。
<br/>　　这心魔之王，之前吞噬了不少的魔族，现在的实力已经是进一步攀升，达到了九劫古帝的层次。
<br/>　　虽然比起龙摩罗相比来说，还差不少，但是在加上楚风眠的手段，也足以弥补了。
<br/>　　“附身！”
<br/>　　楚风眠对着心魔之王命令道，只见心魔之王，依附在了楚风眠的身上。
<br/>　　他身上的魔意，故意是被楚风眠释放显现了出来。
<br/>　　一眼望过去，现在的楚风眠，便是如同一尊真正的魔族一样。
<br/>　　心魔之王早已经是被楚风眠收服认主，现在附身在楚风眠的身上，外人根本是难以察觉的出来。
<br/>　　在加上楚风眠血脉之中，显现而出的若有若无的龙威，现在楚风眠变化成为了龙摩罗。
<br/>　　不管是在外表上，还是身上的气息，力量上，都是跟真正龙摩罗，没有任何的区别。
<br/>　　虽然不能说是天衣无缝，但是楚风眠也有着自信，足以瞒住少帝了。
<br/>　　毕竟龙摩罗，也是魔族之中一位超级强者，少帝也不会太过怀疑他的身份。
<br/>　　“上一次少帝差一点杀了我，这一次也该我送他一份大礼去了！”
<br/>　　楚风眠目光冰冷无比，他之前差一点就被少帝轰杀，这个仇怨楚风眠可是铭记于心。
<br/>　　这一次，这山谷之中如果真的是血邪魔帝的陵墓的话，其中的宝物必不会少。
<br/>　　除了那人皇罐碎片，是楚风眠必须要得到之物之外，其他的东西，楚风眠能夺走几件，也是对少帝的打击。
<br/>　　楚风眠现在的实力虽然无法是跟少帝正面对抗，但是他也不会让少帝好过。
<br/>　　“走。”
<br/>　　想到这里，楚风眠便是不在浪费时间，他的身形连连闪烁，从虚空之中飞了出来，同时也变幻成为了龙摩罗人形的样子。
<br/>　　催动这遁光，向着山谷的方向飞去。
<br/>　　这山谷，应该是被少帝以一种玄妙的手段，所遮掩起来的伪装。
<br/>　　这手段的玄妙，如今楚风眠已经是靠近了过去，都无法看得出，这山谷之内，其实是另有洞天。
<br/>　　半步天人境级别强者的手段，楚风眠想要参悟出来，还是极为勉强。
<br/>　　不过之前楚风眠已经是见到了那一群魔族，进入其中的景象了，自然早有准备，在距离山谷入口处的地方。
<br/>　　楚风眠悄悄减速，从空中落了下去，旋即一步踏入到了山谷之中。
<br/>　　刚刚迈入到了山谷之中，楚风眠便是感觉到了天旋地转，这山谷的背后，是一座座群山，在这群山的最高处，一尊金光闪闪的佛寺，浮现在楚风眠的眼神之中。
<br/>　　“群山之巅，净土佛寺！”
<br/>　　楚风眠脑海之中立刻明白了过来，这金光闪闪的佛寺，到底是什么来历。
<br/>　　同时，除了那一座净土佛寺之外，这群山之中，却是浮现着浓郁的魔意。
<br/>　　这等魔意，比起什么幽鬼域的魔意，要浓郁的多，简直是一片魔域。
<br/>　　这净土佛寺，居然是存在于一方魔域之中，这两种截然不同的力量，居然是相安无事，没有任何的冲突。
<br/>　　这一幕，简直是超出了常理。
<br/>　　不过楚风眠却是清楚，这便是因为净土佛祖，便是血邪魔帝，所以他才足以掌控这一佛一魔的力量，缔造出如此诡异的一处天地来。

<br/>　　楚风眠踏入到了一方诡异魔域之中，也是散发出灵识来，察觉这周围的强者。

<br/>　　几乎每一位都是位于九劫古帝巅峰的存在。
