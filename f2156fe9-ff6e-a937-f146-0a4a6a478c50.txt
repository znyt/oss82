　　彼此间都没有说话，山洞中变得寂静起来，经过刚才的亲密接触，关系也更加的亲近起来，彼此都不点破，但却在心中默认。
<br/>　　“烤魔兽腿都凉了，在烤一下吧，我都有些饿了。”
<br/>　　“好，你等等。”
<br/>　　山洞中，云韵依偎在他的怀中舒适的睡去，没有平日里的高贵，也不是高高在上的云岚宗宗主，反而透露出一丝慵懒的气质。
<br/>　　数日清晨，悠悠转醒，睫毛颤了颤，渐渐睁开美眸，望着映入眼中的温柔面孔，云韵目光也柔和起来。
<br/>　　有些不舍的离开温暖踏实的怀抱，云韵站起身来，惊喜道：“斗气回复的很快，明日一早估计就能完全恢复了，到时候就去找紫晶翼狮王。”
<br/>　　望着云韵曼妙迷人的曲线背影，慕清霄有些后悔，正人君子果然不好当：“我倒是觉得，实力被封印的你，别有一番风味。”
<br/>　　闻言，云韵柳眉顿时一竖，心中尽是温柔，美眸瞪着慕清霄，嗔道：“乌鸦嘴，说什么呢？”
<br/>　　慕清霄微微一笑，道：“现在的你，根本不是紫晶翼狮王的对手，需要我帮忙吗？”
<br/>　　听到慕清霄的话语，云韵没有否认，现在的她确实不是紫晶翼狮王的对手，单独对付它，想要获得紫灵晶恐怕是不可能的。
<br/>　　沉吟片刻，云韵若有所思，道：“紫晶翼狮王一般都将紫灵晶藏在自己山洞中，前几天我本想潜进去，不料却被它发现了。”
<br/>　　“明天，等你实力恢复，你去引开它，我去洞内帮你取紫灵晶，你觉得怎么样？”
<br/>　　说实话，紫晶翼狮王虽是斗皇级别，但对慕清霄来说，根本不存在威胁，他有自信几回合将之击杀剑下，但他却不想暴露太多。
<br/>　　云韵觉得方法可行，但美眸中却充满担忧，道：“魔兽山脉内部太危险了，你能行吗？”
<br/>　　慕清霄剑眉一挑，一把将她搂住就吻了上去，含住她的粉唇，肆无忌惮的品尝起来，一番缠绵后，直到她的粉唇都有红肿，方才停止。
<br/>　　“不要怀疑自己男人的能力，不信你可以试试。”
<br/>　　云韵俏脸上布满诱人的红晕，自然知道他话中有话，道：“哼，随便你吧，你把这块水晶佩戴在身上，接近紫灵晶后，它便会发热，你只需依靠温度高低，就能快速找到。”
<br/>　　说着，她就从玉指上的一枚碧绿色的纳戒中取出一块菱形水晶，直接递给慕清霄。
<br/>　　接过菱形水晶，慕清霄微笑道：“别说是紫灵晶，你想要的话，紫晶翼狮王幼崽我都帮你抓回来。”
<br/>　　云韵白了他一眼，显然是以为他在说大话，若真是将幼崽抓回来，紫晶翼狮王非得跟他们拼命不可。
<br/>　　“你休息吧，我出去看看能不能寻找到灵药，傍晚会回来。”
<br/>　　望着他离去的背影，云韵低声呢喃道：“你自己小心。”
<br/>　　傍晚，慕清霄准时回到山洞中，然后自储存空间中取出一些魔兽肉，以及一些生长在魔兽山脉中的灵果，味道香甜可口。
<br/>　　刚进入山洞，他眼中就闪过一丝惊艳。
<br/>　　此时，云韵盘坐在地上，奇异的长剑平放在玉腿上，不知何时已经换上了一套雪白的素裙，将她的娇躯勾勒出迷人的曲线，优雅迷人。
<br/>　　臻首顶部本有些慵懒的凤凰发饰，再次聚拢，不经意间透露着高贵，美丽动人的脸颊，淡然优雅，先前的柔弱气质，已经完全消逝不见。
<br/>　　慕清霄微微摇头，看来又出去洗澡了，由此可见女子拥有太强的实力也不好，因为不好管教，除非男人有完全压制她的实力。
<br/>　　不过，自她身体上的波动来看，已经恢复的七七八八，出去洗澡倒是没什么问题了。
<br/>　　而且，其中几头五阶魔兽，已经在昨日被他斩杀，除了六阶紫晶翼狮王，也没有魔兽能威胁到她。
<br/>　　不过，看她现在的高贵气质，更是提高了慕清霄的征服欲。
<br/>　　见他回来，云韵也停止修炼，美眸渐渐睁开，美眸扫向慕清霄，温柔道：“回来了。”
<br/>　　依稀是以前清脆悦耳的声音，慕清霄微微点头，道：“刚摘回来的灵果，味道不错，已经洗干净了，你先尝一尝，我这就给你烤肉吃。”
<br/>　　“嗯。”
<br/>　　微微点头，云韵身形微动，已经出现在慕清霄的面前，接过灵果，坐在一旁，不客气的吃了起来，望着他忙碌的背影，浓郁的幸福感涌上心头。
<br/>　　吃完晚餐，慕清霄就不顾脸皮的将云韵抱在怀中，手上占点便宜是自然的，只是想更进一步，立刻就被制止。
<br/>　　竖日清晨，天刚蒙蒙亮，慕清霄抬头望着站在一块巨石上的优雅修长身姿，第一缕阳光倾洒而下，配上她高贵的气质，犹如降临人世的女神。
<br/>　　此时的云韵，就如同当天与紫晶翼狮王战斗时一样，其中雍容与高贵，清冷中参杂着高傲，神圣不可侵犯。
<br/>　　但慕清霄就是喜欢征服这样的女人，成就感满满啊。
<br/>　　似乎是察觉到了他的视线，云韵转过身来，温柔道：“记住，自己小心，保全性命最重要，若是拿不到紫灵晶，我会另想办法。”
<br/>　　话语刚落，慕清霄就轻轻跃上巨石，轻轻的趴在云韵怀中，双手环抱在她纤腰上，如同树懒一样。
<br/>　　惊愕过后，云韵顿时满头黑线，见他死皮赖脸的模样，无奈的拍了拍玉额。
<br/>　　这混蛋家伙，无时不刻不想着占她的便宜，但是想着此处距离紫晶翼狮王的洞穴有几千米的距离，只能任由他抱着。
<br/>　　毕竟，途中随时都有可能遇到强大魔兽的袭击，能不能抵达洞穴都还是两说。
<br/>　　将脑袋埋在硕大柔软的胸口，慕清霄嘴角牵起一丝微不可查的得逞笑容。
<br/>　　察觉到胸口的热气，绝美的嫣红涌上她的脸颊，低头看了一眼无赖似得慕清霄，娇躯微震，背后一对青色能量羽翼伸展而出。
<br/>　　“不要乱动，否则你掉下去我可不管。”
<br/>　　强心压下跳动频率加速的芳心，云韵深吸了口气，背后双翅一振，两人迅速离地而起，转眼就来到高空。
<br/>　　而慕清霄则将脑袋埋的更深，恨不得钻进那雪白的沟壑中一探究竟。
