　　“唤醒血脉？”
<br/>　　寒月漓听到楚风眠的话，喃喃自语一声。
<br/>　　“我的确感觉到，在我的身躯之中，有着一股沉睡的力量，但是这股力量，我却是不懂得如何唤醒。”
<br/>　　“在我踏入到神力境，成就圣者后，这股力量只是苏醒了一些，却是十不足一，大部分的力量依然是在沉睡。”
<br/>　　“不错，你的血脉之力，大部分都在沉睡，若是唤醒，你的资质甚至丝毫不比我要相差多少，甚至可能比起我还要高。”
<br/>　　楚风眠点了点头道。
<br/>　　寒月漓的血脉之力，已经是在沉睡，她可是天地神体，继承先天之源的玄寒之体。
<br/>　　要是一旦彻底唤醒，她的资质将会达到一种恐怖的地步，超越楚风眠都有可能。
<br/>　　“如何才能唤醒？”
<br/>　　寒月漓开口询问到，她相信既然如今楚风眠开口，自然便是有着办法才对。
<br/>　　“血脉融合，以我的血脉，跟你的血脉如何，如此一来，我便是有自信可以激发出你血脉中的力量。”
<br/>　　楚风眠平静的开口道。
<br/>　　“我的血脉，也并非是寻常血脉，乃是远古战龙的血脉，虽然并非先天，但是如今力量已经小成，跟你体内玄寒之体的力量交互，便是可以将你的力量彻底激发出来。”
<br/>　　先天神体之间，可以互相交互血脉，来争取力量。
<br/>　　这也是一些强者，虽然身为先天神体，但是却也在不断的寻找着其他拥有先天神体的天才的原因。
<br/>　　不过对他们来说，他们要做的却不是互相交互，而是掠夺，掠夺先天神体中的力量。
<br/>　　直接夺取，远比互相交互得到的力量要强的多。
<br/>　　但是楚风眠却做不出这种事来，以楚风眠的性格，他绝不会为了力量，来夺取其他人的先天之力，除非是楚风眠的敌人，楚风眠才会毫不犹豫的出手。
<br/>　　先天之力，虽然强大，却是容易被人掠夺，那曾经太子的婚约，便是也想要夺取寒月漓身上的先天之力。
<br/>　　“好，我的血脉，的确还在沉睡，若是我的血脉之力可以彻底唤醒，我也可以再进一步，为你分忧。”
<br/>　　寒月漓听完楚风眠的话，当机立断的开口道。
<br/>　　这毫不犹豫，丝毫没有考虑。
<br/>　　“你就不怕我夺取了你的血脉？”
<br/>　　楚风眠看着寒月漓如此痛快的回答，有些惊讶的说道。
<br/>　　这种先天神体，很容易会被人夺取血脉的，若是楚风眠做点什么手脚，轻而易举便是可以得到寒月漓的血脉。
<br/>　　“你若是想要这么做，早就可以了，当初也不会救我了。”
<br/>　　寒月漓脸一红，笑了笑道。
<br/>　　“开始吧，我都有些迫不及待了。”
<br/>　　“好。”
<br/>　　楚风眠点了点头，突然心神一动，在他的指尖，几滴战龙精血，凝聚出来，一下子点入到了寒月漓的身躯之中去。
<br/>　　“不要抵抗，我会激发出你身躯中的血脉的玄寒之力来。”
<br/>　　楚风眠轻声说道。
<br/>　　他的一切思路，都是融入到了那几滴战龙精血中去，一下子潜入到了寒月漓的身躯之中。
<br/>　　战龙精血入体，顿时在寒月漓的身躯中，最为深层次的血脉，一股股玄寒之力，爆发而出。
<br/>　　无穷无尽的寒气，简直是要将人给硬生生冻死。
<br/>　　寒月漓紧闭双眼，她的衣衫，都被无数寒气冻得粉碎，在这寒气之下，灵器的衣衫，都无法抵挡。
<br/>　　两个人，刹那间都是变的赤身裸体，却是丝毫不知。
<br/>　　楚风眠的身上，远古战龙精血被全部催动起来，来抵挡这无穷无尽的寒气，幸好楚风眠早有准备，他的身躯，已经是被造化之力守护。
<br/>　　同时楚风眠打入到寒月漓身躯中的战龙精血，已经是开始沟通起来，将寒月漓身躯中的力量，彻底唤醒。
<br/>　　“轰！”
<br/>　　一股股灵力，爆发而出，席卷着整个密室。
<br/>　　楚风眠跟寒月漓，坐在其中，每时每刻，都在激发着寒月漓身躯中的力量。
<br/>　　一天，两天，三天。
<br/>　　足足七日时间过去，寒月漓身上涌现出的寒气，已经是达到了极致。
<br/>　　她的样子，虽然未变，但是现在看来，寒月漓看起来整个便是一尊冰雪女神一般。
<br/>　　气质发生了翻天覆地的变化。
<br/>　　她的境界，依然是神力境七重，但是她的力量，却是比起之前，要强大了数倍，十数倍不止。
<br/>　　“这便是唤醒之后的力量？”
<br/>　　许久，寒月漓睁开眼睛，眼神中尽是震惊之色。
<br/>　　她已经感觉到，她的力量比起曾经，要强大了数十倍不止，七日时间，居然是发生了如此翻天覆地的变化。
<br/>　　这便是先天神体被彻底激发力量的一刻。
<br/>　　“我感觉到，我的实力已经是足以跟真正的巅峰圣者一战了。”
<br/>　　“巅峰圣者？以你现在的境界，只怕是一些半步生死境的武者，都可以轰杀了。”
<br/>　　楚风眠的双眼，也是缓缓睁开，面露笑意的说道。
<br/>　　激发先天神体的力量，的确并非是那么容易，也幸好楚风眠了解这玄寒之体，才会进展的如此顺利。
<br/>　　“这一切，也都多亏了你。”
<br/>　　寒月漓看着楚风眠，面露复杂，楚风眠的出现，拯救了她，先是帮她结束了跟太子的婚约，如今又是因为楚风眠，她才唤醒了自己血脉中的力量。
<br/>　　可以说她现在的成就，根本离不开楚风眠。
<br/>　　“我说过你是我的女人，这点事，算得了什么。”
<br/>　　楚风眠笑着开口道。
<br/>　　“更何况，我也从其中得到了不少的好处，血脉交融，也令我的实力，有着提升。”
<br/>　　楚风眠的实力提升虽然不如寒月漓那般明显，但是他也得到了一部分的玄寒之力，算是半个玄寒之体了。
<br/>　　在加上远古战龙血脉的力量，现在的楚风眠，资质已经丝毫不比一些先天身体要差了。
<br/>　　这对他也是有着莫大的好处。
<br/>　　“走吧。”
<br/>　　楚风眠起身，才发现身上空无一物，他的脸色微微一僵，便是灵力凝聚，化为了一件长衫出来。
<br/>　　寒月漓这时也才注意到，脸色不由一红，凝聚出一件长衫，才跟楚风眠，一起离开了这密室之中。
