　　“李峰！”苏莫见道李峰苏醒，顿时疾呼一声，面上露出了一丝笑容。
<br/>　　终于醒了！
<br/>　　几年过去了，他和李峰终于再次相见了！
<br/>　　苏莫心中感慨不已，李峰的命运，真是坎坷到了极致。
<br/>　　先是修为被废，后又身体被夺舍，每一件都是悲惨无比。
<br/>　　而这两件事，多多少少都和苏莫有关系，让他心中倍感亏欠。
<br/>　　李峰苏醒，睁开了眼眸，脑中一时间还有些懵圈，搞不清楚状况。
<br/>　　“苏……苏莫？”少倾，李峰脑中渐渐清明，望着眼前的苏莫，怔怔的开口。
<br/>　　“李峰，你终于醒了！”苏莫微笑道。
<br/>　　“苏莫，这是怎么回事？”李峰瞥了一眼不远处的秦不死三人，又看了看四周，不解的问道。
<br/>　　李峰这几年的记忆完全空白，他只记得，在血罗殿攻打天灵宗之时，他被皇甫擎擒住了。
<br/>　　然后，他就被带到了中洲，最后被他人精神力侵入了识海，再之后他便失去了意识，一无所知了！
<br/>　　没想到，清醒过来之后居然就见到了苏莫。
<br/>　　“李峰，你被人夺舍了肉身，不过现在已经没事了！”苏莫叹了口气说道。
<br/>　　“这……这到底是怎么回事？”李峰依旧不解。
<br/>　　随即，苏莫就为李峰详细的讲述了一番最近几年的事情，并且将他们目前的处境，也详细的解说了一遍。
<br/>　　李峰听完之后，久久无语，苏莫虽然说的随意，但其中的各种凶险，却根本隐藏不住。
<br/>　　“苏莫，带我一起走吧！”良久之后，李峰郑重的说道，他对于苍穹世界，并无任何留念。
<br/>　　苏莫是他今生唯一的朋友，他自然要跟随苏莫离开。
<br/>　　再者，他的父母在苏家，也就是在苍穹神宫之中。
<br/>　　“嗯！”苏莫点了点头，片刻之后，他便将李峰收进了虚界神图之中。
<br/>　　随后，苏莫没有再去处理空间玉镯中的人，再次动身，和秦不死三人向南洲而去。
<br/>　　空间玉镯中的那些人，苏莫现在懒的去杀，待下次需要吞噬之时，那些人便是他的资源。
<br/>　　苏莫几人速度极快，瞬息千里，仅仅三日之后，便到达了南洲。
<br/>　　随后，他们一番探查，便找到了空间通道所在的那片辽阔的平原。
<br/>　　虽然他们不知道空间通道具体出现的位置，但能确定就是在平原上，接下来他们只需安心等待便可。
<br/>　　光头大汉和黑衣中年人，进入了秦不死的空间宝器之中，只剩下了苏莫和秦不死两人。
<br/>　　只要他们两人，能安全逃离出去，所有人就都能逃脱了。
<br/>　　平原上，一处低矮的土丘上，苏莫和秦不死均是盘膝而坐，双眸紧闭，都在静静的修炼。
<br/>　　苏莫的修为已达武皇境九重的巅峰，距离武帝之境，已经不远了。
<br/>　　从武皇境突破到武帝境，有两道屏障需要跨越。
<br/>　　其一自然便是神丹，要将神丹的层次，提升到武帝的级别，蕴含的玄力更加浑厚、更加精纯和强大。
<br/>　　其二，便是武魂需要尽皆成为战魂。
<br/>　　只有跨越了这两大屏障，才能真正晋升到武帝之境。
<br/>　　而这两大屏障，只要跨越了其中一种，便能成为准帝。
<br/>　　苍穹世界的那些准帝，全部都是冲破了第一道屏障，而止步于第二道屏障。
<br/>　　武魂晋阶成为战魂，这才是最困难的事！
<br/>　　目前，苏莫对于神丹突破到武帝层次，没有什么担心，只要根基扎实，资源足够，轻而易举。
<br/>　　而武魂晋阶为战魂，也是他最大的难题！
<br/>　　苏莫曾经详细的了解过，武魂想要晋阶成战魂，不仅要做到自身的神魂和武魂融合，其中还要融合奥义的力量。
<br/>　　这是非常困难的事情，好在他曾经凝聚过战魂种子，对于晋升战魂能提高一定成功率。
<br/>　　不过，目前距离巫族天才试炼结束，通道开启，只有月余的时间了。
<br/>　　无论是神丹突破，还是武魂进化为战魂，苏莫都来不及了。
<br/>　　现在他只能尽力淬炼玄力，巩固根基，实力能强一分便是一分。
<br/>　　时间一点一滴的流逝，一日又一日，很快便过去了一个月。
<br/>　　这一日，苏莫和秦不死均是停止了修炼。
<br/>　　“苏莫小友，通道即将开启，成败在此一举了！”秦不死目光凝视着苏莫，满脸的凝重。
<br/>　　“嗯！”苏莫点了点头，心中同样沉重无比。
<br/>　　秦不死沉吟了片刻，手掌一番，掌中出现了一根羽毛。
<br/>　　这根羽毛呈灰白色，其上却有朦胧的光晕闪烁，传出一股晦涩的气息。
<br/>　　“小友，请将这个收下！”秦不死将羽毛递向苏莫，目光灼灼的说道。
<br/>　　“前辈，这是……？”苏莫有些不解，但也并未推脱，立刻便接过了羽毛。
<br/>　　“这根羽毛能让我复活，就算我死了，只要你能逃出去，我便能借此复活！”秦不死沉声道。
<br/>　　苏莫闻言心中一震，不由的暗暗心惊，此人居然如此不简单，借助一个羽毛就能重生！
<br/>　　难怪对方经历过三次大劫，依然不死，的确厉害！
<br/>　　想必对方不止是有一根这样的羽毛，估计在苍穹世界之内，还有留下了这种羽毛。
<br/>　　到时候，就算他们全部死了，对方依旧能在苍穹世界复活。
<br/>　　不死鸟，果然不凡！
<br/>　　苏莫点了点头，随即郑重的收下了这根奇怪的羽毛。
<br/>　　他也不希望对方陨落，两人若是能一起逃离，那就再好不过了。
<br/>　　“多谢小友！”秦不死向苏莫抱拳道谢。
<br/>　　“前辈无需客气！”
<br/>　　苏莫摆了摆手，道：“我们到时候见机行事吧！”
<br/>　　“嗯！”
<br/>　　秦不死点头，略一沉吟，继续道：“苏莫小友，有一事你必须有心理准备！”
<br/>　　“何事？”苏莫疑惑问道。
<br/>　　“是关于巫族尊者一事！”
<br/>　　秦不死满脸凝重，声音低沉的道：“作为武尊强者，实力恐怖绝伦，往往身上的威压，便能让我们生不起反抗的念头，而想要威胁尊者，难度非常的大！”
<br/>　　苏莫闻言深吸了一口气，这一点他早已想过，尊者何其强大，他们威胁尊者，对方可能瞬间就将他们秒杀。
<br/>　　随后，苏莫又和秦不死商讨了片刻，为了安全起见，两人在巫娴等所有‘人质’的识海中，都留下了一缕玄力。
<br/>　　玄力中蕴含他们的意念，他们不死，玄力会很稳定，不会伤及人质的性命。
<br/>　　若是他们一旦死亡，玄力立刻就会爆炸，直接炸碎人质的识海。
<br/>　　一切准备妥当之后，苏莫和秦不死两人，也没有再修炼，因为距通道开启之日，已经近在眼前了。
<br/>　　站在土丘上，苏莫催动幽冥魔瞳武魂，已经发现了极远之处有几名零散的巫族天才，来到了这片广遨的平原上，显然也在等待通道开启。
