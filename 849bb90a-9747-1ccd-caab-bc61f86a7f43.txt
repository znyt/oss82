
<br/>　　<br/>　　    “他人怎么样？”菜刀文低声问了一句，他那意思是王惊蛰有没有看出丁武的命道来。<br/>　　<br/>　　    王惊蛰摇了摇头，说道：“有惊无险……”<br/>　　<br/>　　    丁武这次真的要是挺不过去的话，王惊蛰早先就能从他的面相观望出来了，特别是昨晚在酒吧里出去的时候，他并未瞧出对方有厄运当头的征兆，所以丁武挨的这十几刀是要不了他命的。<br/>　　<br/>　　    但尽管如此，王惊蛰也挺憋气的，人毕竟还是差点死了！<br/>　　<br/>　　    齐泉叹了口气，说道：“医生说，今晚人要是能醒过来的话就过了危险期了，如果醒不过来可能就有点麻烦了”<br/>　　<br/>　　    王惊蛰阴着脸“嗯”了一声，又接着问道：“吴洋跑了，没抓到人？”<br/>　　<br/>　　    “对，他人太精了，马爷去堵的他，但没想到的是当时吴洋和另外一人直接从三楼就跳了下去，吴洋没追上，不过跟他在一起的人被抓住了，人现在被扣在老板的会所里”<br/>　　<br/>　　    “抓了一个？那也行，有一个在手里了，吴洋就跑不了了……”王惊蛰吐了口气说道。<br/>　　<br/>　　    另外一头，陈重的庄园式会所里，那个跟吴洋在一起的人被逮住之后就给送了过来，陈重得到消息之后就赶来了。<br/>　　<br/>　　    陈重从车里下来，老马就迎了上去，说道：“抓的这个嘴也很硬，我们审了几句他一个字都没往出吐，而且态度狂的很，一点都没有阶下囚的觉悟，还他么的跟我们说了，咱们最后得跪着给他送出去呢”<br/>　　<br/>　　    陈重顿时冷笑道：“他是不脑子进水了？还想着给他送出去？我他么直接给他送到乱坟岗里埋上算了！”<br/>　　<br/>　　    陈重和老马边走边说，进到会所里后，午桥和王生也站了起来，老马给双方介绍了下：“这是北面过来的朋友，出马仙里最牛逼的人物午桥和王生”<br/>　　<br/>　　    陈重伸手跟对方握了一下，笑道：“早就听过您的大名了，这次没想到有机会碰上了，以后再来蜀地你还得找我，我这人就爱交朋友”<br/>　　<br/>　　    午桥说道：“那不是打扰陈老板了么？”<br/>　　<br/>　　    “哈哈，别人找我叫打扰，你们来了叫寒暄！”陈重几句话顿时就给自己和午桥那边拉上了关系，语言艺术运用的非常到位。<br/>　　<br/>　　    午桥也笑了，摇头说道：“这次都有点不好意思了，正主没抓到跑了”<br/>　　<br/>　　    “没事，没事，咱们不还是有收获了么，有一个就行啊……”陈重和午桥聊了几句后，就让人把扣住的那个给带了过来。<br/>　　<br/>　　    这是个大概二十几岁的青年，走过来的时候完全没有任何身为阶下囚的觉悟，迈着步子的时候还昂首阔步的，神情一脸倨傲。<br/>　　<br/>　　    这三堂会审的架势也没有镇住他，这人瞥着眼神，先是看着午桥说道：“北出马的午桥，呵呵，你是不是过界了，你们出马的不是向来都在北边呆着的么，什么时候也敢往南边跑了？”<br/>　　<br/>　　    “过啥界，你是在和我说楚河汉界，还是阴阳两界啊”午桥淡淡的道。<br/>　　<br/>　　    “我说的是巫门和你们当初出马仙划的界限，以前不就说好了的么？出马不出北，巫门不跨南，怎么着你们不会这么快就给忘了吧？”<br/>　　<br/>　　    午桥的脸“唰”的一下就变了，这青年说的是关于几十年前的一场协定，这个规定也不知道怎么冒出来的，大致的意思就是，若论风水生意的话北方都以杨公风水为主，南边则是王朝天家族，而关于巫门和出马仙之间也有个协定，他们之间也是南北划界，不能逾越，大概有几十年了吧这个规定都遵守的挺好，至少在明面上谁都没有触犯过。<br/>　　<br/>　　    午桥皱了皱眉，没有吭声，对方说的话还是挺有道理的，他这么干确实坏了规矩。<br/>　　<br/>　　    “那我也来说说我的规矩？”陈重忽然背着手来到青年身前，盯着他的眼睛说道：“我的规矩很简单，人不犯我我不犯人，但人要犯了我的话我肯定翻着倍的给你还回去，你们之前干的这些事正好撞到我的规矩上了，祸害我妻女，还差点要了我的命，你们巫门的人在蜀地是不是过的日子太舒心有点找不到北了？还是觉得这些年我陈重修身养性让人都把我给忘了？”<br/>　　<br/>　　    那青年斜了着眼睛说道：“管我什么事，又不是我干的，那都是吴洋下的手，跟我有什么关系？你不会以为我跟他在一起呢，就是他的帮凶了吧？多好笑的一件事啊，你不满你去找吴洋啊，你抓我也没用”<br/>　　<br/>　　    老马顿时抬腿一脚就踹在了他的胸口上，指着对方骂道：“你当我们都是傻子呢吧？没你的关系，你跑什么啊？”<br/>　　<br/>　　    青年从地上爬了起来，说道：“这一脚我记着了，你也记着到时候看我怎么给你还回去”<br/>　　<br/>　　    “说吧，吴洋在哪呢？”老马皱眉问道。<br/>　　<br/>　　    “我要是说不知道呢……”<br/>　　<br/>　　    老马看了陈重一眼，他说道：“再给他一次机会，嘴要是还这么硬的话，你就帮他好好回忆下，到底怎么才能给吴洋找出来”<br/>　　<br/>　　    陈重刚说完，身上的电话忽然响了，他拿出来一看后顿时就愣住了：“他怎么给我打电话了？”<br/>　　<br/>　　    电话屏幕上的号码备注很简单，就写了个‘李大秘’三个字，但这三个字的份量在蜀地却价值不菲，因为这个备注明显是个秘书类的角色，而他却是给川中父母官当秘书的。<br/>　　<br/>　　    陈重走到一旁，接通了电话，笑道：“呦，李大秘这么闲啊？知道我这几天没什么事，正打算找你出来喝点呢，你这是打算提前预约一下呗？呵呵，你说地方我随时恭候”<br/>　　<br/>　　    “老陈，喝酒的事改天再说吧，我现在问你个事你老实告诉我”电话里，李秘书声音有点严肃的说道。<br/>　　<br/>　　    陈重嗯了一声，说道：“知无不言啊，什么事？”<br/>　　<br/>　　    “你手里是不是扣了一个人，昨天晚上在汇龙小区外面带走的……”<br/>　　<br/>　　    “唰”听到李秘书的话，陈重瞬间呆愣了，他怎么也没有想到对方是因为这个问题给他打来的电话。<br/>　　<br/>　　    “你说的什么意思，我怎么不懂呢？”<br/>　　<br/>　　    李秘书叹了口气，说道：“老陈，你别和我装糊涂了，我没有拿准的话，会这么问你？”
