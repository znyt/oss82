陆寒霆想了一下苏希这个人，他对苏希印象并不深，苏家作为海城四大豪门之一是最低调的，而苏希这个人的存在感也很低，只有在夏夕绾回海城的最初他出现过，现在想来颇有几分神龙见首不见尾的神秘感。
<br/>
<br/>        陆寒霆觉得自己严重忽略了这个人，这个人其实一直都在，而且在整个事件里起到了一个非常重要的作用。
<br/>
<br/>        陆寒霆将手里的青玉碗递给了夏夕绾，“听说你跟苏希从小一起长大，青梅竹马？”
<br/>
<br/>        “是呀，我妈咪很喜欢苏希，苏希是我妈咪的关门弟子。”
<br/>
<br/>        “你妈咪？”陆寒霆深邃的狭眸里突然闪过了什么，他好像从来没有问过关于她妈咪的事情，她也没有提起过。
<br/>
<br/>        夏夕绾想说些什么，但是这时赵婶走了进来，笑眯眯道，“绾绾小姐，听说你今天想吃面，我刚去磨了一些面粉，现在给你做手擀面。”
<br/>
<br/>        “赵婶，”夏夕绾迅速挽住了赵婶的胳膊，不好意思的眨了眨眼，“不用这么麻烦啦，我已经买了现成的面。”
<br/>
<br/>        赵婶疼爱的嗔道，“那些面不好吃，婶儿给你做手擀面。”
<br/>
<br/>        说着赵婶将刚磨好的上等面粉倒在了碗里，然后加入牛奶鸡蛋和少量水，开始做手擀面。
<br/>
<br/>        很快赵婶手起刀落，一盘手工面条就整齐划一的落在了盘子里。
<br/>
<br/>        陆寒霆在一边看了这个赵婶一眼，如果没有看错，这个赵婶不光是星级大厨那么简单，她应该是豪门贵胄家高薪聘请的那种厨娘。
<br/>
<br/>        “哇赵婶，这么久没见，你这手艺真是越来越好了，每次你都帮我做饭，我都不需要动手了。”
<br/>
<br/>        赵婶将手擀面做好，又动作干练快速的调了一碗面条的酱汁，这酱汁浓香四溢，光酱汁都能喝几碗的感觉。
<br/>
<br/>        “绾绾小姐，我可什么都没有做，这面还是你自己煮的。”
<br/>
<br/>        夏夕绾看着自己要做的事情，她只需要将水煮开，然后将面下进去，灵魂都被赵婶给包了，这面还是她自己煮的吗？
<br/>
<br/>        “赵婶…”夏夕绾是真的过意不去了。
<br/>
<br/>        赵婶迅速露出了一副可怜兮兮的神色，“绾绾小姐，你千万别跟我们客气，你外面那么一大片园林，我们常年吃你的饭，如果这点活都不让我们帮你，那你是不是觉得我们嘴巴太多了，不想给饭我们吃了？”
<br/>
<br/>        夏夕绾摆手，“不是不是，我不是这个意思…”
<br/>
<br/>        “那就行，绾绾小姐，我先走啦。”
<br/>
<br/>        活干完了，赵婶就迅速退下了，这是厨娘最基本的涵养。
<br/>
<br/>        不过赵婶走到陆寒霆身边顿了一下，询问道，“绾绾小姐，这位是？”
<br/>
<br/>        夏夕绾澄亮的翦瞳看着男人，眸里顾盼流转，嗓音俏甜，“他啊，我老公~”
<br/>
<br/>        他啊，我老公~
<br/>
<br/>        陆寒霆狭长的眼梢里染上了缱绻的柔软。
<br/>
<br/>        “原来是姑爷啊，姑爷，绾绾小姐，我先退下了。”赵婶转身离开。
<br/>
<br/>        姑爷？
<br/>
<br/>        陆寒霆伸出健臂，搂住了夏夕绾盈盈一握的纤软腰肢，他挑了一下剑眉，“姑爷？绾绾大小姐，我怎么有一种入赘你家的错觉？”
