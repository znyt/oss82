　　随着阴阳谷内一个个地名的报出，叶永阳的嘴唇就开始不停颤抖，浑浊的目光带上了无尽复杂情绪，尤其是当‘水族墓地’四个字一出，用崩溃的心情来形容叶永阳此时的状态也不为过。
<br/>　　“楚涵竟然连水族墓地都能进去？他没死在里面？”带着吧不可置信，叶永阳出声。
<br/>　　“怎么说话的呢？”顾凉晨适当的表达了一丝不满，而后自豪道：“当然是安然无恙的出来了，不过我们长官也说了，水族墓地里面危机重重，他也是险象环生，差一点就把命搭在里面，并且出口和入口并不是同一条通道，所以至今为止，那个凶险的地方也只有他一个人踏足过。”
<br/>　　听到这一解释，叶永阳目光闪烁：“你们长官，还真是气运绝佳。”
<br/>　　“是这样。”顾凉晨继续道：“长官还说了很多，但由于我没有去过那里，所以无法理解也想象不出来，但似乎是阴阳谷内有什么禁制。”
<br/>　　叶永阳腾的一下·身躯一颤，双目放大：“连禁制都知道？”
<br/>　　“知道啊！”顾凉晨一脸的理所当然，而后故作忽然反应过来的惊讶道：“你这老头似乎知道的也不少啊？”
<br/>　　叶永阳苦笑：“我知道的有一半都被你说了……”
<br/>　　顾凉晨诧异道：“莫非白家这么死盯着你，是因为阴阳谷？”
<br/>　　“不错。”叶永阳也不打算继续死守秘密，在顾凉晨说了这么多辛秘之后，他已经心中下了某种决定，于是道：“既然楚涵上将去过了制裁之地，也知道水族墓地的出入口分别在哪，那他应该提到过制裁之地的最后一层，有八个门吧？”
<br/>　　顾凉晨看到这老头终于上钩，忍者心中的激动，点头道：“是有八个门，这莫非还有什么说法么？”
<br/>　　“呵！”叶永阳轻叹一声，仿佛用尽了力气道：“那八门中有四个门，就是当下四大神秘家族千年前的家族墓地入口。”
<br/>　　顾凉晨顿了顿，而后就着叶永阳的话道：“白家，肖家，罗家和高家，眼下我还知道一个水族，所以说当年真正的秘门家族并非是四个，而是八个？”
<br/>　　叶永阳先是点了点头，而后又忽然摇头：“是八个，但也不是八个。”
<br/>　　“这又是什么说法？”顾凉晨诧异开口，此时他的诧异并非作假，而是的确第一次听到这样的说法。
<br/>　　叶永阳淡淡道：“八门之首为回炉，只是回炉一门的墓地内，埋葬的并非回炉一脉的人，而是其守护者，除了回炉，制裁，水火，白罗高肖，这八门之外，还有一直生活在阴阳谷内的守护者一族，也就是说实际上是九门，只不过回炉与守护者相辅相成，而在传闻里回炉一门的人至始至终都没有出现过，所以阴阳谷里都是历代守护者说了算。”
<br/>　　顾凉晨有些晕，表情也迷茫起来，但很快他的思维又回到了正点上。
<br/>　　“那白家想要活捉你，又是为什么？”顾凉晨好奇问道。
<br/>　　叶永阳看了他一眼，道：“我知道那八门墓地的生死穴，四大神秘家族都想要再次重回阴阳谷，更想进入千年前的墓地，但他们不知道如何安全进去，也不知道如何活着出来。”
<br/>　　第一次听到这一消息的顾凉晨心跳飞快，语气也带上了紧张：“连神秘家族都无法掌控的地方，我长官竟然一个人闯进去，还活着出来了？”
<br/>　　没想到顾凉晨的第一反应竟然是这个，叶永阳先是愣了愣，而后不禁笑道：“所以我才说，楚涵气运滔天。”
<br/>　　顾凉晨点头道：“的确是，大秘密都像是找上门一样，被他一次又一次的撞见，无论多死绝的境地，他也能绝处逢生。”
<br/>　　聊到这里，忽的叶永阳想起了什么，有些激动的问道：“但是他怎么知道他进入的地方是水族墓地？你也似乎对水火两族的事情不惊讶。”
<br/>　　这一问题出口，顾凉晨嘴角微不可查的勾起，道：“我长官跟水族人关系不错，不过火族人就处不好了，据说他一不小心毁了火族的什么东西，还挺重要的，火族人恨不得将他杀之后快呢！”
<br/>　　轰！
<br/>　　叶永阳的心神剧烈震荡起来，好似天都塌了一般，情绪如同过山车般剧烈起伏。
<br/>　　“楚涵见过活的水火两族人？！”叶永阳震惊的声音尖锐，音色都变了样。
<br/>　　“是啊。”顾凉晨再一次回答的理所当然。
<br/>　　叶永阳坐不住了，强行忍者剧痛将身躯前倾：“这件事还有谁知道？”
<br/>　　“就你，我，还有我长官。”顾凉晨笑着回答，而后又摸了摸下巴补充了一句：“可能高家的嫡系小少爷高少辉也知道，我长官跟他是拜把子兄弟，很多东西都是高少辉跟我长官解答的，我长官将水火两族还在世的秘密告诉他也不奇怪。”
<br/>　　又冒出来一个高家的嫡系小少爷，此时的叶永阳已经不想去思考为何楚涵会认识这么多不可思的人，当下他整个心神都放在了水火两族还在世这件事上。
<br/>　　顾凉晨也不顾叶永阳的心脏能不能承受的住，紧接着又道：“你也别说出去，我之所以告诉你这些，也是有我长官授意的。”
<br/>　　“此话又是怎讲？”叶永阳已经失去了思考能力，震惊之下显得有些呆傻。
<br/>　　顾凉晨笑道：“你认识叶默不？”
<br/>　　轰！
<br/>　　叶永阳再次心中震荡，由于激动异常而导致整张脸都不正常的发红，嘴唇哆嗦个不停声音颤抖：“当然，认识！他是我的亲·哥！但是我们已经很多年没见了，他致力于搞研发，我则是管理家族，末世爆发后也彻底失散，叶默现在在哪？”
<br/>　　顾凉晨有些惊讶的愣住，随后有些抱歉的说道：“对不起，叶默已经去世了……”
<br/>　　叶永阳目光黯淡了一下，苦笑道：“也是，他年岁可不小了。”
<br/>　　顾凉晨没有再继续追问，而是放任身后的这个老人陷入悲痛。
<br/>　　也不知过了多久，在一片安静之中叶永阳再次开口：“他的本名叫叶永默，叶默这个名字是为了躲避神秘家族的搜寻，看来楚涵和我族兄关系匪浅，我的事情也都是叶默告诉楚涵的吧？所以才会专门派你来找我。”
