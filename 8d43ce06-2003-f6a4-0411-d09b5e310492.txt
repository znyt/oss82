        晚风吹在脸上有些凉意，明司寒从后座上拿了件外套给连星儿盖上。
<br/>
<br/>        他看着她的睡颜，嘴角情不自禁地微微上扬。
<br/>
<br/>        这丫头安静的时候，倒是像个小仙女。
<br/>
<br/>        说好的不骗她，结果他还是骗了她。
<br/>
<br/>        这个婚约，他不打算取消了。
<br/>
<br/>        爷爷的反转虽然在他意料之外，但是她会选择他的事情，早就在他意料之中。
<br/>
<br/>        时间差不多的时候，明司寒开车将连星儿送了回去。
<br/>
<br/>        私宅门外的前坪。
<br/>
<br/>        童筱颖看到明司寒抱着连星儿走上台阶，还以为自己的宝贝女儿出了什么事，急得踱步上前。
<br/>
<br/>        “阿姨放心，星儿只是睡着了。”明司寒轻声道。
<br/>
<br/>        童筱颖松了口气，立即给明司寒指路：“我带你去她的房间。”
<br/>
<br/>        连星儿从小到大都是这样，只要是睡着了，就是雷打不动。
<br/>
<br/>        这小丫头在一岁大的时候，在家里若是安静了，她童筱颖就得在玩具堆里去找这个睡着的小家伙。
<br/>
<br/>        明司寒将连星儿放到床上后，还很体贴地替她脱了鞋，盖上薄毯。
<br/>
<br/>        这不是他特意装出来的，而是出自本能的反应，童筱颖看得出来。
<br/>
<br/>        “我做了夜宵，你吃点再走？”童筱颖客气道。
<br/>
<br/>        明司寒微微摇了摇头：“谢谢阿姨的热情款待，太晚了，就不打扰你们了。”
<br/>
<br/>        “那你开车回去的路上注意安全。”童筱颖微笑着点了点头。
<br/>
<br/>        明司寒微微颔首，然后转身离去。
<br/>
<br/>        他离开后，傅夜沉才洗完澡从房间里出来，看到童筱颖刚从楼下上来，便关问了一句：“星儿回来了吗？”
<br/>
<br/>        “嗯，回来了。是小寒送她回来的。”童筱颖回答道，“我做了夜宵，大哥要不要尝尝？”
<br/>
<br/>        “不尝了，我倒是有份资料要给你看。”傅夜沉淡淡地说道。
<br/>
<br/>        童筱颖顿时好奇地迎了上去：“什么资料？”
<br/>
<br/>        傅夜沉搂着童筱颖进了屋，将公文包里的一份文件递给童筱颖。
<br/>
<br/>        童筱颖还以为是什么秘密文件，翻开一看时，才发现是明司寒的详细资料。
<br/>
<br/>        资料里面，明司寒的身高体重血型，家庭背景，兴趣爱好等等，被傅夜沉调查得一清二楚。
<br/>
<br/>        “我就觉得这孩子不错，星儿以后真嫁给了他，一定会很幸福。”童筱颖看完资料后，不禁沾沾自喜道。
<br/>
<br/>        “但愿吧！”傅夜沉语重心长起来。
<br/>
<br/>        童筱颖不解地看着傅夜沉：“你似乎仍旧有所顾虑？”
<br/>
<br/>        “这些资料毕竟只是表面上调查到的，他本人怎么样，只有星儿跟他接触后，星儿才知道。”傅夜沉淡淡地说。
<br/>
<br/>        童筱颖顿时忍俊不禁地放下手中的资料，走过去踮脚抱住傅夜沉的脖子：“我觉得，小寒这孩子会是星儿将来值得依靠的人。”
<br/>
<br/>        “嗯，我相信你的直觉。”傅夜沉会心一笑，抬起手来，宠溺地揉了揉童筱颖的头顶。
<br/>
<br/>        回去的路上，因为车辆极少，所以明司寒开车回去的路上车速有些快。
<br/>
<br/>        两边的路灯将马路照得亮如白昼，就在这个时候，前方五十米的地方突然窜出个黑影倒在了路中央，吓得明司寒连忙踩了个急刹车。
<br/>
<br/>        车身稳稳地停下后，车头前那个黑色的人影倒在地上一动不动。
<br/>
<br/>        明司寒皱了皱眉头，拉了手闸，熄了火，打开了驾驶座的车门走了下去。
<br/>
<br/>        “喂，你没事吧？”
<br/>
<br/>        他走近蹲下身去，才发现对方是个女人。
<br/>
<br/>        她身上穿着黑色塑身夜行衣，头上扎着高马尾，若不是她捂着大腿，他还真未发现，她的大腿上正在流血。
<br/>
<br/>        “我送你去医院！”明司寒皱了皱眉头，将女人从地上捞起来抱在了怀中。
<br/>
<br/>        女人用羸弱的声音，气若游丝地说道：“不、不要……求你、求你救救我……不要、不要送我去……医院。”
<br/>
<br/>        “……”明司寒有些无言以对。
<br/>
<br/>        但这个时间点，也没地方可去了。
<br/>
<br/>        明司寒不得不将这个女人带回自己的别墅，并叫了私人医生过来给这个女人包扎。
<br/>
<br/>        私人医生应光济给女人包扎完伤口后，又开了几服药给明司寒。
<br/>
<br/>        “还好枪伤没伤到她大腿的动脉，不然真的会失血过多而死。子弹已经取出来了，伤口未愈合之前，都不要碰水。”应光济一边收拾着医药箱，一边叮嘱道，顿了顿后，他瞥了一眼明司寒，会心一笑，“你女朋友？”
