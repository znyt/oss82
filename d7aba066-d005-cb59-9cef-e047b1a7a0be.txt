        “那去工作吧！”孙莉莉偏了偏头。
<br/>
<br/>        诺筱颖微微欠身，拿着资料转身离开了孙莉莉的独立办公室。
<br/>
<br/>        集中办公室里，靠窗边的格子间，诺筱颖安安静静地坐在办公桌前，仔仔细细地翻看刚刚从孙莉莉那儿拿过来的资料。
<br/>
<br/>        总裁办的面积比较大，而且还是复式楼，要求设施要齐全，风格不许复古，最好是超现代风格。
<br/>
<br/>        诺筱颖将资料翻看完后，才发现，对方提出的要求居然整整有十页纸！
<br/>
<br/>        “哎——”
<br/>
<br/>        诺筱颖不得不深沉地叹了口气，看样子，孙副部长是给了她一个棘手的单子。
<br/>
<br/>        起初在孙副部长办公室里时，她没太仔细过目，现在再看的时候，感觉自己掉进了一个大坑里。
<br/>
<br/>        这傅氏集团未来的大总裁，要求还真多啊！
<br/>
<br/>        诺筱颖绞尽脑汁地将资料研究了三天，才在孙莉莉那儿申请了外出假条，背着作画工具去了傅氏集团的总部大厦。
<br/>
<br/>        傅氏集团的总部大厦离地利置业大厦并不远，却比地利置业大厦更加雄伟，并且地处市中心地带，本身就拥有三层地下车库以及前坪大型广场，还有五层裙楼商场。
<br/>
<br/>        诺筱颖在一楼大厅门卫那儿做了登记，以及出示了自己的临时工作证后，才得进入傅氏集团总部。
<br/>
<br/>        往楼上去的电梯，并排有三部，最左边的门框上注明了VIP专属电梯。
<br/>
<br/>        自古左右，以左为尊。
<br/>
<br/>        诺筱颖非常识趣地上了最右边的员工电梯。
<br/>
<br/>        新任总裁的办公室在大厦顶楼，具体地址，资料里已经写得一清二楚。
<br/>
<br/>        诺筱颖很快就找到了新总裁办的位置，那里连张玻璃门也没有，走进去就是原始的毛坯房。
<br/>
<br/>        不过，东边朝左向的落地窗，倒是吸引了她。
<br/>
<br/>        落地窗从里可以看到外面，但是从外面却看不到里面。窗外是天台，女儿墙的旁边还摆满了绿色盆景植物。
<br/>
<br/>        而东边朝右向的地方，则是一个全玻璃阳台。
<br/>
<br/>        这个地方非常好，虽然是顶楼，但无论采光还是通风，窗户和门的布局很符合风水学。
<br/>
<br/>        看到这里，诺筱颖放下背包，拿出笔和纸还有尺子，在这空房子里认真地测量设计起来。
<br/>
<br/>        外婆曾经希望她长大以后去学医，以后当一名有名的中医。但是，她让外婆和老妈失望了，因为父亲，她落下了课程又没时间复习，所以，没能考起外婆喜欢的那所中医大学，更没考上老妈所希冀的那所医科大学。
<br/>
<br/>        学室内设计，完全是因为，她有这方面的兴趣爱好。
<br/>
<br/>        诺筱颖很想专升本，继续念大学，但是，妈妈不同意，因为，家里就算有积蓄，妈妈也不会拿出来用在她身上开销。
<br/>
<br/>        就连她读高中，都是爸爸向妈妈求来了。
<br/>
<br/>        对，如果不是爸爸，她连个高中都读不了。
<br/>
<br/>        每每想到这里，诺筱颖心里格外难受，鼻子酸酸的，眼睛瞬间就湿润了。
<br/>
<br/>        诺筱颖深吸了口气，缓了缓糟糕的心情后，继续对着资料和现场进行设计。
<br/>
<br/>        这位新任总裁对办公室的装修要求，还真不是一般的高。
<br/>
<br/>        既要符合这个要求，又要符合那个要求，这得使劲地绞尽脑汁，都不一定能设计出这位新任总裁想要的样子。
<br/>
<br/>        诺筱颖又叹了口气，开始怀疑是不是自己技不如人，学艺不够精湛。
<br/>
<br/>        在这空屋子里待了半晌，诺筱颖才动笔开始在画板上进行初略的设计。
<br/>
<br/>        她对待自己的每一次单子都非常地用心，用心到可以完全沉浸在自己的设计世界里，完全不受外界的干扰。
<br/>
<br/>        门框外，传来了稀疏的脚步声。
<br/>
<br/>        “大少爷，这就是老爷为您准备的新办公室。老爷说，您亲自过来一趟，看看这间办公室的位置，您喜不喜欢，若是不喜欢，我们还可以给您再换。”
<br/>
<br/>        傅氏集团大厦的主管一边毕恭毕敬地介绍，一边点头哈腰地跟在了傅夜沉的身侧。
<br/>
<br/>        他真的很忙，但是爷爷非得要他抽空出来，看一下他新办公室的选址。
<br/>
<br/>        傅夜沉被逼无奈了，才不得已来这一趟。
<br/>
<br/>        男主管就像供着皇帝一样，巴结着傅夜沉。
<br/>
<br/>        毕竟，傅夜沉是这傅氏集团未来的继承人，他丝毫不敢怠慢。
<br/>
<br/>        傅夜沉刚走到门口，看到屋内那抹熟悉的背影，立即抬起手来，示意身旁的男主管闭嘴。
<br/>
<br/>        男主管瞬间闭上了嘴，顺着傅夜沉的目光往屋内看去后，又小声地向傅夜沉介绍道：“这位是给您设计办公室装修图的设计师。”
<br/>
<br/>        “你先去忙吧！这儿，没你什么事了。”傅夜沉淡淡地说道。
<br/>
<br/>        男主管犹豫了一下，还是颔首欠身，默默地退下了。
<br/>
<br/>        傅夜沉见那个烦人又聒噪的男主管离开后，回眸望向室内那抹令他日思夜想的背影，薄唇情不自禁地微微扬了扬。
<br/>
<br/>        他一步步轻轻地朝她靠近，而她却坐在那里，完全沉浸在了她的设计中。
<br/>
<br/>        刘管家说她去韩国旅游，弄了半天，她原来是想给他惊喜，亲自为他设计办公室的装修。
<br/>
<br/>        傅夜沉从刘管家那儿了解到了苏漫雪的所有资料，她是学室内设计的，但是还未毕业。
<br/>
<br/>        他很快就要退役从商的事情，苏漫雪一定早就听刘管家说了。
<br/>
<br/>        她是不是也跟他一样，期待着他不再冒着生命危险在外奔波，回到她的身边，为她赚钱养家。
<br/>
<br/>        诺筱颖完全不知道身后有人在慢慢地靠近，等她发现的时候，身子忽然一轻，双腿凌空，落入了一个温暖的怀抱中。
<br/>
<br/>        “宝贝，没想到，你会为了给我这么大一个惊喜，连刘管家都骗过去了！”傅夜沉轻轻松松地将诺筱颖来了个公主抱，十分宠溺地抱在了怀中。
<br/>
<br/>        诺筱颖被他这一举动吓得手里的画板都掉落到了地上，手指间只捏着一块炭笔。
<br/>
<br/>        就在她定睛抬眸看去的时候，她撞进了一双深邃冰冷的眼眸中。
<br/>
<br/>        男人的眉宇间藏着犀利的霸气，深邃犹如夜狼般的黑眸下，是高挺的鼻子，如刀削一般的薄唇微微上扬，一抹似笑非笑，让他看似冷酷，又看似柔情。
