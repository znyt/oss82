　　云熙双手环在罗昊脖子上，轻轻皱了皱眉头，问道：“现在是不是没刚才那么郁闷了？”
<br/>　　罗昊这才明白，原来这个女人是因为看到自己郁闷，所以才故意来戏耍自己的，罗昊心中感动，用额头顶在云熙额头上，说道：“有你在，我就算有再多的郁闷也全都烟消云散了。”
<br/>　　“既然不郁闷了，那就赶紧去洗个澡，然后好好睡一觉。”云熙嗅着鼻子，说道：“身上都是汗味，难闻死了，再不去洗干净都要发臭了。你要真是发臭了，看我还让不让你爬上我的床。”
<br/>　　“你也别顾着说我。”罗昊捏了下云熙的鼻子，说道：“我知道你昨天晚上肯定也是一晚上没睡，这几天又是你特殊生理期，你应该知道自己的状况不能太累，好好休息，你要是突然晕倒了，你打算要吓死我，谋杀亲夫啊！”
<br/>　　云熙噗嗤一笑，说道：“你要是这么容易就能被吓死，你也不是我认识的狂龙了。行了，我知道了，一会儿我去洗澡然后睡觉！本来我身上都干净的，现在都沾到你汗味了。”
<br/>　　“嘁，怪我咯？”
<br/>　　“不怪你难道还怪我咯？”云熙反驳道。
<br/>　　罗昊：“……”
<br/>　　罗昊洗了个澡就回帐篷休息去了，他知道，只要西伦科特没死，事情就没完，而且现在西伦科特手里很有可能掌握着P病毒这种恐怖的生化武器，后面将要面对的绝对会是一场硬仗，所以现在罗昊必须要抓紧时间休息，恢复状态以应对后面要发生的情况。
<br/>　　罗昊一觉醒来是当地时间下午三点钟，穿上衣服，匆匆赶到联合行动司令部，了解目前的情况。
<br/>　　通过对拉蒙的突击审讯，联合行动司令部得到了一些有价值的东西，至于是用什么办法撬开拉蒙嘴的，罗昊没有去过问，有了“八爪鱼”这个例子在前面，就算是用脚趾头也能想到是怎么让拉蒙开口的了，现在被关在审讯室里的拉蒙估计也跟“八爪鱼”一样，成了一个目光呆滞，智商只有七八岁儿童程度的白痴了。
<br/>　　不过，拉蒙会比“八爪鱼”惨的多，毕竟“八爪鱼”身上还有一个美国公民的身份保护他，詹姆斯不能直接处决掉“八爪鱼”，必须要把他送回国接受审讯；但是拉蒙就不同了，失去了价值的拉蒙等到联合行动司令部从这个厂区里撤离的时候，或许就会变成一具尸体被丢弃这里，逐渐腐烂，最后变成一具白骨。
<br/>　　“通过对拉蒙审讯，我们目前可以得知，西伦科特会在明天早上搭乘一艘货轮，从拉普拉塔河的一个港口出发，离开南美洲。”詹姆斯将那艘西伦科特将会搭乘的货轮放大在三维电子沙盘上，说道：“这艘在这个国家注册的货轮表面上运送的是当地的特产，但实际上，这艘货轮的主人是布克尔特，上面实际运送的是一批高纯度的cocaine。DEA（美国缉毒局）和国际刑警组织也已经盯上了这艘货轮，准备在拉普拉塔河上对这艘货轮动手。”
<br/>　　“DEA也要插手？”罗昊皱眉看着詹姆斯，说道：“这样会把情况变得复杂，如果西伦科特手中真的掌握着P病毒，把他逼急了直接在拉普拉塔河上释放病毒，病毒就会沿着运河进行传播，其后果难以预料。”
<br/>　　“我跟DEA沟通过。”詹姆斯耸了耸肩，说道：“但是他们并不打算卖我这个面子，执意要按照计划对这艘货轮动手。不过通过协调，DEA方面答应这次行动以我们为主，我们先上船对西伦科特采取行动，DEA进行收尾。”
<br/>　　“白捡一份功劳？”罗昊戏谑的看着詹姆斯。
<br/>　　詹姆斯略显尴尬的摸了摸他的鹰钩鼻，随后说道：“也不能说DEA白捡一份功劳，他们也给我们提供了一些有价值的东西，在这艘货轮上有DEA的卧底，卧底告诉了我们西伦科特手中生化武器的位置。”
<br/>　　说着，詹姆斯把另外几张照片放在三维电子沙盘上，照片应该是用手机之类的工具，在比较远的地方拍摄的，所以并不是特别清晰，但是依然可以通过西伦科特从液氮低温瓶中拿出来的套着金属外壳的玻璃瓶大小判断，这就是西伦科特从山顶别墅地下室冷藏柜中拿走的东西，而且根据玻璃瓶中紫红色的液体，也基本可以确定，在这玻璃瓶中装得就是P病毒！
<br/>　　至少罗昊不会认为，西伦科特会随身带着三管葡萄汁到处跑！
<br/>　　“这次行动，除了Alpha和Bravo小队外，我会让Force-Recon进行配合。”詹姆斯看着罗昊和麦克米兰，说道：“等到这艘货轮进入大西洋后再动手，DEA在货轮上的卧底已经安装了追踪器，我们的卫星也会死死地咬住它，行动完成后，会有直升机接应你们撤离。”
<br/>　　詹姆斯走到罗昊面前，对他伸出手说道：“少校，如果你们这次行动顺利的话，这将会是我们最后一次见面和握手了。”
<br/>　　“那我希望这就是我们的最后一次见面和握手。”
<br/>　　罗昊笑着伸出手跟詹姆斯握了握。
<br/>　　“少校，我还是那句话，希望我们以后不会以对立的身份在战场上见面。”詹姆斯说道：“跟你这样的人交手，会让上帝不再保佑美利坚。”
<br/>　　“我们国家从来都是热爱和平，并且努力去维护和平。”罗昊握着詹姆斯的手，突然用力，盯着詹姆斯说道：“虽然我们不会主动去挑起战争，但是我们也不会害怕别人挑起战争。如果真的有一天战火爆发，而我还能拿起武器走向战场，将军，我可以向你保证，我和我的小队一定是第一个进入战场并且最后从战场撤离的部队。”
<br/>　　“我丝毫不怀疑这一点。”詹姆斯率先松开了手，那双深邃的蓝色眼睛盯着罗昊，说道：“如果你们国家的指挥官都跟你一样，那就太可怕了。”
<br/>　　“狂龙也许只有一个，但是比我优秀的指挥官会有很多，将军。”罗昊回答道。
