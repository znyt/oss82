        冷夫人意味深长地看着童筱颖，终于不再继续这个话题。
<br/>
<br/>        从冷夫人的屋子里出来后，童筱颖只觉自己的心口在隐隐作痛，她不明白自己为何会有一种难以言喻的心痛感，就像被什么虫子钻了心，很难受，但是又说不出来。
<br/>
<br/>        不知何时，眼睛湿润的时候，模糊的视野里，突然出现了一抹身影。
<br/>
<br/>        童筱颖定了定睛，抬手拭去眼角的泪花时，男人的样子，映入了自己的瞳仁。
<br/>
<br/>        橘色的庭院灯下，傅夜沉玉树临风，五官清晰立体，他身上散发的气息，透着与生俱来的矜贵。
<br/>
<br/>        在她沉睡的那段日子里，她一直在做梦，梦里只有这个男人的影子，与她若即若离。
<br/>
<br/>        “嗨，大哥。”童筱颖若无其事地展开笑颜，打个招呼都显得与他的关系有些生疏。
<br/>
<br/>        傅夜沉朝她迎面走过来，直到来至她的跟前，才停下脚步，高大的身影，将她的娇躯完全笼罩。
<br/>
<br/>        童筱颖不得不仰头看向傅夜沉，两只不知所措的纤纤玉手背到身后，尴尬地替自己辩解：“我急着来见婆婆，忘记给你打电话了。”
<br/>
<br/>        他发现，她虽然失忆了，但是她并不排斥她是他的妻子的这件事。
<br/>
<br/>        “我妈跟你说什么了？”傅夜沉的语气凝重，似乎是不许她对他撒谎。
<br/>
<br/>        童筱颖也不打算藏着掖着，直接跟傅夜沉坦白：“婆婆说，我不能生育的话，可以找个女人来代孕。”
<br/>
<br/>        “你确定你不能生育吗？”傅夜沉的声音淡定清冷。
<br/>
<br/>        如果，连华生的父亲能让筱颖活下来，还治好了她的听力，那么她……
<br/>
<br/>        心里想到这块时，傅夜沉点漆的眸子，下意识地落向了童筱颖的肚子。
<br/>
<br/>        他觉得，她可以生，只是现在，她可能会不大愿意跟他做。
<br/>
<br/>        童筱颖见傅夜沉盯着自己的下方，本能地抬起手来护着自己的肚子，微微侧了侧身：“我又没生过孩子，我怎么知道自己能不能生。”
<br/>
<br/>        “我觉得你可以。”傅夜沉会心一笑。
<br/>
<br/>        童筱颖忽然间想到了什么，心直口快地说道：“要不这样吧！明天我去医院做检查，如果确定我不能生孩子，我们俩就离婚，你再重新娶一个能生孩子的老婆。我觉得你人很好，长得又帅，还有家企，不能无后。”
<br/>
<br/>        “不用去医院，我也可以帮你做检查。”傅夜沉嘴角微扬。
<br/>
<br/>        他微微俯身，一张俊脸瞬间在童筱颖的眼前放大。
<br/>
<br/>        童筱颖惊怔地缩了缩脖子，又有些好奇：“难道你也是医生？”
<br/>
<br/>        “……”傅夜沉顿时汗颜了。
<br/>
<br/>        他的小女人变得纯情了，嗯？
<br/>
<br/>        是不是在她眼里，她现在所遇见的人，全都是医生？
<br/>
<br/>        事实上，也差不多了。
<br/>
<br/>        醒来后，满屋子都是医生，一天到晚都在抽她的血，研究她身上的基因。唯一不是医生的裴金澈，却跟她一样，都是实验品。
<br/>
<br/>        人生的悲哀，莫过于此。
<br/>
<br/>        童筱颖对“医生”的潜意识里，有些敏感，也有些后怕。
<br/>
<br/>        “你难道不想知道我们俩的过去吗？”傅夜沉接着问道。
<br/>
<br/>        童筱颖却摇了摇头，反问：“我为什么要知道过去的事情？现在不是挺好的吗？”
<br/>
<br/>        从她在实验室里苏醒过来后，她就从来没有想过要去了解自己的过去。不过，她倒是对他挺好奇的。
<br/>
<br/>        “对了，我还不知道你叫什么名字。”童筱颖尴尬地接着说，认识他也有一段时间了，她确实没有主动问过他有关于他的事情，而且他也没告诉过她。
<br/>
<br/>        “傅夜沉。”傅夜沉垂着眸子，淡淡地应道。
<br/>
<br/>        童筱颖莞尔一笑：“你的名字还挺好听。”
<br/>
<br/>        “我妈跟你说的那些事情，你不用放心上。你放心，我不会跟你离婚。你不是还想继续环游世界吗？你若是跟我离婚了，谁给你钱去环游世界，对吧？”傅夜沉打趣地说。
<br/>
<br/>        童筱颖不好意思地挠了挠耳后的长发：“我……我不想去旅游了。”
<br/>
<br/>        其实，她一个人去旅游，虽然轻松自在，但是总觉得少了点什么。
<br/>
<br/>        “那你接下来想做什么？”傅夜沉直起身子，温和地问道。
<br/>
<br/>        童筱颖托起下巴，歪着脑袋，仔细地想了想后，樱桃小嘴微微上扬：“嗯——想去吃好吃的！”
<br/>
<br/>        “那我带你出去吃好吃的东西。”傅夜沉的黑眸里透着久违的温柔，语气也是十足的宠溺。
<br/>
<br/>        当他向她伸出手，仅仅只是想牵着她的时候，她却只是沉浸在自己的欢喜中，眼睛眯得像两弯月牙，然后自顾自地拍着手转身往那边大门口走去。
