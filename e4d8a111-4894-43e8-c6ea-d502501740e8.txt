　　苏莫闻言，这沁儿姑娘，好奇心居然如此之重。
<br/>　　“玄天长老，要对我进行考验！”苏莫同样神念传音，向沁儿说道。
<br/>　　“什么考验？”沁儿闻言，眼眸一亮。
<br/>　　“就是看看我的天赋实力，值不值得大力培养！”苏莫传音说道。
<br/>　　“原来如此！”沁儿了然，看来父亲要对段惊天考验一番，再做决定。
<br/>　　不过，他心中还有疑惑，这点小事，为何父亲要瞒着她，一切都闭口不提呢。
<br/>　　这真是奇怪，看来段惊天还有所保留，没有告知她详细情况。
<br/>　　不过，沁儿也不没有再问，毕竟，她再问也问不出什么。
<br/>　　“沁儿姑娘，我们来轮回古城祭拜天尊！六宫弟子之间，会有哪种比斗？”苏莫向沁儿问道。
<br/>　　他心中清楚，肯定不是单纯的比武，不然也不至于来轮回古城。
<br/>　　“祭拜天尊大人，可以得到天尊大人的福泽，但是，并不是所有人都能祭拜的！”
<br/>　　沁儿并未隐瞒，徐徐开口，继续道：“想要祭拜天尊，首先要不满百岁的真神，一生只能受到一次福泽，其次，要祭拜天尊，要先闯过三关！”
<br/>　　“哪三关？”苏莫好奇的问道。
<br/>　　沁儿微微一笑，一双灵动的大眼睛极为闪亮，笑着道：“这三关，被称为轮回三关。”
<br/>　　稍稍停顿，夕儿继续道：“轮回三关，第一关是天门关、第二关是地元关、第三关是人神关！”
<br/>　　“天地人三关，和天王宫、地王宫、人王宫有关系吗？”苏莫好奇的问道。
<br/>　　“并没有关系！”
<br/>　　沁儿摇了摇头，道：“这三关，只是祭拜天尊大人的门槛，是一种考验，闯过了这三关，才能祭拜天尊大人！”
<br/>　　“原来如此！”
<br/>　　苏莫点了点头，心中了然，又问道：“还有呢？”
<br/>　　他知道，绝对不可能这么简单，不然，他怎么超越六宫妖孽，证明自己的天赋实力。
<br/>　　“闯过三关之后，剩下的人，才有资格祭拜天尊，不过，每个人能得到的福泽，就看自身的实力了！”沁儿凝视说道。
<br/>　　“是要互相比斗吗？”苏莫沉声问道。
<br/>　　“算是吧！到时候你就知道了！”沁儿笑了笑。
<br/>　　“多谢沁儿姑娘告知！”苏莫表示感谢。
<br/>　　“你无需客气，希望你能通过父亲的考验！”
<br/>　　沁儿微微一笑，得意洋洋的道：“这一次，我姬月沁也要向父亲展示一番。”
<br/>　　苏莫轻笑一声，这个姑娘心思真是简单。
<br/>　　不远处，褚彦、骆妤和姬月松，看到沁儿和苏莫相谈甚欢，均是眉头微皱。
<br/>　　这个沁儿，未免和段惊天走的太近了吧！
<br/>　　“段惊天，这次让你好看！”
<br/>　　褚彦心中暗怒，心中思量着，如何给予段惊天教训。
<br/>　　他要让段惊天，在沁儿面前，颜面扫地。
<br/>　　“月松哥，这个段惊天来路不明，沁儿和其走的太近，难免以后出问题！”骆妤向姬月松传音说道。
<br/>　　“放心吧，待会有机会，我敲打敲打段惊天！”姬月松传音道。
<br/>　　“嗯！”骆妤点了点头。
<br/>　　另一边，玄天长老长身而立，凝视着前方的轮回古城，心有感触。
<br/>　　除非轮回天尊再生，不然，轮回古城将永远不会彻底开放。
<br/>　　他们六宫之人，也只能勉强开启一小部分。
<br/>　　叹了口气，玄天长老转头，瞥了一眼沁儿和苏莫。
<br/>　　对于沁儿和苏莫走的近，他倒是没有什么意见。
<br/>　　嗖嗖嗖！！
<br/>　　就在此刻，天空中传来犀利的破风声，众人转头看去，便看到近百名武者，齐齐飞天而来。
<br/>　　“是地王宫的人！”
<br/>　　“地王宫的人，比我们稍多一些！”
<br/>　　“很多妖孽天才！”
<br/>　　众人王宫的弟子，见到这飞来的近百人，均是凝目观望。
<br/>　　这百位到来之人，均是地王宫之人，同样由一位地王宫的中年长老带领，来势汹汹。
<br/>　　嗖嗖嗖！
<br/>　　片刻之间，百名地王宫的弟子，纷纷落在了轮回古城的城门之前。
<br/>　　一个个地王宫的弟子，均是气势如虹，大有唯我独尊的气势。
<br/>　　众人王宫的弟子，均是在打量地王宫之人，苏莫的目光，亦是落在了地王宫的人群之中。
<br/>　　他的目光，首先就落在了一位平凡的青年身上。
<br/>　　地王宫绝世妖孽，陈凡！
<br/>　　此刻的陈凡，修为也已经有所突破，达到了二品低级。
<br/>　　苏莫扫了一眼，发现地王宫之中，修为达到二品高级的也有两人，整体阵容比人王宫强不少。
<br/>　　陈凡，以及那三名二品高级的弟子，都是众人关注的交点。
<br/>　　“姬玄天，这次你居然来了！”
<br/>　　地王宫的长老，看向玄天长老，面带意外的说道。
<br/>　　因为，他完全没想到，堂堂人王宫三长老，居然会亲来轮回古城。
<br/>　　地王宫的这位长老，是一位高高瘦瘦的中年人，一身灰袍，留着一缕三羊胡，给人一种仙风道骨的之感。
<br/>　　“姚长老，姬某索性无事，便带队来此，并且看看小女姬月沁的本事！”玄天长老道。
<br/>　　“原来如此！”地王宫的姚长老点了点头，他也并未多想。
<br/>　　在两位长老交谈之间，人王宫的众弟子，和地王宫的众弟子，互相凝视，战意在弥漫。
<br/>　　当然，大家都在克制，毕竟，现在不是动手的时候。
<br/>　　“姬月松，上次你去天浑虚界，我没能前去领教，这次希望有这个机会！”
<br/>　　地王宫的阵营之中，一名二品高级修为的青年，目光凝视着姬月松，微笑着说道。
<br/>　　这名青年高大魁梧，浓眉大眼，气息浑厚无比。
<br/>　　所有人王宫的弟子之中，能被他看在眼中的，也就姬月松一人而已。
<br/>　　虽然人王宫，还有一位二品高级的林无霜，但是，此女实力不如姬月松。
<br/>　　“狄央，你会领教的！”姬月松面色淡淡，但是心中，却也是充满凝重。
<br/>　　这个地王宫的狄央，绝对是一个劲敌，不仅战力强大，并且在二品高级已经沉淀了好几年，距离三品都不远了。
<br/>　　此人，是地王宫之中，最强的一人。
<br/>　　“呵呵，那可不一定，你若是连陈凡都不如，我也不会有兴趣！”狄央笑了笑。
<br/>　　“陈凡！”姬月松脸色一沉，随即看向狄央身边，面色冷淡的陈凡。

<br/>　　“狄央，你们地王宫，也就你能让我稍稍重视！”姬月松淡淡的说道，言语毫不客气，自信无比。
<br/>　　“彼此彼此，你人王宫，也就你有点本事！”狄央冷笑一声说道。

<br/>　　尤其是鬼王宫，非常的强大，远超任何一宫。

<br/>　　二品高级，已经是天赋的极限了！
