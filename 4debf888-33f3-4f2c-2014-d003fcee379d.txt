　　人有两根中指，能发挥效力的，自然是男左女右。
<br/>　　至于人眼，法术界中叫“招子”，是人精气神的所在，不过一般人的眼对邪物没什么伤害，只有开了天眼，才能发挥出人眼的潜能，阴阳眼，只是天眼的最基本的形态。
<br/>　　古龙曾经著名的一本小说《陆小凤传奇》，其中陆小凤的绝招灵犀一指，就是用中指夹住任何兵刃，这就是有典故的。
<br/>　　叶少阳听梧桐说到灵犀指，当下点了点头，听她继续往下说。
<br/>　　“坛城，不管是在藏传佛教，还是****之中，都是一种具有灵性的法阵，一般都是用来祈福或者加持某些开光仪式，供奉溢美，但是我们面前这座坛城，并不是那种坛城，跟藏传佛教和****都没关系，而是西域的一种原始的巫术……
<br/>　　西域在佛教入住之前，是信仰苯教的，不过当时还有很多各种原始信仰，后来随着佛教入住，这些宗教都被影响，不管是正教还是邪/教，都从佛教里吸收了不少，死灵坛城，就是从模仿佛教的坛城，再配合原始的巫术，所形成的一种法阵……
<br/>　　如果我没猜错，我们面前这个，就是这种法阵，就是死灵坛城……我从来只是听说过，还是第一次亲眼见到。”
<br/>　　叶少阳听她说完，一张嘴早就张得老大，说道：“死灵坛城，到底是什么，****又是什么？”
<br/>　　“****就是苯教啊，西藏有五大教派，黄红白花黑，黄教就是格鲁派，也就是现在藏区的主流，****，就是苯教，不过严格说并不算是佛门，只是吸收了很多佛门的东西，在不懂的看来还以为是佛教的一支。
<br/>　　还是说死灵坛城……这个东西不知道是谁创造的，坛城本身，就像中原法术界中的塔一样，本身就具有很强的灵力，如果开光加持，就能成为法阵，只不过这尊死灵坛城供奉的不是佛，而是某种邪神，再取一百〇一根人男子左手的灵犀指，嗯，这些人必须是童男子。
<br/>　　传闻，这些男子要先用法术，将一身阳气都逼到灵犀之上，用丝带扎好，再直接跺下，作法困住其中的魂力，再摆成不同的姿势……你看这一堆堆的手指摆成的不同图案，就像是符文一样，组成在一起，就是一个法阵，靠的是储藏在手指内的人的阳气来维持……
<br/>　　之后，在阵法形成之后，气息内敛，自称阴阳小周天，丝毫也不会卸去，就成了一个以自己为核心的法阵。我们现在看到的，就是这样。”
<br/>　　叶少阳听她说完，再看这一百多根手指……想到它们是从一百个人手上砍下来的，叶少阳只能用四个字来形容这种感觉：触目惊心。
<br/>　　“所以，这个法阵的作用是什么？”叶少阳问出核心的问题。
<br/>　　“作用，就是激活并且保证棺材在这里的风水不变，里面的尸体享受风水灵气，只要法阵不会破坏，尸身可以千年不腐，同时也能照应到子孙后代。”
<br/>　　叶少阳听到这里，总算是明白了，手摸着下巴，说道：“你的意思是说，有人在这里造了这个东西，把他祖宗放在这棺材里，为了福荫后人？”
<br/>　　“没错，这里本是龙脉，水月洞天，这坛城还有采集风水灵气的作用，加上自身的法阵，能更好地吸收这里的风水妙处……”梧桐说着转头看了看四周，道：“这么好的风水，再加上这座坛城，后代一定会出一个枭雄。”
<br/>　　叶少阳皱眉道：“为什么不是皇帝？这不是龙脉吗？”
<br/>　　梧桐笑道：“龙脉是没错，不过帝王者，福临天下，容不得妖邪之气，这里为了这座坛城，杀了一百个人，煞气太重，后人就算出头，也只是个杀伐天下的枭雄，绝对当不了真正的皇帝。”
<br/>　　叶少阳还是好奇，道：“那为什么不直接弄个能出帝王的什么法阵之类的？”
<br/>　　梧桐笑起来：“你以为，帝王的灵台就那么好搭吗？不是说给你龙脉，埋在这里就一定能当皇帝的，很多龙脉从外表看就是荒山野岭，不少当地人死了也都埋在那里，为什么没有出皇帝？”
<br/>　　一句话问得叶少阳说不出话来。
<br/>　　因为之前被他救了一命的缘故，梧桐跟他亲近了许多，不爱说话的她，也对叶少阳打开了话匣子，大致说了龙脉聚养帝王之气的灵台是怎么搭的。
<br/>　　“我听我爷爷说，大清国的龙脉，是在长白山龙脉下面，修建了一座金龙吐水，不过涂的不是水，而是金沙，并且有极其高超的工匠制作，金沙循环使用，不断被喷涂出来，将风水印活，长盛不衰……”
<br/>　　叶少阳听了这话，当场震惊。
<br/>　　“所以他们家做了几百年的皇帝？那难道古代那些开过皇帝，都是因为祖上大费周章地在龙脉埋过人？”
<br/>　　梧桐忍不住发笑，“你本末倒置了。风水只有禳补，哪有能决定一切的，找龙脉埋先人，只是希望将来能出一位帝王，哪里是所有皇帝都是这么来的。靠风水术想当皇帝，其实是非常难的，因为风水学中，百密一疏，只要有一处不对，整个风水盘也就毁了，达不到最好的效果。
<br/>　　而人算总有穷尽，就算是我们地师家族，也不能保证做到，就算做到，一些天灾地动、甚至一只钻进古墓的穿山甲或者老鼠，都能毁坏整个风水盘，再像禳补就不可能了。”
<br/>　　叶少阳缓缓点头，这些知识，大部分大本来是懂的，只是因为梧桐刚给他说了太多新奇的东西，一时间有点晕，问了傻问题，当下说道：“其实，这也算是一种天命吧，有时候机关算尽，但是并不一定就能成功。”
<br/>　　梧桐点点头，表示赞同他的意见。
<br/>　　叶少阳再看那些用来凑阵的手指，想到他们当初是被人从一百个人手上生生斩下来的，顿时有点头皮发麻。
<br/>　　两人的目光，不约而同落在那副棺材上。
<br/>　　叶少阳道：“你之前说，这里卖的是大宋皇族的后人？可是，为什么会用西域的邪术？而且我们之前看到的就是。为什么会这样？”
<br/>(本章完)
