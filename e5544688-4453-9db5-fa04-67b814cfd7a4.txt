        &lt;h3&gt;第三卷一百三十六章  陈子澜到甸城&lt;/h3&gt;
<br/>
<br/>        陈子澜一行四人到达甸城。
<br/>
<br/>        他们是第一次到甸城，因为上次的案子，泉城那个小县城随着社会的发展，已经不再是一片清静之地，虽然他们的工作做得很细致，但是总是难免有漏网之鱼。
<br/>
<br/>        上头的很是重视这件事情，陈子澜工作细心，在他们行内是出了名的，再加上身手也不错，这一次就让她带队去甸城看看和甸城的人碰头，双方配合着调查一些事情。
<br/>
<br/>        甸城地势复杂，人口分布范围太大，从来都是灰色聚集地，后来因为秦寿接手倒是隐秘了一些，很多灰色，势力被端掉，秦寿在他们业内可是被大家公认的大英雄。
<br/>
<br/>        世界上邪恶和正义是并存的，不过是此消彼长而已，谁能将邪恶一网打尽？但是正义永远不会缺席。
<br/>
<br/>        无论邪恶如何气焰嚣张，正义之剑永远悬挂前方。
<br/>
<br/>        陈子澜到达甸城，接待他们的是甸城的一个部门老总，这个人叫廖国华，是跟着秦寿一起很多年的人，从一个小人物被秦寿提拔起来的。
<br/>
<br/>        到达的当天，廖国华安排陈子澜他们住下，并请陈子澜他们吃了顿晚上。
<br/>
<br/>        “陈总，你们远道是客，今天就好好休息一晚上，明天我安排人将资料拿给你，配合你们的工作。”廖国华客气的说。
<br/>
<br/>        “真是谢谢廖总了，早就听说廖总是秦总手下得力的帮手，人帅工作能力强，今天一见真是不虚其名。”陈子澜笑着说。
<br/>
<br/>        “就是，不知道我们能不能见上秦总呢？秦总可是我们仰慕的对象，也是我们学习的榜样。”陈子澜的一个同事接话。
<br/>
<br/>        “秦总现在事情特别多，我有时候都会见不到他，等过几天我找他汇报一下，看看有没有时间接待你们。”廖国华客气的说。
<br/>
<br/>        “啊，那太好了，要是能见到秦总也不枉来一趟甸城。”另外一个同事充满了喜悦。
<br/>
<br/>        “唉，没有想到秦总名气这么大，就连你们那样的小县城都知道。”廖国华笑着说。
<br/>
<br/>        “你不知道啊，秦总是我们的教育材料呢，是我们的偶像也是我们学习的榜样。”另一人接着廖国华的话说，语气里满是崇拜。
<br/>
<br/>        “嘿嘿，秦总也是我们学习的榜样，多面手！”廖国华开心的应付着。
<br/>
<br/>        一餐饭下来拉近了大家的距离，明天的工作是不是很顺利？
<br/>
<br/>        晚上，陈子澜对大家说：“甸城看着一片祥和，但是大家也要注意点，我们不能仅仅靠着兄弟单位帮忙，我们必须要自己主动出击。”
<br/>
<br/>        “陈总，那你说我们该怎么办？”同事们问陈子澜。
<br/>
<br/>        “我们这一次为期十五天的调查，也许时间更长，大家做好思想准备。我们需要去一些小的地方看看，也许能找到一些线索。”陈子澜皱着眉头。
<br/>
<br/>        “就是从小地方查起，甸城有秦寿坐镇，一些不法分子肯定不敢在他们的眼皮子底下活动，只能去周边的小地方看看。也许那些脏东西是从小地方流出去的。”
<br/>
<br/>        “也不一定是从这里出去的，毕竟地方太大，我们的线索也不是很明显，只是只要有一丝希望我们就得去寻找。但愿这个地方安安宁宁，干干净净。”陈子澜叹了口气。
<br/>
<br/>        廖国华坐在办公室里，寻思着这几天怎么安排陈子澜他们。
<br/>
<br/>        这也有很多年了，甸城基本都是太太平平的，没有太多的事情，偶尔就是小打小闹一番，表明甸城只不过是在他们的工作范围内比之前要好了很多，但是要是说一个地方真的是干干净净反而惹人怀疑。
<br/>
<br/>        陈子澜他们的事情说小也小，说大也大，看怎么样能将大事化小，小事化了，像这种事情是不用汇报给秦寿的，他直接就搞定了。
<br/>
<br/>        在他的心目之中敲定了要配合陈子澜需要去的地方和需要找的档案，安排上几个人陪着陈子澜他们转转，过几天送他们回家就可以了。
<br/>
<br/>        第二天，廖国华安排人配合陈子澜他们将很多之前的档案拿出来，先看看有没有和之前相似的地方，因为每一个团伙都有其独到的手法和行事方式。
<br/>
<br/>        看着那么多的档案，陈子澜眉头紧皱，这些档案他们没个十天半月是看不完的。
<br/>
<br/>        等廖国华的人一走，陈子澜对同事们说，我们今天先看看档案，明天留俩个人继续看档案明天俩个人去外面转转。
<br/>
<br/>        陈子澜他们安心的查了一天档案，什么都没有发现。
<br/>
<br/>        第二天陈子澜想出去，可是廖国华阻止了。
<br/>
<br/>        “你们先看完这些，找一找有没有相似的地方，况且你们对这里人生地不熟的，要想出去我们应该配合你们，只要你们说想去哪里我们便去哪里，这样无头苍蝇那样到处乱飞是不行的。”
<br/>
<br/>        四天过去了，陈子澜他们一无所获，八天过去了，他们在档案面前摇头叹息。
<br/>
<br/>        “我们不能光在这里看这些没有用的东西，明天无论如何我们要走出去。”陈子澜指着一个档案，这是几年前的甸村，离甸城一百余公里，曾经被北京某部门带人捣毁了一个团伙，此后几年这里安安宁宁，再也没有任何不法的事情。
<br/>
<br/>        “这个地方有些蹊跷，当初北京来的人竟然没有和当地的部门联系，都是悄无声息的的来的，知道事情结束之后，当地部门才赶过去。”陈子澜看着大家说。
<br/>
<br/>        “这说明了，当初的哪个人有可能知道联系了当地不会有任何作用。”陈子澜的同事说。
<br/>
<br/>        他说完之后大家都不再说话，低着头继续查资料。
<br/>
<br/>        “明天我要去这里看看。”陈子澜语气坚定。
<br/>
<br/>        “我和你一起去吧。”另一个同事郑培说。
<br/>
<br/>        “可以，你们俩个人就在这里，我们俩个出去。”陈子澜考虑明天怎么出去。
<br/>
<br/>        陈子澜和郑培第二天说是想出去逛逛甸城，留下俩个同事查档案，廖国华想要安排人陪同的时候被陈子澜拒绝了。
<br/>
<br/>        陈子澜说：“我们就是出去逛街，这几天把人闷得不行了，我出去买点私人用品。”
<br/>
<br/>        廖国华看着陈子澜笑了笑：“那你好好转转，工作的事情也不急于一时，我给你派一辆车。你们去转转。”
<br/>
<br/>        “不用派车，我们就走走。也好看看甸城的风景。”陈子澜微笑着拒绝了。
<br/>
<br/>        陈子澜和郑培租了当地的一辆车，让人将他们拉到两百公里外的甸村。
<br/>
<br/>        廖国华看着远去的陈子澜嘴角噙着笑。
<br/>
<br/>        拿起手机发了一个短信，转身哼着小曲走进档案室。
<br/>
<br/>        “你们那俩个同事出去转了，你们不去吗？”廖国华看着陈子澜的同事说。
<br/>
<br/>        “我们都来这么久了，一点头绪都没有，那有心思转的啊。”俩个同事苦恼的说。
<br/>
<br/>        “这么说，陈总他们是出去暗查去了？”廖国华问道。
<br/>
<br/>        “啊，没有，陈总说出去买东西去了。”
<br/>
<br/>        “那就好，我就怕他们私下里出去，这里是山区，不太安全，别出事情就好。我不打扰你们工作了。”廖国华转身走了。
