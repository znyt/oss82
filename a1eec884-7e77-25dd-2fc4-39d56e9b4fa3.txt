　　叶少阳道：“那就多派几个人，一路赶过去就是了。”
<br/>　　张道长叹息道：“就是人手不够，最近兵荒马乱，死的人多，几个赶尸匠分头赶尸，目前山上只有一个赶尸匠，想等其余赶尸匠，但时间不够，一来苦主等不了，二来就算有灵符加持，也没法长期封印尸体……”
<br/>　　叶少阳听到这里，才算真正明白了，说道：“道兄是想让我帮忙赶这一趟。”
<br/>　　张道长道：“以老弟你的实力和见识，有你赶这一趟尸体，那一定会出什么问题，最重要的是，这一批尸体正好就是往东北方向去的，跟你要去的地方算是顺路，所以……”
<br/>　　“那没问题，反正我自己走也是走。”叶少阳爽快答应下来。
<br/>　　张道长也没想到他答应得这么干脆，睁大眼睛看着他，说道：“老弟，你真的肯帮忙？”
<br/>　　叶少阳笑道：“这有什么不肯的，不是什么大事。”
<br/>　　“老弟仗义！那就有劳老弟了！”
<br/>　　张道长又跟他说了一些细节。本来这件事情已经耽误了很久，既然叶少阳答应，当然是越快越好。叶少阳提出这就去干活。
<br/>　　张道长叫上两个道童，帮叶少阳背着行李，几个人一起出门，前往公堂义庄。
<br/>　　公堂义庄离清风观几十里的山路，路上闲着也是闲着，叶少阳借口自己一直在山里清修，对外界情况不甚了解，跟他打听起当今法术界的情况。
<br/>　　张道长身份有限，无法接触到几大宗门的上层，但因为在俗世里混，对大致的情况还是有所了解，一路上侃侃而谈，把自己了解的尽数向叶少阳道出。
<br/>　　叶少阳听的很认真。
<br/>　　对于民国时期法术界的情况，他之前从典籍记录里也看过，但是典籍上记载的多是几大宗门上层中间的情况，以及一些不世出的恶鬼厉妖出没的这类奇闻异事，对于真正世俗之间的法术界的情况，记载的并不多。
<br/>　　叶少阳听了张道长的讲述，脑海中自动浮现出了四个字：龙蛇混杂。
<br/>　　民国时期的法术界，远远比他想象中要复杂的多，乱世出妖邪，同时也是法术界最辉煌的时期。
<br/>　　“叶老弟，你是茅山弟子，你们接触的，应该都是几大门派的高人吧？”张道长满怀羡慕地说道。
<br/>　　叶少阳只好说是。
<br/>　　张道长于是说了几个对他来说都是传说中的人物，大部分叶少阳也都知道——从典籍里看到过，都是自己的前辈，只好敷衍他说听说，但是没有细谈。
<br/>　　“那叶老弟与张晓寒认识吗？”张道长突然问道。
<br/>　　“张晓寒？”叶少阳仔细回忆了一下，自己从未听说过这个名字。
<br/>　　“对啊，张晓寒，号称佛门第一天才，你不知道？”张道长有些吃惊。
<br/>　　“哦，你怎么会知道他？”叶少阳故意反问了一下，心中啧啧称奇，第一天才这种说法，居然在民国的时候就有了。
<br/>　　“嘿嘿，张晓寒的名字，只要是法师，哪个没听说啊。佛门几百年一遇的天才，说起来还是我本家呢，可惜像我们这样的，连见一面的机会都没有。”
<br/>　　叶少阳更加好奇，问他：“你们民间是怎么说他的，说来听听。”
<br/>　　张道长于是说起了听来的，有关张晓寒的信息。张晓寒，二十多岁，九华山俗家弟子，据说二十岁不到就受封禅师，法力精深，前途无量，是法术界公认的最有前途的年轻人。三年前下山历练，走遍南北，到处斩妖除魔，也找同辈的知名法师比试，无一败绩，遂名动天下……
<br/>　　叶少阳听了这番解说，顿时想起了一个熟悉的人名：道风。
<br/>　　听张道长说的天花乱坠，这个张晓寒应该是个极厉害的人物，但是令叶少阳不解的是，他从来没听过这么名字，之后张道长还说了好几个少年天才的名字，叶少阳都听过，这些之后都成了法术界的大人物，唯独这个张晓寒，叶少阳确定自己从来没在任何典籍上看过他的名字。
<br/>　　叶少阳想不通这是为什么，只是觉得其中肯定有什么缘故。他暗暗把张晓寒这个名字记在了心里。
<br/>　　法师的黄金时代……无数天才涌现，在乱世中互相竞争，各显神通。
<br/>　　叶少阳听张道长说了一路，简直就是一卷民国法术界的人物志，各门派的奇才高人，在这个妖孽迭出的乱世粉墨登场，竞相生存……如果不是自己有太多的事情要去办。叶少阳其实挺想留在这个世界，跟这些所谓的天才们竞争一番。
<br/>　　翠云听不懂他们说的这些，在旁边也不插话，脑海中有自己的盘算。那两个道童也不敢插嘴，就只有叶少阳和张道长两人一个问一个答，几十里的山路，就听见两人一路嘚啵嘚啵地说。
<br/>　　天擦黑的时候，他们总算来到公堂义庄。
<br/>　　义庄是在一个山坳里，几个道士，占据了一片无人荒村，找了一些稍微好点的房子居住，还雇了几个村民在这里住，养了点鸡鸭，种了些菜，整个地方破是破了点，不过看上去倒也有一点世外桃源的感觉。
<br/>　　在村子的四周，确实种了一圈桃树，还有一圈枣树，将村子围起来，并且布置成了迷宫一样的所在，从前山俯瞰了一会，叶少阳立刻看出了门道，说道：“九曲双煞阵。”
<br/>　　张道长点头道：“老弟不愧是茅山天师，一眼就看明白了，的确是九曲连环，尸鬼双煞。”
<br/>　　这里的“煞”，是针对鬼和僵尸说的，桃木压鬼，枣木克尸。义庄毕竟是停放尸体的地方，跟鬼本来没什么相干，但因为阴气重，又是在深山里，不免会吸引到一些孤魂野鬼过来吸收阴气修炼，或者干脆借尸还魂。
<br/>　　用枣木和桃木来布置这样一个阵法，能防止外鬼进来，也能防止里面的僵尸跑出去。
<br/>　　至于九曲连环，其实就是一个弯弯绕绕的道路，并不复杂，人进去之后，可以见弯拐弯，一条路走到底就可以，但是对鬼怪和僵尸来说，这些树木就成了浩瀚的森林，如果修为不够，很容易就会迷失在中间，怎么也转不出来。
