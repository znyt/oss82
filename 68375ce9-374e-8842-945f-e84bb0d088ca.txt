　　幻象！
<br/>　　木落真人的理智提醒自己，这一切都是幻象，他握着道剑的手迟疑了一下，便用力刺下去。
<br/>　　道剑贯穿了女子的脑袋，伤口从附近碎裂，像破碎的瓷器一样，一片片掉下去，路出空空的内部。
<br/>　　果然是幻象。
<br/>　　木落真人松了口气，就在这时，面前那尊还没有完全碎掉的女子，嘴角突然扬起，露出一个诡异的微笑。
<br/>　　她笑什么？
<br/>　　突然，木落真人感觉自己腋下一阵剧痛，低头看时，从腰部往上，有绿色的火焰燃烧起来，自己的肉身在迅速融化。
<br/>　　刚才女子手中的香炉……
<br/>　　一瞬间，木落道人明白过来，就是方才自己那一瞬间的迟疑，失去了先机，被那女子得手，在自己身上种下了这可怕的火焰。
<br/>　　他用力拍打起来。
<br/>　　突然后脑勺一凉，他凭着本能猛然转身，同时双手交叉，托举上去。
<br/>　　一道金光，从上面打下来，落在交叉的双臂上，被他修为阻挡，停顿了几秒钟，然后直接震碎了双臂，击打在他脑袋上。
<br/>　　眼前一黑，神识也在慢慢褪去。
<br/>　　月华……你在哪里啊？
<br/>　　无尽的黑暗之中，出现了一点光明，那是赤月罗刹的脸，在黑暗中漂浮着，逐渐远去，木落真人试图伸手抓住，但手刚抬起来，就垂了下去。
<br/>　　他的魂体，就在四宝的面前，缓缓散去了。
<br/>　　“你心有牵挂，必死无疑。”四宝看了看他，心中又想，你到底是个多情之人，只是走错了路。当即拿出金玟钵盂，将他的魂魄收了进去。
<br/>　　赤月罗刹和木落真人，这对太阴山的有名的情侣，就此殒命。
<br/>　　一道风雪，飞卷而来。
<br/>　　雪魔已经发现了这边的情况，惊愕之下，急忙震退了冲和子，朝高台飞过去，一边喊话让更多人追随自己，杀了高台上三个人，这才是他们此行最重要的任务。
<br/>　　“我技穷了。”林三生望着台下蜂拥而至的鬼兵，淡淡说道。
<br/>　　他说的是实话。本来，一切都按照他的计划发展，防线严密，有叶少阳等人加持结界，就算对方来的邪物的实力和数量远远超过他们预料，一样能够坚持到徐文长完工。
<br/>　　唯一没想到的，就是秋莹的死……
<br/>　　让叶少阳等人失去了理智，主动出击，反而被围困，金刚三藏在结界破碎之后，也潜入水中，不知道跟什么人战斗去了，现在高台上只剩下他跟四宝两个，面对以赤月罗刹为首的太阴山大军，显得有点螳臂当车。
<br/>　　但是他一点也不怪叶少阳。
<br/>　　面对伙伴的死，他表现出这样，才符合他的性格，否则就不是叶少阳了。
<br/>　　“有我有我，不怕。”四宝拍了拍胸脯，干脆坐在了地上，双手握着念珠，开始作法，金身罗汉在空中再现，双手在胸前虚握，中指弯曲相对，从双手中间，飞出了一颗黑色灵珠，不断旋转着放大，形成了一个黑洞，将雪魔作法召唤的风雪完全吸进去，消失不见。
<br/>　　雪魔迟疑了一下，一咬牙，冲了上去，立刻感到一股强大的吸力，将他吸入了黑洞之中。
<br/>　　那些随后感到的鬼兵，一个个都被吸了进去。
<br/>　　被吸入幻境中的，是他们的元神，他们的本尊都还站在原地，一动不动。
<br/>　　林三生呆呆看着这一幕，手持天地规尺，试图敲打雪魔，耳边突然响起四宝的声音：“不要动，他们的神识被我的诸天观想术困住，与我神念相连，若有外力杀了他们，我也会死……”
<br/>　　林三生便不敢动了，朝下面看去，那些被幻想困住的邪物之中，有一些身影越来越淡，直到消失，化作精魄。
<br/>　　他心中感到震撼，但不清楚四宝用了什么手段，居然来一个困一个。
<br/>　　四宝在轩辕山那段时间，得到鸠摩罗什真传，学到了完整的诸天观想术，一段时间的修炼，虽然没到完满，但已经大成，然后，他将诸天观想术与罗汉金身融合在一起——之前他就进行这方面尝试并且成功，如今修炼到完成的诸天观想术，一个是绝代失传的心法，一个是顶级法术，两者结合，威力极大。
<br/>　　所有被幻境困住的人，都会与他神识相连，他在神识中观想出了二十四诸天，因此这些人都会受到二十四诸天的攻击，能承受多久，全看道心和修为。
<br/>　　但每多一个敌人，四宝的神识就要多分出来一份，进行神识的操控，这是一项很累的事情，控制的人越多，也就越累。
<br/>　　虽然不断有邪物死在幻境中，元神俱灭，但鬼兵源源不断涌来，人也是越来越多，四宝逐渐感到力不从心了，林三生在一旁看到他全身都在流汗，身体也微微颤抖着，急在心头，却将无能为力-——如果没有四宝，技穷的他也只能眼睁睁看着鬼兵们冲向高台，干扰徐文长，如今，四宝已经是最后的希望。
<br/>　　“兄弟，你可要顶住！”
<br/>　　林三生咬了咬牙，飞下高台，不断斩杀着后方用来的邪物，为四宝尽可能分担压力。慕寒等一干人也在杀敌，只是邪物实在太多，大家都应顾不暇。
<br/>　　好在这些邪物的目标是攻上高台，否则如何对他们进行围攻的话，他们还真顶不住。
<br/>　　至于捉鬼联盟那边……林三生根本来不及管了。
<br/>　　“噗……”
<br/>　　四宝身子往前一松，喷出了一口血，然后继续咬牙苦撑。
<br/>　　“噗……”
<br/>　　雪魔大喝一声，也喷出一口血，然后双眼猛然睁开，不愧是雪魔，他战胜了幻境中的敌人，已经醒来了。
<br/>　　双手作法，凝结空气，化作一根巨大的冰锥，朝四宝刺去。
<br/>　　林三生一个纵身飞过去，手中天地规尺落下去，打在冰锥上。
<br/>　　冰锥斩断，但后半截去势不减，砸在了四宝额头上。
<br/>　　“啊！”
<br/>　　四宝一声惨叫，向后倒下。
<br/>　　那些被幻象困住的邪物，在一瞬间清醒过来，先迷茫地看了看四周，然后狂吼者朝四宝冲过去。
<br/>　　没办法了。
<br/>　　林三生哀叹一声，抓住四宝，努力将他拉到一边去。
