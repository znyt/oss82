        如今，她崔素妮手里已经有了范本，再设计一套总裁办的装修图，对她来说一点都不难。
<br/>
<br/>        崔素妮高高兴兴地从孙莉莉的办公室里走出来时，诺筱颖还坐在她那办公桌前发呆。
<br/>
<br/>        真是活该！
<br/>
<br/>        崔素妮心里洋洋得意地白了诺筱颖一眼。
<br/>
<br/>        辞职信……
<br/>
<br/>        她诺筱颖是不会写的！
<br/>
<br/>        诺筱颖突然从办公桌前站起身来，疾步走到了崔素妮的格子间里，二话不说，便直接抬起手来，掴了崔素妮一巴掌。
<br/>
<br/>        “诺筱颖，你简直就是个疯女人！凭什么打我？”崔素妮捂着脸，惊叫着跳了起来。
<br/>
<br/>        诺筱颖愤恨地瞪着崔素妮，声音哽咽地说道：“崔素妮，今日你陷害我，来日，我也不会让你好过！”
<br/>
<br/>        “呵？我陷害你？！诺筱颖，别给你脸不要脸了！你抄袭我的设计图，人证物证，证据确凿，还在这反咬我一口。我要报警！我还要上法庭告你！”崔素妮说着，连忙抓起桌上的手机，趁势要拨号报警。
<br/>
<br/>        大家一个个不敢吭声，孙莉莉得知集中办公室里又闹腾了起来，于是从独立办公室里走出来看了看，见崔素妮扬言要报警，便开口制止道：“小崔，家丑不可外扬，这么简单的道理你不懂？”
<br/>
<br/>        “我……”崔素妮欲言又止，不得不放下了手机。
<br/>
<br/>        孙莉莉踏着高跟鞋，抬头挺胸地走到了诺筱颖的面前，冷冷地说道：“念在你还未毕业，只是个实习生，我就给你留一份面子。你直接走好了，什么辞职信，辞退信，都不用写了。以后，我们就当，我们的公司里从未有过你诺筱颖这个人！”
<br/>
<br/>        “是啊！还不快滚噢！抄袭狗！赶紧滚啊！”此时，集中办公室里不知道是哪个人最先起了哄。
<br/>
<br/>        下一秒，大家各个对诺筱颖指指点点，纷纷在口头上用言语驱赶她离开。
<br/>
<br/>        诺筱颖看着这群人的嘴脸，心痛到想要呕吐，这种反胃的滋味，难受到让她泪眼模糊。
<br/>
<br/>        “滚啊！”
<br/>
<br/>        “滚啊！”
<br/>
<br/>        “还不滚噢！”
<br/>
<br/>        “就是，快滚！”
<br/>
<br/>        大家异口同声地呵斥。
<br/>
<br/>        诺筱颖顿时感到一阵天旋地转，不得不胡乱收拾着自己的手提包，抱着自己的东西，仓皇而逃。
<br/>
<br/>        明明她没有抄袭，却被人害得如此狼狈不堪。
<br/>
<br/>        诺筱颖离开办公室后，走到一个垃圾桶边，痛心到反胃，不得不扶着垃圾桶，把肚子里的食物吐了个干净，差点连胆汁都吐出来了。
<br/>
<br/>        职场的尔虞我诈，她第一次见识到。
<br/>
<br/>        而这种手段，只不过是小儿把戏。
<br/>
<br/>        诺筱颖背靠着墙，缓缓地往下滑落，瘫坐在了地上，目光呆滞。
<br/>
<br/>        她只是想坐在这儿缓缓自己这糟糕透顶的心情，却没想到，待了还不到十分钟，就有保安过来驱赶她离开。
<br/>
<br/>        诺筱颖不得不从地上站起身来，踉跄着身子，犹如行尸走肉般离开这里。
<br/>
<br/>        她想哭，却哭不出来。
<br/>
<br/>        诺筱颖甚至觉得自己就像一只人人喊打的过街老鼠。
<br/>
<br/>        回到家中，一如既往，诺筱颖装作若无其事地收拾屋子打扫卫生。
<br/>
<br/>        只是，今天，她比以往打扫得更勤。
<br/>
<br/>        她内心的情绪非常糟糕，只能用这种方式来发泄自己的冤屈、无辜与无助。
<br/>
<br/>        诺筱颖做好了晚饭，愁眉不展地坐在餐桌前，等着哥哥嫂嫂下班回来。
<br/>
<br/>        但时间差不多的时候，开门进屋的人，却只有哥哥诺司峻一个人。
<br/>
<br/>        “嫂嫂呢？”诺筱颖下意识地问。
<br/>
<br/>        诺司峻嘟囔着回答：“她今晚要加班。”
<br/>
<br/>        随后，哥哥诺司峻换了双拖鞋走进来后，便深沉着脸对诺筱颖，接着说道：“筱颍，明天跟我回老家一趟。”
<br/>
<br/>        哥哥诺司峻说完这话后，着实把诺筱颖吓了一跳。
<br/>
<br/>        “发生什么事情了吗？”
<br/>
<br/>        “家里出大事了！”
<br/>
<br/>        “大事？什么大事？是不是爸爸他……”
<br/>
<br/>        “跟爸爸无关。”
<br/>
<br/>        “那跟谁有关？”
<br/>
<br/>        “你！”
<br/>
<br/>        “我？”
<br/>
<br/>        昨晚还在纳闷家里出的大事为什么会跟自己有关，翌日诺筱颖便跟着诺司峻踏上了回老家的旅途。
<br/>
<br/>        坐在火车上，诺筱颖仍旧在问哥哥诺司峻，家里到底发生了什么事情，是跟她有关系的。可诺司峻也只是微微耸肩，表示什么都不知道。
<br/>
<br/>        一切都只有等到家后，才能知道具体情况。
<br/>
<br/>        “爸妈，我和哥回来了。”
<br/>
<br/>        诺筱颖和哥哥诺司峻拖着行李箱回到家后，诺母只是冷不丁地说道：“回来了，还不赶快去做饭，待会你表妹梦萌也会来。”
<br/>
<br/>        “哦！”诺筱颖面无表情地应了声，这时，只见小姨妈和小姨父从内堂走了出来。
<br/>
<br/>        “筱颖和司峻回来了。”小姨父和小姨妈几乎是异口同声。
<br/>
<br/>        诺筱颖和诺司峻也纷纷向两位长辈问好。
<br/>
<br/>        今天家里是有什么喜事吗？很久没有来往的小姨父和小姨妈竟然也来家里了！
<br/>
<br/>        诺筱颖一边做饭一边就在想这件事情。
<br/>
<br/>        当她把饭菜做好，一碗碗端上桌的时候，表妹童梦萌背着包拖着行李箱从院外走了进来。
<br/>
<br/>        “爸妈、大姨妈、大姨父，我回来了！”
<br/>
<br/>        听到表妹童梦萌那清甜的呼唤声，第一个冲上去的永远都是诺母。
<br/>
<br/>        童梦萌只比她诺筱颖小几分钟，身为表妹，在两个家中是格外受宠。
<br/>
<br/>        诺筱颖看到诺母拉着童梦萌嘘寒问暖，莫名其妙地感到心凉。
<br/>
<br/>        其实，任谁看了，心都会凉的吧！
<br/>
<br/>        自己的老妈，对自己的表妹比对自己更好。
<br/>
<br/>        诺筱颖深吸了口气，脸上仍旧挂着微笑，毕竟家里，许久未见的小姨妈和小姨父也在。
<br/>
<br/>        好久都没有大团圆的聚在一起，这一次，倒是全部都齐了。
<br/>
<br/>        刚风尘仆仆地回到家中，还要做一桌子的饭菜，此刻一家人聚在桌前吃饭，最累的人应该是她诺筱颖，可诺筱颖却只能眼睁睁地看着老妈不停地往表妹童梦萌的碗中夹菜。
<br/>
<br/>        桌上那一桌子的菜，诺筱颖全是按诺母的要求做的，同时也是表妹童梦萌爱吃的。
<br/>
<br/>        诺筱颖一看到老妈对童梦萌格外好，心里便有一种说不出的心酸。
<br/>
<br/>        从小到大，老妈对这个表妹的喜爱程度有增无减。
