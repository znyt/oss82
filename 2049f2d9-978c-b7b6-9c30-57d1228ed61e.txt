        &lt;h3&gt;第三卷第三百二十章  “知心人”宋长生&lt;/h3&gt;
<br/>
<br/>        夏颜看着许弋澄和顾恩赐离开，对宋长生说：“长生，呼延临庆遇到麻烦了，你能不能帮帮他？”
<br/>
<br/>        “怎么回事？”宋长生漫不经心地问。
<br/>
<br/>        “洛城的两个项目迟迟完不了工，有人在查他们了，盯得很紧，那两个项目当时是我一手促成的。”夏颜此时已经完全把宋长生当了自己的知心人。
<br/>
<br/>        “你拿好处了？”宋长生抬头看了一眼夏颜脖子上的心型钻石项链。
<br/>
<br/>        “嗯，我将东西都送出去了，你知道我现在这样的地位其实是拿不到什么的。”夏颜一副小女人寻求帮助的样子。
<br/>
<br/>        “给老马了还是陆家了？”
<br/>
<br/>        “陆家啊，现在的老马还不是为陆家服务。”夏颜不以为意的说，她没有想到宋长生会知道这么多，完全都是下意识的回答。
<br/>
<br/>        “那就好办了，天塌下来有他们顶着呢，你别怕，到时候大不了你找找他们将情况说一下，重新换一个岗位。”宋长生看着夏颜。
<br/>
<br/>        “重新换岗位，我挺喜欢这个岗位的，况且现在和你们在一起也很好，我不想离开。”夏颜话里话外都带了一丝丝撒娇的味道。
<br/>
<br/>        “培养你自己的人，你走了还不是要听你的？真是个傻女人。”宋长生笑着骂夏颜。
<br/>
<br/>        “我就在想，怎么样将洛城的事情消灭在萌芽状态，不让他们发酵，可是呼延临庆那头猪，做事情真是让人很恼火，他用了陈子昂的方案，却拿不下来项目，还从陈子昂公司挖了几个人，可是项目似乎是进了死胡同，表面看起来都很不错，可是已接入运行设备的时候就不行了。”夏颜苦恼的说。
<br/>
<br/>        “说实话，我对软件不了解，我用的都是代理别人的，你们当时就应该找一家代理做做，干嘛要剽窃陈子昂他们的方案？还真以为自己能拿下来呢，呼延临庆和陈子昂也不是第一次对着干了，他们的第一个项目就是将陈子昂他们的方案改得乱七八糟的，后来还是呼延寰宇出头帮他解决了的。”宋长生嗤笑了一下。
<br/>
<br/>        夏颜对呼延临庆的第一个项目不是很熟悉，那个时候她还是名不见经传的小人物。
<br/>
<br/>        “你竟然知道的这么清楚？”夏颜后知后觉的看着宋长生。
<br/>
<br/>        “知己知彼百战百胜，呼延临庆自己有几斤几两不知道？非要自己搞研发，这几年研发了什么出路了？他自己做的项目哪一个不是蹩脚的，售后服务多得很。是不是经常让你给他收拾乱摊子？你呀，迟早会被他拖累的。”宋长生一副真心替夏颜着想的样子，满脸透着关切，那双桃花眼却滴滴乱转。
<br/>
<br/>        “我以后会注意的，但是目前还得想好怎么解决，总不能让他出问题啊。”
<br/>
<br/>        “用的陈子昂的方案，那就去找陈子昂他们帮忙了，出除了陈子昂他们别人怕是不行吧？呼延临庆和呼延寰宇是兄弟，就让呼延临庆去找他的哥哥了。”宋长生看了一眼夏颜。
<br/>
<br/>        “让呼延临庆去找陈子昂？”夏颜睁大了眼睛，她是多么的不想听到陈子昂的名字，可是今天宋长生已经说了好几次了。
<br/>
<br/>        “是让呼延临庆去找，又不是你去找，你怕什么？”宋长生看了一眼夏颜，他总是弄不明白夏颜为什么会那么排斥陈子昂。
<br/>
<br/>        “那行吧，让呼延临庆去找，我就不管了。”夏颜最近被洛城的项目折腾得有些担心，要是一切顺利还好说，可是偏偏呼延临庆是个猪队友，到现在项目都搞不定。
<br/>
<br/>        “你也别太担心了，如果有想法就换个岗位，你现在手头上有可靠的人选吗？”宋长生关切的问夏颜。
<br/>
<br/>        “现在有两个人选，一个是陈伟冯，一个是孙毅。俩个人都还不错，只是我有些担心。”
<br/>
<br/>        “担心什么？”
<br/>
<br/>        “陈伟冯是陈子昂的侄子，后期会不会向着陈子昂他们？孙毅的母亲曾经在陈子昂的公司，现在反出来在呼延临庆的公司上班。”
<br/>
<br/>        “你担心孙毅和他妈妈一样？”
<br/>
<br/>        “是啊！”
<br/>
<br/>        “陈伟冯你不能用，再怎么说血缘关系断不了，真有什么事情他肯定会向着陈家的，那陈子霖他们真的就在西城一家独大了。孙毅的妈妈反出来肯定是你给的诱惑够大，做母亲的肯定会为了自己的孩子着想，有情可原，据我所知，孙毅的人品不错。”宋长生为夏颜分析着。
<br/>
<br/>        “你说的我不是没有考虑过，所以我一直在想着以后推荐谁对我们最有利。”夏颜咬着手中的笔头。
<br/>
<br/>        “你自己先考虑考虑，观察一段时间，不急。”宋长生安慰着夏颜。
<br/>
<br/>        “唉，才发现这么多人里面还能和你说说话，不在其职不谋其位，其实我也很难的。”夏颜开始倒苦水了。
<br/>
<br/>        “是啊，当一个女强人是很辛苦的，上面要应付，下面要对付，真不容易。放心吧，以后有什么事情你尽管给我说。我做不了你的靠山，做个知己还是可以的。”宋长生桃花眼秋波泛滥，看得夏颜心里发慌。
<br/>
<br/>        “晚上一起吃饭？我想吃老城根。”夏颜轻轻的说。
<br/>
<br/>        “呀，今天不能陪你了，今天约了好几个另外部门的人，晚上要一起吃饭，你介意不介意和我一起去？”宋长生拒绝了夏颜的邀请。
<br/>
<br/>        “算了，你约的人我又不认识，就不去了。”
<br/>
<br/>        “其实也有你认识的人，只是........”宋长生犹豫了下。
<br/>
<br/>        “那我就更不能去了，冒冒失失的和你一起去算什么？”夏颜目前还不打算和宋长生走得那么近。
<br/>
<br/>        “那改天约个时间，我好好陪陪你。我现在回去将这份策划书落实了，等事情完结之后好好庆祝一下。”宋长生站起来看了夏颜一眼。
<br/>
<br/>        “行，那你走吧，改天见。”
<br/>
<br/>        “有事情给我打电话，我会随叫随到的。”宋长生挥挥手，飘飘然而去。
<br/>
<br/>        夏颜看着宋长生离开，叹了口气，拿起手机给老马打电话。
<br/>
<br/>        宋长生走出夏颜办公室的大楼，坐到自己的车里，掏出手机：“喂，兄弟，哥哥一定会将你推到重要的位置上去，让那些威胁二姨的人滚蛋吧！”
<br/>
<br/>        在北京忙忙碌碌的陈子昂并不知道西城发生的一切，她总是在不经意间被人提起，也不知道洛城已经开始了调查，事情已经按照她的部署在有序的进行，不但在进行，冥冥之中还有了外界的助力。
