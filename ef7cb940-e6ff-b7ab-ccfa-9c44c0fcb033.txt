　　铛！
<br/>　　宛若金属碰撞，苏重四十米黑红刀芒，骤然停在蚁王头顶。一只漆黑手掌，轻飘飘的抓住刀芒。
<br/>　　梅路艾姆神色不变，右手举过头顶。整条手臂黑的发亮，好似金属铸就！
<br/>　　“这招已对本王无用了！”
<br/>　　砰！
<br/>　　五指合拢，凌厉强悍的刀芒，瞬间崩碎！
<br/>　　“你的移动方式，本王学会了。你的攻击手段，毫无作用了。你可以去死。”不知何时，梅路艾姆竟出现在苏重身后！
<br/>　　轰！
<br/>　　黑色拳头骤然突破音障，狠狠砸在苏重背脊。他好似炮弹，整个人被打飞半空。身体向后倒成一个弓形，脊椎发出吱吱呀呀危险声音。
<br/>　　尚未落地，蚁王化作残影，再次出现在苏重身边。被漆黑亮色覆盖的双臂，再次砸向苏重背脊。只这一下，苏重整个身体近乎对折！
<br/>　　轰的一声，就像陨石落地。苏重被蚁王狠狠的砸进地底。一个数十米的巨坑，凭空印在大地上！
<br/>　　……
<br/>　　尼特罗盘膝坐在东拓国皇宫屋顶。身边不远处，尼飞比多和枭亚普夫两蚁，好整以暇的站着。对于尼特罗的虎视眈眈，没有丝毫紧张。
<br/>　　“虚张声势，还是确实有底牌？”尼特罗拿不定主意。
<br/>　　最强战斗力蚁王，已经被苏重引走。这里只剩下两个蚁王护卫。
<br/>　　通过刚才的瞬间交手，尼特罗清楚感觉到，两个护卫很强大。念能力庞大的让人心惊。
<br/>　　但他丝毫不惧。战斗，可不只看念气量。武道才是他的根本。念能力只不过是武道的一部分。更何况，他的念气量同样不差。
<br/>　　他有信心拿下两个蚁王护卫。但对方的态度让尼特罗拿不准，那种有恃无恐。是对蚁王充满信心，还是真的有什么底牌？
<br/>　　尼特罗念力缓缓散出，缓缓感知着整座皇宫。一股淡淡的危险感觉，袭上心头。
<br/>　　盯着皇宫深处，尼特罗脸色阴晴不定。果然有所依仗！
<br/>　　不能轻举妄动！尼特罗只希望，苏重能够顺利搞定蚁王。他不敢发动攻击。两个护卫没什么，但那个给他危险感的人，超出了预估。
<br/>　　一旦激烈交战，身后数万人类，必然遭殃。苏重和蚁王不在乎，他在乎！
<br/>　　嗡嗡嗡！
<br/>　　一阵扇叶旋转声由远及近的传来。尼特罗脸色一变，转头看去。数百大型飞艇，从远处天空飞来。
<br/>　　飞艇很快就落在皇宫门前，一群穿着黑色战斗服的人，从飞艇上下来。拍着队伍，训练有素的冲进皇宫。一看就是久经训练的精锐士兵。
<br/>　　黑衣人根本就不理会屋顶对峙双方。数百人分工明确，直接动手搬运兵蚁蚕茧。
<br/>　　噗噗噗……
<br/>　　突然间，十多个刚刚碰到蚕茧的黑衣人，头颅猛然炸开，其他人顿时停下动作。
<br/>　　砰！
<br/>　　一道黑影从半空站闪现，炮弹被砸飞。轰的一下，把厚厚的皇宫墙壁砸开一个大洞。砖石碎块落了一地。
<br/>　　哗啦一声，碎石堆破开，尼飞比多满脸满脸杀气的一跃而起。死死瞪着站在不远处的尼特罗。
<br/>　　刚才他瞬间出手，直接抓爆黑衣人头颅。如果不是尼特罗阻拦，他能眨眼间就把这些家伙全灭掉！
<br/>　　“尼特罗！嵌合蚁必须消灭干净，你到到底在干什么！”一个魁梧冷酷的中年人越众而出，厉声呵斥道。
<br/>　　尼特罗扫了一眼对方，眼睛始终盯着尼飞比多，和不知何时出现在一旁的枭亚普夫。
<br/>　　本以为这些家伙是来救人。没想到，竟是来运兵蚁。消灭嵌合蚁？谁知道对方打什么主意。
<br/>　　“继续！”中年人冷然命令道，根本不管旁边虎视眈眈的尼飞比多：“尼特罗，现在很多大人物都在关注这里。如果我的人出了问题，下一刻就会迎来炸弹洗地。所有人的命，都在你的手里。你看着办吧。”
<br/>　　说完转身开始指挥黑衣人搬运兵蚁。
<br/>　　尼特罗脸色难看，他当然知道所谓的炸弹是什么。那是吝啬的蔷薇，是只需要一颗。就能让皇宫附近所有生物，全部死绝的大杀伤性武器！
<br/>　　竟然用皇宫外数万人的性命，来胁迫自己帮对方搬运兵蚁！这是一个两难的选择。
<br/>　　噗！
<br/>　　一个刚刚碰到蚕茧的家伙，整个人都变成了血雾！
<br/>　　“尼特罗！”冷酷中年人猛然转身，咬牙切齿喝道：“你难道想让这些人，都陪着你去死！你最好……”
<br/>　　噗！
<br/>　　一声轻响，刚才还叫嚣不已的中年人，头颅突兀飞上半空。
<br/>　　尼特罗脸色大变：“你怎么会在这里！”
<br/>　　站在尼特罗面前的，正是蚁王梅路艾姆！
<br/>　　“不对，有些不对劲！”尼特罗仔细观察，眼前这个蚁王，是一个全身漆黑的家伙。和他之前见的蚁王完全不一样。
<br/>　　之前的蚁王给他的感觉，是残暴凶悍。而眼前这个蚁王，残暴有之，却又充斥着一股惊人的混乱。好像他那漆黑体内，藏着无数暴虐灵魂一般！
<br/>　　“尼特罗，不用为难，我替你做了选择！”
<br/>　　好似无数声音混杂在一起，重重叠叠的声音从梅路艾姆口中传出。只是声音，就让人心里躁动不安。
<br/>　　“你是谁！”尼特罗满脸凝重。这种感觉，怎么那么像比杨德的变异百式观音？！
<br/>　　“你可以称呼我为——鬼！”最后一个字，漆黑蚁王突兀抬高声调。
<br/>　　层层叠叠的诡异声音，如索命魔音，直灌众人大脑。
<br/>　　啊……
<br/>　　一连串惨叫声。数百黑衣人，顿时双眼翻白，轰然倒地。血液从七窍内，缓缓流出，眼看活不成啦。
<br/>　　只是简单一吼，竟瞬杀数百精锐士兵！
<br/>　　念兽！一种另类的分身念兽！瞬间，尼特罗明白了眼前蚁王的身份。
<br/>　　蚁王竟塑造了一尊念分身，看对方浑身漆黑如墨，周身缭绕淡淡黑烟，只是逸散出来的威势，就已经让尼特罗浑身发紧。
<br/>　　还有那犹如混乱灵魂的感觉，这到底是个什么鬼东西！
<br/>　　……
<br/>　　噗！
<br/>　　苏重猛然突出一口鲜血，他整个人近乎对折！倒伏大坑内衣衫破烂，狼狈不堪。
<br/>　　呼！一双墨绿色的脚，出现在苏重视线之内。
<br/>　　“数月以来，我潜入人类社会。吞噬能力者近千，各种各样的人类数十万。吸取精华，分离糟粕。实力完全进入另一个层面。能够承受我这么多拳，你足以自傲了。”蚁王淡然道：“不过，也就到此为止了。”
<br/>　　右脚上墨绿色甲壳皮肤，慢慢氤氲起黑色烟气。右小腿，眨眼间就变成了漆黑色。金属般的黑亮，充满坚硬冰冷之感。
<br/>　　去死吧！漆黑右脚，对着苏重的头颅狠狠踢去。
<br/>　　啪！
<br/>　　一声轻微闷响。梅路艾姆一怔。低头看去，一只手掌，牢牢抓住他的漆黑右腿。
<br/>　　咔嚓咔嚓，伴随着毛骨悚然的的骨骼碰撞声。对折起来的苏重，瞬间恢复正常。
<br/>　　“死？那还早得很呢！哈！”
<br/>　　苏重竟抡起梅路艾姆，好似拿着一个破布娃娃，左右前后疯狂轰砸！
<br/>　　轰轰轰……
<br/>　　大地一阵剧烈颤抖，竟如地震一般！
