<br/>				陈默并不知道凌经理运回去的蔬菜引发了小范围的轰动，有很多人都对港区餐厅逐渐的趋之若鹜。<br/>　　<br/>　　他还是按部就班的生活着。在他接收到席止涵最终的设计稿，都已经按照他的要求做了修改后，最终确定了方案。<br/>　　<br/>　　在确定了方案后，陈默也约了他们建筑部门，双方见面详谈。这里就需要席止涵的沟通了，所以为了让她动起来，直接就开车将上百瓶的爽肤水邮寄了过去。<br/>　　<br/>　　何老和吴老，两个人要回一趟市区，做身体检查，另外就是想将家里老伴也接过来，所以两个人在今天早上就走了。<br/>　　<br/>　　好机会，早就嫌这两个老头在自己这里住宿呢，还时不时的要说教一下自己，陈默还要老实认真的接受。<br/>　　<br/>　　嘿嘿！直接叫了陈明过来，不动自己的住房，只是将其他客房移动了一下，然后在自己房屋和客房之间隔离开来，重新开了院门。<br/>　　<br/>　　客房这里就成为一个单独的院落，拥有八间客房，多功能室，娱乐室，读书室，多功能厨房等房间，不够的都让于经理一一补齐。而且为了以后的扩展，都留下了扩展需求的位置。<br/>　　<br/>　　然后在晚上的时候，又移栽了不同的植物，添置了凉亭，花架，走廊、水塘等等的一些室外休闲区域，然后就是聚灵阵和感应阵的布设。要是没有这个阵法，别人怎么可能掏十万元居住呢？<br/>　　<br/>　　反正听他们说要把老伴接过来，所以这些房子干脆都移动一下，也好让他们住着自由一些。<br/>　　<br/>　　房子的移动非常简单，只要将一辆吊车就可以将这种集装箱房移动起来，非常方便。而且院落的布置都没有用水泥，都是用空心砖敷设，还有石头等等，弄得非常的舒服。<br/>　　<br/>　　陈默还趁着他们都走了，大海也去采购小牛和猪苗的机会，将乾坤珠内的一些鱼移动到了外边的鱼塘中，鲈鱼、鳗鱼、虹鳟鱼、黄鳝、鲢鱼、草鱼鲤鱼、桂鱼等都移出来一批。<br/>　　<br/>　　鱼的体型都是长的比较大的，不过也没有移出很多，因为在乾坤珠内生长的比较快速，到了外边就慢多了。先在乾坤珠内养着，等差不多了再放出来，就可以直接上市了。<br/>　　<br/>　　还给家里鱼塘和隔壁鱼塘内放了些锦鲤，这些锦鲤在乾坤珠内吃的好生活好，不仅体型长大不少，有近二十多厘米了，而且鱼身体上的颜色也是非常的周正，尤其是鸿运当头和大正三色，还有黄金色非常的纯正。在鱼塘中游来游去的，非常的漂亮。<br/>　　<br/>　　另外就是，李瑞回去后，就派人过来拉走了一车的蔬菜。不知道他那里是怎么安排的，反正听他说蔬菜非常的受欢迎。<br/>　　<br/>　　而且是因为在其他分店都不采用，有些人反对不能用。他就自作主张在自己负责的店铺里用了。没有想到的是，却获得了极大的成功。<br/>　　<br/>　　他和本家的那帮人是如何竞争的就不知道了，陈默也不想去问，但是现在有了这种品质的蔬菜后，那效果绝对是非常厉害的，就不知道那些人到时候会不会找到自己这里来。<br/>　　<br/>　　下午的时候，席止涵带着施工管理部门的人过来和他谈判。主要是设计规划方案已经确定后，就没有必要拖下去了。<br/>　　<br/>　　由于席止涵在中间协调，另外就是这个公司其实就是他们席家的自己公司，所以很多事情就非常的好解决。当场将陈默的一些要求记录下来后，就差不多能开工了。<br/>　　<br/>　　陈默其实也没有太多的要求，除了质量上要求会找个监理公司来监督质量以外，最主要的就是建设的先后。将道路和葫芦谷的改造放在首位，然后将西面整个的山坡地全部开发出来。<br/>　　<br/>　　他想先利用山坡地种植东西，建筑什么的其实后面盖好就成。现在蔬菜出货量一日大于一日，每个两天就会来拉辆车，所以两百亩的菜地根本就不够用。<br/>　　<br/>　　都谈的差不多以后，双方也就互签协议，确认了这个事情，最迟在后天所有施工队伍就要进场。陈默也需要支付第一批款项，大概在5000万元，用于修路和平整葫芦谷，修建山谷道路，平整山坡地，山谷中间做成水库等等，还有建设一些桥梁啊什么的。<br/>　　<br/>　　对于这些钱的花费，陈默早就有心里准备的，所以签完协议后直接支付了事。不过席止涵在离开的时候，又从他这里拿了100瓶爽肤水。<br/>　　<br/>　　现在西市有钱人的太太圈里，这种爽肤水已经流传开来，要不是她上次收到陈默邮寄的100瓶，恐怕连自己用的都没有了，今天她跟过来的主要目的也是那爽肤水，至于次要的目的的，当然就是签协议合作了。<br/>　　<br/>　　看看席止涵现在的皮肤就知道爽肤水的功效了，自从用了爽肤水以后，以前熬夜和吃酸辣等刺激性食物所引起的，黑头、粉刺、黑色素沉淀等等都变没有了，现在肌肤不仅红~润白~皙，而且还非常的水嫩光滑，简直能一把掐出~水来一样。<br/>　　<br/>　　爽肤水的效果，都不用介绍，只要指着自己的脸说是用过爽肤水的，这就是效果，那么起到的宣传作用也就杠杠的。<br/>　　<br/>　　所以席止涵从陈默这里拿了这么多瓶的爽肤水，上次的100瓶在手里没有待半天的时间，就以已经没有了。而且还有人不断的给她钱，让她购买爽肤水，甚至有的女人直接转给他5万元，说是卖一瓶。<br/>　　<br/>　　席止涵现在都有些感叹，自己还不如去做爽肤水的生意，来钱快不说还没有任何的责任，还不累。这次问陈默要了100瓶，也是为了应付增加的人，都是认识，要是不能满足的话怕说不过去。<br/>　　<br/>　　这也让陈默见识到了女人的疯狂，为了美，还真舍得花钱。<br/>　　<br/>　　辛妍哪里，其实今天也给他了电话。这几天就是在凑钱，想从他这里多进一些爽肤水，所以直接凑了一百八十万，准备进个100瓶，等资金回笼以后在继续。<br/>　　<br/>　　陈默其实可以让辛妍赊欠的，这些东西在他眼中虽然不值钱，制作也非常的简单，除了回春草的叶子外，其实也没有什么好东西，但是却被辛妍拒绝了。<br/>　　<br/>　　他知道这是辛妍的自尊心在起作用，不过他还是尊重她的选择。所以既然如此的话，还有什么好说的，那就这样呗!<br/>　　<br/>　　对于这个女同学，他还是非常认可的。以前都是低头学习了，丝毫没有注意到班级里的女同学。<br/>　　<br/>　　从这次回来后，接触了辛妍以后，才发现有些时候有些同学还是没有什么改变。现在社会上有一些一开同学会就成炫耀会，一见老同学就是攀比！其实有些人，还是不会变的。<br/>　　<br/>　　他和辛妍接触这么久，从内心中也没有感觉到什么别扭，感觉还是比较聊的来的。这也是他愿意和辛妍继续接触，聊天等等。<br/>　　<br/>　　不过和辛妍发展一下，或者继续下去再加深一下彼此之间的关系，对于陈默来说还没有想好，也许再过一段时间再说吧。感情问题，似乎总是有些复杂的说！<br/>　　<br/>　　：。：<br/><br/><br/>Ps:书友们，我是小学嗣业，推荐一款免费小说App，支持小说下载、听书、零广告、多种阅读模式。书友们快关注起来吧！<br/><br/><br/>
