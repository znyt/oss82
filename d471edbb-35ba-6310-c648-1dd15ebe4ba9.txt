　　周遭遍地尸骸。
<br/>　　猩红的血液，流淌一地；剧烈的打斗痕迹，布满了整个万剑碑林。
<br/>　　尸体上，面容大多透漏着恐慌、绝望。
<br/>　　萧逸不禁暗暗腹诽，自己之前屠杀时，当真如此面容可怖吗？
<br/>　　萧逸摇了摇头，没有再多想。
<br/>　　眼睛蓦地一闭，又猛地一睁。
<br/>　　睁开之时，那双原本清澈凌厉的眼眸，已然变了。
<br/>　　太阴太阳之眼凝视下，一个个无形漩涡凭空而现。
<br/>　　漩涡，快速吸收着周遭尸体的武魂力量。
<br/>　　这数千剑修的武魂，萧逸本就没打算放过。
<br/>　　原本在他的打算中，是先于云渊长老二人分别后，再独自回来吸收。
<br/>　　武魂，乃是天地规则所化。
<br/>　　越是修为高深的武者，其武魂即便死后也越难消散。
<br/>　　所以萧逸并不担心。
<br/>　　但后来既然出现了变故，又感知到背后有莫名的强者在凝视他，他便只能先远遁百万里，再绕一圈回来万剑碑林这里。
<br/>　　一分钟后。
<br/>　　地面一具具尸体的武魂力量，已被吸收殆尽。
<br/>　　不得不说，能来剑域的剑修，都绝非泛泛之辈。
<br/>　　数千剑修的武魂，其中几乎清一色都在蓝色巅峰阶品以上。
<br/>　　紫色武魂，占了大半。
<br/>　　还有不少人，拥有着深紫色阶品。
<br/>　　另外，还有几个黑色武魂。
<br/>　　萧逸眼眸一闭，散去了太阴太阳之眼，眼眸恢复了正常。
<br/>　　内视一番，体内小世界处，控火兽呆萌地蹦蹦跳跳，显然有些雀跃。
<br/>　　“要突破吗？”萧逸双眼一眯，有些期待。
<br/>　　不过，十数秒后。
<br/>　　控火兽停下了跃动，身上光芒，散发着浓郁的深紫色。
<br/>　　但也很显然，距离突破，仍旧远远不够。
<br/>　　“按理说，应该足够力量了。”萧逸皱了皱眉。
<br/>　　数千高品阶的武魂，其中不乏深紫色与黑色，理应足够一个紫色武魂突破了。
<br/>　　如今控火兽并未突破，可能只有一个，那边是突破的方式和层次。
<br/>　　一般武者，武魂第二次觉醒，越是出色的天骄或妖孽，所需的武道圣丹便层次越高。
<br/>　　若是武道圣丹的层次低了，失败率就会极大。
<br/>　　拿当初在血光洞府见的那八个少府主来说，他们的武魂，原本应该只是普通紫色。
<br/>　　他们这个层次的天骄，虽出色，但还说不上惊天彻底。
<br/>　　故他们觉醒所需用的武道圣丹，大概绝世强者层次的圣丹即可。
<br/>　　也就是他们府内的太上长老或者府主之类的强者，即可帮他们突破。
<br/>　　若是普通圣皇境巅峰为他们突破，想必他们的失败几率会极高。
<br/>　　又如莫悠这个层次。
<br/>　　很显然，莫悠的天资，绝对远强于十八府的天骄。
<br/>　　这个十数年前便名震中域，年纪轻轻就力压五大学宫天骄的妖孽，绝对算得上是当世最强妖孽之一。
<br/>　　他的武魂要第二层觉醒，若是绝世强者层次的武道圣丹替他突破，失败率绝对极大。
<br/>　　故洛前辈那等层次的传奇人物，亲自替他突破觉醒。
<br/>　　除了成功率之外，武道圣丹的层次强弱，也决定着觉醒的武魂强弱与否、完美与否。
<br/>　　越是强的武道圣丹，自然能帮觉醒者觉醒出更完美、更强的武魂。
<br/>　　如果萧逸没有猜错的话，此刻的控火兽，必然是在太阴太阳之眼的帮助下，以最完美的方式准备着觉醒。
<br/>　　否则，绝不可能那么庞大的武魂力量都无法让它突破。
<br/>　　不过，以完美形式突破的控火兽，将会变成什么样子？
<br/>　　萧逸心头不禁暗暗期待。
<br/>　　他有着太阴太阳之眼这一逆天手段，根本无需外人替他觉醒。
<br/>　　只要吸收足够的武魂力量即可。
<br/>　　这也是萧逸不回总殿‘避难’的原因，他更宁愿在外闯荡历练。
<br/>　　“希望能在进入古帝墓之前突破吧。”萧逸自语一声。
<br/>　　嘭…
<br/>　　萧逸手中一股火焰凝聚而出，试了试如今控火兽的威力。
<br/>　　火焰一出，惊人的温度，还有那惊人的绝对控制力，让得萧逸脸色一变。
<br/>　　“好强。”萧逸身躯一震。
<br/>　　萧逸再次内视了一番。
<br/>　　小世界处，总共有两个武魂，一是控火兽，二则是冰鸾剑。
<br/>　　冰鸾剑武魂的品阶，只是紫色。
<br/>　　但却紫得异常浓郁，紫得异常深色。
<br/>　　这抹紫，几乎比任何紫色武魂都要浓郁得多。
<br/>　　而且，冰鸾剑本身还处于残损状态。
<br/>　　萧逸对比了一下二者。
<br/>　　控火兽的深紫色，仍旧比不上冰鸾剑的紫色光芒。
<br/>　　但这已经比萧逸见过的其他各类紫色武魂，要深色许多了。
<br/>　　恐怕，紫色武魂阶品中，现今能比控火兽更强的，也就只有冰鸾剑。
<br/>　　单纯论紫色阶品武魂，控火兽想必已经可以碾压其他各种紫色武魂了。
<br/>　　虽未到黑色，也比不上别的黑色武魂。
<br/>　　但控火兽，作为炎龙大陆最弱的妖兽，没有任何别的‘特点’，只有一种纯粹的控火专精。
<br/>　　这控火专精，在现今深紫色的阶品下，足够给予萧逸异常恐怖的控火能力了。
<br/>　　“如今控火能力大增，说不定可以…”萧逸似是想到了什么，脸色陡然一喜。
<br/>　　控火兽的再次提升，无疑给了萧逸远胜上一次以‘易霄’身份出现时的更强的控火能力。
<br/>　　嗖…
<br/>　　萧逸化作紫炎火翼，火翼一振，振翅飞离。
<br/>　　数千剑修的乾坤戒和乾坤袋，早在之前与云渊长老二人离开万剑碑林时，便已经被他悉数拿了。
<br/>　　数个时辰后，萧逸直接横跨了剑域，亦出了剑域。
<br/>　　控火能力大增下，他化出紫炎火翼的飞行速度亦大增。
<br/>　　短短数个时辰，便能飞跃一个庞大地域。
<br/>　　剑域之外，某个地域内。
<br/>　　萧逸随便在一座妖兽森林上降落，随后寻了座小山，开辟出一个山洞，走了进去。
<br/>　　接下来，也该修炼了。
<br/>　　万剑碑林，已然是是非之地；剑域之内，想必随着那数千剑修的陨落，也会有不少动静。
<br/>　　故萧逸并未选择在剑域修炼。
<br/>　　而是特地出了剑域，才在别的地域开始修炼。
<br/>　　嗖…嗖…嗖…
<br/>　　萧逸手中光芒一闪，一份份修炼资源凭空而现。
<br/>　　其中，不乏世间万年难遇的天材地宝；不乏一些实力滔天的凶险之内丹；更不乏一些蕴含精纯力量的重宝。
<br/>　　当然，最令人瞩目的，无异于那一条条灵脉。
