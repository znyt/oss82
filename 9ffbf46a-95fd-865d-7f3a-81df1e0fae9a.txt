        &lt;h3&gt;第三卷第二百四十五章  我想去西城&lt;/h3&gt;
<br/>
<br/>        我们生活在这个世界上，其实有很多的事情不是我们能主宰的，生命如春花秋月，亦如暴风骤雨，幸和不幸都是在一念之间，谁也不知道将来面临的是什么。
<br/>
<br/>        我们能做的，大概就是把握好我们现在的时间，用心的、用力的去生活去爱，爱自己爱他人。
<br/>
<br/>        当黎明来临的时候，我们伸手微笑着迎接那一缕晨曦，当黑夜降临的时候我们也要敞开心怀拥抱黑夜所给予我们的孤独和黑暗。
<br/>
<br/>        陈子昂在忙忙碌碌的生活中不知疲倦的奔波着，她就是一批脱缰的马，努力的奔跑着，只是因为她内心有光、有爱！
<br/>
<br/>        在她上学时的梦想一直根深蒂固的隐藏在她的心灵深处，她现在所做的每一件事情其实都没有脱离她那个时候的方向，她赚到的钱都被她投入到各种研发中，虽然她只是一个私企。
<br/>
<br/>        陈子昂在和赵绾绾及呼延寰宇聊天时说的那句话：“其实，我觉得我要是一个程序猿肯定作用比现在还大，只是命运弄人。”反映了她的骨子里其实很想有一番作为，也许她认为从另一个层面上发展比她现在还要好。
<br/>
<br/>        人生之路不会按照既定的路线走下去，谁也不知道下一秒会发生什么，作为陈子昂来说，她现在要做的就是防患于未然。
<br/>
<br/>        表面上的一切看似平静，但是都是暗潮汹涌。
<br/>
<br/>        陈子昂将安男带回了北京，她并没有让安男一个人回他们的家里也没有将安男送回杭州，寒伯安几天都没有给安男信息，证明寒伯安也有可能遇到了麻烦，谁都知道落家和寒家的关系极好。
<br/>
<br/>        寒冰玉是寒家的孩子却拥有落家的股份，在落家很多年。能动寒冰玉和落雪，寒伯安怕也不会例外，只是祈祷寒伯安能够安全。
<br/>
<br/>        当然，她的这些分析也只是她的分析，她是不会告诉安男的，告诉安男也是让安男徒增担忧罢了。
<br/>
<br/>        “安男，你就不要回杭州了，我带你去北京陪着陈虞和落妍呆几天，这两个淘气包天天闹着想见安男阿姨呢。”
<br/>
<br/>        “好啊，我也想见他们呢，能和他们在一起是件快乐的事情。”安男何等聪明，她知道陈子昂能赶到上海将她连夜带走，事情怕不那么简单。
<br/>
<br/>        她很担心寒伯安！
<br/>
<br/>        只是陈子昂什么都不说，她也什么都不问。
<br/>
<br/>        陈子昂的家里又多了一个安男。
<br/>
<br/>        陈子昂接到父亲生病的电话了，在陈子昂的记忆中，父亲一直都是铜墙铁壁，就没有过生病的迹象，这一次生病了，让她的心里有些发慌，从离家到现在，她只顾着孩子，只顾着创业，很少去关注父母亲，突然就觉得时光匆匆，让她很怀念从前，怀念儿时被爸爸背着，被妈妈抱着的感觉。
<br/>
<br/>        陈子昂拿着电话发呆，眼泪就不知不觉的流下来了。
<br/>
<br/>        不知如何，她最近总是控制不住自己的泪水，微微有一点触动，眼泪就会不由自主的掉落。
<br/>
<br/>        她得回去陪陪父亲，她对安男说：“我父亲生病了，我得回去看看，你这几天就在我家呆着，帮我看着这两个孩子，我可能得几天才能回来。”
<br/>
<br/>        陈子昂怕安男回家，在寒伯安没有回来之前，她得保证安男是安全的，安男也给她说了在寒家的情况，她认为那个叫苓希的人有问题，有可能是处于嫉妒，女人的嫉妒心也是很可怕的。
<br/>
<br/>        “我能不能和你一起去看看？”安男听说陈子昂的父亲生病了心里一动，她可是知道手镯的事情，她很想见一见陈子昂的父亲。
<br/>
<br/>        “你想和我一起？”
<br/>
<br/>        “是的，我有一次回家给我妈妈说过你们家乡，我妈妈很希望过去，但是一直没有时间，我能不能打电话将我爸爸和妈妈叫上一起过去看看？这一段时间刚好他们休假。”安男觉得她提这样的要求有点过分，但是她真的很想带着妈妈一起去看看，她不想给母亲留下遗憾。
<br/>
<br/>        “这个时候过去，那边的天气不怎么好，我的父亲是在西城看病。”陈子悦他们已经将陈爸爸接到了西城了。
<br/>
<br/>        “西城啊，十三朝古都，子昂，你带着我们一起过去吧，你陪叔叔，我陪我爸爸妈妈转一转。”
<br/>
<br/>        陈子昂见安男这样说，也就不再坚持：“那我们是不是要去接阿姨和叔叔？”
<br/>
<br/>        “不用，不用，我打电话给他们，让他们直接到西城，我们在西城集合。”安男心里一喜，无论情况怎么样，她都得让母亲见见陈子昂的父亲。
<br/>
<br/>        “那就这样吧，你看看机票，然后看看阿姨他们到西城的时间，我让我哥去机场接他们，如果我们的飞机到达时间差不多，那就等着我们一起走。”
<br/>
<br/>        “好，我来看机票，你准备什么时候走？”
<br/>
<br/>        “那就明天吧，我下午再去公司处理一下事情，你给我们看机票。”陈子昂不知道安男怎么安排她的父母亲的时间，只能将买机票的事情交给安男来做。
<br/>
<br/>        “行，子昂，你去忙着，我来看机票。”
<br/>
<br/>        看着陈子昂出了家门。
<br/>
<br/>        杭州，安家。
<br/>
<br/>        安男给她母亲打电话；“妈，我带你去西城吧？”
<br/>
<br/>        “你带我去西城干嘛？”代凉和安林曦正在书房讨论问题，俩个人就一个问题争论不休。
<br/>
<br/>        “去见你哥哥，但是我不确定是不是。”安男是真的不确定。
<br/>
<br/>        “你说什么？”代凉有些激动，之前让寒伯安寻找手镯的事情，后来一直都没有结果。
<br/>
<br/>        “怎么了？”安林曦问代凉。
<br/>
<br/>        “安男说她带我去见我哥哥。”
<br/>
<br/>        “你把电话给我，我来问。”安林曦接过电话。
<br/>
<br/>        “男男，怎么回事？”
<br/>
<br/>        “爸爸，陈子昂他们家曾经有一个和妈妈一模一样的手镯，不过后来手镯送人了，现在不在陈家的手中，陈爸爸生病了，我想带着妈妈和你去见一见他们，别留下遗憾。”
<br/>
<br/>        “见一见也是好了，你准备什么时候去？”
<br/>
<br/>        “明天吧，我来买机票，你们从杭州走，我从北京走，我们到西城汇合。”
<br/>
<br/>        “你怎么在北京？不是应该在上海吗？”
<br/>
<br/>        “我挺喜欢陈子昂的，就跑到她家里来了，你都不知道，陈子昂俩个孩子有多么可爱。”安男一直都没有告诉她的父母亲，她在寒家的一切。
<br/>
<br/>        “陈子昂都有孩子了？”安林曦曾经拿陈子昂做为教材给他的学生上过课。
<br/>
<br/>        “啊，爸爸，这个是一个秘密，你不能告诉任何人。”
<br/>
<br/>        “知道了，那我们明天见，你定好机票给我们说。”
<br/>
<br/>        安男定好了机票，心里有些忐忑有些激动，现在的技术很发达，有没有手镯无所谓，让他们先见面再说。
