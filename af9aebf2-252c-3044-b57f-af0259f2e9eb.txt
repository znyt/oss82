　　叶少阳目光又落在门的两边，有两条长幅的红纸从上面垂下来，上面写着两行烫金大字：
<br/>　　净土莲花一花一佛一世界，
<br/>　　牟尼珠献三摩三藐三菩提。
<br/>　　像是一副工整的对联，但上面光影流动，似乎蕴含着雄浑的灵力。边上还有一些道家或佛家的符文，贴在门的四周。
<br/>　　“这么多道符文，这里头到底是什么物资啊。”小马吐槽了一句，回头看了一眼，让后面那些跟着来的生灵都站住了。
<br/>　　叶少阳念了一道护身咒，眉心处释放出一道紫光，将周身裹住，抵抗着仓库门上和外墙上那些法器和灵符的威力，走到门下，把所有灵符都揭了，八卦镜也取下来，拿在手中的感觉，至少是八段光的法器，翻到后面，看到上面几个篆字：一掌山河。
<br/>　　不知道这是谁的法器，被当成了定门钟来用，叶少阳擦掉正面镜面上的朱砂符文，八卦镜的灵力立刻收敛起来。叶少阳丢给吴嘉伟，他法器少，拿着这个可以防身，自己有功能相近的阴阳镜，倒是用不上这个。
<br/>　　仓库门上有两道铁链，接口处挂着巨大的明锁，叶少阳把锁拉起来看钥匙孔，发现锁上面印着奥迪四环。
<br/>　　“这不是当代的产品吗！”叶少阳怔住了，虽然不知道这是啥牌子，但四环标志的门锁，自己小时候就见过的。
<br/>　　这门锁必然是圣灵会的人从人间带过来的，这也从侧面证明了这不是一般的仓库，不然没必要这么讲究。
<br/>　　“管他什么锁，放着我来！”
<br/>　　小马撸着袖子走上来，让叶少阳帮他拉直铁链，按在门上，操起板砖就砸起来，他这板砖是玄铁的，坚硬无比，用了不一会工夫，两把铁锁都砸开了。然后一脚把门踹开，脸庞顿时就被光线照亮了。
<br/>　　小马还得意洋洋地跟叶少阳夸耀自己的搬砖，突然发现他眼睛直了，仔细往里面瞅了一眼，立刻就懵逼了，嘴巴缓缓长大，冒出了一句感慨：“你爸爸的！”
<br/>　　这简直就是金库！
<br/>　　仓库里面，四面都是货架，上面摆满了法器，不是一件件，而是一堆堆的。
<br/>　　仓库没有窗户，也只有这一个门，里面没灯，但是上品法器发出的光，将这个偌大的仓库照得透亮——大部分法器是不会发光的，但是七段光以上的法器，都会发出灵光，这种灵光普通人类看不见，邪物和法师却能一眼看出，并且通过光华的强弱明灭，看出法器的价值，几段光几段光的叫法，就是这么来的。
<br/>　　当然，法师在行走江湖的时候，一般都是把法器的光华收敛起来，仓库里的这些都没了主人，自然就是原始状态。
<br/>　　这仓库里的法器，大部分都是不发光的，但因为数量实在太多，总有几件七段光以上的。这么多的法器……大伙都是头一次见到。
<br/>　　“我来挑几个玩！”小马说着要走进去，被叶少阳一把拉住，“先别进去，你不想想，这么多法器就在这堆着等人来拿？法术公会就这么大方？”
<br/>　　小马怔了一下说道：“这门上不是有铁锁和灵符什么的。”
<br/>　　“那算什么，以前这边有白虎堂的人驻守，再加上灵符和八卦镜就够了，现在人都走了，这些都是死东西，顶什么用，再说他们为什么不带走。”
<br/>　　“不是说，带走不了这么多……”
<br/>　　“那真正值钱的也该带走吧，这里八段光的都好几件，不可能这么浪费。”
<br/>　　“呃，那咋办？”
<br/>　　“看看再说。”
<br/>　　叶少阳站在门口，念咒感召七星龙泉剑，惊喜地发现，七星龙泉剑就这里头不远的地方！
<br/>　　叶少阳按捺着狂喜，摸出了几道灵符，朝里面打过去，灵符四散飞开，火苗一直都是青色的，说明没有邪气，也没有法阵什么的存在。
<br/>　　这个世界奇怪的很，法师在这里，就像鼻子失灵了一样，只要对方不动手，就算面对面一个人，都感觉不到对方身上一点邪气。
<br/>　　叶少阳身上法器都不在，只能靠灵符来探路。
<br/>　　“你们等着啊，我先去瞅瞅。”叶少阳说完，自己小心翼翼地走进去，一步，两步……一直走到仓库正中间，并没有什么问题。
<br/>　　“七星归位！”
<br/>　　叶少阳念咒，召唤七星龙泉剑，结果感应到了龙泉剑的位置，但是……宝剑好像被什么东西给困住了，于是朝着感应到的位置走过去。
<br/>　　在正对着大门那边的货架下面。
<br/>　　叶少阳趴在地上看，最下面紧贴着地面那一层，堆着成堆的法器，叶少阳扒拉了几下，突然摸到了自己的背包，急忙抓过来，打开一看，阴阳镜什么的都在里头，心头大喜，继续扒拉，又找到了腰带，上面插着的法器法药之类的也都还在，急忙绕在腰上。
<br/>　　这熟悉的感觉……
<br/>　　叶少阳拍了拍腰带，心中说不出有多惬意。现在就差一把七星龙泉剑了。
<br/>　　叶少阳再度念咒，只听见法器中间一阵响动，七星龙泉剑怕是被埋在了下面，位置太深，不把这些法器都清理掉，一时很难拿出来。
<br/>　　“喂喂，小叶子，没问题吧，我们可进来了！”
<br/>　　不能叶少阳回答，小马已经闯了进来。
<br/>　　吴嘉伟也跟进来，感知到了自己的藏锋宝剑，从一堆法器中将它找出来，连剑鞘都还在，锵的一声拔剑出鞘，剑锋微微抖动，发出轻吟，仿佛也在庆祝与主人重逢。
<br/>　　小马招呼大伙一起进来寻找法器，顿时几百号人一下子都往里涌，快把仓库都挤爆了，后面的人根本进不来。
<br/>　　小马从货架上拿了一杆不知道谁的大砍刀，跳到货架上，用力砍在墙上，打出了一个洞，然后跟个拆迁队长似的额，指挥大家一起动手，很快就将整个仓库都拆了，人群一窝蜂涌上来，寻找自己的法器。
<br/>　　场面热闹而又混乱。
<br/>　　“大家不要乱，卧槽，先找自己的，找不到的一边站着，等大家都拿完了再说！”小马站在货架上面维持秩序。
<br/>(本章完)
