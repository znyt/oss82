　　把崔钰的事情放到一边，李承乾决定回头找人去长子县把这家伙给调到长安县来。
<br/>　　他自己能到大唐，成为大唐太子李承乾，从某些方面来讲，应该好好‘感谢’一下老崔。所以把他调到长安县，放到自己眼皮子底下，好好‘报答’一下，也是题中应有之意。
<br/>　　想到这里，李承乾再次莫名的嘿嘿冷笑出声，看在夜魅与白蝙蝠两人眼中，浑身直起鸡皮疙瘩，对礼一眼，纷纷猜测到底是谁又得罪了这位殿下，被他惦记上了。
<br/>　　连续两天，李淳风一直晕迷未醒，李二急的不行，却也没有什么好办法，只能一遍遍不停追问，但却每一次都得到相同的答复。
<br/>　　直到第三天中午，有些烦躁的老李得到了通知，李淳风醒了。
<br/>　　“什么？贪狼当空，破军相映是什么意思？说清楚一些！”看着正抱着寒瓜（西瓜）啃的儿子，李二蹙眉说道。
<br/>　　“就是说，今年不宜出行呗，贪狼星主侵略，破军星主杀戮，李淳风正是算到这些因为一时激动，急火攻心才晕过去的。”丢掉手中的瓜皮，再次从老头子的盘子里拿起一块。
<br/>　　“你少吃点，别吃坏了肚子。”李二瞅了边上一堆的瓜皮，有些不满儿子贪吃的样子，感觉这小子现在就像贪吃星转世。
<br/>　　“嘿嘿，最后一茬了，再不吃就要等到明年。”李承乾嘻皮笑脸的说着，又往嘴里塞了一口寒瓜，咕哝道：“父皇，要不这一次就别去了，看样子风险不小，否则小李也不至于吓晕过去。”
<br/>　　“这些事儿你怎么知道的？”李二对去九成宫的事情不可置否，只是追问李承乾是如何知道这事儿的。
<br/>　　“儿臣这不是和秦英老道关系好么，昨天去他那里，听他说的。”李承乾不以为意的说着，尽量表现的正常一些。
<br/>　　李二眼看着寒瓜越来越少，再不吃就没了，忍不住也拿起一块，赌气似的咬了一口，乜着儿子说道：“朕看你是不想留在京里监国吧？还想着偷懒？”
<br/>　　“这怎么可能，那可是欺君之罪，谁敢干啊！”李承乾一边叫着撞天屈，一边努力往嘴里塞着寒瓜。
<br/>　　大唐时的西瓜可没有现在那么大，人头大小的基本可以算是最大的了，所以真要吃的话，一人一个还不怎么够。
<br/>　　现在只有半个寒瓜，更是少之又少，哪有不拼命抢的道理。
<br/>　　“你欺君的事儿还少干了？”李二狠狠瞪了儿子一眼。
<br/>　　这混小子，拼了命的抢老子的半个寒瓜，难道这就不算是欺君了？答应老子的两米大镜子到现在也没送来，就不算欺君了？再往前数数，一桩桩一件件欺君的事儿这混小子干了多少，现在竟然还有脸跳出来说什么欺君之罪。
<br/>　　越想越生气，最后一拍桌子：“来人，去把李淳风给朕叫来，朕要当面问问他。”
<br/>　　“喏！”方老太监给李承乾传了一个爱莫能助的眼神，退出去安排人去太史局传召李淳风。
<br/>　　关于李承乾的说法，老太监和李二的态度一起，也是认为是李承乾想要偷懒，不想留在京里监国，所以才极力阻止。
<br/>　　那么一旦李淳风被叫来，很可能他的说辞立刻就会被拆穿。
<br/>　　可是李承乾会怕么？已经和道门有了良好沟通，彼此早就对好了说辞，验证啥的完全不成问题。
<br/>　　果然，李淳风被招来之后，唠唠叨叨的一番长篇大论，其结论和李承乾说的一样，不宜出行。
<br/>　　天罡、地煞、二十八星宿，牛逼整整吹了大概有半个时辰，李淳风最后总结道：“陛下，此行凶险异常，还请陛下三思。”
<br/>　　李二被李淳风整的有些懵，道家典籍他有所涉猎，但总是不多的，长篇大论之下，开始还可以，说的多了他也有些反应不过来，沉默了好长时间才皱眉问道：“如你所言的话，是有兵灾？”
<br/>　　“陛下，这个就不好说了，天相显示只不宜出行，但具体事宜无法测得。”李淳风摇头说道。
<br/>　　他也不傻，虽然李承乾嘱咐过，如果被问的没办法就说兵灾起自北边，但这种没有依据的话不到万不得以他是不会说的。
<br/>　　“你下去吧！”李二摆摆手，让李淳风退出去，随后若有所思的看着李承乾。
<br/>　　“父皇，您看我干嘛啊，我啥也没干。”李承乾苦着脸说道。
<br/>　　“朕有说什么么？”李二不动声色的问道。
<br/>　　李承乾小声咕哝着：“您是什么都没说，可是您这比说什么都厉害。”
<br/>　　李二嗯了一声，也不知道他想表达什么意思，隔了一会儿才说道：“行了，你回去吧，这件事朕会考虑的。”
<br/>　　李承乾从老头子那里离开之后，意识到一个很严重的问题：他考虑到一切条件，但是却忘记了考虑老头子的性格。
<br/>　　以李二那种天老大，他老二的情格来说，如果不知道有危险还则罢了，知道了估计更要迎难而上。
<br/>　　贪狼主侵略，破军主杀伐，作为天策上将的李二会怕这些东西？不会，这是李承乾替李二作出的回答。
<br/>　　老头子如果知道有人会起兵反叛的话，估计连想都不会想就会亲自以身作饵，吸引叛军来攻，到时候来个一网打尽。
<br/>　　可是这和李承乾开始的计划是完全相反的，他是想要阻止老头子去九成宫，没想到现在似乎弄巧成拙，成了老头子非去不可的结局。
<br/>　　李二去不去九成宫，李承乾实际上并不怎么在乎，反正老头子身边能人无数，就算有什么事情，估计也伤不到他。
<br/>　　但是长孙皇后呢？老头子去九成宫，长孙皇后总不能不去吧？
<br/>　　这就像是一个死循环，想要阻止长孙皇后去九成宫，那就必须要阻止李二，想要阻止李二就必须让他知道事情的严重性。
<br/>　　可当李二知道事情的严重性之后，结局却是非去不可，因为老李想要证明自己不畏惧任何的挑战。
<br/>　　怎么办？现在事情又再一次回到原点，阻止不成，反而更加坚定了老头子去九成宫的信念，这让李承乾恨不能扇自己一个耳光。
<br/>　　异想天开果真是要不得！
