陆寒霆听到了她软弱的声音，抬头看她，现在女孩三千青丝散落在他雪白的枕头上，她巴掌大的绝色小脸退去了所有的血色，显得十分的苍白。
<br/>
<br/>        她出了一脸的冷汗，汗水将清纯的长发凌乱的缠在了她的小脸上，她整个人有一种被蹂躏过后的惊心动魄的美。
<br/>
<br/>        陆寒霆低头，吻她的红唇。
<br/>
<br/>        夏夕绾没有避开，乖巧而青涩的回应了他两下，然后小心翼翼的试探，“陆寒霆，我不会走的，我会很听话，你能不能将我先松开，我的手很疼…”
<br/>
<br/>        在女孩的安抚诱哄下，陆寒霆身上的戾气减少了几分，他伸手，将黑色皮带给解了下来。
<br/>
<br/>        夏夕绾活动了一下小手，然后将小手放在了枕头下。
<br/>
<br/>        那里有她的针。
<br/>
<br/>        但是陆寒霆速度更快，他将她的小手一把摁住了，嗓音犀利沙哑，“想干什么，恩？”
<br/>
<br/>        他相当警觉，丁点的风吹草动都能惊动他，夏夕绾打开小手，将自己纤白的手指穿进了他修长的五指里，跟他十指相扣，无辜道，“我没有干什么…”
<br/>
<br/>        “你这张小嘴很会骗人，小骗子！”陆寒霆粗粝的拇指按上了她的菱唇…
<br/>
<br/>        ……
<br/>
<br/>        陆寒霆翻身下去，已经睡着了，夏夕绾将刚刚刺进他脑袋里的一根银针给抽了出来，然后坐起身。
<br/>
<br/>        睡衣纽扣已经坏了几颗，她白腻的肌肤上全是触目惊心的伤口，夏夕绾起身下床，进了沐浴间。
<br/>
<br/>        她站在盥洗台前看了一下自己，自己脸色苍白的像一张纸，身体的疼痛加失血过多让她两眼都有些发黑。
<br/>
<br/>        她伸手捂了一下脖间被他咬过的地方，他咬的是血管，留下了深深的齿印。
<br/>
<br/>        如果谁看到她这一副模样，绝对会报警的。
<br/>
<br/>        夏夕绾拿着牙刷开始刷牙，刷了十几遍，牙龈都出血了，她才停止了动作。
<br/>
<br/>        出了沐浴间，夏夕绾重新上了床，在陆寒霆身边躺了下来，她不敢离开，怕他夜里还会出状况。
<br/>
<br/>        他的病情比她想象的还要更严重，最糟糕的是他嗅觉灵敏，想给他施针都已经很难了，刚才若不是他到了…那样的关头，她根本就不能得手。
<br/>
<br/>        夏夕绾脑袋里乱乱的，她躺着不敢动，怕吵醒他，惊动奶奶，而且身上的伤口一动就有撕裂般的疼痛感传来。
<br/>
<br/>        这时身边的陆寒霆动了一下，夏夕绾迅速屏住了呼吸，只见他将健臂伸了过来，习惯性的将她抱入了自己的怀里，又睡去了。
<br/>
<br/>        夏夕绾苍白的小脸贴着他心房的位置，退去了刚才的阴郁狠戾，他又恢复了往日里矜贵的模样。
<br/>
<br/>        听着他心脏那里传来的“咚咚”有力搏动，疲倦的夏夕绾也进入了梦乡。
<br/>
<br/>        ……
<br/>
<br/>        夏夕绾在凌晨五点钟的时候睁开了眼，身边的陆寒霆还在睡，她起身下床，用一件大衣将自己裹得严严实实的，然后离开了幽兰苑。
<br/>
<br/>        待会儿幽兰苑的佣人们就会醒，她必须在之前离开，不然这一身的伤是瞒不住的。
<br/>
<br/>        她没有去叶翎那里，因为不敢让翎翎看到，她和翎翎的友情与日月同长，但是陆寒霆的身体状况她不敢告诉她。
<br/>
<br/>        所以夏夕绾去了枢密研究院，来到了药房。
