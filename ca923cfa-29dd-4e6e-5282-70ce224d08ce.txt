　　嗖！嗖！嗖！！
<br/>　　一百八十余名天才，纷纷飞出了血煞山，身形落在了广场上。
<br/>　　有的天才和苏莫一样已经将血阳花收了起来，比如巫人王、巫玛、巫幽兰。
<br/>　　但大多数的人，都是将血阳花捧在自己的掌心中。
<br/>　　其中血阳花大小不足一尺的人，足有三分之二。
<br/>　　这些人个个垂头丧气，显然知道自己要被淘汰了。
<br/>　　那些血阳花超过一尺的人，大多都是一尺出头的样子，只有寥寥十数人能达到两尺以上。
<br/>　　比如巫穹的血阳花，达到了惊人的七尺，非常的壮观。
<br/>　　也有几人的血阳花，达到了五、六尺的程度。
<br/>　　苏莫扫视了一眼，发现巫天御的血阳花很小，只有一尺出头的样子，不过勉强算是过关了。
<br/>　　“巫穹的血阳花有七尺，好厉害！”
<br/>　　“巫墨羽的也有六尺大小！”
<br/>　　“不知道巫人王和巫玛、巫幽兰的血阳花达到了什么地步？”
<br/>　　广场上嘈杂一片，议论声四起。
<br/>　　很多人都被巫穹的血阳花震惊了，其他很多天才的血阳花，连一尺都没有达到，而巫穹的血阳花却达到七尺大小，当真是惊人。
<br/>　　不过，众人最好奇的还是巫人王、巫玛和巫幽兰的血阳花大小，三人是天赋最高的人，想必血阳花绝对超过了七尺。
<br/>　　也有一些人好奇苏莫的血阳花大小，毕竟苏莫在第一关之时，表现很是抢眼。
<br/>　　“所有人，取出你们的血阳花！血阳花不足一尺之人，自行离场！”大祭司目光扫视众人，朗声喝道。
<br/>　　而后，一些血阳花不足一尺之人，纷纷黯然离开，退到了远处。
<br/>　　一些没有取出血阳花的人，则是纷纷取出自己的血阳花，大多都是一尺以上，三尺以下的大小。
<br/>　　苏莫并未率先取出自己的血阳花，而是准备先看看巫人王、巫玛和巫幽兰三人的血阳花有多大。
<br/>　　只见巫玛和巫幽兰手掌一翻，同时取出了自己的血阳花，霎时之间血光爆射，映照一方。
<br/>　　两人的血阳花，均是达到了一丈，其中巫玛的血阳花，足有一丈一的样子，而巫幽兰的血阳花，也堪堪的达到了一丈。
<br/>　　两人的血阳花，仿佛两轮血色的耀日，血光耀眼无比，吸引了所有人的目光。
<br/>　　“一丈一、一丈，果然厉害啊！”
<br/>　　“不愧是巫玛和巫幽兰，真称得上是我族的绝世天才，将其他人彻底的甩开了！”
<br/>　　“这等大小的血阳花，估计只有巫人王能压他们一头了！”
<br/>　　广场上炸开了锅，众人惊讶不已，参照巫穹等人的血阳花大小，众人本来猜测，巫玛和巫幽兰的血阳花大小，应该在八尺到九尺样子。
<br/>　　而结果却超出了他们的预料，两人的血阳花均是达到了一丈以上。
<br/>　　巫玛转头看向身边的巫人王，见对方还未取出血阳花，眸中露出冷然的笑意，道：“巫人王，你的血阳花呢？不会没超过一丈吧？”
<br/>　　巫人王面色淡淡，瞥了一眼巫玛和巫幽兰手上的血阳花，眸中露出不屑的笑意，道：“没想到你们这么废，真是无趣！”
<br/>　　巫玛和巫幽兰闻言，顿时脸色一沉，对方的言语实在是刺耳。
<br/>　　众人闻言惊讶，巫人王这么自信，难道血阳花大小远超巫玛和巫幽兰不成？
<br/>　　“巫人王，亮出结果吧！孰强孰弱一目了然！”巫幽兰开口了，声音空灵，如同空谷幽兰。
<br/>　　“哼！就让你们看看差距！”巫人王冷笑一声，随即手掌一番，血色的骄阳陡然出现，血光映照八方。
<br/>　　嘶！
<br/>　　霎时之间，倒吸冷气的声音此起彼伏，所有人都凝视这着巫人王的手上。
<br/>　　血色的骄阳，直径足有一丈九，接近两丈的样子。
<br/>　　众人心中震惊，巫人王的血阳花，居然达到了一丈九，将巫玛和巫幽兰彻底的甩开了。
<br/>　　就连看台上的众多核心长老，以及大祭司和四祭司，都很是惊讶，他们非常清楚催动血阳花的难度。
<br/>　　一个时辰的时间，巫人王能将血阳花催生到一丈九，这种吸纳血煞之气的速度，非常的恐怖。
<br/>　　“不错！”大祭司赞赏般的点了点头，面上露出了一丝笑意。
<br/>　　“怎么样？服气了吧？”
<br/>　　巫人王面带戏虐的对巫玛和巫幽兰说道，言罢，他眼眸扫视四方，傲然喝道：“你们这些所谓的天才，根本没有和我竞争的资格；第一名，舍我其谁！”
<br/>　　舍我其谁！
<br/>　　巫人王言语充满了霸气，这是无敌的自信，纵观全族，武帝境的族人之中，谁的天赋能与他争锋？
<br/>　　不少人都被巫人王无敌的自信感染了，心中充满了崇敬。
<br/>　　绝世的天才，谁人不敬畏？
<br/>　　一时间，巫人王成为了全场的焦点，无比的瞩目。
<br/>　　巫玛和巫幽兰的脸色难看了下来，他们本以为这一关能撼动巫人王，没想到差距居然如此之大。
<br/>　　“巫尘，你的血阳花呢？为何还不取出来？”
<br/>　　就在此时，一道喝声突然响起，立刻打破了现场的气氛，是巫天御的声音。
<br/>　　此刻，巫天御正紧紧的盯着苏莫，他很好奇苏莫血阳花的大小，但所有人都取出了血阳花，唯独苏莫迟迟不取出血阳花，这让他有些焦急了起来，所以出言催促。
<br/>　　下一刻，所有人的目光都落在了苏莫的身上。
<br/>　　众人也是有些好奇，巫尘第一关表现的如此抢眼，这第二关应该也不错吧！
<br/>　　虽然不可能超过巫玛和巫幽兰，但血阳花达到五、六尺大小，还是很有可能的。
<br/>　　巫人王、巫玛和巫幽兰三人，亦是将目光投到了苏莫的身上。
<br/>　　不过，三人都没有太过在意，这个突然崛起的巫尘或许不错，但和他们相比，还不是一个层次的人物。
<br/>　　见众人望来，苏莫不禁摸了摸鼻子，心中有些无奈，他的血阳花有些搞大了。
<br/>　　他没想到巫人王这么废，血阳花才一丈九，若是早知如此的话，他就不会让自己的血阳花长这么大了。
<br/>　　他这血阳花若是取出来，怕是有点太夸张了！
<br/>　　但此刻事已至此，他后悔已经晚了！
<br/>　　叹了口气，苏莫手掌一番，便将自己的血阳花取了出来。
<br/>　　霎时之间，一轮巨大的血色太阳陡然出现，映红了整个圣巫广场。
<br/>　　静！
<br/>　　整个圣巫广场，突兀的静了下来，死一般的寂静！
