　　“好了。”
<br/>　　萧逸笑笑，“开始参悟吧。”
<br/>　　“第一层尽头见。”
<br/>　　“好。”丁秋月点了点头。
<br/>　　“嗯？不对。”萧逸忽然反应过来，面露疑惑之色。
<br/>　　“我们是参悟普通剑碑进来的。”
<br/>　　“那子风师弟他们呢？”
<br/>　　如果都是进入这剑帝碑空间的话，那子风师弟他们会出现在什么剑碑面前？
<br/>　　他们参悟的是剑帝碑。
<br/>　　剑帝碑本就在外面。
<br/>　　丁秋月回答道，“子风师兄和桌万宗师兄，他们会直接进入第二层空间的。”
<br/>　　“他们参悟的是剑帝碑上的剑道知识，不必进入第一层。”
<br/>　　原来如此。
<br/>　　他们其他参悟者，怕是就算真的过了第一层，得了这些剑碑主人的一生剑道知识；亦远不如林子风和桌万宗二人在剑帝碑上得到的些许剑道知识。
<br/>　　毕竟，那是剑帝这位大陆剑道第一人的剑道知识。
<br/>　　其他剑碑的主人，与这位剑道一帝相比，怕是天差地别。
<br/>　　“我相信你不比他们差。”萧逸看着丁秋月，笑笑。
<br/>　　“开始吧。”
<br/>　　“嗯。”丁秋月认真地点了点头。
<br/>　　二人走向剑碑之前，开始参悟。
<br/>　　轰…
<br/>　　几乎是二人闭目参悟的瞬间。
<br/>　　两条剑气纵横之路，轰然而起。
<br/>　　从剑碑开始，一路延伸至这片空间的尽头。
<br/>　　同一时间。
<br/>　　远方，一声声爆响声，惊天剑气蔓延。
<br/>　　很显然，在这片空间之内的其余剑修，也开始了参悟之路。
<br/>　　萧逸闭目，脚步，却踏上了剑碑之后的这条剑气之路。
<br/>　　轰…
<br/>　　进入剑气之路后，萧逸只觉脑海一阵轰鸣，随后一白。
<br/>　　待他恢复了清明后。
<br/>　　面前，一个握剑的年轻人，忽然一剑袭来。
<br/>　　萧逸先是一惊，随后眼眸一冷。
<br/>　　他并不认识面前这个年轻人。
<br/>　　但这个年轻人，萧逸一眼便看出，只是个后天境的小子。
<br/>　　后天境，对萧逸而言，是很久前的事了。
<br/>　　这等层次，萧逸怕是一口气就能将他轰飞。
<br/>　　不过，刚要出手，却猛然发现，自己体内一身元力，一身修为，悉数尽消。
<br/>　　自己体内的元力，微弱得可怜，顶多只到后天境层次。
<br/>　　萧逸脸色大变。
<br/>　　面前年轻人，冷笑一声，“怎么，知道怕了？你这等庶出之子，哪来的资格与本公子争家主之位？”
<br/>　　年轻人的剑，已然袭来。
<br/>　　萧逸双眼一眯，即便他只有后天境修为，也不是面前这小子能比的。
<br/>　　刚要双手成爪，夺过年轻人之剑，并将他击败。
<br/>　　却陡然发现，自己根本用不了形意五绝。
<br/>　　“该死。”萧逸低骂一声，一个侧身，堪堪躲过这一剑。
<br/>　　这时，脑海中缓缓传来一阵记忆。
<br/>　　“玄阶功法，剑气决。”
<br/>　　“玄阶剑技，裂石斩。”
<br/>　　萧逸不禁老脸一抽，也只得快速参悟脑海中的剑技功法。
<br/>　　这时，年轻人再次仗剑而来。
<br/>　　“滚。”萧逸低喝一声。
<br/>　　几乎是数个呼吸间，萧逸便一剑将面前年轻人击败。
<br/>　　下一秒，身影一闪，忽然消失在原地。
<br/>　　又是身影一闪。
<br/>　　周遭景象，已然变了。
<br/>　　周遭，是一深山野林。
<br/>　　不远处，数个老者掌风猎猎，全力攻来。
<br/>　　“三个先天境？”
<br/>　　萧逸下意识地未放在眼中，但想起之前的事，陡然内视了一番。
<br/>　　内视之后，萧逸再次脸庞一抽。
<br/>　　“后天境九重修为？”
<br/>　　不错，他现在的修为，只有后天境九重。
<br/>　　后天战先天？
<br/>　　这对于普通人来说，几乎不可能。
<br/>　　但对于天骄来说，绝非难事。
<br/>　　不过萧逸下一秒又发现，自己能用的，除了那门玄阶剑技外；他的其他武技，根本使用不得。
<br/>　　此时，三个老者全力的一掌，已然轰来。
<br/>　　三掌，悉数轰向萧逸要害。
<br/>　　显然，这三个老者，要取‘他’性命。
<br/>　　“滚。”萧逸低喝一声。
<br/>　　他腰间，一直挂着一柄剑。
<br/>　　服饰，也并非他自己原本的服饰。
<br/>　　他忽然渐渐意识到自己现今境地。
<br/>　　锵…
<br/>　　萧逸一剑劈出，却被三个老者轻松轰退。
<br/>　　还未反应过来，又是一掌，直接将他轰飞数十米。
<br/>　　萧逸‘狼狈’地在地面翻了几圈，这才稳下身影。
<br/>　　咔咔…
<br/>　　萧逸站起身，握了握拳头。
<br/>　　手中之剑，紧了紧。
<br/>　　“也罢，好久没这般战斗过了。”萧逸嘴角咧过一道冷笑。
<br/>　　这抹冷笑，代表着，他要动真格了。
<br/>　　这此，未等三个老者攻来，萧逸已仗剑而出。
<br/>　　双方，战了起来。
<br/>　　但，双方的战斗，并没有什么华丽的元力爆发，也没有什么惊人的武技。
<br/>　　三个先天境加一个后天境九重，战斗激烈程度极其有限。
<br/>　　足足一个时辰后。
<br/>　　萧逸满头大汗，几乎力竭。
<br/>　　而面前，三个老者已然消失，取而代之的，是三具尸体。
<br/>　　“呼。”萧逸深呼吸一口气。
<br/>　　“醒来。”萧逸心头低喝一声，心神一凛。
<br/>　　“呼。”萧逸从参悟状态醒来，再次深呼吸一口气。
<br/>　　感知了一番，自己的一身修为实力，并未消失。
<br/>　　同时，自己已经在这条剑气纵横之路上，走了几步。
<br/>　　“所谓的感悟一生，原来是如此。”萧逸明白了过来。
<br/>　　毫无疑问，刚才萧逸的战斗，尽皆是这位斩星前辈曾经经历过的战斗。
<br/>　　当然，那是这位前辈年轻时的战斗。
<br/>　　而自己所能用的，便也只有这位前辈掌握的功法和武技。
<br/>　　萧逸刚要闭目，继续参悟下去。
<br/>　　忽然，乾坤戒内一阵躁动。
<br/>　　萧逸手中光芒一闪，冷焱剑凭空而现。
<br/>　　“小子，你确定真要继续走下去？”冷焱剑内剑灵，忽然问道。
<br/>　　“怎么？”萧逸皱了皱眉。
<br/>　　剑灵传音道，“你知道这位斩星前辈，活了多少年吗？”
<br/>　　“他的一生所经历之战斗，又有多少？”
<br/>　　“你走这条剑气之路，将代表着，你会代入他的身份中，将他一生走过的路，悉数走一遍。”
<br/>　　“然后呢？”萧逸皱眉问道。
<br/>　　剑灵桀桀一笑，“小子，你才活了多少年？不过二十来年吧。”
<br/>　　“而这位斩星剑道，又活了多少年了？与他相比，你活过的岁月，不过沧海一粟。”
<br/>　　“你怕我迷失其中？”萧逸耸耸肩，“只要剑心坚固，这算不得问题。”
<br/>　　“再者，即便迷失了，半月之后，便会被轰出这片空间。”
<br/>　　“对，你会被安然无恙地轰出去，恢复萧逸的身份。”剑灵冷笑道。
<br/>　　“但，你脑海中的记忆，可不会消失。”
<br/>　　“对比你自己活过的区区20多年岁月，这位斩星前辈的无数岁月经历，你觉得谁的记忆更多？”
<br/>　　“那时，你还是‘你’吗？”
<br/>　　剑灵的话，瞬间让萧逸打了个冷颤。
