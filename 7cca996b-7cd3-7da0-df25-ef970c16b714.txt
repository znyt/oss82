　　瞬间惨死数十人，众巫族天才立刻惊醒，纷纷急速暴退，各自与自己的对手拉开了距离。
<br/>　　一时间，混战暂时停止了下来。
<br/>　　一众巫族天才，实在是被苏莫的十种武魂震惊了，暂时没有在出手。
<br/>　　苍穹神宫的众强者见此，也纷纷的缓了口气，他们的目光都望向了苏莫。
<br/>　　不知苏莫能不能打败那名巫族最强的天才，若是苏莫胜了，那一切就好办了。
<br/>　　但若是败了，他们怕是只能逃命了。
<br/>　　动用了十种武魂之后，苏莫整个人仿佛化为了一柄人形宝剑，锋芒绝世，剑威无敌。
<br/>　　脚步抬起，苏莫虚空踏步，一步万丈，大步向巫天御走了过去。
<br/>　　嗤~~~
<br/>　　苏莫一路所过之处，空间被他的身躯直接划破，犹如利箭洞穿长空。
<br/>　　咻咻咻！！
<br/>　　二十八柄本命灵剑急速返回，环绕苏莫身周，随时准备出击。
<br/>　　巫天御眼眸微眯了下来，虽然刚才他也被震惊的无疑附加，但他可没有丝毫惧意。
<br/>　　“小小的苍穹世界，能出现一个你这种妖孽，的确是奇迹！”巫天御淡漠的说道，虽然他不知道苏莫怎么会有这么多武魂，但苏莫逆天的实力，已经让他彻底重视了起来。
<br/>　　此刻，他不禁心中暗叹，辛亏他也参加了这次试炼，不然的话族中上千名试炼子弟，还真有可能栽在此地！
<br/>　　虽能想的到，一向孱弱无比，没有什么强者的苍穹世界，会出现此等妖孽。
<br/>　　“土著，我可以给你一个机会，带你离开苍穹世界！”巫天御朗声说道。
<br/>　　苏莫闻言一怔，随即停下了身形，面带惊疑的问道：“你会这么好心？”
<br/>　　苏莫不太相信对方的话，不过，还是开口询问，想看看对方到底要如何！
<br/>　　“你天赋不凡，就算在外界，也是绝顶天才，自然可以破例！”巫天御笑道，他的确是起了爱才之心，如此天赋的土著，若是直接杀了，他就太可惜了。
<br/>　　“是吗？”苏莫眼眸微眯，心中却并没有高兴，天下哪有免费的午餐。
<br/>　　果然，便见巫天御微微一笑，道：“只要你认我为主，成为我的奴仆，永生永世效忠于我，我便可保你安然离开苍穹世界！”
<br/>　　巫天御依旧高高在上，即便苏莫现在展现的实力，已经无比逆天，他依旧是以上位者的姿态，在藐视着苏莫。
<br/>　　苍穹神宫众人闻言，均是眸中光芒爆闪，听此人之言，不似作假。
<br/>　　不过，他们都了解苏莫的为人，怎么可能甘愿做别人的奴仆！
<br/>　　“呵呵！”苏莫笑了起来，眸中闪过凌厉的杀机，收他为奴？永世效忠？当真可笑！
<br/>　　“星宿剑爆！”突然间，苏莫一挥手，二十八柄本命灵剑，瞬间形成大星宿剑阵，向巫天御绞杀而去。
<br/>　　轰隆隆！！
<br/>　　大星宿剑阵化为了一股浩大的剑气风暴，笼罩一方空间，威势滔天，凌厉无比。
<br/>　　对于收为奴仆一事，苏莫根本就不会有半分考虑，就算是生死道消，他也不会沦落为奴。
<br/>　　“自寻死路！”巫天御见此，心中杀机大起，他不惜放下身段，收苏莫为奴，对方居然还不同意！
<br/>　　在他看来，他收苏莫为奴，已经是莫大的恩赐了。
<br/>　　“让你见识见识什么才是真正的实力！”眼见剑气风暴绞杀而来，巫天御的背后虚引升腾而起。
<br/>　　巫天御同样动用了自己的武魂，其武魂是一道飓风，高大数十丈，急速旋转，将他包裹在内。
<br/>　　而最惊人的是，这飓风武魂之上，居然闪烁着九道金色的光环。
<br/>　　天级九阶武魂！
<br/>　　下一刻，巫天御催动了他的飓风武魂。
<br/>　　刹那之间，飓风武魂飞速旋转，急速暴涨，立刻形成了通天彻地的飓风，冲上了万里高空。
<br/>　　这飓风极为恐怖，连天接地，如同一根通天之柱，骇人无比。
<br/>　　下方死亡之海的海水，瞬间便被飓风卷起，逆冲九天，巨大的飓风变成了一根通天的水柱。
<br/>　　飓风卷动，无边的狂风肆虐九天十地，横扫方圆数万里。
<br/>　　数千里外的众人，遭到狂风的扫荡，都控制不住身形，直接向远处抛飞而去。
<br/>　　轰隆隆！！
<br/>　　星宿剑暴击在了飓风之上，顿时发出一连串的爆响，但丝毫无法撼动飓风。
<br/>　　二十八柄本命灵剑，挡不住飓风的力量，直接被卷飞了出去。
<br/>　　“这是……！”
<br/>　　苏莫面带惊色，他一头黑发飞舞，全身衣衫烈烈作响，猛烈的狂风如同惊涛骇浪，不断的拍击在他的身上，让得他无法伫立虚空，身形止不住的向后飘退。
<br/>　　苏莫心中震撼不已，这飓风武魂居然如此恐怖？
<br/>　　“土著，死吧！”立身于飓风中的巫天御，厉喝一声，手臂猛然一挥。
<br/>　　随即，通天彻地的浩大飓风，立刻开始分解，分解出一股高达万丈，席卷方圆千丈的小型飓风。
<br/>　　小型飓风如同一只巨兽，撕碎一切，急速向苏莫横扫而来，速度快到极致。
<br/>　　飓风扫荡形成的狂风，如同一柄柄天刀，锋锐无比，扑天盖地。
<br/>　　“给我碎——星辰陨！”
<br/>　　苏莫怒喝一声，一剑狠狠的劈出，三色剑气爆射长空，直接向飓风斩去。
<br/>　　恐怖的剑气如同星辰陨落，横贯天地，撕开猛烈的狂风，直接斩在了飓风之上。
<br/>　　轰！
<br/>　　一声爆响，剑气立刻崩碎，化为漫天剑雨，纷纷洒洒。
<br/>　　而小型的飓风，也被一剑劈开大半，风力急速溃散，威势大减。
<br/>　　“碎！”苏莫再次击出一剑，才将这道飓风彻底斩灭。
<br/>　　“呵呵！不错，但你又能挡住几招呢！”
<br/>　　通天彻地的飓风中，传出巫天御的轻笑声，随即，一道又一道比先前那道小型飓风还要强大的飓风，不断的从通天飓风中分解而出。
<br/>　　霎时之间，便有十余道小型飓风分解而出。
<br/>　　十余道小型飓风，呈扇形向苏莫碾压而去，强大无比的风力，简直扫荡一切。
<br/>　　苏莫脸色阴沉如水，他想打败此人，估计有些难了！
<br/>　　就算他底牌尽出，动用大三宝术，也没有十足的把握。
<br/>　　毕竟，谁知道对方还有没有什么强大的绝招！
<br/>　　不到迫不得已，苏莫不愿意动用大三宝术，因为一旦动用了大三宝术，若是还拿不下对方。
<br/>　　待大三宝术的时效一过，他虚弱不已，必死无疑！
