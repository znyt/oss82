　　宫殿中，苏莫盘膝坐在蒲团上，他的手上出现了一枚储物戒指。
<br/>　　这便是那名黄泉魔宗元老鲁桂的储物戒，苏莫意念探入其中，仔细查探其内的物品。
<br/>　　不愧是武皇境九重的武者，此人非常的富有，其储物戒中，极品灵石十多亿，上品灵石一百多亿，皇级的灵草灵药也有数十株。
<br/>　　除此之外，很多杂七杂八的物品，也是非常的多。
<br/>　　不过，此人的身家虽然丰厚，但对于如今的苏莫而言，也只能算是还看的上眼。
<br/>　　其储物戒内的之源，苏莫只需一个多时辰，就能将其耗尽。
<br/>　　“这是何物？”苏莫手掌一翻，将储物戒内的一物取了出来。
<br/>　　这是一杆小旗，旗杆只有五寸长，旗帜长宽均不足三寸，漆黑如墨其上传出阵阵鬼气。
<br/>　　苏莫意念赋予旗帜之上，顿时发现其内传出阵阵鬼哭狼嚎之声。
<br/>　　“居然有这么多冤魂！”苏莫心中一惊，因为他发现这小旗之内，足足有数十万冤魂。
<br/>　　这小旗内怨气冲天，鬼气森森，非同寻常。
<br/>　　少倾之后，苏莫便将这小旗收了起来，准备有时间问问金一等傀儡人，这小旗是什么邪物。
<br/>　　随即，苏莫收敛心神，静静的打坐了起来。
<br/>　　他消耗了大量的精气神，身体疲惫不堪，虚弱不已，必须静养一段时日了。
<br/>　　接下来一段时间，苏莫便闭关不出，一直在宫殿中静修。
<br/>　　足足过了一个多月的时间，他才结束了静修，走出了宫殿。
<br/>　　此刻苍穹神宫中已经大变了模样，处处张灯结彩，一片喜庆的景象。
<br/>　　因为苏莫大婚之日将近，苍穹神宫中早已开始了布置，为大婚做准备。
<br/>　　苏莫离开宫殿之中，便径直前往水月宫，想去看看夕儿的情况。
<br/>　　不过，在水月宫苏莫并未见到夕儿，因为夕儿接受水月大帝的传承，正在闭关。
<br/>　　水月大殿中，苏莫和水一相对而坐。
<br/>　　“水一前辈，夕儿还要多久能出关，不会耽搁婚期吧？”苏莫向水一问道。
<br/>　　“宫主放心，夕儿姑娘五日内便能出关，不会耽搁婚期。”水一说道。
<br/>　　苏莫闻言点头，距婚期还有十余日，完全能来的急。
<br/>　　略一沉吟，苏莫将那杆黑色的小旗取了出来，向水一问道：“水一前辈，你可知这是什么宝物？”
<br/>　　“这是……！”水一见到苏莫手中的小旗，神情一怔，随即立刻来到苏莫身前，仔细的打量了起来。
<br/>　　“宫主，你从何处得到此物？”少倾之后，水一面色凝重的说道。
<br/>　　“从黄泉魔宗之人的手中所得。”苏莫道，他看到水一凝重的样子，也是疑惑了起来，难道这小旗还是什么惊天宝物不成。
<br/>　　不过，这种邪物，就算是至宝，也是邪宝。
<br/>　　“如果我所料不错的话，此旗名为尊魂幡，乃是巫族的特殊宝器！”
<br/>　　水一满面凝重的说道，不知为何，苏莫在水一的眼眸中，看到了一丝哀伤。
<br/>　　“巫族的特殊宝器？”苏莫闻言一惊，巫族就是试炼者的种族，那这尊魂幡就是试炼者一族的宝器了！
<br/>　　这小旗，居然如此不凡！
<br/>　　这时，水一又开口了：“不过，这杆尊魂幡却不是巫族的那种尊魂幡，而是仿制的半成品，威力与真正的尊魂幡相比，天差地别。”
<br/>　　苏莫点了点头，若是此小旗真的是巫族的尊魂幡，那还得了？
<br/>　　“这种尊魂幡有何强大之处？”苏莫又问道。
<br/>　　“尊魂幡，据说是用亿万生灵的神魂炼制而成，无比的强大！”
<br/>　　说到此处，水一略显模糊的脸庞上，露出了哀伤之色，道：“当初远古大劫之时，宫主便是陨落在尊魂幡之下！”
<br/>　　“哦？水月大帝吗？”苏莫闻言惊讶，这尊魂幡果然强大。
<br/>　　“不，是五行帝尊！”水一摇了摇头道。
<br/>　　“什么？”苏莫闻听此言，顿时倒吸了一口凉气，五行帝尊居然是死在尊魂幡之下！
<br/>　　“宫主，你有所不知，五行帝尊之所以被称为是帝尊，其实是超越了武帝的存在，达到了武尊的境界，但依旧陨落在尊魂幡之下，便可知尊魂幡的恐怖！”水一叹了一口气，幽幽的说道。
<br/>　　“武尊！”苏莫闻言沉默了，原来武帝之上是武尊！
<br/>　　不过，苏莫亦是心中暗叹，五行帝尊达到了武尊之境，堪称苍穹世界第一强者，依旧难逃陨落的命运。
<br/>　　这巫族，到底是什么种族，居然这般强大，苍穹世界之外，又是怎样的世界？
<br/>　　不由的，苏莫对于外界产生了好奇。
<br/>　　苍穹世界既然是巫族的试炼地，那就并非久待之地，他必须要出去。
<br/>　　除非他打败整个巫族，才能让苍穹世界脱离巫族的掌控，但这显然不太可能。
<br/>　　所以，苏莫必须要离开苍穹世界，并且将亲人朋友全部带出去。
<br/>　　这是个很重的任务，如同千斤重担一般沉重！
<br/>　　“水一前辈，你可了解苍穹世界之外的世界？”苏莫好奇的问道。
<br/>　　“了解一些！”
<br/>　　水一点了点头，道：“在远古大劫之前，苍穹世界和外界相通，苍穹世界的武者，只要是有武皇境界以上，便能自由出入！”
<br/>　　“那外面是什么样的世界？巫族又是什么样的种族？”苏莫目光灼灼，沉声问道。
<br/>　　“无尽宇宙，万界星空！”
<br/>　　水一目光望向殿外，凝神道：“无尽宇宙之中，星辰无数，生存着亿万生灵，百万种族，而巫族正是其中之一，而且是宇宙万族榜中有名的种族！”
<br/>　　耳中听着水一的话语，苏莫脑中不由的想到了一个浩瀚的宇宙世界，亿万种族共存的世界。
<br/>　　“宫主，我苍穹神宫，只有冲出苍穹世界，才有生存的希望！”水一突然望着苏莫，声音低沉而凝重。
<br/>　　“嗯！”苏莫重重的点了点头。
<br/>　　“苍穹世界的入口已经被封死，唯有巫族试炼者降临，才会开启，所以，千年才有一次机会，一旦错过这次机会，就要永远留在苍穹世界，哪怕躲过了试炼者的杀机，也终要埋骨于此！”水一告诫道。
<br/>　　听了水一的话，苏莫沉默不语，心中沉重无比。
<br/>　　水一之言非常在理，苍穹世界毕竟是小地方，资源有限，武道落后，留在苍穹世界，就算不被试炼者杀死，最终也会消散在岁月的长河中。
<br/>　　只有冲出苍穹世界，才能继续追寻武道之路，追寻那武道的巅峰。
<br/>　　“水一前辈，我不会让你失望的！”片刻之后，苏莫面色郑重的向水一道。
<br/>　　“嗯，你还有近十年的时间准备！”水一道。
<br/>　　苏莫点头，随即，他又和水一聊了片刻，向对方详细的了解了一番巫族的情况，最后才心思沉重的离开了水月宫。
