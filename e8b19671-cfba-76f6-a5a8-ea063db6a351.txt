　　表面变得如此狰狞恐怖，慕清霄自然不知道，他只能感受到一阵阵锥心的刺痛，好在他的定力强，经历过无数次改造，虽然痛苦，却依旧能够坚持下来。
<br/>　　慕清霄知道，能量冲刷肉身，对他来说是机缘，只要忍耐过去，修为越高好处就越能体现出来，而且除了忍耐，好像也没有任何办法。
<br/>　　忍耐痛苦的同时，他也只能注视着体内的动静，尽量提高炼化能量的速度，以此来减短痛苦。
<br/>　　精纯的能量顺着心法路线运转，一丝丝的化作慕清霄的力量，而他的气息也在逐渐的攀升，半个小时下来，便已经抵达元婴六重巅峰。
<br/>　　此刻，他体内的能量貌似根本没有炼化多少，不知道过了多久，刺痛感减轻，但能量却依旧庞大，慕清霄都有些麻木了。
<br/>　　就在这时，心脏中忽然爆发出一股能量，慕清霄心间颤了颤，九霄琉璃焱怎会在此刻出来，而且还是自行涌出，不受控制。
<br/>　　没有一丝的温度，但刺痛感远比刚才要浓郁，在骨髓中蔓延，最后扩散到身体中每一个角落。
<br/>　　牙齿紧咬，慕清霄就静静的盘坐在空地上，没有发出一丝的声音，但身体却忍不住细微的颤抖起来，皮肤上也浮现一阵异样的红润。
<br/>　　额头上细汗顺着脸庞滑落而下，早就将衣襟打湿一片。
<br/>　　神药中所蕴含的能量之强，即使慕清霄早预料到不会弱，可如今真正的亲身体验，却依旧觉得骇然。
<br/>　　真不愧是超级强者所练出的神药，若真无厘头一样的吞服下去，恐怕他早就尸骨无存。
<br/>　　伴随着一阵刺痛，不知何时，九霄琉璃焱已经将慕清霄的身躯全部包裹起来，衣物焚烧殆尽，只剩下洁白如玉，线条弧度完美的身躯。
<br/>　　九霄琉璃焱出现的瞬间，千米范围内的生灵，便以肉眼可见的速度化为灰烬，而他所在的地面也凹了下去，冒起一阵白烟。
<br/>　　元婴六重巅峰！
<br/>　　元婴七重巅峰！
<br/>　　元婴八重巅峰！
<br/>　　元婴九重巅峰......
<br/>　　慕清霄的气息在一瞬间攀升到元婴九重巅峰，而且气息并没有停止的趋势，而是继续向化神期壁障冲撞了上去。
<br/>　　“轰！”
<br/>　　天空上猛的响起一道嘹亮巨响，随即整片天地一下子就暗了下来，上空，云层迅速从四面八方凝聚而来。
<br/>　　慕清霄抬头望向天空，神色凝重，道：“雷劫！”
<br/>　　凝聚而来云层，并非乌黑色，而是一种极端深邃的暗黑色，黑云弥漫整片天空，其中并没有雷雳声传出。
<br/>　　慕清霄凝望着天上不断蠕动的黑云，来自黑云中，他能感觉到一种极具毁灭性的力量，仅仅是这股气息就让他汗毛炸立！
<br/>　　暗黑色云层不断的蠕动，笼罩在森林上空，好似是某种前兆，整片天地都是一片寂静，没有半点雷鸣声响起。
<br/>　　然而，就是这种死一样的寂静，才让他觉得可怕！
<br/>　　慕清霄心头颤了颤，真是恐怖的雷劫，他可是第一次遇到雷劫，顿时有些不知所措起来。
<br/>　　然而，慕清霄所不知道的是，他并非没有渡过雷劫。
<br/>　　只是，当初在‘秦时明月’位面，渡劫的时候晕厥了过去，待他醒来的时候，雷劫已经消失。
<br/>　　伴随着暗黑色云层不断的蠕动，大约十几分钟后，云层中心缓缓裂开一个圆形空洞，空洞内部，依旧是一片黑漆漆的模样，却隐藏着一股令人余悸的气息。
<br/>　　然而，就在圆形空洞出现的一瞬间，慕清霄浑身寒毛都竖了起来，元婴九重巅峰的气息，毫无保留的自体内席卷而出。
<br/>　　“蓬！”
<br/>　　黑色羽翼自背后弹出，黑色晶体自在额头上浮现，整个人都充斥着神圣的气息，在他气息暴涨的时候，上空空洞之中，也传来一阵异响。
<br/>　　“轰！”
<br/>　　慕清霄凌空而立，空中一声巨响，旋即一道约莫手臂粗细的黑色闪电，以迅雷不及掩耳之势飙射而出，瞬间就劈在他的胸膛上。
<br/>　　在黑色雷霆的恐怖一击下，慕清霄整个胸膛都皮开肉绽，恐怖的力量，瞬间将他击落，压着他向地面坠去。
<br/>　　伴随着‘砰’的一声巨响，碎石飞射，尘埃弥漫，恐怖的力量直接将地面击穿，形成一个深三米的人形坑洞，坑洞中冒着白烟，可见这一击的恐怖。
<br/>　　“咳咳！”
<br/>　　慕清霄拖着残破的身躯自坑洞中飞出，嘴中咳出两口鲜血，胸膛上的裂痕恐怖狰狞，然而却没有血液流出，而是燃起了黑色的火焰。
<br/>　　在火焰的燃烧下，伤口坏死的肌肉蠕动几下，随后以肉眼可见的速度恢复起来，短短几秒就恢复的差不多了。
<br/>　　脸上肌肉狠狠的抽了抽，慕清霄道：“真是恐怖的雷劫，若非自己拥有凤凰血脉，在这一击下，不死也得重伤，哪能像现在这样活蹦乱跳。”
<br/>　　待伤势恢复如初，慕清霄捏了捏拳头，发现自己的力量又增强许多，而且挡在前面化神期的壁障忽然松动了一些。
<br/>　　“有趣，雷劫竟能助我突破，不愧是诸天万界最巅峰的血脉，肉身扛雷劫，简直太变态了！”
<br/>　　想到这里，慕清霄心中隐隐兴奋起来，再次抬头看向天空上的黑云时，眼中的一丝惊惧也消失不见，取而代之的是兴奋，激动，唯独没有恐惧！
<br/>　　雷劫好似受到了挑衅，雷鸣声作响，“砰砰砰”三声巨响，比先前还要粗的黑色雷霆瞬间落下。
<br/>　　慕清霄咬紧牙关，运转心法的同时，卯足了劲，化作一道剑芒向雷劫撞去，没有任何的防御，携带者破风声，肉身与雷霆相撞！
<br/>　　“轰！”
<br/>　　当雷霆降落在身上，慕清霄瞬间坠落，黑色羽翼扇动，在距离地面两米的时候，终于停了下来。
<br/>　　忍不住喷出一口鲜血，胸膛早已炸裂开来，焦黑一片，面目全非，伤势比先前更加严重，内脏与森森白骨隐隐可见，看似极其可怖。
<br/>　　黑色雷弧在伤口跳动，由于痛觉太过强烈，慕清霄剑眉紧皱在一起，脸色早已变得惨败一片，但嘴角却泛起一丝弧度。
