　　从几天前一个消息已经是传遍整个中域了。
<br/>　　归墟天中所有七大宗门弟子，都被一名叫做楚风眠的年轻人所斩杀，楚风眠的凶名，倒是已经在中域之中响彻而起了。
<br/>　　古家家主也是之前知道了这个消息，如今听到楚风眠的名字，才发现这尊凶人，居然是来到了中域了，更是来到了宁州古城之中。
<br/>　　敢一个人将归墟天之中所有的七大宗门弟子都给轰杀。
<br/>　　楚风眠简直就是一个无法无天的人物，狂妄到极点的一代凶人。
<br/>　　现在楚风眠居然是放过了三位监察使，这其实便是给了古家家主一个面子了。
<br/>　　古家家主，毕竟可是这宁州古城之主，要是这三位监察使，现在都死在了宁州古城之中，那他们古家可也会引来大麻烦。
<br/>　　虽然他可以出手阻拦，但是古家家主的心中也没有想要跟楚风眠起冲突的想法，楚风眠就连七大宗门都不放在眼中，他们古家，自然也不会被楚风眠放在眼中。
<br/>　　到时候鱼死网破，对谁都不是一件好处。
<br/>　　“三个不懂事的小家伙罢了，给他们一个教训就行了，楚某还没打算在这宁州古城中染血。”
<br/>　　楚风眠轻描淡写的开口道。
<br/>　　这古家家主的顾虑，他自然清楚。
<br/>　　这古家家主既然是没有想要出手阻拦楚风眠的心思，也正是代表着他没有打算跟楚风眠冲突。
<br/>　　反正楚风眠也没有打算杀了这三个监察使的想法，倒不如卖一个顺水人情。
<br/>　　毕竟楚风眠想要购买一些圣药的话，若是有着古家相助，也会方便不少。
<br/>　　“这里不是说话的地方，楚公子，请跟我来。”
<br/>　　古家家主看着楚风眠，出声说道，便是大手一挥，凝聚出了一道空间隧道，一步踏入了其中。
<br/>　　“楚前辈，这麻烦您了，给您找了这么大的麻烦。”
<br/>　　古锦看着楚风眠，有些歉意的开口道。
<br/>　　她将楚风眠带到宁州古城来，本来是想要报答楚风眠，现在却是惹来了这种灾祸。
<br/>　　“没事，古小姐不用在意。”
<br/>　　楚风眠随意笑了笑道。
<br/>　　楚风眠现在身份反正暴露出来，自然会有数不过来的麻烦过来，几个小小的七大宗门弟子，楚风眠还没有放在眼中，这次饶了他们一命，楚风眠心中还有着其他计划。
<br/>　　看着那空间隧道，楚风眠没有任何犹豫，便是一步踏入了其中。
<br/>　　踏入空间隧道的一刻，楚风眠便是来到了一处密室之中，这密室似乎是从虚空深处硬生生的开辟出来的，除非是知晓这密室的所在，不然任何人不经过空间隧道，都无法进入其中。
<br/>　　这密室周围，还布满这无数禁制，层层叠叠的，令这密室的隐蔽性又更强一筹。
<br/>　　看来这里应该就是古家之中的一处密室了。
<br/>　　“锦儿，你先下去吧，我跟这楚公子，有话要说。”
<br/>　　古家家主看着古锦跟了进来，便是出言开口道。
<br/>　　“是。”
<br/>　　古锦点了点头，退了出去。
<br/>　　在这密室之中，只剩下了楚风眠跟古家家主两个人。
<br/>　　“楚公子倒是好胆色，这古家密室，那三个监察使都从来不敢进来。”
<br/>　　古家家主看着楚风眠出言道。
<br/>　　他的心中也有些惊讶，楚风眠的胆子居然这么大，居然是丝毫惧怕他，大摇大摆的走到了这密室之中。
<br/>　　要知道这空间隧道的背后到底是什么，从外面是根本无法看出的，寻常武者看到都要忌惮几分，仔细分析才敢进入。
<br/>　　而像是楚风眠这样大摇大摆之间就直接进来的，古家家主还是碰到了第一个。
<br/>　　从楚风眠的神色上来看，甚至都是没有任何一点的慌张，有着一种极端的自信，似乎是根本不惧怕他，会选择加害楚风眠一样。
<br/>　　这种极端的自信，也让古家家主感叹，倒是不愧敢跟七大宗门都抗争的人，居然有着如此胆量。
<br/>　　“三个胆小怕事的废物罢了。”
<br/>　　楚风眠随意开口道。
<br/>　　龙潭虎穴，楚风眠照样不怕，想要成就强者，做事决不能畏畏缩缩。
<br/>　　不过楚风眠心中也有些好气，这古家家主，如今既然已经是知道了楚风眠的身份，居然是没有让楚风眠离开宁州古城，反倒是带到了这古家密室之中来。
<br/>　　这古家家主到底是想要干嘛？拉拢？
<br/>　　楚风眠的身份，可以说是一个炸药包一样，跟楚风眠走得近，便是很有可能都会受到七大宗门的敌视，甚至是会被七大宗门视为敌人。
<br/>　　“不知古家主叫楚某前来，所为何事？”
<br/>　　楚风眠直接出言道。
<br/>　　既然到了这密室之中，那也没什么好在隐瞒的了，直接开口便是。
<br/>　　现在既然这古家家主将楚风眠请到这密室之中来，肯定便是有着他的目的。
<br/>　　“也没什么事，只是想要问问楚公子，你这次来我宁州古城，到对是为的何事？”
<br/>　　古家家主看着楚风眠，缓缓的开口道。
<br/>　　楚风眠绝对是他所见到过的，最难以看穿的年轻人，中域之中的绝代天才，古家家主也见到过不少，可是却没有任何一个人像是楚风眠一样，令他一点都看不穿。
<br/>　　他身为宁州古城之主，简直是人精一般的人物。
<br/>　　可是他看楚风眠，却是一片混沌，无法看穿楚风眠一举一动。
<br/>　　这让他也是极为好奇，在北域之中，居然还会诞生出如此可怕的人物来。
<br/>　　古家家主再度开口道。
<br/>　　“楚公子不用担心，我宁州古城，虽然被七大宗门所监管，不过却并不属于七大宗门，你跟七大宗门之间的恩怨，跟我们没有任何关系，不过楚公子救了小女，那老夫，自然是要来问一问了。”
<br/>　　“救了古小姐，也不过是偶然罢了。”
<br/>　　楚风眠听到这里，也听出了古家家主的想法，以及他的顾忌，楚风眠的身份实在是太敏感，他也担心，楚风眠到底是不是冲着他们古家，才来的宁州古城。
<br/>　　“至于楚某来到宁州古城，很简单，楚某只是想要购买一些炼药的药材罢了。”
<br/>　　说到这里的时候，楚风眠的手中突然出现了一个玉符，递给了古家家主。
