　　那名双手被绑在一起，吊在房间天花板上的“雷鸣”特种部队队员低垂着脑袋，他身上几乎没有一块皮肤是好的，身上遍布伤痕。
<br/>　　在他左腿内侧，股动脉的位置被差了一段一头削尖的透明硬塑料管子，管子内被塞了一小截香烟里的滤嘴，管子下方放着一个塑料桶，从管子里流出的鲜血“嘀嗒嘀嗒”的滴落到塑料桶中。刚才罗昊在房间门口闻到的血腥味就是从这个塑料桶中飘出来的，罗昊没有去伸手去探查被吊在房间天花板上的“雷鸣”特种部队队员的生命迹象，只是低头看了眼他脚下那只塑料桶，就知道这个“雷鸣”特种部队队员已经没救了。
<br/>　　罗昊走到那道铁栅栏前，让铁栅栏背后的齐凡、梅庆伟、林晓阳等人退开后，拔出手枪对着挂在铁栅栏上的门锁就是一枪，把锁打烂，然后拉开栅栏门，把人放了出来。
<br/>　　“你们没事吧？”罗昊看着齐凡、梅庆伟、孙涛三个人问道。
<br/>　　“能跑能跳的。”梅庆伟回答道。
<br/>　　孙涛抬头看了眼被吊在天花板上的“雷鸣”特种部队队员，说道：“如果你再来晚一点，吊在天花板上的人可能就不止是他一个了。我们这些人可能都要被挂腊肠了。”
<br/>　　“西伦科特没有打算在这里待很长的时间，这里只是中转，他把我们带在身边只是作为要挟联合行动司令部的筹码。”齐凡向罗昊解释道：“等到他联系好怎么离开，他就会把我们这些人全部解决掉，他是不会带着我们这些人质跑路的。”
<br/>　　“现在该让西伦科特这混蛋偿命了。”罗昊把手枪递给齐凡，问道：“还能打吗？”
<br/>　　“没有任何问题。”
<br/>　　齐凡按下弹匣扣，退出手枪中的弹匣看了眼弹匣中剩余的子弹，然后把弹匣重新塞回到手枪中。
<br/>　　同样被关在铁栅栏背后三名“雷鸣”特种部队队员，把他们吊在房间天花板上的同伴放了下来，去对面房间找来一条床单，盖在阵亡的“雷鸣”特种部队队员身上，他们也知道现在不是带着他一起离开的时候，所以只能让他继续在这里待上一段时间。
<br/>　　“哥，其他人情况还好吗？”
<br/>　　跟在罗昊身后的齐凡，突然对罗昊问道，“我们被分开关押，相互之间都不知道彼此的情况，就连被带到这个山顶别墅的时候，我们也是被分开押送的。”
<br/>　　听到齐凡的问题，罗昊脚步停了下来，脸上闪过犹豫和挣扎，最后还是对齐凡、梅庆伟、林晓阳三个人说道：“当时除了我、狐狸、无常、噬蜂逃出来外，虎鲨、电光、冰枪三个人当场阵亡，剩下的全被抓了。噬蜂伤得很重，已经被送回国接受治疗，最坏的情况就是跟当时的妖瞳一样，再也醒不过来。在我过来的时候，龙猫、冰狼、蜂鸟、罗刹、金丝猴已经获救，只是他们的情况比起你们要糟糕，伤口出现了溃烂可能已经诱发感染。”
<br/>　　“你说电光、虎鲨、冰枪死了？”齐凡有些难以置信的看着罗昊问道。
<br/>　　罗昊眼圈有些发红的点了点头，回答道：“凤凰姐带人去现场把他们的尸体带回了联合行动司令部，并且送回了国。看到冰狼被救回来的那一刻，我都不知道等到他醒来之后，我要怎么跟他说明。”
<br/>　　梅庆伟按了按罗昊的肩膀，说道：“逝者已矣，我们现在要考虑的是怎么抓住西伦科特这个混蛋，让他血债血偿！”
<br/>　　“妈的！”孙涛一脚踢在走廊墙壁上，狠狠地说道：“肯定有人出卖了我们，要不然西伦科特根本不可能知道我们坐标！”
<br/>　　“是美国佬那边出了问题。”罗昊回答道：“汉密尔顿在出事之后就被要求回国接受审问，现在由太平洋舰队司令詹姆斯接替汉密尔顿的位置。”
<br/>　　“这群美国佬，出了事情就想要包庇！？”孙涛脸色很难看。
<br/>　　孙远、沈飞、张子涛三个人的阵亡让他很愤怒，尤其是孙远，这是孙涛最难以接受的事情。
<br/>　　“这次美国佬不会包庇。”罗昊摇头道：“在这次袭击中，海豹六队的一个战术小组遭到舰载攻击战斗机扔下的炸弹攻击，全军覆没无一幸存，现在美国国内天天在游行和声讨，如果汉密尔顿真的有问题，在这样的情况下，没有人能包庇他，现在白宫方面估计正想着把汉密尔顿给推出去顶罪，因为他们现在需要给愤怒的美国人民一个解释，而有些东西又是不能公开的，汉密尔顿则是最好的顶罪工具。”
<br/>　　对于汉密尔顿后面可能会遭到的待遇，在场的四个人没有人同情他，如果他当时能够听从李然的话，在行动前仔细地再对情况进行分析，而不是急于对西伦科特进行斩首，后面的事情就不会发生，各国特种部队也不会有现在这样的巨大损失。
<br/>　　罗昊带着被营救出来的六个人下楼来，楼下，李毅峰看到齐凡、梅庆伟、孙涛三个人也是非常高兴，虽然他们也受了伤，但是并不算严重，还能自由行动并且具有战斗力。
<br/>　　被罗昊救出来的六个人从客厅里那些死去的影子部队队员身上获得了武器和弹药，他们将影子部队队员尸体上的战术背心给脱了下来，然后穿在自己身上。
<br/>　　进入左侧那栋别墅的李然和慕容飞也是救出了一些被关押起来的特战队员，而其中就有周柏。
<br/>　　罗昊大致看了下周围的人，加上那个因为被刺破股动脉，失血过多死亡的“雷鸣”特种部队队员，没有找到的人质全都被找了回来，而且值得庆幸的是，这些从两栋别墅中被救出来的特种部队队员全都具有战斗力，这下子让罗昊身边的战斗力得到了很大的提升。
<br/>　　别墅中不缺武器和弹药，每个被救出来的特种部队队员都拿到了自动步枪，充足的弹药和手雷。
<br/>　　“你们找到西伦科特没有？”罗昊对李然、李毅峰、慕容飞问道。
