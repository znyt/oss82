        就在仲正浩和邢乐游不知道该怎么安慰贺思思的时候，张子晋却在这火上浇油地说道：“小学妹，你别跟阿寒计较。阿寒虽然没有承认他喜欢那个连星儿，但是我们这当兄弟的，一眼就能看得出，他的眼里和心里，除了那个连星儿，就装不下其他女孩。再说了……”
<br/>
<br/>        “你给我闭嘴！”邢乐游和仲正浩几乎是异口同声。
<br/>
<br/>        张子晋识趣地耸了耸肩，抿着嘴不再吭声。
<br/>
<br/>        “思思同学，你别听子晋这家伙瞎说！那个连星儿哪能比得上你！”仲正浩一本正经地夸赞贺思思。
<br/>
<br/>        邢乐游咧嘴一笑：“她除了长得比你好看那么一丁点，其他没一样比你强。”
<br/>
<br/>        他这话一落，仲正浩忍不住踢了邢乐游一脚：“你怎么说话的？”
<br/>
<br/>        贺思思吸了吸鼻子，看着面前这两个好心安慰自己的男生，柔柔弱弱地说道：“谢谢两位学长的宽慰。”
<br/>
<br/>        “我跟你说！你要想打动阿寒的心，你得坚持不懈。”仲正浩忍不住给贺思思支招道。
<br/>
<br/>        邢乐游点了点头，接着说道：“没有哪个男生不喜欢温柔体贴的女孩子。我跟你说，连星儿那丫头野得狠！大大咧咧的，除了长相以外，性格真的不招人喜欢。”
<br/>
<br/>        “照你这么说，阿寒岂不是不是人呢？”张子晋见缝插针，补充了一句。
<br/>
<br/>        结果遭来邢乐游一记白眼。
<br/>
<br/>        仲正浩拿来了纸巾，递向贺思思。
<br/>
<br/>        “嗯，我今天就呆在这儿，等他回来。”贺思思一边说着，一边接过仲正浩手中的纸巾，自顾自地擦眼泪。
<br/>
<br/>        仲正浩和邢乐游都没再说什么了，直到午饭的时候，他们俩还特意去食堂多打了一份饭菜上来送给贺思思吃。
<br/>
<br/>        女生寝室里，连星儿终于等到了顾千凝的电话，只不过这个时候已经下午四点了。
<br/>
<br/>        “星儿，你来市中心的索菲斯西餐厅，要打扮得漂亮一点噢！我们今天出门可都是打扮的漂漂亮亮的，所以，你也得跟上我们的节拍。我们等你一起吃饭噢！”顾千凝笑嘻嘻地说道。
<br/>
<br/>        连星儿应了声“嗯”，收了线后，开始打开衣柜挑起衣服来。
<br/>
<br/>        挑来挑去，她还是比较喜欢粉色。
<br/>
<br/>        换好了衣服，又化了个淡妆。
<br/>
<br/>        连星儿十分满意地站在镜子前转了一圈，自信地咧嘴一笑。
<br/>
<br/>        当她抵达顾千凝所说的索菲亚西餐厅时，偌大而时尚的西餐厅里却只有她一个顾客。
<br/>
<br/>        其实除了她以外，中央的三角钢琴架前还坐着一个身着白色衬衫，背影优雅的男生。
<br/>
<br/>        连星儿觉得这个男人的背影有些令她眼熟，但当她走近的时候，男生却开始挥动着臂膀，在钢琴键上弹奏了起来。
<br/>
<br/>        他的双手，修长而很有骨感，在黑白键上雀跃时，手影相随，速度快得令人发指。
<br/>
<br/>        其实她也学过钢琴，也考过级，但是以这个男生的水平，绝对在她之上。
<br/>
<br/>        不过话说回来，这男生弹的曲子，她却是第一次听。
<br/>
<br/>        连星儿默默地站在这个男生的身后，安安静静地听男生把曲子弹完。
<br/>
<br/>        若不是这曲完毕，男生起身转向她，她还真没发现面前的人，竟然是明司寒。
<br/>
<br/>        “喜欢吗？”明司寒从三角钢琴的平台上下来，手中还握着一支黄玫瑰。
<br/>
<br/>        但是，连星儿却不等他走到自己的跟前，脸色一变，转身就想离开。
<br/>
<br/>        明司寒见状，大步流星地跑上前去，伸手抓住连星儿的臂弯，手下稍稍一个力度，便将她的身子旋转回了自己的跟前。
<br/>
<br/>        “对不起！”他送上手中的黄玫瑰，诚恳地向她道歉。
<br/>
<br/>        连星儿瞥了明司寒手中的黄玫瑰一眼，微微扬起下巴，目光落向了一旁。
<br/>
<br/>        明司寒会心一笑，耐着性子解释道：“上午的事情，是我不对，没有弄清楚事情的来龙去脉就责怪你！”
<br/>
<br/>        “对啊！你有什么资格责怪我？”连星儿回眸瞪着明司寒，一张晶莹剔透的粉唇，因为生气而嘟了起来。
<br/>
<br/>        明司寒看着她的那张秀色可餐的小嘴，很想凑上去咬一口。
<br/>
<br/>        但他也就只是有胆子在心里意淫一下，不敢对她付诸行动。
<br/>
<br/>        “是是是！千错万错，都是我的错！不知道我的大小姐，是否愿意原谅我？”明司寒的俊脸上挂满了虔诚的笑容。
<br/>
<br/>        俗话说，伸手不打笑脸人。
<br/>
<br/>        连星儿看着明司寒脸上那好看的微笑，心里也莫名其妙地跟着愉悦了起来：“那，本大小姐看在你今天生日的份上，就不跟你计较了！”
<br/>
<br/>        “嗯，请星儿大小姐收下鄙人的黄玫瑰。”明司寒微微欠身，优雅地行了一个绅士礼。
