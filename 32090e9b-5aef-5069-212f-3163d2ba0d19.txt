
<br/>　　<br/>　　    陈夫人肩头的两盏阳火也在这时突然烧的大旺起来，身上的阴气顿时一扫而光，冰凉的身体也逐渐恢复了正常的温度。<br/>　　<br/>　　    卧室外面，楼下的客厅里，尽管知道王惊蛰在上面能让自己挺托底的，但陈重和陈臣两父女俩仍旧坐立不安的等着，一脸的忧愁和担忧，这种事你摊到谁身上谁都放不下心，人对于未知总是最恐惧的。<br/>　　<br/>　　    不知过了多久，楼上传来脚步声，两人同时抬起头，就看见王惊蛰手里捧着个碗走了下来，等他下来的时候，才看见碗中居然诡异的立着一根筷子，筷子上头始终朝着一个方向偏着，指向了门外。<br/>　　<br/>　　    陈臣连忙迎了过来，开口问道：“惊蛰，我妈？”<br/>　　<br/>　　    “你家在这住了有多长的时间了？”王惊蛰问道。<br/>　　<br/>　　    陈重说道：“有七八年的时间了吧，我们很少搬家”<br/>　　<br/>　　    “那就好，这碗你拿着，跟着这个筷子指的方向走，等着碗中插的筷子什么时候不动了，你也就停下来不用走了，然后喊三声‘妈，跟我回家吧’就行了”王惊蛰把碗递给陈臣，她接到手里之后，诧异的问道：“这就行了？”<br/>　　<br/>　　    陈夫人被鬼给缠了身，鬼刚才被王惊蛰给收拾了，但她被惊走的魂魄自然还没有回来，一般情况下惊魂之后魂基本都在自己最熟悉的一带游荡，只是不知道该往哪走而已，这时候要是有个最熟悉的人领她回家，那自然就能找到本体然后再附上就可以了。<br/>　　<br/>　　    人都有三魂，少了一魂一般都是行为呆滞，少两则长睡难醒，要是三魂全失这人基本就是死透了，怎么招都没有用，人的七魄，越少越弱，基本就是恶疾霉运缠身，所以有很多人出了啥问题在医院却查不出病因，那基本就是魂魄被惊没了。<br/>　　<br/>　　    特别是七岁以下的小孩子是最容易被惊魂的，因为那时人刚出娘胎没几年魂魄还不稳呢，有时哪怕就是被吓住了，都容易把魂给惊走了，其次就是古来稀的老人，这时身体渐弱魂魄不稳，也是很容易被惊了的。<br/>　　<br/>　　    要说惊魂走失的表现其实也挺明显的，基本就是没有精神，浑身没劲比较容易嗜睡头脑不清醒，然后脾气暴躁，特别是睡觉的时候特容易盗汗和惊醒，常人总说人不在状态的时候就会说，你怎么跟丢了魂似的呢，基本就是这个意思了。<br/>　　<br/>　　    小孩丢魂其实很好招的，因为魂胆小都没有走远，肯定就在家附近，民间就有很多能叫魂的方式而且基本都很灵验，家里有养公鸡的就把公鸡抹脖子杀了，将鸡血滴到门口然后念几声小命说回家吧，就能孩子丢的魂叫回来了，要不就是像王惊蛰告诉陈臣的这样，碗中盛满清水插上一根筷子，念着丢魂人的名字就能给引回来了。<br/>　　<br/>　　    我记得我小时候也曾犯过这病，我妈骑车的时候把我摔在了地上，正好有一辆马车过来，马蹄子差点踩到我脑袋上，当时就给吓了个半死，人直接就傻了，回到家里不吃饭不睡觉就坐在那发愣，后来找人看过说是魂丢了，得给找回来才行，但用的却不是上面这些方式，就是老人找了一张老旧的邮票烧成纸灰然后送水喝下去就行了。<br/>　　<br/>　　    陈臣端着碗，顺着筷子头指的方向从家里出去了，陈重指了指沙发说了声坐，然后拿出烟递了一根过去，跟他随意的聊了起来。<br/>　　<br/>　　    “惊蛰今年看起来年岁不大，跟小臣差不多？”<br/>　　<br/>　　    “是差不多，二十四了”<br/>　　<br/>　　    “哦”陈重点了点头，弹了下烟灰后，又随意的问道：“你是哪人啊，听口音不像南方的呢？”<br/>　　<br/>　　    “我家在豫中的朝歌，跟丁武是一个地方的，不过这些年我也没在家呆着，一直在外面闲逛来着”<br/>　　<br/>　　    “呵呵，有本事的年轻人，有几个守家带地的？早就跑到外面去了，我要不是因为家里的原因，年纪轻轻的也肯定跑出去了”陈重笑着问道：“你这个年纪，不管是在南方还是北方，家里老人都该着急催婚了吧？哎，我跟你说做父母的关于这一点，全天下的爹妈基本都是一个心思，就拿小臣来讲吧，我对她的要求不高，只希望能找个老实本分人品好点的就行了，但这丫头也不知道是处于什么心态，这都二十三了我也没见她往家里带过一个男朋友，我跟你说，我也快要到急的不行了·····”<br/>　　<br/>　　    王惊蛰挠了挠鼻子，总觉得跟陈重聊天的节奏有点不太对了：“呵呵，也是这么回事”<br/>　　<br/>　　    陈重意味深长的说道：“我对小臣的个人问题也不是特别在意，毕竟家里条件还行么，所以还是得看人”<br/>　　<br/>　　    王惊蛰瞥了眼陈重的眼神，忽然冒出一股毛骨悚然的感觉来，总觉得对方有种要看透他内心本质的念头。<br/>　　<br/>　　    “小蛰啊……”陈重掐了烟头，跟个长辈似的语重心长的问道：“你想找个什么样的姑娘呢？你陈叔的阅历和人面还行，你有什么中意的类型，我能帮你留意下，方便的话可以介绍认识认识，我听小臣说你平时也不太出门，老是在家呆着可不行啊，天上不可能总掉下来林妹妹吧，步子还是迈出去才行啊”<br/>　　<br/>　　    “呃……”王惊蛰憋了半天，也没憋出个屁来，他就算在傻也听明白陈重话里的意思了。<br/>　　<br/>　　    他这是要招驸马爷了啊？<br/>　　<br/>　　    王惊蛰寻思了下，说道：“呃，我还不急，不急，这几年身上的事比较多，等我捋一捋的吧”<br/>　　<br/>　　    “哈哈，人过了二十，以后的日子就快了，一晃几年时间就过去就奔着三十去了，到时候想再找那就麻烦了”陈重看见外面的陈臣已经回来的，就拍了拍王惊蛰的腿，说道：“就这么定了，陈叔有空帮你看看有没有合适的姑娘，我对你这个年轻人，还是很看好的”<br/>　　<br/>　　    王惊蛰就寻思了，你还不如干脆说想把你姑娘给推给我算了，在这挂什么羊头卖什么狗肉啊。<br/>　　<br/>　　    陈臣端着碗进来了：“惊蛰，我按照你的说的做完了”<br/>　　<br/>　　    王惊蛰叹了口气，说道：“你直接上楼去卧室就行了，你妈被惊走的魂，自然就能回去了，事后好好静养一下，找个中医开个补气血的方子就可以了……”
