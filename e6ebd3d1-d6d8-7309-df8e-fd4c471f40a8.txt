　　众人之中，唯独庄菲菲和红姐隐约猜到，叶仙人应该就是叶尘，毕竟她们见识过叶尘的神鬼手段。
<br/>　　那高总监反应过来之后，想起之前叶尘说过，杨国强会过来，先是两眼一瞪，继而猛然摇了摇头，
<br/>　　“不！这不可能！绝对不可能是杨董事长！”
<br/>　　咆哮了一声之后，高总监深吸了一口气，稳定了一下激动的情绪，然后直接冲着向门口，满脸煞气道：
<br/>　　“我倒要看看，在临安市的地头上，哪个吃了熊心豹子胆，敢冒充杨董事……”
<br/>　　高总监的话还没有说完，房门打开之后，眼前出现了一名相貌威严的中年男子，顿时吓得浑身一哆嗦，差点直接尿了，
<br/>　　“杨、杨董事长，您、您怎么来了？”
<br/>　　眼前这人，赫然正是杨氏集团的董事长杨国强。
<br/>　　杨国强听到高总监这话，顿时眉头一皱，
<br/>　　“你是哪位？”
<br/>　　高总监此时也顾不得，杨国强为何会出现在这里，连忙陪着笑脸，道：
<br/>　　“杨董事长，您好！我是华艺天皇娱乐的商务总监高建人呐！您叫我小高就行！”
<br/>　　“哦，”
<br/>　　杨国强随口应了一声，目光已经转移到后方等人身上，等他看到叶尘的时候，顿时两眼一亮，脸上露出一抹敬畏之色，还不忘整了整了自己的衣冠，犹如小粉丝见到自己的偶像一般。
<br/>　　“杨董事长，不知道您来这里所为何事？有什么需要小高我效劳的，您尽管吩咐！”
<br/>　　高建人依旧在喋喋不休的献着媚，虽然他隐隐觉得事情似乎有些不妙，可是依旧不愿意相信，杨国强真的是被一个少年喊过来的，心中一直在告诉自己，
<br/>　　‘巧合！这一定是个巧合！’
<br/>　　然而，
<br/>　　现实却给了他狠狠一击！
<br/>　　只见，杨国强完全无视高建人的谄媚，直接从他跟前越过，然后快步朝那名‘狂妄少年’走了过去。
<br/>　　继而，
<br/>　　在众人无比震惊的目光下，杨国强竟然冲着那少年，直接缓缓跪倒在了地上，满脸的激动神色，
<br/>　　“叶仙人在上！小人杨建国拜见！”
<br/>　　哗！
<br/>　　包厢里的所有人，顿时全都傻眼了。
<br/>　　就连庄菲菲事先知道叶尘身份超然，看到杨国强的这番举动，也震惊的捂住了嘴巴。
<br/>　　杨国强是何许人也？
<br/>　　那可是整个东江省，跺一跺脚，都要抖三抖的大人物！
<br/>　　在整个华夏商界，也有着一定的地位。
<br/>　　如此大佬级的人物，居然向叶尘跪拜行礼！
<br/>　　这怎能不让人感到震撼？
<br/>　　至于那位美女律师，看到这一幕，更是震惊的无以复加。
<br/>　　若非亲眼所见，她打死也不愿意相信，叶尘居然真的只用一个电话，就把大佬级人物杨国强，给喊了过来！
<br/>　　而且从眼前的情景来看，这个少年的地位，似乎远远在杨国强之上！
<br/>　　‘这家伙到底是个什么妖孽啊？难道真的是某个超级大家族的公子哥？’
<br/>　　“噗通！”
<br/>　　高建人看到眼前这一幕，更是两腿一软，一屁股坐在了地上，脸色比死了爹妈还要难看。
<br/>　　若非他之前，和杨国强见过几次面，他肯定认为眼前这个人是冒充的。
<br/>　　堂堂杨氏集团的董事长，东江省的顶级商业商业大佬，居然向一名少年跪拜行礼！
<br/>　　这实在是太梦幻了！
<br/>　　然而，
<br/>　　面对杨国强的跪拜，叶尘却连眼皮都没有抬一下，依旧翘着腿，坐在椅子上，缓缓品着茶水。
<br/>　　杨国强见状，顿时心头一禀，悄悄望了一眼周围众人的情景，脸上不由得浮现出一抹惶恐之色。
<br/>　　作为东江省商界的大佬级人物，杨国强自然也不是泛泛之辈，大致看了一眼周围，便已经猜到了大概。
<br/>　　但是见叶尘一言不发，他也不敢多说什么。
<br/>　　足足过了好一会，
<br/>　　叶尘这才缓缓将手中的茶杯放心，瞥了一眼依旧跪俯在地上的杨国强，
<br/>　　“杨董事长请起，这么着急把你喊了，没有耽误你的大事吧？”
<br/>　　说出这话的同时，叶尘体内的真元自动放出，已经将杨国强从地上托起。
<br/>　　杨国强心头惊骇不已，神色顿时也越发的恭谨，连忙赔笑道：
<br/>　　“叶仙人说笑了！什么事也没有您的事重要，不知道杨某有什么地方，可以为叶仙人效劳的，您尽管吩咐！”
<br/>　　叶尘见杨国强如此识趣，也不想过分难为他，于是冲着一旁的庄菲菲道：
<br/>　　“庄小姐，把你的事情，向杨董事长说一说吧！”
<br/>　　庄菲菲此刻，对叶尘早已经佩服的五体投地，当即乖巧的应了一声是，然后把她和华艺天皇娱乐以及高总监之间的矛盾，一五一十的讲了出来。
<br/>　　她口才本来就很好，而且又常年混迹娱乐圈，说话滴水不漏，又恰到好处，箭头直指高建人。
<br/>　　杨国强听完之后，顿时额头冷汗直冒，不由得再次狠狠瞪了一眼身后，已经快要吓尿的高建人，真的是想杀人的心都有了！
<br/>　　华艺天皇娱乐公司，是杨氏集团旗下的分公司，一直都是由他的弟弟在打理，

<br/>　　只不过，
<br/>　　在他这样的商业大佬面前，庄菲菲不过就是一个戏子，根本微不足道，所以压根就不会放在眼里。
<br/>　　可是，
<br/>　　他怎么也想不到，这个女人竟然请出了叶尘这尊大佛！

<br/>　　高建人感受到杨国强身上的杀气，顿时吓得浑身肥肉乱颤，继而跪着爬到杨国强的面前，
<br/>　　“杨董事长，我错了！我知道错了！可是这真的不管我的事啊，这一切，都是杨总的意思啊！”
<br/>　　听到高建人这话，杨国强先是微微一愣，继而脸色大变。
