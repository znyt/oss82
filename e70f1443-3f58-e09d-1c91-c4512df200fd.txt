　　离开庄园后，兰博基尼一路疾驰，半小时后就抵达辉煌集团门口。
<br/>　　此时，烈阳高照，辉煌集团正处于上班时间，门口非常肃穆，完全看不到人影。
<br/>　　当慕清霄将车停在门口，保安室中就有两名中年男子走了出来，无论是气质或者步伐，都给人稳如泰山的感觉。
<br/>　　关上车门，慕清霄脸上掀起一丝弧度。
<br/>　　“林雷，好久不见。”
<br/>　　来者正是独狼佣兵团的首领，独狼林雷，与他的弟弟林鸣。
<br/>　　此时，他们锐气尽数散去，添了一分柔和，虽看似平凡，但身手相比两年前，却是天差地别。
<br/>　　原本两人就是雇佣兵，经历过枪林弹雨……
<br/>　　如若是普通人，完全安定下来后，身上的锐气恐怕早就被消磨的一干二净，但林雷等人不同。
<br/>　　虽说，他们在辉煌集团的职务只是安保人员，平时也是闲得慌，但他们却将空闲的时间，都用在了修炼上。
<br/>　　有慕清霄给的简洁版的《易筋经》，他们的实力不仅没有退化，反倒是比两年前强了数倍。
<br/>　　林雷与林鸣见到慕清霄后，严肃的脸庞也松懈下来，露出真心诚意的微笑。
<br/>　　“董事长。”
<br/>　　拍了拍林雷的肩膀，慕清霄打量了两人一眼，明显能感觉出，他们气息都沉稳强悍许多。
<br/>　　“最近这段时间，过的怎么样？”
<br/>　　“都是老样子，日子倒是舒坦，就是身子快生锈了，有些怀念以前的生活。”
<br/>　　林雷回想起，往日里跟兄弟们枪林弹雨的日子，不禁有些感慨。
<br/>　　边走边聊，互相之间不像上下级关系，倒是像许久不见的好朋友，不多时就来到辉煌集团后面的训练场地。
<br/>　　此处，是与辉煌集团一同完工的训练场，专门供给安保人员训练使用，林雷他们的住处就在训练场对面的宿舍中。
<br/>　　慕清霄的到来，独狼佣兵团的弟兄们都非常高兴，嚷嚷着要一起喝酒，不过却被他委婉拒绝。
<br/>　　离开训练场后，慕清霄带着林雷来到集团的后院，此处是一处人工湖，两岸种植着绿竹，中间有一个凉亭。
<br/>　　慕清霄负手而立，站在人工湖旁沉思，林雷则默默的站在身后。
<br/>　　原本，他并不打算找林雷等人，毕竟，哪怕没有独狼佣兵团，辉煌集团的安全也不可能有问题。
<br/>　　欧阳紫嫣，雅儿阿朱，都是元婴级别的修士，实力虽说不上排山倒海，但上天入地已经不在话下。
<br/>　　而且，有他留下的资源，她们的实力，每一天都在进步。
<br/>　　至于林雷他们，都是一群普通人，若说的难听点，他现在就是养了一群废物。
<br/>　　如果是平时，一年花几亿养一群雇佣兵，价格对辉煌集团来说，不过是杯水车薪。
<br/>　　不过，轮回者的出现，倒是让他警惕了起来，林雷他们也终于能派的上用场。
<br/>　　他们的资质虽不高，但只要有足够的灵药和功法，哪怕是废物，也能用灵药砸成实力不弱的修士。
<br/>　　恰巧的是，慕清霄最不缺的就是资源，培养几十个金丹修士，对他来说，也不算太难。
<br/>　　沉默片刻，慕清霄凌空一指，人工湖中的水花就顺着轨迹快速升起，最后在空中凝聚成一个个图案。
<br/>　　站在他的身后，林雷浑身僵硬，目瞪口呆的看着眼前不可思议的一幕。
<br/>　　许久，空中的水花滑落，后院中的气氛变得有些压抑。
<br/>　　林雷是聪明人，根据慕清霄以前给他的修炼法门，与现令人无法理解的一幕，已经多少猜测出了接下来的情况。
<br/>　　察觉到林雷的神色，慕清霄微微一笑，屈指一弹，一道灵光就射入林雷脑海中。
<br/>　　霎时间，林雷眼中闪烁着震撼之色，道：“《太极玄清道》！”
<br/>　　“你先尝试着修炼，也同样可以传给你的兄弟们，有些事情，你们现在不必知道，也没资格知道太多。”
<br/>　　说罢，慕清霄就转身离去，留下林雷一人，愣愣出神。
<br/>　　慕清霄没有说太多，在他眼中，林雷是个聪明人，知道该怎么做。
<br/>　　将《太极玄清道》传给他们，只是想尝试下，如果他们在一段时间内，达不到他想要的要求，那也就没有培养下去的必要。
<br/>　　如果，他们在一段时间内，达到了自己想要的要求，继续培养他们，也未尝不可。
<br/>　　否则，慕清霄传授的就不是《太极玄清道》，而是更高级的修炼心法，比如《空明幻虚剑》这种。
<br/>　　欧阳紫嫣她们虽有实力，但慕清霄并不希望自己的女人涉及其中，若是林雷等人到时候有实力应对，最好不过。
<br/>　　当然，现在只是实验阶段，是虫是龙就要看他们自己了。
<br/>　　离开后院，慕清霄前往了一趟总裁办公室，并且将林雷等人的情况解释了一下。
<br/>　　最后，两人撑着空闲时间，耳鬓厮磨在一起，慕清霄尝到一些甜头后，并没有乘胜追击，而是径直离开。
