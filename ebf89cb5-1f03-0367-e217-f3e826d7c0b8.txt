　　苏莫和付元青两人，在峡谷中等了两个时辰，后续又有三批人到来。
<br/>　　毫无意外，全部被苏莫镇压，被迫交出了道剑。
<br/>　　当然，苏莫和他们无仇，也没有伤他们的性命。
<br/>　　三批人之后，便基本没有人来此了。
<br/>　　“十一柄道剑！”
<br/>　　苏莫盘膝坐在峡谷之中，低声自喃，目前为止，他共计得到了十一柄道剑，算是已经合格了。
<br/>　　当然，据说得到的道剑越多，奖励则越丰富，他自然还想得到更多的道剑。
<br/>　　“付元青，此地大概有多大？为何为何无人来此了？”苏莫转头看向付元青，沉声问道。
<br/>　　“此地很大，方圆千万里！”
<br/>　　付元青面色郑重，继续道：“估计很多人，抱着和我们一样的心态，守株待兔，所以没有人来此了，而剩下拥有道剑之人，必然都是强者，弱者不可能得到道剑！”
<br/>　　“你认为哪些地方有人，带我过去！”苏莫站了起来。
<br/>　　“好！”
<br/>　　付元青点了点了点头，随即，便带着苏莫，离开了这片峡谷，向其它地方飞去。
<br/>　　随即，两人的身形辗转各处，寻找其他人。
<br/>　　以苏莫的实力，基本上是扫荡，只要遇到其他人，必然是强势压服。
<br/>　　只要是遇到稍微厉害的武者，基本上都有道剑在身，而且是拥有多柄，而一些弱者，则是并没有道剑。
<br/>　　连续扫荡了十几个时辰的时间，苏莫得到的道剑，达到了五百多柄。
<br/>　　因为，有的厉害的妖孽，身上都拥有十数柄道剑在身，只要掠夺一人，便能得到大量的道剑，
<br/>　　苏莫的大肆掠夺，自然是引起了一阵喧嚣，让得很多天才震惊，只因为他修为太低了。
<br/>　　整片空间之中，如拓跋流明、云沐、韩生等强者，虽然都在掠夺，但是那些人均是众所周知的强者，自然在情理之中，但是，苏莫却是没有名气。
<br/>　　当然，苏莫在掠夺期间，曾远远的感受到了云沐的气息，不过他选择了退避，此时还不是和对方硬碰硬的时候。
<br/>　　毕竟，他战胜千幻邪君，都是在攻其不意之下，想要击败云沐，好像有些不太可能。
<br/>　　一片郁郁葱葱的丛林之中，苏莫和付元青，一前一后的前进。
<br/>　　“秦大哥……差不多了，再想找到更多的道剑，已经不太可能了！”付元青向苏莫说道。
<br/>　　“为什么？”苏莫问道。
<br/>　　“因为，有些实力偏弱的人，得到十柄道剑之后，就会立刻前往下一关，估计很多都已经走了，而拓跋流明和云沐等人，想抢夺她们的道剑根本不可能！”付元青沉声说道。
<br/>　　他虽然清楚的了解了苏莫的实力，但是，也不认为苏莫能战胜云沐等人，这根本不切实际。
<br/>　　“你是说，可以进入下一关了？”苏莫停下了脚步，既然如此，他就不多留了。
<br/>　　整片空间，只有近三千柄道剑，他一个人就得到了五百多柄，已经很多了。
<br/>　　“可以了，就算再继续寻找，也不可能得到多少了！”付元青点了点头。
<br/>　　“那好！”苏莫叹了口气，既然如此，这一关就到此为止吧！
<br/>　　随即，苏莫意念一动，所有道剑，全部出现在了他的身前。
<br/>　　这些道剑，共有五百零六柄，每一柄蕴含的道，都不相同。
<br/>　　“给你！”
<br/>　　苏莫手臂一挥，其中的十柄道剑，便飞到了付元青的面前。
<br/>　　“这……给我？”付元青见此，顿时一怔，面带不可置信之色。
<br/>　　他从未想过，苏莫会分给他十柄道剑，这太让他意外了。
<br/>　　“你帮了我，我自然不会亏待你！”
<br/>　　苏莫沉声道，随即，他看向自己剩下的四百九十八柄道剑，大手一伸，浑厚的玄力喷涌，将所有道剑都包裹了起来。
<br/>　　在混沌源力的笼罩之下，四百九十八柄道剑，立刻纷纷动了起来，形成了一个特殊的阵法，弥漫出玄妙的气息。
<br/>　　而付元青见此，深深的吸了口气，没有多言，也开始行动。
<br/>　　他虽然很意外，但是，既然苏莫给他道剑，他自然不会拒绝。
<br/>　　他心中有些感激，因为，若是没有跟着苏莫，他很难凑齐十柄道剑，因为他的实力不够，很容易被人抢夺。
<br/>　　由四百九十八柄道剑组成的阵法，呈圆形缓缓旋转，飞到了苏莫的头顶之上。
<br/>　　阵法之中，一股玄妙的气息，越发的浓郁，最后化为淡淡的气流，向下倾斜而下，涌入了苏莫的体内。
<br/>　　这个时候，阵法和苏莫，同时的消失了。
<br/>　　而付元青，用十柄道剑组成的阵法，同样弥漫出玄妙的气息，裹挟着付元青，消失无踪。
<br/>　　一片虚无的世界之中，苏莫双眸微闭，头顶之上巨大的剑阵，不断的涌入他的体内。
<br/>　　而他的意识，进入了一处玄奇的世界之中。
<br/>　　这片玄奇的世界之中，有很多人在练剑，练习各种各样的剑道。
<br/>　　有水属性的剑道，剑劲绵绵不绝，有重力剑道，每一斩下，都仿佛是裹挟了几百个星辰的重量，还有追踪剑道，一剑击出，不管是对手在何地，不管对手如何躲避，都无法躲开。
<br/>　　总之，各种稀奇古怪的剑道，在这里都应有尽有，而种类刚好是四百九十八种。
<br/>　　这些人均是在练剑，各种剑道的真意，都能显露而出。
<br/>　　望着这各式各样的剑道，苏莫陷入了静静的感悟之中。
<br/>　　不错，他在学习，学习每一个人的剑道。
<br/>　　当然，他并不是要将几百种剑道都学会，因为这根本不可能，他相信其他人也不可能。
<br/>　　而苏莫所要学习的，是这些人如何施展这各种剑道，他要学习到其中的要点，融会贯通，精炼自己的剑道。
<br/>　　时间不断的流逝，苏莫仔细的观察每一种剑道，了解其中的玄妙。
<br/>　　不知不觉，好像是过去了几日，又好像是过去了几年，他的心中有所明悟，自己的剑道也在不知不觉之中精进。
<br/>　　不过，苏莫心中的疑惑，却也是更多了。
<br/>　　“终极剑道，是怎么融合的？或者说是怎么施展的？”苏莫心中低声自喃。
<br/>　　因为，按照他的猜测，终极剑道是终极剑尊收集天下所有剑道，融合而出。
<br/>　　但是，这具体要怎么融合？
<br/>　　比如，此地有四百多种剑道之法，如何将他们彻底融合？
<br/>　　在苏莫想来，所谓的终极剑尊，也没有修炼成终极剑道，因为终极剑道根本就没有定义。
<br/>　　比如，你掌握了一百种剑道，将其融合在了一起，可以称其为终极剑道，但是，你掌握了一千种剑道，融合在了一起，也可以称之为终极剑道。
<br/>　　而到底有多少种剑道，没有人知晓，估计就连终极剑尊，都不知道到底有多少种剑道。
<br/>　　换言之，终极剑道就是一条无尽的深渊，没有底线，加入的剑道越多，终极剑道便越多。
<br/>　　“想不通！”苏莫没有紧皱了起来，他实在想不通终极剑尊，是如何做到的。
<br/>　　要是想要把这所有的剑道，全部一一学会，根本不可能。
<br/>　　所以，必然是有特殊的捷径，只是到底是何种捷径，任何人都不得而知。
<br/>　　不然的话，早就有人学会终极剑道了。
<br/>　　少倾，苏莫不再多想，也不再观察眼前的各种剑道，心神收了回来。
<br/>　　霎时之间，眼前的一切，全部都烟消雨散，他的身形出现在了一处特殊的地方。
