　　“但你却想当叶少阳。”芮冷玉道。
<br/>　　“应该说，是想当真正的叶少阳。”
<br/>　　芮冷玉鼻孔里哼了一声，十分轻蔑地看着他，说道：“你永远成不了他。他之所有有今天，是因为他骨子里，是一个不屈的人，他有他的坚守和信念，而你，什么都没有。”
<br/>　　“活下去，就是我的信念。”
<br/>　　“那你还在等什么。”
<br/>　　“我在等道风死。”
<br/>　　叶少阳诡谲地笑了笑，“挺讽刺的是不是，我们在这里闲聊，道风却在外面为我死撑着，他会一直撑下去，直到灯枯油尽……”
<br/>　　有清长风等一干绝世强者对付他，有大洞真经碑文，好有彩云仙子的《胡笳十八拍》的仙乐大阵，道风根本没有幸存的道理。
<br/>　　关键是，他为了保护自己，根本不会走。那就只能战死了……
<br/>　　“道风一死，风之谷大乱，鬼域三足鼎立的局面也会被打破，必然波及人间，天劫来临，群魔乱舞，诸方势力都会被波及，到时候天下大乱……法术公会入主人间，必然一呼百应，而我，作为天选之子，必将成为力挽狂澜额英雄！”
<br/>　　“但你毕竟不是叶少阳，你不是天选之子。”
<br/>　　“哪有什么天选之子！”
<br/>　　叶少阳大笑，“那不过无极鬼王和酆都大帝之间的一个笑话！我说我是叶少阳，谁敢说我不是？”
<br/>　　“我。”
<br/>　　一个淡淡的声音，从身后传来。两人都是一惊。叶少阳转头看过去，一袭青衣，缓缓走来，头顶三花绽放，手里提着打神鞭，气息却十分内敛，脸上也没什么表情。
<br/>　　“道风！！”
<br/>　　叶少阳震惊，仔细听外面，打斗声依旧，口中喃喃说道：“不可能！凭混沌天魔和你那个手下，怎么可能挡得住他们！”
<br/>　　芮冷玉看到道风一瞬间，心花怒放，长长地吐出一口气，望着突然出现的道风，这一幕，真的有点天神降临的感觉。
<br/>　　道风站住，一只手背在伸手，静静地看着叶少阳，“从一开始，我就知道，你是假的。”
<br/>　　“不可能！”叶少阳暗自思忖了片刻，异常坚定的说道，“我没露出任何破绽！”
<br/>　　道风从袖子里摸出了一个东西来，是一枚亮晶晶的匕首。
<br/>　　灭灵钉！
<br/>　　叶少阳怔住，下意识地伸手到兜里，摸到了一个同样的物件。
<br/>　　两把灭灵钉！
<br/>　　“你可能不知道，真正的少阳，早就把灭灵钉送给了那个女警，你表现的很好，天衣无缝，但唯独一处败笔，便是在她面前拿出了灭灵钉。”
<br/>　　叶少阳愕然，随后很快冷静下来，苦笑着摇了摇头，“没想到竟然输给了细节，天意吧。所以，是她提前告诉了你真相？”
<br/>　　道风没有回答。真相是显而易见的：谢雨晴发现自己身边的叶少阳是假的，但是没有打草惊蛇，而是暗中找机会告诉了老郭，老郭自然赶紧告诉道风。
<br/>　　道风也偷偷观察过他，但没有揭破，一是想看看他到底要干什么，二来，也是想将计就计。
<br/>　　这个假叶少阳，趁着叶少阳被困古墓，冒充她回到石城家中，目标是山海印——在古墓的空间里，已经确定了叶少阳的山海印不在身上。
<br/>　　他当然不敢去找道风他们，怕露出马脚，只有找谢雨晴，之后听到法术公会的人汇报，叶少阳已经说出山海印的下落，然后，就如林三生推测的那样，他们暗中观察，看到道风一行人进了山河社稷图，出来时候虽然什么都没说，但反而更让他们确定，山海印被道风所得。
<br/>　　于是，才有了今天的计划，一来是为了山海印，二来也是为了除掉道风这个心头大患、让他好继续冒充叶少阳，算是一举三得。
<br/>　　天衣无缝的计划，没输在了一个细节上。
<br/>　　叶少阳心思百转，皱眉望着道风：“你既然早猜到了这一切，为何还要跟来？我想不到，你有什么办法逃出去，难道你真自信到以为你们能逃得走？”
<br/>　　“我来，是要确定一点，眼前的你，是不是真的叶少阳。”
<br/>　　叶少阳一惊，“确定？”
<br/>　　“毕竟，你们完全一样，我怕杀错了，而这件事，容不得一丝意外。”
<br/>　　叶少阳愣住，只为了确定自己是不是真的叶少阳，就主动投入圈套……他不能明白，道风是什么样的心态。
<br/>　　但是芮冷玉却能理解。
<br/>　　对道风而言，就算明知面前这个叶少阳百分之九十九是假的，但只要有一点怀疑，他就不敢下手，因为，对方是叶少阳。
<br/>　　他想要杀一个跟叶少阳一模一样的人，必须过自己内心那一关。所以他还是来了。
<br/>　　不过……这里头的确还有些让人不解的地方。但现在不是想这些的时候，芮冷玉冲道风说道：“你还等什么！”
<br/>　　道风手中的打神鞭一瞬间放出万丈光华，朝叶少阳打过来。
<br/>　　“真打，我也未必怕你！”
<br/>　　叶少阳锵的一声拔剑，一时间紫光萦绕，道风也忍不住多看了一眼。
<br/>　　七星龙泉剑！
<br/>　　不过，是不属于这个世界的。
<br/>　　轰！
<br/>　　两把神兵撞击在一起。叶少阳被震退了几步，用身法稳住，一身戾气爆发，双眼血红地盯着道风，邪恶地笑起来。
<br/>　　“你当我是这个世界的叶少阳？我早已入魔，几乎堪破上仙牌位，道风，我早就想与你一战！”
<br/>　　叶少阳左手在七星龙泉剑的剑锋上擦了一下，献血涌出，喷洒在剑锋上，宝剑就像一块吸水纸一样，让血在剑锋上漾开，成为了一把赤红色的血剑，多余的鲜血流淌在上面，都被蒸发成了血雾，环绕在剑锋周围，这画面看上去，充满了一种邪恶的意象。
<br/>　　“这七星龙泉剑，早已被我炼成血剑，怎么，你看到这个，很气愤吗？”
<br/>　　叶少阳对着道风挥剑，无数剑光飞起，将道风包围起来。他自己也仗剑冲了进去，念着咒语，将宝剑的威势发挥到了极致。
<br/>　　一道青影，在血光剑影之中穿梭，快得难以捕捉。
<br/>　　芮冷玉被定魂符封印，脑袋无法转动，根本看不清打斗的双方，只能感受到一阵阵的真空波动，在这狭小的石洞中层层叠叠的涌动着。
