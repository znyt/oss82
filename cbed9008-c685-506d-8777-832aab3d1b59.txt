　　　　猴儿山的半空中，雷电呼啸撕扯，形成一个巨大的漩涡肆虐，但突然，其中迸发一声铿锵有力的呐喊声。
<br/>　　“给劳资破”
<br/>　　巨大的雷电漩涡顷刻间溃散，一股雷浪激荡的席卷出去，涂小安完好无损的立于天地之间，人形态还好好的维持住。
<br/>　　“哈哈...天雷之威，不过如此而已”
<br/>　　此时，涂小安整个人灿灿生辉，眼中精光四射，刚才他被紫色雷电疯狂的折磨，好像有无数吃虫子撕咬着他。
<br/>　　但是，涂小安并没有感觉到痛苦，本身拥有雷电之力的他，对于雷一点都不陌生，区别在于，他用雷电劈别人，现在雷电劈他。
<br/>　　天雷蕴含天威，能对灵魂产生影响，一些心智不高的人，灵魂一错乱，顾不得抵挡天雷，就灰飞烟灭了。
<br/>　　第一道雷电，看似非常的可怕，但是涂小安真正接触了，反而没有感到惧怕惶恐了。
<br/>　　其实未知的事情才是最吓人的。
<br/>　　他有先天优势雷电属性，对天雷拥有一定的免疫力，这是非常巨大的依仗。
<br/>　　这场仗，他涂小安只能胜不能败，败了就是惨败，所有的努力就付之东流，灰飞烟灭。
<br/>　　涂小安重新出现，让无数妖精纷纷动容，在远处山巅目视着这一切，震撼无比，他不但活下来了，而且还完好无损？
<br/>　　生物逆天而行，自然要受天地之罚，也便有了天劫，经历天劫之人实力愈是厉害，天地之罚威力愈是大，对方在第一道紫色雷电下活了下来，也不是没有道理。
<br/>　　要知道杀鸡焉用牛刀，既然用了牛刀，那么你的皮就一定是坚硬的。
<br/>　　“轰”
<br/>　　又一声巨大的雷响，第二朵劫云花盛开，一道紫色粗大的雷电仿佛一条蛟龙蜿蜒从劫云中穿出，而后带着一往无前的气势，按照直线一头朝下方冲去，仿佛神挡杀神魔挡诛魔一般，威力明显比第一道更大。
<br/>　　涂小安掀开眼，眉宇间似是凝着化不开的冰霜，一双妖治的眸子此时像是两汪平静无波的湖水，淡然冷漠，通透到了极致。
<br/>　　蛟龙一般的雷电霹雳而下，涂小安居然仰头而上，一只白皙的朝着蛟龙雷电抓去，居然被实质化的抓住了。
<br/>　　这一画面让人震撼的难以复加，倒吸了一口口凉气。
<br/>　　“给我滚”
<br/>　　只见涂小安立于半空之中，抓起蛟龙雷电就往外面扔，天哪，还被扔飞出去了。
<br/>　　一代猛人也。
<br/>　　远处猴儿山的两位猴王顿时破口大骂，转身便逃，这缺德的东西，要装逼就算了，抓起蛟龙雷电却朝着他们这片山域扔了过去。
<br/>　　轰...
<br/>　　这片山域瞬间笼罩在巨大的雷海之中，所有生物顷刻间死绝，直接被真空了一样。
<br/>　　“哈哈...继续劈”
<br/>　　涂小安仰天呐喊，愈发的自信起来，拥有雷电属性的他，天雷在让他的面前没有那么的恐惧，不过这仅仅是一个开始。
<br/>　　要知道整整三十六道天雷，一道比一道强，劈在你身上，好像一拳比一拳重。
<br/>　　轰...轰...
<br/>　　暗红色的天，直接两朵劫云花盛开，滚滚雷声，翻腾如海，只见两条巨大蛟龙般的雷电相互咆哮，相互缭绕，雷光灿灿，威压赫赫，朝着涂小安就铺天盖地劈下。
<br/>　　好似电光火石间，涂小安又想起了上古巴蛇跟他说的话。
<br/>　　它说涂小安身上拥有雷电属性，要是对抗天雷，说不定不是劫难，也有可能转化为机遇。
<br/>　　要是吸收了天雷之中的天威，那么他以后释放出现的就是天雷了。
<br/>　　或许，这也是涂小安强化自己雷电属性的最好机会。
<br/>　　趁着现在天雷的威力还没有强大到让他害怕，那便试一试吧。
<br/>　　其实现在天雷的杀伤力早就超出了一般渡劫生物的威力了，只是涂小安自己不知道。
<br/>　　想通这点，涂小安反而张开了双手，做出一副迎接天雷，要拥抱天雷的架势。
<br/>　　这一举动，让无数妖精大跌眼镜，瞠目结舌。
<br/>　　草泥马的，你这属于死前装逼不成？
<br/>　　纵观历史上，都从来没人敢如此面对天雷的，都是调动全力力气对抗天雷，生怕一着不慎，就会落的一个灰飞烟灭的下场。
<br/>　　旋即，两道蛟龙雷电直接将涂小安给吞没了，雷音滚滚，巨大的雷海漩涡翻滚如潮，天空闪电嘶鸣。
<br/>　　“劈死他，劈死他，从来没见过如此装逼之人”
<br/>　　天雷面前，用肉身去迎接，去硬抗，如同儿戏，是应该说人比人气死人，还是这货不知好歹。
<br/>　　无数生物在天劫的威压下吓都被吓死了，你一个正主那么随意？
<br/>　　“变态，变态，这条蛇精真是成精了”赤阳巨猿躲到另一边，气的说话都语无伦次了。
<br/>　　猿猴王恨不得抢在天雷降下之前，先杀了这条蛇，刚才居然将天雷扔给了它们。
<br/>　　它们是有惊无险的躲过去了，但是它的猴子猴孙就没有那么好命了，在一顿酸爽之下，直接往生极乐。
<br/>　　“喂，你们两个猴说话注意点，当老牛我是死人啊”
<br/>　　两只猴王这会很聪明，居然躲到了牛妖王跟树精的区域，它们不相信那条蛇精还能扔一道天雷到这边来。
<br/>　　“牛妖王，这条蛇精惹的祸，一切的锅都要你来背，你等着死吧”猿猴王冷冷的说。
<br/>　　牛妖王顿时不寒而栗：“上帝保佑我的主人渡劫成功”
<br/>　　要是失败了，他就惨了，他拿什么对付两只猴王，一只都够呛。
<br/>　　“哈哈，爽，痛快，痛快”
<br/>　　涂小安身在雷海漩涡之中，没有任何防御，甚至完全没有抵挡，像是借用天雷之力淬炼自己的身躯似的。
<br/>　　但是他的身躯是虚假的，不存在淬炼，不过涂小安运转雷电之力，偷偷的吸收天雷之威。
<br/>　　全身上下一通噼里啪啦，雷电穿梭其中，涂小安为了保持这具身体的抗性，用金元素加持，暗地之中不少雷电能量被他偷偷吞噬了进去。
<br/>　　这种吞噬是不可思议的，涂小安还在潜移默化的实验着，外界看来，他仿佛在经历巨大的痛苦，雷电贯穿他全身肆虐。
<br/>　　隐约间，涂小安感觉自己的雷电之力容量变大了，这种大怎么说呢，如果说他正常情况下，可以劈出一百道威力相同的雷电。
<br/>　　那么现在，随着吞噬天雷的能量，他已经可以释放一百一十道雷电了。
