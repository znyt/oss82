　　哗…哗…哗…
<br/>　　方圆数千米内的海面，顿时一阵翻腾。
<br/>　　一连窜密集的水流冲击声，有些震耳。
<br/>　　看真切些，竟是无数吞灵冰鱼飞跃而来。
<br/>　　密密麻麻，数之不尽，如过江之鲫。
<br/>　　那样子，像极了闻到腥味的猫一般，甚至似乎有些疯狂。
<br/>　　“怎…怎么可能…”风花月等人，瞪大了眼睛，满脸不可置信之色。
<br/>　　原本在他们周围密集的吞灵冰鱼，早已游走一空，全往萧逸等人的方向疾游而去。
<br/>　　甚至于，冰海底下，还陆续有吞灵冰鱼疯狂地被吸引而出。
<br/>　　不过半晌，出现的吞灵冰鱼，数量比之前多了十倍都不止。
<br/>　　且全部聚集于萧逸等人的范围。
<br/>　　另一边的白冰雪、钟无忧、叶铭等人，也瞪大了眼睛。
<br/>　　那种感觉，就像眼前忽然冒出了一座座‘金山’般，让人惊喜。
<br/>　　萧逸淡淡一笑，道，“还愣着做什么？”
<br/>　　白冰雪等四人立刻反应了过来。
<br/>　　大手一挥，便是数十条吞灵冰鱼被吸摄而去。
<br/>　　“吸。”叶铭一边吸取中吞灵冰鱼中的精纯灵气，一边继续‘捉鱼’。
<br/>　　只是，围拢在周遭的吞灵冰鱼，太多太密集了。
<br/>　　还未曾来得及将刚捉的数十条吞灵冰鱼吸收殆尽，又是密密麻麻一群涌来。
<br/>　　高空中，冰宫的执事愣了愣，随即快速给五人记录着分数。
<br/>　　远处，几位冰宫长老也将瞩视的目光，看向这边。
<br/>　　“那小子怎么回事？好精纯的寒冰属性力量。”一冰宫长老皱起了眉头。
<br/>　　“那等精纯度和澎湃度，连我们这一身天元境后期修为都不可能拥有。”
<br/>　　另一冰宫长老摇了摇头，道，“那小子不过地元九重修为，不可能有这般本事。”
<br/>　　“估计，是他身上有什么重宝吧。”
<br/>　　“据我所知，他们几人是裂天剑宗的天骄。”
<br/>　　“有此能耐，或者身怀如此重宝，也是正常。”
<br/>　　“也对。”另一位冰宫长老也出言道，“毕竟裂天剑宗是炎武王国第一武道圣地。”
<br/>　　“再者，或者是那位前辈赠予的重宝也不一定。”
<br/>　　“那位前辈？”刚才皱眉的那位冰宫长老出言道，“那个女人？”
<br/>　　话音刚出，他连忙闭上了嘴巴，不再多言。
<br/>　　原本皱着的眉头，也立马一松。
<br/>　　看向萧逸等人的目光，也变为了理所当然，而非疑惑。
<br/>　　另一边，风花月等人，脸色冰冷至极。
<br/>　　“该死，裂天剑宗的杂碎使了什么妖法？”
<br/>　　数百米外，一位冰河教弟子也冷声道，“这里可是极寒之地。”
<br/>　　“是我们的天下，何时轮到一些外来者嚣张。”
<br/>　　“风花月，我等联手给他们点颜色瞧瞧。”
<br/>　　“好。”风花月点了点头。
<br/>　　只见这几人，忽然猛地跺了跺脚下冰块。
<br/>　　一股莫名力量，陡然而出。
<br/>　　远处，钟无忧等人，还在欢快而快速地捕捉着吞灵冰鱼。
<br/>　　萧逸也同样如此。
<br/>　　但半晌，他忽然皱了皱眉头。
<br/>　　“嗯？”萧逸明显感觉到一丝不对劲。
<br/>　　脚下冰块，似在微微颤动。
<br/>　　这种颤动，非常微弱，若非敏锐过人，很难发现。
<br/>　　而造成脚下冰块颤动的原因…
<br/>　　“是水流。”萧逸眉头皱了起来。
<br/>　　下一秒，脸色忽然一变。
<br/>　　“是妖兽。”萧逸一惊，大喝一声，道，“小心。”
<br/>　　果然，他的话音刚落，一大群身躯庞大的冰海妖兽破冰而出。
<br/>　　白冰雪等人，听到提醒，早有准备。
<br/>　　一道道强悍攻击打出，瞬间将这些原本准备袭击的海兽击杀。
<br/>　　萧逸亦是早有准备，数道剑气从他手指间激扬而出。
<br/>　　直直打向叶铭的方向，替实力较弱的叶铭挡下了冰海妖兽。
<br/>　　不过，冰海妖兽是挡下了。
<br/>　　但，极其敏感的的吞灵冰鱼却立刻受惊，纷纷欲要逃回冰海之下。
<br/>　　“哼。”萧逸冷哼一声，“来了，便休想走。”
<br/>　　手中寒霜剑，凭空而现。
<br/>　　“寒冰覆海斩。”萧逸低喝一声。
<br/>　　如潮剑音，瞬间覆盖了方圆千米范围。
<br/>　　剑音如滔天巨浪，将冰海的海面搅得翻滚剧烈。
<br/>　　无数吞灵冰鱼，被强行震飞。
<br/>　　随后，在剑音之下，悉数被一股寒霜所包裹。
<br/>　　这些吞灵冰鱼，很是奇妙，一旦击杀，将会化为精纯灵气，自动消散。
<br/>　　萧逸只得将它们全部冰封起来。
<br/>　　跨啦啦…
<br/>　　一阵阵冰雹降落的声音。
<br/>　　钟无忧等人，连忙取出乾坤袋，将这些被冰封的吞灵冰鱼吸摄其内。
<br/>　　萧逸也没闲着，大手一挥，无数‘冰坨’，被吸摄入乾坤戒中。
<br/>　　直至将所有被冰封的吞灵冰鱼收光。
<br/>　　萧逸才将目光看向风花月等人。
<br/>　　明显能看到，现在的风花月等人，脸上一阵气急败坏之色。
<br/>　　虽然不知道他们用的什么手段，吸引了大量冰海妖兽袭击自己等人。
<br/>　　但无论是萧逸还是钟无忧、白冰雪等人，都很清楚，此事和他们脱不了干系。
<br/>　　第一轮冰海之缘规定，有名额的参赛武者间，不得互相干扰。
<br/>　　风花月等人，算是违规了。
<br/>　　不过，他们的手段又有些隐秘，高空中的冰宫执事们，并未发现这里的端倪。
<br/>　　远处。
<br/>　　风花月等人，脸色难看至极。
<br/>　　“该死，他们怎么可能这么快发现有冰海妖兽袭击？”
<br/>　　数百米外的一冰河教弟子沉声道，“那个叫萧逸的，反应如此之快。”
<br/>　　“我感觉，他是个猎妖师，而且经验相当老道。”
<br/>　　虽然彼此间隔着数百米距离。
<br/>　　但，对于地元境以上武者来说，隔空传音并非难事。
<br/>　　正当这几人脸色阴沉时。
<br/>　　风花月忽然蓦地一笑，目光直视萧逸那个方向。
<br/>　　“哈哈，那群杂碎完了。”
<br/>　　“你们看。”
<br/>　　远处。
<br/>　　萧逸五人的那个方向。
<br/>　　冰海之下，忽然出现了一抹极其巨大的黑影。
<br/>　　黑影之巨，竟是笼罩了方圆千米范围。
<br/>　　“什么东西？”萧逸猛地一惊。
<br/>　　明显能感觉到，一股澎湃的力量，逐渐在这周遭出现。
<br/>　　萧逸看了眼下方，脸色一变。
<br/>　　下一秒，一庞然大物，冲天而起。
<br/>　　“是吞灵冰鲸。”萧逸低喝一声。
<br/>　　吞灵冰鲸，与吞灵冰鱼一样，是这极寒冰海处，有吞噬灵气之力的奇妙存在。
<br/>　　但，它们有着与冰鲸一模一样的实力。
<br/>　　特别如今是在这冰海当中。
<br/>　　吞灵冰鲸的实力，将更加强悍。
<br/>　　“哈哈哈哈，你们完蛋了。”风花月等人阴冷大笑。
<br/>　　“你那些精纯的寒冰力量，不仅吸引了大量吞灵冰鱼。”
<br/>　　“连吞灵冰鲸这样的恐怖家伙都吸引了过来。”
