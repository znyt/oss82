夏夕绾没有想到她跟柳招娣第一次交锋，柳招娣竟然来势这么凶猛，完全杀的她猝手不及。
<br/>
<br/>        不过，等她找到叶翎，缓过神，她一定会找柳招娣算账的！
<br/>
<br/>        ……
<br/>
<br/>        跟夏夕绾挂断通话后，花姐迅速拨出了顾夜瑾的手机号，她有顾夜瑾的私人手机号，可以畅通无阻。
<br/>
<br/>        悠扬的手机铃声响了一遍，那端的电话就被接通了，顾夜瑾低醇悦耳的嗓音徐徐传递了过来，“喂。”
<br/>
<br/>        “顾总，不好了，翎翎出事！”花姐迅速道。
<br/>
<br/>        此时在三万英尺的国外，六星级总统套房的落地窗前，顾夜瑾刚结束一场商务会议回到了房间里，单手捏着手机，单手扯了一下脖间的领带，这时就听到花姐说叶翎出事了。
<br/>
<br/>        总统套房里的私人秘书当即就感觉整个房间里的温度骤然降到了冰点，他吓得屏住了呼吸。
<br/>
<br/>        顾夜瑾扯领带的动作顿了一下，薄凉的唇瓣快速掀动，“出什么事了？”
<br/>
<br/>        花姐将事情的大概说给顾夜瑾听，又将地址发送了过来，顾夜瑾俊美如玉的面容上没什么情绪起伏，不过看着十分阴森，他淡漠道，“我知道了。”
<br/>
<br/>        丢下了这么一句，顾夜瑾直接挂断了电话。
<br/>
<br/>        他转过身，清寒的黑眸看向身后的私人秘书，将地址递过去，“现在就去查这个地址，还有取消一切商务行程，准备专机，我要飞回帝都。”
<br/>
<br/>        “是，总裁。”
<br/>
<br/>        私人秘书以最快的速度做好了这一切，然后将查到的消息汇报上去，“总裁，这个地址是一个私人奢.淫酒会。”
<br/>
<br/>        顾夜瑾俊美的眉心一蹙，清寒的黑眸如渲染开的墨汁，浓稠的深不见底，他拿出手机，拨出了一个号码。
<br/>
<br/>        很快，那端接通了，陆寒霆低沉磁性的嗓音传递了过来，“喂。”
<br/>
<br/>        顾夜瑾捏着手机，“你那个后妈这些年守活寡，是不是已经神志不清了，敢挑我的人动手？”
<br/>
<br/>        “叶翎怎么了？”陆寒霆问。
<br/>
<br/>        “叶翎被送到私人酒会上去了。”
<br/>
<br/>        陆寒霆和顾夜瑾都是簪缨贵胄之家出身的太子爷，他们没参加过这类的私人酒会，但是该懂的都懂，“…看来她最近是挺闲的，想找点事做了。”
<br/>
<br/>        “你也别闲着，我飞回去要两个小时，你先赶去私人酒会。”
<br/>
<br/>        “顾夜瑾，这就是你找我帮忙的态度？”
<br/>
<br/>        “哦，忘了告诉你，夏夕绾已经去了。”
<br/>
<br/>        那端的陆寒霆直接默了两秒，然后“嘟嘟”两声，他把电话给挂了。
<br/>
<br/>        ……
<br/>
<br/>        夏夕绾到了，这是帝都的郊外，这里坐落着一个几千平的私人豪华酒庄，现在酒庄外面停满了各种豪车，酒庄里面灯火通明，看着就纸醉金迷。
<br/>
<br/>        叶翎就在里面。
<br/>
<br/>        夏夕绾走上前，有侍者在那里发面具，来往所有的人都戴着面具。
<br/>
<br/>        “美女，你想要什么样的面具？”侍者问。
<br/>
<br/>        夏夕绾看了一下，面具有很多种类，小狐狸的，小兔子的，火焰的，妖魔鬼怪的，什么都有。
<br/>
<br/>        “拿什么面具都可以？”
<br/>
<br/>        “都可以，不过，这男士面具和女士面具都是一样的，如果你跟某个男士挑选了一样的面具，那你们就是主仆关系，男为主，女为仆，主人可以要求仆人做一样事情，什么事情都可以，不可以拒绝。”
