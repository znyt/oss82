　　在所有人的注视之下，苏莫远离了演武场，来到了殿主塔楼之前。
<br/>　　“来顶层！”
<br/>　　塔楼的大门大开，殿主的声音，在苏莫的耳畔响起。
<br/>　　苏莫沉默不语，进入了塔楼之中，顺着其内的楼梯，向顶层而去。
<br/>　　片刻之后，苏莫来到了塔楼的顶层，也就是第十三层。
<br/>　　“殿主，我来了！”
<br/>　　眼前是一扇石门，苏莫微微抱拳，面向石门轻声说道。
<br/>　　“进来吧！”
<br/>　　殿主的声音再次响起，随即石门缓缓向一侧移开了。
<br/>　　映入眼帘的是一座房间，朴实无华，房间中央有一张白玉打造的玉桌，一身锦袍、面色刚毅的殿主正端坐在玉桌之前。
<br/>　　“殿主！”苏莫来到了殿主的身前，面色郑重，不过，他也没有太多的担心，毕竟对方对他不利的可能性非常低。
<br/>　　“坐吧！”殿主抬了抬手，示意苏莫在他身前坐下。
<br/>　　“谢殿主！”苏莫点了点头，然后便端坐在了玉桌前。
<br/>　　殿主沉吟了一番，道：“上官昊，如果本座没有看错的话，你不仅修炼了混沌神帝的道统，还拥有不败战气元神，以及万化神剑元神！”
<br/>　　“嗯！”苏莫点了点头，大方的承认了，现如今，他也没有继续隐瞒的必要了。
<br/>　　“不错，你的天赋在神域，也足以被称为是妖孽了！”殿主微微颌首，苏莫的不败战气元神，他感知到那澎湃的战意，便一清二楚。
<br/>　　至于万化神剑元神，他之前并不能肯定，毕竟，剑类的元神实在太多了。
<br/>　　只不过，苏莫的剑威太强，他才猜测，是万化神剑元神。
<br/>　　除了万化神剑元神，他想不出还有那种剑类元神，可以让苏莫拥有如此强的剑魂之力。
<br/>　　“殿主，你让我来如此，不知所谓何事？”苏莫直接切入正题，他要离开千渊殿，不想再继续拖延了。
<br/>　　“你既然修炼了混沌神帝的道统，为何不去混沌神朝？”殿主沉声问道。
<br/>　　“我与混沌神帝并无关系，或许将来会去吧！”苏莫沉声道，不是他不想去，而是他不敢去。
<br/>　　帝一魂绝对知晓他修炼了混沌神帝的道统，说不定布下了天罗地网，贸然前去混沌神朝，很可能会自投罗网。
<br/>　　“嗯！”殿主点了点头，对此他倒是没有什么疑惑，毕竟下界与神域隔绝，下界武者得到机缘，修炼了混沌神帝的道统，并不一要去混沌神朝。
<br/>　　事实上，神域修炼混沌神帝道统的人，非常之多。
<br/>　　不仅是在混沌神朝，在各大神朝都有。
<br/>　　因为，混沌神帝的道统，经过无数万年的传承和分化，已经产生了大量的分支。
<br/>　　更有很多前辈大能，模仿混沌神帝的道统，穿创造出新的混沌武学。
<br/>　　现如今的神域，混沌之力的修炼之法，何止千百种？早已百花齐放。
<br/>　　当然，在神域，修炼混沌神帝最正统的大混沌五行神诀之人，那就相当少了，只有混沌神帝的各代传人。
<br/>　　至于眼前的上官昊，修炼的混沌道统，是正统还是分支，亦或者只是其他人模拟出的新武学，殿主就不得而知了。
<br/>　　气氛稍稍凝重，苏莫静等殿主开口，而殿主却是沉默了下来。
<br/>　　少倾，殿主仔细的打量了一眼苏莫，道：“上官昊，之前你和连霆战斗之时，本座感知到你的体内，还有一种奇异的元神，你三生元神？”
<br/>　　苏莫闻言，顿时心中一惊，看来是吞噬元神被发现了。
<br/>　　他的天梦元神在吞噬元神之中，不可能被发现，那么被发现的肯定是吞噬元神。
<br/>　　而他在之前的战斗中，并没有动用吞噬元神，只是动用不败战气元神之时，稍稍催动吞噬元神，将不败战气元神从吞噬漩涡之中吐出来。
<br/>　　这一切，之时在电光火石之间，居然就被发现了。
<br/>　　苏莫面无表情，他不知道是否要承认，若是承认了，对方会不会追问到底？
<br/>　　“是！”最终，苏莫还是点了点头，承认了还有第三种元神。
<br/>　　他觉得，坦白或许比死不承认更好。
<br/>　　“哦？是什么元神？”殿主闻言，眸中精光一闪，心中惊讶不已。
<br/>　　这个上官昊的天赋，未免也太惊人了吧？
<br/>　　混沌道统、三生元神、而且其中两种元神还是强大的万化神剑和不败战气。
<br/>　　如此天赋，整个神域都极其少见，堪称绝世妖孽了。
<br/>　　不过，他也发现了一些问题，比如，这个上官昊的战力，全部来自于混沌之力、战魂之力和血脉之力。
<br/>　　至于奥义、奥义武学，几乎没有。
<br/>　　当然，这也可能是没有动用。
<br/>　　“吞天圣魂元神！”苏莫沉声道，他不可能告诉对方自己吞噬元神的底细。
<br/>　　“吞天圣魂！”殿主闻言，心中大震，居然又是一种太古武魂榜排名前二十的武魂！
<br/>　　殿主是彻底的震惊了，他相信苏莫不是吹嘘，因为以苏莫低调且隐藏天赋的心思，不可能故意夸大。
<br/>　　但是，这也未免太过骇人了。
<br/>　　三生元神的妖孽，神域不是没有，而且还不在少数。
<br/>　　但是，三生元神全部都是太古武魂榜排名前二十的存在，这就有点骇人了。
<br/>　　即便殿主修为高深，此刻也不禁心神剧震，掀起滔天巨浪。
<br/>　　少倾，殿主压下心中的惊骇，道：“上官昊，你有如此惊人天赋，我带你去总殿，举荐你进入大夏神王府如何？”
<br/>　　“大夏神王府？”苏莫闻言一怔，随即心中不禁暗暗叹息，不管是什么府，他如今都不能留下来。
<br/>　　“殿主，此事以后再说吧，我想外出历练几年，领略一番神域大地的风采！”苏莫沉声说道。
<br/>　　“外出历练？”殿主闻言，稍稍沉默，外出历练这是很正常的事情。
<br/>　　只不过，他心中清楚，苏莫若是外出历练，怕是就不会再回来了。
<br/>　　如此妖孽的天赋，无论是去哪个界域、哪个神朝，都会受到疯狂的招揽。
<br/>　　“你准备去哪里历练？”殿主沉声问道。
<br/>　　“不一定，走到哪是哪，过些年再回来！”苏莫道。
<br/>　　“好吧，本座不拦你，你可以去找自己的机缘！”
<br/>　　殿主点了点头，继续道：“不过，天九殿会给你留着，本殿主希望你尽快回来！”
<br/>　　“多谢殿主！”苏莫急忙起身，向殿主抱拳行礼，郑重的道：“我是千渊殿的弟子，永远都是！”
<br/>　　对于千渊殿，苏莫虽然没有什么感情，但这位殿主，却也算是帮了他。
<br/>　　没有天九殿，他修为不会提升的这么快。
<br/>　　“嗯！你有此心最好！”
<br/>　　殿主微微颌首，笑着道：“本座夏千渊，期待你历练归来的那一日。”
<br/>　　“我会的！”苏莫郑重的说道。
<br/>　　“你对神域不熟，本座推荐你去几个地方如何？”殿主夏千渊笑着说道。
<br/>　　“请殿主告知！”苏莫道。
<br/>　　“混沌神朝的混沌神山、天池神朝的时空界域、以及终极剑域。”夏千渊道。
<br/>　　“这个地方有何不凡之处？”苏莫好奇的问道，既然对方推荐，那这三个地方定然不简单。
<br/>　　“混沌神山，自然对你修炼混沌之力有极大的帮助，至于时空界域和终极剑域，可以让你在大道规则和剑道之上，有飞速的提升！”夏千渊道。
<br/>　　“原来如此！”苏莫点了点头，面现沉吟之色，他本来想问问对方，可知道绫裳的情况。
<br/>　　但是，很快他就放弃了，他要去哪里，绝对不能泄露。
<br/>　　因为这一次离开，他会先去找绫裳。
<br/>　　“殿主，既然如此，那弟子就走了！”苏莫再次抱拳，恭敬的说道。
<br/>　　“去吧！”夏千渊没有再言挥了挥手。
<br/>　　“告辞！”苏莫再次一拜，转身离开了房间。
<br/>　　望着苏莫离去的背影，夏千渊微微叹了口气，对于眼前的青年，他是真的看重。
<br/>　　但是，这个上官昊如此的妖孽，又修炼了混沌神帝的道统，迟早要加入混沌神朝的。
<br/>　　所以，他没有强行挽留，也没有这个必要。
<br/>　　“此子，堪比那些万古天骄！”
<br/>　　夏千渊低声自喃，在神域大地，能百岁成神之人，都是厉害的天才人物。
<br/>　　而这些天才人物之中，最顶尖的那一小撮人，便是万古天骄，天赋冠绝神域，惊艳九大神朝。
<br/>　　而这个上官昊，并不比那些万古天骄差，差的只是出生而已。
