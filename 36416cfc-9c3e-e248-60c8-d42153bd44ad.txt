　　“呼。”萧逸深呼吸一口气。
<br/>　　无论如何，他现在已经掌握了一丝霸道剑道的天地武道力量了。
<br/>　　虽然只有不足半成。
<br/>　　但给他的实力增幅，已经相当强大。
<br/>　　以前是借用，现在是真正自己施展，两者的威力，自是不同。
<br/>　　萧逸看了眼那汪水泉。
<br/>　　虽里头的寒冰之髓还有很多。
<br/>　　但萧逸并不认为这足够自己填满小世界。
<br/>　　毕竟，寒冰之髓对武者的效果，会逐渐减弱。
<br/>　　此时，萧逸从脑海中提取了一些信息。
<br/>　　吸收了冰尊者的记忆后，萧逸得到了一门功法。
<br/>　　功法名叫‘冰尊圣纹诀’，由冰尊者所创。
<br/>　　这是一门功法，却也是秘法。
<br/>　　这门功法的品阶，冰尊者的记忆中，并未阐明。
<br/>　　但萧逸对比了一下，大概和九阳轮回诀是一个层次。
<br/>　　九阳轮回诀，本是裂天剑魔前辈从一个古老遗迹中所得。
<br/>　　毫无疑问，冰尊圣纹诀，绝对极其强大。
<br/>　　萧逸将功法认真看了一遍，以他的领悟力，不过多时，便已参悟。
<br/>　　而此时，萧逸的脸上，已经写满了惊喜。
<br/>　　“好厉害的功法。”萧逸一惊。
<br/>　　冰尊圣纹诀，以武道力量为基，庞大力量为源，在身上烙印下‘冰纹’。
<br/>　　冰纹成，鬼神惊，威力滔天。
<br/>　　此时，萧逸微微思索了一下，算了算时间。
<br/>　　冰尊圣纹诀，修炼起来不会太难。
<br/>　　起码对他而言是这样的。
<br/>　　没有犹豫，萧逸立刻开始了修炼。
<br/>　　“吸。”萧逸低喝一声。
<br/>　　冰尊圣纹诀修炼的第一步，乃是凝聚出‘冰纹’基石。
<br/>　　这个基石，由武道力量而成。
<br/>　　不过必须是寒冰一道的武道。
<br/>　　萧逸第一时间想到的，自然是寒冰剑道。
<br/>　　毕竟，这是现今自己除了霸道剑道以外的最强剑道。
<br/>　　如同之前吸摄霸道剑道的天地力量那般。
<br/>　　天地间的寒冰剑道力量，正在快速涌向萧逸。
<br/>　　不过萧逸并未吸入体内，而是将之凝聚于手臂上。
<br/>　　按照冰尊圣纹诀的修炼办法，萧逸精准地控制着这些天地力量，在手臂上烙印着一个玄妙的图案。
<br/>　　半个时辰后。
<br/>　　萧逸停下了吸收。
<br/>　　而此时，手臂上，已多了一个散发着白色光芒的奇妙图案。
<br/>　　这个图案，便是‘冰纹’基石。
<br/>　　接下来，便是为这个基石灌输力量。
<br/>　　“吸。”萧逸再次低喝一声。
<br/>　　这次，吸收的是水泉内的寒冰之髓。
<br/>　　两大武魂齐开下，寒冰之髓的吸收速度，相当惊人。
<br/>　　当然，这些寒冰之髓，并未吸入体内。
<br/>　　而是吸入手臂中的冰纹基石中。
<br/>　　水泉内的寒冰之髓，几乎是以肉眼可见的速度，不断减少着。
<br/>　　待得吸收了近万斤寒冰之髓时。
<br/>　　手臂上那个冰纹图案，陡然间散发出刺眼的白光。
<br/>　　这是冰纹即将成型的征兆。
<br/>　　又是两千斤寒冰之髓被吸入图案。
<br/>　　‘冰纹’彻底成型。
<br/>　　这也代表着，冰尊圣纹诀已练成。
<br/>　　事实上，若是其他人修炼冰尊圣纹诀，绝不需要这么多寒冰之髓。
<br/>　　只不过，‘冰纹’中蕴含的力量，与气泉挂钩。
<br/>　　故萧逸修炼起来，需要的力量特别多。
<br/>　　这也是冰尊圣纹诀厉害的一点，冰纹，简直就是武者的另一个气泉。
<br/>　　只不过，萧逸气泉内的元力，是极冰、极热两种元力。
<br/>　　而此时冰纹内，只有极冰一种力量。
<br/>　　毕竟是有寒冰剑道为基石凝聚而成的。
<br/>　　另外一点，便是‘冰纹’中的力量，可一瞬间爆发，用于战斗。
<br/>　　这一点，相当恐怖，亦是萧逸为之惊骇的地方。
<br/>　　正常来说，武者，绝不可能一瞬间爆发体内气泉力量用于战斗。
<br/>　　肉体、经脉等等，会承受不住。
<br/>　　哪怕是肉体过人的萧逸也一样。
<br/>　　以往他调动‘冰山火海’的力量，虽曾经试过压缩力量。
<br/>　　但那种压缩，是以释放武技或者攻击的形式。
<br/>　　比如以前的‘紫炎压缩’，体内元力已经释放出来了，化为紫炎了。
<br/>　　他真正压缩的，只是紫炎。
<br/>　　而并非一瞬间让如此庞大的元力在自己体内爆发。
<br/>　　但，冰尊圣纹诀，却能无视这一点。
<br/>　　‘冰纹’，以武道力量的方式，调动里头庞大的力量。
<br/>　　让得使用者可以无任何代价或者反噬。
<br/>　　“了不起，当真了不起。”萧逸不禁赞叹着。
<br/>　　这门功法的创造者，确实厉害。
<br/>　　本身功法的玄妙，竟能契合天地武道，以让得武者可以一瞬间爆发里头力量。
<br/>　　而不会有任何代价。
<br/>　　当年的冰尊者，绝对是个惊天动地的武道强者。
<br/>　　“隐。”萧逸低喝了一声。
<br/>　　手臂上的冰纹，瞬间消失。
<br/>　　如今，冰纹已成。
<br/>　　不用时，可覆盖于皮肤之下。
<br/>　　萧逸已经忍不住想要试试‘冰纹’的威力了。
<br/>　　等同于自己三千三百三十三丈庞大气泉的力量，一瞬间爆发。
<br/>　　战力，将会飙升到何种程度？
<br/>　　另外一点，冰尊圣纹诀，共可凝聚九道冰纹，威力层层叠加。
<br/>　　现今这第一个冰纹，代表的是寒冰剑道的冰纹。
<br/>　　而第二个、第三个，乃至之后的冰纹，需要新的寒冰武道。
<br/>　　萧逸手头上掌握的寒冰武道，并不多。
<br/>　　风雪剑主的风雪一道，暴雪庄主的暴雪一道，都是寒冰武道的范畴。
<br/>　　只不过，在萧逸看来，这两种寒冰武道，太弱，远远比不上寒冰剑道。
<br/>　　冰纹的威力，也与这种武道的强弱有关。
<br/>　　萧逸并不想让这些不够强的寒冰武道，占据了自己冰纹的数量。
<br/>　　毕竟，每凝聚一个冰纹，需要的力量与自身气泉挂钩，代价太大。
<br/>　　值得一提的是，寒冰之髓，对武者的效果，会渐渐变弱。
<br/>　　但在凝聚冰纹时，却并没有。
<br/>　　毕竟，冰纹是一个基石，只是为其填充力量。
<br/>　　寒冰之髓拥有多少力量，便为其填充多少力量。
<br/>　　“是时候离去了。”萧逸双眼一眯，站了起来。
<br/>　　如今，修为已突破天元七重，掌控霸道剑道的天地力量。
<br/>　　又修炼了冰尊圣纹诀，凝聚了第一道冰纹。
<br/>　　此次收获极大，实力更是暴增。
<br/>　　萧逸走到洞穴前。
<br/>　　洞穴外，那凛冽的罡风，无尽的幽暗，让人心头颤栗。
<br/>　　那些罡风，便如黑暗中的幽魂，呜呜作响，阴森恐怖。
<br/>　　嗖…萧逸身影一闪，飞跃而出，打算回到裂缝之上。
<br/>　　不过，他的身影，才刚刚离开洞穴不远。
<br/>　　一阵猛烈的吸力，陡然间将他吸摄，再次将他吸回洞中。
<br/>　　踏…踏…踏…
<br/>　　在那股吸力之下，萧逸在洞中足足退了十数步才稳下身影。
<br/>　　“该死。”萧逸脸色大变。
<br/>　　他忘了，这个洞穴，便是极寒裂缝中所有凛冽罡风的‘风眼’位置。
<br/>　　洞穴之外，方圆百米之内，皆是风眼的覆盖范围。
<br/>　　这个范围的吸力，极其恐怖。
<br/>　　当初萧逸便是被这股吸力吸入动中。
<br/>　　无数年来，死在洞穴中的这些强者，当初亦是被这股吸力吸来，再无法逃出升天。
<br/>　　“给我破。”萧逸大喝一声。
<br/>　　蕴含天地力量，以及澎湃元力的寒冰裂天斩，瞬间从暴雪剑中劈出。
<br/>　　前方无尽罡风，瞬间破散。
<br/>　　萧逸连忙飞跃而出。
<br/>　　但下一秒，萧逸以同样的速度，被吸回洞中。
<br/>　　“不对劲。”萧逸双眼一眯，脸色凝重无比。
