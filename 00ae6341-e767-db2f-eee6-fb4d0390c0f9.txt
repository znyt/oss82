
<br/>　　<br/>　　    奔驰小跑里的胡胡掐着手机，屏幕上显示着三个数字“110”，手指按在了播出键上：“这他么的，一字马和小兔子还没享受到呢，我是真有点不太甘心啊，小遥遥你胡哥不能亲自出手救你，那就得借警察叔叔的手了，我得为这床友的关系，尽一份力吧”<br/>　　<br/>　　    胡胡肯定不能出面，但他还想把人救出来，这时候最合适的，就肯定是有事找警察了，借警方的手绝对不费劲就能把人给救出来。<br/>　　<br/>　　    报警电话还没有播出的时候，奔驰小跑后面一束车灯透了过来，胡胡手指往回缩了一下，眯着眼睛回头就看见一辆车子开了过来，然后转到了院里。<br/>　　<br/>　　    胡胡寻思了一下，手就抽了回来，下了车朝着那边走去，探着脑袋往里面看了一眼，正好瞧见司机来到后面拉开车门，沈君从车上下来了。<br/>　　<br/>　　    胡胡当即就一愣，沈君他不认识但以前曾经见到过一次，知道对方长什么样。<br/>　　<br/>　　    “事漏了？”胡胡皱眉嘀咕了一句。<br/>　　<br/>　　    本来，他还以为刘珺瑶被绑走了是因为其他什么事，但这时候看见沈君从车里下来，他就知道这是给沈沉鱼下灵媒的事漏了，沈君顺藤摸瓜抓到了刘珺瑶，这么一来的话很明显就把胡胡也给牵扯了进来。<br/>　　<br/>　　    “有点麻烦了啊……”胡胡烦躁的叹了口气，他本以为这事做的神不知鬼不觉的，现在来看明显是要露馅的节奏了。<br/>　　<br/>　　    面对一个袍哥大佬，胡胡是很不愿意跟对方结仇的，沈君在蜀地的能量很大，关系网非常的复杂，基本上你在蜀地要是跟他结上了仇，那就意味着后患无穷了。<br/>　　<br/>　　    “沈君身边还有这样的人？居然还能发现沈沉鱼被接了灵媒，也没听说袍哥里有这种高手啊，这他么的……为了胯下那点事，招惹上这么个麻烦，是真不划算啊”胡胡一脸的惆怅，这个炮约的约出麻烦来了。<br/>　　<br/>　　    仓库里面，沈君手插着口袋进来，站到刘珺瑶的面前，皱眉说道：“小鱼儿的朋友不多，你之前肯定算一个，但我挺想不通的是，你为什么背后非得捅她一刀把她往悬崖下面推呢，你既然是她的朋友，那就肯定知道小鱼儿有个哥哥叫沈君，你还敢这么干的话，胆子真不小啊”<br/>　　<br/>　　    刘珺瑶惊恐的咬着嘴唇，连连摇头说道：“不，不是我，我没有，你误会了，我没对小鱼儿怎么样”<br/>　　<br/>　　    沈君叹了口气，揉了揉脑袋说道：“我既然都已经让人把你带过来了，你还有什么可狡辩的？你可能不太了解我的风格，就是碰到麻烦了，我不一定非得要有什么确凿的证据就对你下手，只要怀疑就可以了，你听明白了？我不需要人证物证都指向你，仅仅是怀疑就行了”<br/>　　<br/>　　    刘珺瑶突然就哭了，往后退了几步，咬着嘴唇说道：“不是的，你听我解释……”<br/>　　<br/>　　    “我现在还给你说话的机会，就是想问问，是谁帮你对付小鱼儿的，你自己肯定没有这个能力，背后得有人给你帮忙才是，说吧，这人是谁啊”<br/>　　<br/>　　    “不能说，我不能说的，求求你放了我吧，我再也不敢了，以后见到小鱼儿我都绕着走，我再也不敢对她干什么了”<br/>　　<br/>　　    沈君抿了抿嘴，抬头跟两个马仔说道：“她有点不太听话，那这样吧，用你们的方式帮她想想，自己到底能不能开口”<br/>　　<br/>　　    马仔顿时呲牙就笑了，掏出手机调出摄像，交给另外一个人说道：“91论坛最近挺火的，拍个片咱也卖过去……当红女明星的圈圈叉叉，这个标题打出来，绝对吸引眼球”<br/>　　<br/>　　    “你完事了，我接棒！”<br/>　　<br/>　　    刘珺瑶当即就傻了，哭喊着说道：“不要，不要，我说，我说还不行么”<br/>　　<br/>　　    这一招对她来说绝对管用，刘珺瑶绝对不会怀疑这帮人有没有胆子敢这么干，本身袍哥的性质就摆在这里呢。<br/>　　<br/>　　    在她之前，有多少前辈倒在了这上面，当年港岛姓刘的那个女明星跟她现在的状况如出一辙，这都过去快二十年了，有的时候那件事还会被翻出来呢，想洗都洗不干净。<br/>　　<br/>　　    沈君摆了摆手，说道：“让她说吧”<br/>　　<br/>　　    “姓胡，他叫胡胡，我之前忌恨小鱼儿，就跟胡胡提了一下有没有什么办法对付她，他就说交给他来办就行了，没过几天胡胡就给了我一个吊坠，告诉我说把这个东西给小鱼儿，只要她戴在身上就可以了……”<br/>　　<br/>　　    “胡胡，什么人？”沈君皱眉问道。<br/>　　<br/>　　    这个时候，仓库门外，两台警车开了进来。<br/>　　<br/>　　    “咣当，咣当”几个警察推开车门，直接就奔着仓库闯了进来。<br/>　　<br/>　　    沈君回头，看见警察来了就一愣，没想到警方的动作这么快，刘珺瑶失踪前后也不过就一个小时左右，居然就找过来了。<br/>　　<br/>　　    警方带队的人进到仓库里后，看见沈君也愣了，他在蜀地本来就属于新闻人物，有太多的警察跟他打过交道有过接触了。<br/>　　<br/>　　    “沈老板？”带队警察看了他一眼，然后望向刘珺瑶说道：“有人报警，说有个绑架案，你别告诉我以你的身份和地位，还会干这种事呢？”<br/>　　<br/>　　    沈君摆了摆手，淡淡的说道：“这是有人跟你们开玩笑呢吧？扯淡，我能绑架谁啊，我就是找这位小姐过来问问话，是不有点误会了啊？”<br/>　　<br/>　　    “不是那就最好了，现在扫黑除恶正在关键时刻呢”带队的警察冲着刘珺瑶问道：“小姐，是这么回事嘛？”<br/>　　<br/>　　    刘珺瑶见到警察来，揪着的心顿时就放下了，她看着表情平淡的沈君，沉默了半晌后，点头说道：“是的，沈先生就是过来找我问问，没有人绑架我”<br/>　　<br/>　　    刘珺瑶肯定不敢指证沈君绑架他，她也不傻，人又不是他绑的，警方最多只能把他的马仔给抓了，对于沈君本人，一点影响都没有。<br/>　　<br/>　　    “现在问完了么？”警察问了一嘴。<br/>　　<br/>　　    警方的人心里也明镜着呢，他们一看就知道是怎么回事了，但当事人不追究，他们也不会拿沈君硬办。<br/>　　<br/>　　    “问完了”刘珺瑶点头说道。<br/>　　<br/>　　    带队的警察看着沈君说道：“下次你问谁话的时候，别搞得跟拍电影似的，换个方式吧……”<br/>　　<br/>　　    沈君笑了笑，说道：“不好意思，给你们添麻烦了”
