        &lt;h3&gt;第三卷第一百二十七章  悄然改变&lt;/h3&gt;
<br/>
<br/>        宁雅这几天有些心神不宁。
<br/>
<br/>        文件发给陈子昂的时候错了好几个地方，陈子昂将宁雅叫到了办公室。
<br/>
<br/>        “这几天有心思吗？”陈子昂问宁雅。
<br/>
<br/>        “没，没有。”宁雅总是集中不起来精神，她的脑海里总是那个接电话女孩的声音。
<br/>
<br/>        那个是凌辰的女朋友。
<br/>
<br/>        原来凌辰都有女朋友了，那么这几年给她发的信息又怎么说？
<br/>
<br/>        他们也会见面，只是见面的时候总是客客气气的，宁雅一直以为凌辰是爱她的，要不然怎么会说哪样的话?要不然怎么会接听她的电话，哄着她入睡？
<br/>
<br/>        “最近这几天的工作态度好像不是你的风格，这些文件我过的时候会发现一些问题，你还是拿回去仔细看看吧。”陈子昂打回了她提交的文件。
<br/>
<br/>        她问了宁雅，既然宁雅不愿意说，作为上司，她也不好多说什么，但是工作还是工作，无论你有天大的事情，既然你在工作中，那就一定要做好本职工作。
<br/>
<br/>        宁雅叹了口气，转身离开，她知道她不能这样，可是思想总是会抛锚。
<br/>
<br/>        其实，陈子昂也发现了宁雅的状态不好，不光光是工作上，有时候走路都会撞到人而不自觉，这样的状态是不行，况且宁雅还开车。
<br/>
<br/>        她决定下班找宁雅聊一聊。
<br/>
<br/>        宁雅回到办公室，努力的集中精神，把手中的工作重新做了一遍，拿去给陈子昂的时候，陈子昂看着她，将文件又看了一遍，皱着眉头叫来了另外的一个人：
<br/>
<br/>        “小吕，你把这个文件拿下去重新过一遍，一个字都不准出差错。”
<br/>
<br/>        “是。”小吕接过文件，转身回到办公室，嘴里嘀咕着：“宁主任不知道怎么回事，这一段时间心神不宁的，怕是要挨批评了。”
<br/>
<br/>        “下午有事情吗？”陈子昂问宁雅，宁雅虽然化着妆也掩盖不了苍白的脸色，似乎没有休息好。
<br/>
<br/>        这种状态和前一段满脸水润的宁雅相差太远了。
<br/>
<br/>        “啊，没事情。”宁雅想着陈子昂是不是会教训她，毕竟她最近频频出错，没有想到陈子昂会问她下午有没有事情。
<br/>
<br/>        “没事情，下午一起吃饭，我请你。”陈子昂说完看都不看宁雅，又盯着电脑。
<br/>
<br/>        宁雅看着陈子昂再也不理她，便退出了陈子昂的办公室，心里嘀咕着：“都没有问人家同不同意，直接下命令。”
<br/>
<br/>        虽然宁雅心里嘀咕着，但是毕竟陈子昂是她的上司，不同意也得同意。
<br/>
<br/>        宁雅这几天回家就喝酒，她的酒量太大，怎么都喝不醉，如果喝醉了倒头就睡也好啊，可是一个人喝酒越喝越清醒，想到凌辰有女朋友了她就很难过，就想哭。
<br/>
<br/>        原来还可以打电话给凌辰，现在人家都有女朋友了她还能打电话吗？
<br/>
<br/>        凌辰那个混蛋地地道道的就是个骗子，骗了她这么几年，难怪他从来都没有说过：他爱她的话，说的那些情话就是搞暖昧，她可能就是个备胎，不，连备胎都不是。
<br/>
<br/>        那个冷血冷情的人！
<br/>
<br/>        从陈子昂的办公室出来，想到她好歹是宁家长女，为何就沦落到现在这种地步？
<br/>
<br/>        在别人的公司委曲求全，被“男朋友”劈腿，虽然他们都没有挑明他们是男女朋友关系，可是宁雅在心里已经当凌辰是男朋友了啊，她一直在等着他向她表白。
<br/>
<br/>        虽然两家的实力差距有些悬殊，但是她可以等他啊。
<br/>
<br/>        宁雅的鼻子发酸，眼眶发涩，可是这里是在公司，她低着头转身去了卫生间，镜子里是苍白着脸眼眶红红的自己。
<br/>
<br/>        努力的平复了一下自己的心情，装着若无其事的样子走到办公桌坐下，努力的集中注意力。
<br/>
<br/>        有时候，她想打电话给凌辰，可是又不想掉份，打电话怎么对凌辰说：“你有女朋友了？你女朋友接我电话了。”
<br/>
<br/>        她做不到！
<br/>
<br/>        下班，陈子昂拿着车钥匙站在她们办公室门口看着宁雅：“你不要开车了，我开车。”
<br/>
<br/>        “好吧。”宁雅看出来陈子昂有话要和她说，刚好她也想和她聊一聊。
<br/>
<br/>        陈子昂带着宁雅到了一个小店，虽说是小店，但是环境真的很好，每个座位都是独立的空间，说话也互不干扰，饭店的装修宁静而古典，一走进去，人突然就感觉静下来了，里面座位基本都有人，但是却没有一点喧哗的声音。
<br/>
<br/>        在北京能有这样的一个宁静的港湾真的不错呢。
<br/>
<br/>        “这个是要提前定位置的，不过我不用定。”陈子昂对宁雅微微一笑，可是笑容有点苦涩。
<br/>
<br/>        “之前和一个朋友经常来这里吃饭，都吃不腻。”陈子昂想起了陆宇霆，他们已经好多年都没有联系了。
<br/>
<br/>        这个饭店就是经常和陆宇霆一起来了，看着这么隐秘而有安宁的环境，陈子昂突然就明白了陆宇霆当初的用心。
<br/>
<br/>        陆宇霆怕别人发现他和她在一起。
<br/>
<br/>        陈子昂苦笑了一下，人生啊！
<br/>
<br/>        “我和一个朋友以朋友的关系相处了两年，说对彼此没有感觉那是假话，曾经我还想着能不能嫁给他，只是造化弄人，我和他隔着跨不过去的鸿沟。”陈子昂将点菜单子递给宁雅，她已经给她自己点了一个素菜，一份鳕鱼。
<br/>
<br/>        宁雅看了一眼，她不能吃海鲜，所以她给她自己点了一份素菜，一份牛柳。
<br/>
<br/>        “所以，有些事情没有必要纠结，在我很小的时候，我的二嫂曾经告诉过我，爱情不是生活的全部，她要我无论面对什么样的状况都要好好的生活，可惜当初给我说这话的人，现在已经不是我的二嫂了。”陈子昂看着宁雅，人生很漫长，改变是必然的，我们都必须得接受改变。
<br/>
<br/>        “人生不是一成不变的，只看你自己怎么走。悲伤几天就够了，没有必要一直把自己困在悲伤里。”
<br/>
<br/>        “我.....”宁雅眼泪下来了，陈子昂递给她纸巾。
<br/>
<br/>        “凌辰有女朋友了，我心里有些难过。我一直等了他这么多年，我以为他的努力都是为了我，我看着他成长，等着他有一天去向我家里提亲，可是，却等来了他女朋友的电话。”
<br/>
<br/>        “为什么要等着他向你提亲呢？你从宁家出来两年了，为什么这两年不去争取呢？”陈子昂觉得还是需要提点一下宁雅。
<br/>
<br/>        宁雅无言以对，她是从宁家出来两年了，但是她毕竟还是宁家的女儿，婚姻之事她怕是做不了主。
<br/>
<br/>        “过去了就过去了吧，既然凌辰有了选择，你也没有必要再纠结了，活出你自己的样子才是真理。”陈子昂是破茧成蝶，她从一个山村里出来的，和宁雅这样天生含着金汤勺的人还是有差别的。
<br/>
<br/>        宁雅是骄傲的。
<br/>
<br/>        但是恒古不变的真理是：改变才是唯一出路。
<br/>
<br/>        “谢谢你，陈总！”宁雅极力的忍住泪。
<br/>
<br/>        “你之前提议我的事情，我想好了，我想自己干一番事业，你说的很对，人要活出自己的样子。我总是想着要凌辰去改变，却没有想到我要自己去改变。我要像你一样，掌握自己的人生。”宁雅梨花带雨的笑着。
<br/>
<br/>        让陈子昂看着有些心疼，她们都是女人，在我们五千年的历史文化中，女人始终是弱者，是附属品。想要摆脱这样观念和身份只有你自己加倍努力。
<br/>
<br/>        成长会痛苦一时，但是不成长会痛苦一生。
