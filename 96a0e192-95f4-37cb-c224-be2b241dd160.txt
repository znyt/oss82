　　第141章
<br/>　　“备轿。”
<br/>　　李如香唇中吐出两个字，将苏樱留下的两份字据收在袖子里。她今日与落叶城城主有约，时间是万万不可耽误的。
<br/>　　一乘小轿来到城主府前，通报过后，李如被城主府的仆人引到大厅。
<br/>　　约有一炷香的光景，落叶城城主方伯山才从外面走进来。
<br/>　　方伯山生得大腹便便，与其说是个城主，倒更像一个商人。
<br/>　　“方大人。”李如起身，深深道了个万福。
<br/>　　“苏夫人，坐，请坐。”方伯山道：“府里有些事，便耽搁了，害得苏夫人久等。对了，苏兄刚刚过世，按理夫人该在家中守孝，苏夫人怎么来了我这里？”
<br/>　　“哎，若不是有过不去的坎儿，我一介女流怎会顶着人们的流言蜚语来寻城主大人。”
<br/>　　“是出什么事了？”方伯山略倾身问。
<br/>　　“那程大雷作恶多端，如今被抓，也是城主英明，为民除害。可那群恶人却把事情怪到我头上，屡屡在家中捣乱。今日丢过来一把刀子，明天泼门上一桶血。”
<br/>　　方伯山微微撇嘴，他又不傻，丢刀子泼血，你把蛤蟆寨的山贼当混混么，山贼也是有尊严的好不好。
<br/>　　“我来就是想问一声，城主准备什么时候杀那恶贼，一了百了，杀了也就断了他们的念想。”
<br/>　　“这个嚒……”方伯山没准备杀程大雷，抓到程大雷是大功一件，报到上面去，自然是活着的要比死的值钱。而且，留个活口，万一蛤蟆寨的山贼发疯，这也是个要挟。
<br/>　　李如不懂方伯山的心理，她取出一个木盒，道：“除此之外，还有一件事，亡夫临死前，答应送给城主一颗夜明珠，这是亡夫生前吩咐过的，今日我特意给城主送来。”
<br/>　　“夜明珠，什么……”
<br/>　　方伯山还有些困惑，忽见那木盒打开，一枚拳头大小的宝珠发出柔和静谧的光芒。
<br/>　　方伯山顿时笑得合不拢嘴：“是这样，是这样，苏兄的为人真没得说，欠我的东西死之前还记得。”
<br/>　　“亡夫那样的为人，自然是和方大人这样重情义的人交朋友，那程大雷的事情……”
<br/>　　“放心，我会好好处理的。”方伯山大包大揽。
<br/>　　方伯山手捧着夜明珠爱不释手，见李如还坐在哪里，表情欲言又止。于是他问道：“苏夫人，是不是还有别的事？”
<br/>　　“哎，本来不该告诉城主的，一点家务事，也平白惹得城主不开心。”
<br/>　　“究竟是什么事？”方伯山已经开始觉得这女人有些烦了。
<br/>　　“还不是我那女儿苏樱，她和山贼交往已经令我苏家蒙羞。亡夫一死，更是没人能管住她。今日她寻到我，要用家产换三千两银子出去挥霍。那家产是亡夫一点一点打下的，本是等着她再年长几岁，便传给她。可看她现在的性情，我实在不放心把苏家交给她呀。”
<br/>　　“竟有这样的事情！”
<br/>　　“怎会有假。”李如叹口气：“我怕她不认账，便让她写了份收据，待会还要差人把银子给她送去。当娘的还能让孩子饿着不成么，可我怕的是今日三千两，明日五千两，回回找我要，我怎么忍心不给。但苏家抵不住她这样挥霍啊。”
<br/>　　方伯山将那收据看了一遍，顿时勃然大怒，手掌重重的一拍桌子。
<br/>　　“胡闹，这样的孩子，就该好好管。”
<br/>　　“亡夫去世后，虽敢管她啊，我毕竟是个后妈，说的重说得轻都不行。”
<br/>　　“我替你做主了，我不能眼瞅着看她把苏兄的家产败光，这份收据留在我这里，她再找你要钱，你就让她来找我！”
<br/>　　……
<br/>　　苏樱左等右等，等不来李如送来的银子，便又找到苏府。毫无疑问，又遭到一顿奚落，却一文钱都要不到。
<br/>　　她从苏家离开，凉风一吹，更感伤心。本想凑笔钱，救程大雷出来，如今却是竹篮打水一场空。
<br/>　　一个人形单影只走在路上，忍不住眼泪便落了下来。正在此时，有人在身后轻拍她的肩膀。
<br/>　　苏樱吓了一跳，回过头来，看到小白狼站在自己身后。
<br/>　　“嘘，跟我来。”
<br/>　　小白狼冲苏樱挥挥手，引着苏樱去往酒楼，路上苏樱就把自己的经历说了。
<br/>　　“我这几天一直暗中看着你，见你去苏家就想拦你，最后果然还是出事了。不过也没什么关系，等过去这一节，与苏家的账可以慢慢算。”
<br/>　　苏樱还不知道小白狼竟一直在保护自己，等她走进湖底捞更是吓了一跳。只见院子里挤着许多人，此刻磨刀的磨刀，喝酒的喝酒。
<br/>　　“你们竟来了这么多人？”
<br/>　　“嫂夫人莫怪，这件事我们不想把你们卷进来，所以也没在你面前露面。”秦蛮。
<br/>　　苏樱这几日伤心过度，心里想的是如何尽己所能，救出程大雷，却也没来湖底捞，不知道蛤蟆寨的人已经到了。
<br/>　　众人听到苏樱的经历后，颇是有几个人气得哇哇大叫，声言要血洗苏家。最后，还是被刘悲制止了，找李如解气可以，但也就暴露了众人存在。接下来想要救程大雷，便更加困难。
<br/>　　“那你们有什么办法救他？”
<br/>　　苏樱这个问题难住了众人，银子花出去无数，可救程大雷的办法还是没有。凭蛤蟆寨的兵力，却也打不破大牢。本想等着押送程大雷时动手，但方伯山那边也没动静，不知道他打算怎么处理程大雷。如今多一日，程大雷就多一分危险。
<br/>　　“其实也不能说没有办法。”苏樱。
<br/>　　众人的目光都落在她身上，众人一直将她当被保护的对象，从没人觉得她会有什么主意。
<br/>　　“这些日子我没出门，一直在家想怎么救他，有些办法根本行不通，我也是瞎想而已。”苏樱慢慢道：“但我觉得有个法子或许可行。”
<br/>　　“什么法子？”小白狼问道，但心里还是陪着苏樱说会儿话，安慰她的心理多些。
<br/>　　“他不能离开大牢，但方伯山却是会出来的，每月二十，他都会去码头捕鱼，如果能绑了，也许能用方伯山换回他。”
<br/>　　众人眼睛一亮，刘悲迫不及待问道：“今天什么日子？”
<br/>　　“十九。”有人回答。
<br/>　　二十就是明天。
