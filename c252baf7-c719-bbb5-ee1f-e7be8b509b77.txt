　　第213章
<br/>　　武将扬眉吐气，文官哗然，各自绞尽脑汁想如何反驳。
<br/>　　“呵，倒是有几分辩才。”宋天函干笑一声，发现自己可能小看了这长须浓眉的糙汉子。
<br/>　　“辩才么？”百里胜挑眉：“我帝国军人镇守边关，靠的是辩才么？不，靠的是这些破盔锈甲，靠的是七尺之躯，靠的是一腔热血。我也想问一句，刚才谁在笑？”
<br/>　　百里胜摊开双臂，做拥抱山河状。
<br/>　　孟中丞语结，其实刚才很多人都对程大雷发出嘲笑，不仅文官，武将也有。如今是武将同仇敌忾，文官摇头叹气，一副『你怎么可以这样子』的表情。
<br/>　　两大集团的交锋地，炮灰是可以牺牲的。
<br/>　　“很好笑么？”百里胜又问：“着破甲与戎族血战可笑？一腔忠义可笑？还是我帝国军人之荣耀可笑？”
<br/>　　“呃……”
<br/>　　百里胜心里那叫一个痛快，在文官和武将的争执中，武将从来输多胜少，很多次都被文官骂的狗血喷头。为此自己卧薪尝胆，苦读诗书，为的是什么，为的不就是像今天一样，也可以把文官集团骂得狗血喷头么。
<br/>　　如此，便越发觉得程大雷可爱。轻拍他的肩膀道：“牛校尉，我有一副盔甲，可以送给你，我不如你，不曾穿着它杀过戎族。我本想让你穿着崭新的盔甲上阵杀敌，可惜今日你的确有罪，我也无法说什么，只请你上刑场砍头时，穿着我的盔甲去，也以一个帝国军人的身份体面的死。”
<br/>　　无论程大雷还是孟中丞，都是炮灰，都可以牺牲。
<br/>　　宋天函内心咆哮，还特么不说什么，你丫说得够多了。这要真砍了牛三斤的脑袋，还不被将军府将牛三斤竖立成典型，说什么一个连盔甲都打不起，只念着杀敌的帝国英雄被砍头了。喔，你问他为什么被砍头，我也不知道，好像是一个叫宋天函的人定了他的罪。
<br/>　　这还不勾得民怨沸腾，以此来撬动文官集团的蛋糕。
<br/>　　“伯爷，严重了不是，不过几下推搡，何至于到砍头的地步，伯爷说笑。”
<br/>　　“推搡？我看他鼻子都流血了？”
<br/>　　“上火！天气这么燥，孟大人又年轻气盛，上火流鼻血不是很正常么。”
<br/>　　“那这件事就算了？”
<br/>　　不能杀了程大雷借题发挥，百里胜还是很遗憾的。
<br/>　　“本无所谓有事，接下来还有事情，诸位还是不要在这里耽搁。”
<br/>　　众人一起往里走，只留下头破血流的孟中丞立在原地。
<br/>　　程大雷是初来乍到，对于官场很多规矩都不懂，也不知道接下来还有什么事情，不是说报个到就可以走么。
<br/>　　穿过屋子，来到后院，程大雷和其他考官走进一座大殿，而徐神机和徐灵儿则被隔在外面。
<br/>　　“看这情况，是要考试么？”
<br/>　　程大雷猜的不错，的确是要考试。在这些考官负责监考之前，朝廷要先对这些考官来一次小武举，看看各人水平如何，也好确定各人在考官团体中的位置。
<br/>　　只是一次小考，自然不会考得太复杂，只考一项，也就是笔试。
<br/>　　程大雷被安排到座位上，观察考场的情形。这次武科举的考官一共一百零三位，其中一百位都在这个考场内，大家都坐在案前，互相离得很远，如此就杜绝了作弊的可能。
<br/>　　据说这次题目是明帝陛下亲自出的，考场里是太监监考。
<br/>　　程大雷坐在那里，有些挠头。扯淡他是不惧的，以他多年键盘侠的资质，又有跨时代的经验，即便在明帝面前，也能怼得他口吐白沫。
<br/>　　只不过笔试……实在不是程大雷擅长的，他的毛笔字一直没下功夫练，关键是原先在蛤蟆寨，他的文化程度已经可是说是傲视群雄，实在没必要再追求孤独求败，而程大雷又不太懂这个时代之乎者也的官样文章。
<br/>　　这要怎么蒙混过去呢，在这个时代卷面分可是很重要的，实在不行就交白卷。
<br/>　　其实程大雷不用太担忧，他的水平在文官里自然提鞋都不配，可是在武将里不说力拔头筹，至少也属于中上游。
<br/>　　这些带兵打仗的大老粗，不识字的有的是，一个个抓耳挠腮，头发成缕成缕的往下揪。
<br/>　　相比之下，文官队伍就很兴奋了，他们的本职工作就是咬文嚼字，真要扯起淡来能扯得你亲妈都不认识你。
<br/>　　他们一个个磨拳霍霍，誓要在这次考试中给武将一个教训，在本次考官队伍中，占得更重要的席位。
<br/>　　有太监将考卷发下来，程大雷已经抱定破罐子破摔的心态。忽然，就这时候，分发试卷的小太监轻轻踢了他一下。
<br/>　　程大雷下意识抬起头，只见这小太监生得眉清目秀，唇红齿白……我擦！
<br/>　　不是程大雷弯了，而是这太监乃是帝国明玉公主假扮的。
<br/>　　她胆子也真够大的，这要是被人认出来，以文官集团那张破嘴，非得把她的公主封号削了不可。
<br/>　　“这个……有标准答案没得？”程大雷低声问。
<br/>　　李婉儿白他一眼，假装没有听清楚。
<br/>　　原来不是来帮自己作弊，而是监工自己有没有卖力干活的，程大雷撇撇嘴，开始查看试卷上的题目。
<br/>　　题目很简单，简单到只有两个字：治兵。
<br/>　　最简单，却也是最复杂，不愧是明帝陛下的手笔，把每个人都搞得无从下笔。
<br/>　　既可以聊如何训练一支百人伍的小队，也可以谈一支十万人大军的控制，再往大了说，可以扯帝国军人的精神建设。
<br/>　　可大可小，可深可广，那究竟什么是正确答案呢，或者，什么才是明帝想要的正确答案。
<br/>　　武将挠头，文官踟蹰，考场内十分安静，只要长吁短叹的声音。
<br/>　　而关于这一点，程大雷是有发言权的。
<br/>　　他带着蛤蟆寨两千弟兄和戎族打了三个多月的仗，最后还打胜了，这至少说明，统领一支两千人的军队，程大雷是绝对没有问题的。而对于兵力调配，后勤供给，伤兵医治等等这些，程大雷都从实践中摸索出很多，也并非一窍不通。
<br/>　　问题是该如何将这些落在纸上。
<br/>　　程大雷想了想，忽然心中一动，饱蘸浓墨，在纸上落下几个墨字。
<br/>　　『浅谈亮剑精神』
<br/>
