　　“黑鹰”直升机位于山腰位置的山路上方下降，螺旋桨在泥石山路上刮起一阵沙尘暴，生长在山坡上的树木也是被吹得东倒西歪发出“沙沙”的声响。
<br/>　　直升机悬停在距离泥石山路二十米左右的高度，两根索降绳被扔下直升机，罗昊、李毅峰、李然、慕容飞四个人分别抓着两侧舱门垂下的索降绳滑降到地面，落地后，罗昊对着直升机中的机组成员竖起大拇指，“黑鹰”直升机立刻拉升高度，在空中调转方向离开。
<br/>　　漆黑的泥石山路上，只剩下罗昊、李毅峰、慕容飞、李然四个人。李然拉起围在脖子上的阿拉伯方巾遮住口鼻，打开多功能战术眼睛的夜视功能，眼前这片笼罩在黑夜之中的山林立刻变成了翠绿之色，山脚下的小村庄中有着点点明亮的发白的光点。
<br/>　　罗昊端着G36C自动步枪，眼睛盯着安装在战术导轨上的全息瞄准镜，打量了一下四周，听到有汽车引擎声从山上传来，立刻变得警惕起来，对着身边其他三个人沉声说道：“上山！”
<br/>　　四个人起身，快速进入山林之中隐藏了起来。罗昊、李然、李毅峰、慕容飞四个人刚进入山林隐蔽起来，就有两道明亮到让打开了多功能战术眼镜夜视功能的四个人感觉到刺眼的汽车灯光，扫进山林，令得隐藏在山林中的四个人不得不侧头避开。
<br/>　　很快，一辆车窗上安装了粗铁丝网，引擎盖、车门、车顶等位置加装了厚钢板，后车厢顶开了天窗，加装了一听M134转管机枪的改装SUV出现在泥石山路上，一名影子部队士兵控制着架设在车顶的M134转管机枪，警惕的打量着泥石山路周围。
<br/>　　“看来这就是情报上所说的机动巡逻队了。”罗昊看着远去的改装SUV，说道：“妈的，还真是武装到了牙齿，就这样的改装，轻武器短时间内根本不可能摧毁它。”
<br/>　　根据情报所示，在这条除了穿越山林唯一可以通往山顶别墅的泥石山路上有机动巡逻队在活动，一般都是一辆跟刚才四个人所见的那种改装SUV加上四到五名的影子部队士兵所构成，机动巡逻队的活动没有规律，作战参谋团对机动巡逻队的评定是具有一定威胁，避免正面接触。
<br/>　　为了避免跟山路上的机动巡逻队接触，罗昊、李然、李毅峰、慕容飞四个人选择了穿越山林接近山顶别墅，虽然会比走泥石山路花费更多的时间，但是却要更加安全，避免了在抵达山顶别墅前发生交火的可能。
<br/>　　漆黑的山林中，四个人如幽灵般朝着灯火通明的山顶别墅悄然靠近，而此时藏身在别墅中的西伦科特并不知道，一支四个人的特种部队战术小组正向他靠近，准备对他实施斩首行动。
<br/>　　装修豪华的别墅客厅中，巨大的水晶吊灯在四周吸顶灯灯光的照射下，十分炫目。
<br/>　　西伦科特有些烦躁的扯掉自己脖子上的领带，仍在沙发上，站起身走到从桌子上拿起一个玻璃杯，用夹子夹了几块冰块扔进杯子中。
<br/>　　当啷，当啷。
<br/>　　冰块跟玻璃杯碰撞发出清脆的声响，很快玻璃杯上就因为冰块凝结了一层水汽。西伦科特拿起桌子上醒酒器，往手里的玻璃杯中倒了三分之一杯红酒，轻轻晃动着酒杯，随后仰头，将杯中的红酒一饮而尽，把还装着冰块的玻璃杯重重的放在桌子上，对站在客厅中的一名影子部队士兵问道：“拉蒙那边还没有回复吗？”
<br/>　　“暂时还没有，将军！”那名影子部队士兵站直身体，回答道。
<br/>　　“该死的混蛋！”西伦科特脸上闪过怒色，狠声说道：“这家伙不会是拿了钱不想做事吧。”
<br/>　　“需要我们去把他干掉吗？”这名负责对西伦科特贴身进行保护的影子部队士兵声音冷酷的问道。
<br/>　　西伦科特身边有一支对他进行贴身保护的护卫队，这支护卫队队员的平均年龄还不到二十岁，但是他们身上透露出来的那种冷酷，那股杀气，以及对生命的冷漠完全不像是他们这个年龄应该有的表现。这支护卫队的队员全都是被西伦科特领养，并且送进影子部队训练基地接受训练的孤儿，所以他们对西伦科特是绝对忠诚的。
<br/>　　“不。”西伦科特冷静下来，对着那名影子部队士兵摆了摆手，说道：“拉蒙在这里的势力庞大，他手下的第3装甲部队非常精锐，现在跟他交手，对我不利！要不是‘暗月’把我出卖了，我也不会有这么大的损失！这些混蛋！”
<br/>　　这名影子部队队员闭上嘴，沉默着站在一旁。
<br/>　　西伦科特走到落地窗前，看着外面的花园和灯光下的游泳池，说道：“今天晚上好安静，太安静了，安静到让我感觉到不安。”
<br/>　　西伦科特折身返回到客厅，对客厅中的两名影子部队队员说道：“我去地下室看看，你们在这里看着。”
<br/>　　“Yes，sir！”
<br/>　　两名影子部队队员“啪”的一声站直身体，目送着西伦科特离开客厅。
<br/>　　这次的斩首行动是绝密，西伦科特绝对不可能再次提前知道，但是他的直觉告诉他今天晚上，会是一个不平静的夜晚，他察觉到了危机，所以他需要作出一点准备。
<br/>　　在往别墅地下室走去的时候，西伦科特用卫星电话拨通了一个电话，“布克尔特，是我……”
<br/>　　山林中，罗昊、李然、李毅峰、慕容飞四个人依旧谨慎在山林中穿越着前进，他们已经可以看到从山顶别墅中透出的灯光。
<br/>　　罗昊关闭了多功能战术眼镜的夜视功能，抬头看了眼山顶别墅的方向，正打算继续往前行进的时候，罗昊把跨出去的脚又收了回来，然后对着自己脚下的草丛指了指。
<br/>　　李然上前，蹲下身，伸手在罗昊手指的草丛轻轻摸索着，很快就摸到了一根隐藏在草丛中的细钢丝。李然脱去战术手套，在细钢丝上轻轻滑过，嘴上露出不屑的微笑。
