　　第335章
<br/>　　“嗯，大家都这么说。”程大雷点点头：“前面的考核你勉强及格，但想要加入蛤蟆寨，还有最后一项考核，只要你能通过，你就是我们的人了。”
<br/>　　“大当家请讲？”
<br/>　　程大雷坐直了身子，问：“依你看，蛤蟆寨目前最大的问题是什么？”
<br/>　　和珅几乎没有任何犹豫，也向程大雷倾了倾身子。
<br/>　　“大当家想要什么？”
<br/>　　程大雷微微点头，在太师椅上做好，轻轻道出一字：
<br/>　　“然。”
<br/>　　大家都是聪明人，说话一点就透。当然，和珅肯定是聪明人，至于程大雷，他觉得自己也是。
<br/>　　和珅问程大雷想做什么，确切来说，是问蛤蟆寨的目标是什么？
<br/>　　如今的蛤蟆寨已经不是之前三五百弟兄时候，大家称兄道弟也就把所有问题都解决了。而现在山寨人口过万，琴川关附近八百里无主之地都是程大雷的地盘。
<br/>　　这些人凭什么跟着程大雷，当然，是为了吃口饱饭。关于这个需求，程大雷目前是能满足的。但吃饱饭之后呢……程大雷招兵买马，开辟商路，募集人口……是打算乱世逐鹿，还是安稳做自己的土皇帝。
<br/>　　程大雷究竟想做什么，至少要让大家都知道。一个共同的目标，才能将人心凝聚在一起，大家也该知道接下来的路该往哪里走。
<br/>　　和珅刚来第一天，就一句话点中蛤蟆寨的要害，让程大雷不得不佩服。
<br/>　　“好啦，你成功通过考核了，以后你就留在山寨，今天先找房子住下，明天再给你安排具体工作。”
<br/>　　“徐军师已经帮我安排好住处了。”和珅问：“大当家，你究竟想做什么？”
<br/>　　程大雷一挑眉毛：“你管我。”
<br/>　　“呃……”和珅快要摸出程大雷脾气了。
<br/>　　程大雷之所以没有回答，原因是因为，他自己也没能想好。
<br/>　　自己才刚刚起步，某些东西现在想是不是有些太不知天高地厚。
<br/>　　虽然把和珅留下了，可程大雷对这满清第一巨贪怎么放心得下。所以程大雷一直关注着对方，果然，和珅留下后，简直比吴用好用一万倍。
<br/>　　程大雷发现这和珅简直是个工作狂，什么事都愿意做，什么事都还做得不错，颇有些废寝忘食的感觉。当然啦，一个万万人的满清人家都管得了，何况程大雷现在这万把人。
<br/>　　会不会，和珅就是那种乱世之能臣，盛世之贪官。
<br/>　　可是和珅越能干，程大雷也就越揪心，这货别把我的家底都搬到他自己家去。
<br/>　　程大雷决定向系统求助。
<br/>　　“系统大大，有没有什么药物，让人吃了就死心塌地忠诚我的药物？”
<br/>　　良久，程大雷收到了系统回复。
<br/>　　『无』
<br/>　　“怎么可能没有呢？类似于三尸脑神丸、生死符之类的？”
<br/>　　『世上本无所谓忠诚』
<br/>　　“呃……怎么说？”
<br/>　　『人非草木，孰能无情。有人贪名，有人逐利，有人好色，有人以天下为己任，有人只有一片私心。当你能给的和他们志向相同，你就可以获得所谓的忠诚，而道不同时，则分道扬镳。老犬尚且吠主，何况人乎，人并不是机器。』
<br/>　　“那有没有什么可以控制其他人的道具，或者技能？”
<br/>　　『有，交换吗？』
<br/>　　又是这样魔鬼一般的字眼，搞得程大雷心惊胆战。
<br/>　　“都有什么呀？”程大雷准备先看看货再说。
<br/>　　『顶级技能驭人之术，能帮你看破人心，操纵人心』
<br/>　　“感觉操作难度很大呀，有没有简单一点的傻瓜道具？”
<br/>　　『有，魅力光环，对身周范围二十米的人有效，大幅度提升你的魅力值，每天可开启三次，每次持续时间三炷香』
<br/>　　“有什么用？”
<br/>　　『提高你的魅力，魅力值提高，无知者视你为神明，强者视你为领路人，愿为你牵马执鞍，女人愿为你铺床叠被，以身相许。当然，也可能女人为你牵马执鞍，男人以身相许』
<br/>　　程大雷一阵恶寒：“这么牛的属性，交换价格肯定不菲，还有没有别的。”
<br/>　　『首领光环，覆盖范围五十米，每天开启三次，每次持续一炷香，范围提升所有人好感度，只对己方人员有效』
<br/>　　这就是一个削弱般的魅力光环呐，程大雷忽然心中一动，道：“既然有魅力光环，是不是还有弱智光环之类的，我一站出来，所有人智商为零。”
<br/>　　『有。』
<br/>　　“什么价码？”程大雷立刻问。
<br/>　　『目前你所拥有的，不够交换价值』
<br/>　　呃……那好吧。
<br/>　　『可否决定交换什么？』
<br/>　　“我想好了，交换魅力光环。”程大雷道。
<br/>　　『不够交换价值』
<br/>　　“呃……那首领光环呢，够不够？”
<br/>　　『可以，交换么？』
<br/>　　“需要交换什么？”程大雷这次多长了一个心眼。
<br/>　　『随机交换，目前无可奉告，但交换的必然是你不想失去的』
<br/>　　“呃……这么残酷么，对了，为什么我这次会召唤到和珅这样的奸臣，是不是和我失去隐藏属性有关。”
<br/>　　『是的，从系统开启，你一直在获得邪恶值，抢夺压寨夫人，杀人灭口都会增加你的邪恶值，邪恶值影响很多事情，包括你召唤到的人。但之前因为隐藏属性『一个好人』的存在，你的邪恶值一直在被清零，你是怎样的人自然会吸引什么人。』
<br/>　　“也就是说，我和和珅一样都是坏人？”程大雷咧咧嘴。
<br/>　　『你不是坏人』
<br/>　　“哈，我就说么。”
<br/>　　『恶人』
<br/>　　程大雷觉得有些头疼了，也就是说，以后自己能召唤到的只能是奸臣、贪官之类的反面角色，自己的山寨日后该不会成为一个反派集中营吧。
<br/>　　“我是不是可以把一个好人再交换回来？”程大雷抖了个机灵，谁知道这隐藏属性有这么大用处呐。
<br/>　　『不可，请你记住，失去的东西永远不可能再拥有。』
<br/>　　这话就有些残酷了，令程大雷有些戚戚然。
<br/>　　『是否交换首领光环？』
<br/>　　“交换。”程大雷一咬牙，恶人就恶人吧，为了活下去，做个恶人也没啥了。
<br/>　　『获得首领光环，失去技能雄辩』
<br/>　　程大雷松了口气，还好还好，很显然这雄辩没有首领光环来得值钱。
<br/>　　
<br/>(本章完)
