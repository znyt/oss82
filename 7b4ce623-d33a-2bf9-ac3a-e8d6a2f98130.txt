　　罗昊自然看出了李霄力把刘炯文件夹撞掉的举动，是他故意而为之，为的就是进行试探。
<br/>　　不得不说，李霄力这个人的疑心很重，非常谨慎，也正是因为他的疑心和谨慎，没有让杭山市警方抓到太多的证据，多次让他从杭山市警方的眼皮子底下动作频频。
<br/>　　在刘炯和李霄力接触的时候，并且被撞掉文件夹的时候，罗昊已经是把手指扣在了扳机上，如果刘炯被李霄力识破，罗昊会毫不犹豫立即采取行动，警方抓捕李霄力需要证据，但是龙隐部队动手不需要证据，李霄力跟玛丽科娃有关系，只需要这一条就足够了，就是这么不讲道理；但所幸的是，刘炯很顺利的骗过了李霄力的试探。
<br/>　　罗昊将压下二道火的扳机重新松开，刚才如果动手必然不是最好的选择，可是如果刘炯真的遇到危险，罗昊肯定会直接采取行动，不会有任何的犹豫。
<br/>　　“我们的人已经全部撤出来了，随时可以行动。”
<br/>　　罗昊耳麦中传来周宋捷的声音。
<br/>　　“狐狸，把‘探路鸟’放出去，看清楚货轮上的情况。”罗昊按住耳麦，对潜伏在树林子里的李毅峰吩咐道。
<br/>　　旋翼式无人机再次从树林子中飞出，朝着那艘货轮飞去，并且在货轮上盘旋着。周围的龙隐部队队员通过跟无人机相连的PDA看到了货轮上的情况。
<br/>　　货轮因为需要在港口停留一夜的时间，明天装卸完货物后再离开，所以货轮上绝大部分船员都选择下船，去港口附近的旅馆或者酒吧过夜。
<br/>　　只不过这些船员此刻还不知道，他们今天要去酒吧找乐子的想法是要落空了，因为他们很快就会被守在路口的特警控制起来，在港口中的行动没有结束前，他们不会获得自由。
<br/>　　货轮上只剩下了为数不多的几个船员，一个黑人船员走到走到船舷边，对着站在货轮下的李霄力点了点头。
<br/>　　李霄力让几个马仔留在原地警戒后，他自己带着刘三还有剩下的马仔走上货轮，甲板上的黑人船员带着李霄力走向那堆堆放在甲板上的圆木，很快就找到了那根被布克尔特做过记号的圆木，对着圆木指了指，然后自己离开了甲板。
<br/>　　刘三对着跟在自己身后的马仔打了个手势，几个马仔走上前，把那根圆木搬了出来，刘三走到圆木前蹲下身，用手在圆木上摸了摸，找到了圆木上开口的位置，从口袋中拿出一把弹簧刀，弹出刀刃，沿着缝隙小心翼翼地切入，然后把那块盖在cocaine上的树皮撬了起来。
<br/>　　四块包装好的高纯度cocaine出现在刘三面前，刘三用刀划开包装，用刀刃挑了一点出来，用舌尖舔了舔，对着站在自己身边刘三点了点头。
<br/>　　“把东西拿走。”
<br/>　　李霄力身后一个马仔打开随身携带来的背包，将圆木中的四块高纯度cocaine装进了背包中。
<br/>　　发生在货轮上的一幕被无人机没有任何遗漏的拍了下来；而就在李毅峰准备把无人机收回来的时候，李霄力突然间抬头，看到了悬停在货轮上的无人机。
<br/>　　在看到无人机的一刻，李霄力先是一愣，大概只是间隔了不到一秒钟的时间，李霄力就反应了过来，拔出了藏在腰后的手枪，指向天空中的无人机。
<br/>　　“妈的！”
<br/>　　李毅峰咒骂一声，立刻控制着无人机往一侧避开。
<br/>　　嘡！
<br/>　　李霄力手中那本通过特殊渠道从境外弄来的贝雷塔92F手枪枪口爆出一团膛口焰，子弹飞向无人机。
<br/>　　咻！
<br/>　　子弹在夜空中划出一道明亮的弹道轨迹，擦着无人机飞过，消失在夜空中。
<br/>　　天空中的无人机堪堪避开了飞来的子弹，朝着树林子的方向飞去。这无人机体积虽然不大，但是造价可不便宜，要是被一颗手枪子弹给打穿了，那就亏大了。
<br/>　　这种目前只有小批量生产，价格相当于一辆轿车价格的无人机，除了可以用来执行行动前侦察任务外，通过改装，加装特制的微型飞弹，可以破坏电源系统或者提供行动支援。这种旋翼式无人机，体积虽小，但是用处却不小。
<br/>　　负责封锁路口的刘东来听到从港口方向传来的枪声，眼睛微眯，看向港口的方向。
<br/>　　李霄力这种人在刑侦队眼中可能是个“硬茬子”，但是在龙隐部队的眼中，这只不过是一个小角色，一个小到不能再小的小角色。刘东来不清楚，为什么李霄力会把龙隐部队招来，但是刘东来清楚，龙隐部队的目标绝对不会是李霄力，李霄力的面子还真没有那么大，能把军方最精锐的特种部队引来。
<br/>　　“有警察，快走！”
<br/>　　李霄力大喊一声，转身就跑下货轮。
<br/>　　“动手！”
<br/>　　罗昊身体微弓，手中端着那把经过了高度战术改装的56-1，朝着已经逃下货轮的李霄力包围过去。
<br/>　　货轮下，两个手中端着土制猎枪的马仔试图拦截住朝着港口包围而来的罗昊、齐凡等人；可是他们手中的武器和他们本身的战斗力就注定了他们的拦截起不到任何作用。
<br/>　　这两个马仔手中的土制猎枪装得是自制的鹿弹，铅弹的覆盖面虽然广，但是射程很近，超过五十米，基本也就是听个响根本不具备任何的威胁。
<br/>　　咻！咻！
<br/>　　两颗狙击步枪子弹，分别从树林子和龙门吊平台的方向飞来，准确的击中了那两个拿着土制猎枪的马仔。
<br/>　　子弹从两名马仔的前胸穿入，空尖结构的弹头在两名马仔身体中形成比普通钢芯弹更大的空腔效应。虽然不像反器材狙击步枪子弹，一颗子弹就能把一个大活人打成两截，但是子弹穿出两个马仔身体，在他们后背形成的创口，还是足以用“恐怖”两个字来形容，创口皮肉外翻，能看到白森森的骨头，鲜血大股大股的往外喷涌着，两名马仔已然已经气绝……
<br/>　　“妈的，不是警察，是特种部队！”
<br/>　　李霄力看了眼倒在自己不远处的马仔的尸体，心中一惊。
