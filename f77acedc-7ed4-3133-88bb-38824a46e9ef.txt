        &lt;h3&gt;第三卷第一百二十章  聊出来的商机&lt;/h3&gt;
<br/>
<br/>        宁雅元旦的一天假，是胡婷陪着度过了。
<br/>
<br/>        俩个人在学校就是死党，不过胡婷现在嫁人了，虽然也是两家因为事业上相结合，但是他们还是同学，关系还是不错的。
<br/>
<br/>        俩个人在元旦这一天除了买买买还是买买买，似乎女人在一起除了买就是买。
<br/>
<br/>        大包小包的东西塞了一车，俩个人心满意足。
<br/>
<br/>        “哎呀，累死姐了，小雅，走，我们去做个spa，放松放松。”胡婷拽着宁雅又去作个spa。
<br/>
<br/>        “小雅，你也该找个人嫁了。”胡婷说。
<br/>
<br/>        “着什么急啊。”宁雅想起来凌辰，凌辰好像好久都没有消息了，也不知道从贵州回没回来，晚上打电话问一下。
<br/>
<br/>        “你是不是还是不能做主啊？宁叔叔还给你介绍相亲对象吗？”
<br/>
<br/>        “你以为我们的婚姻自己能做主？你不过是比我幸运一些而已，刚巧碰上了老同学。”宁雅淡淡的说。
<br/>
<br/>        “那你打算怎么办啊？你以后是不是就不回去上班了？”
<br/>
<br/>        “有他们在，我还能回去吗？不过这样也好，我安安宁宁的上个班也还不错。”宁雅微微叹了口气。
<br/>
<br/>        “那你甘心吗？”
<br/>
<br/>        “我不甘心能怎么样？我可不想亲者疼仇者快，再说家里给我的够我生活了。”宁雅语气里一点温度都没有。
<br/>
<br/>        “唉。”胡婷叹了口气，宁家比他们家要强很多了。
<br/>
<br/>        “胡婷，那一天陈疯子鼓励我自己做一份事情，她说要是我自己不做事情的话，她给我一份更适合我的工作。”宁雅忧郁的对胡婷说。
<br/>
<br/>        “你是想出来自己做事情？”胡婷是宁雅的闺蜜，当然知道宁雅的心里，纠结着呢。
<br/>
<br/>        “我被她说动了，你说哪有老板鼓励自己的下属出去闯的？”宁雅想到陈子昂的话又觉得好笑。
<br/>
<br/>        “同样都是女人，我也想像她那样活得多姿多彩。”
<br/>
<br/>        “那你想干什么？”胡婷抬头看着认真的宁雅，宁雅的能力她是知道的。
<br/>
<br/>        “我还没有考虑好。”宁雅皱着眉头。
<br/>
<br/>        “你们有钱的话可以开一个美容院啊，听说那个可赚钱了。”给她们做spa的姑娘说。
<br/>
<br/>        “哇，这个可以有，小雅，你赶快开个美容院吧，我是你的第一个顾客，你要给我整个双眼皮，对，眉毛也给我整一下。”胡婷开心的笑着，似乎宁雅真的开了美容院。
<br/>
<br/>        “这个，不是不可以啊。”宁雅也像是发现了新大陆。
<br/>
<br/>        现在的人们都有钱了，无论是年轻人还是上了年纪的人都喜欢美美的。
<br/>
<br/>        有些人都跑国外去。
<br/>
<br/>        要做就做大一点的。
<br/>
<br/>        宁雅的心思活络起来了。改天要好好考察一下，看看市场前景怎么样。
<br/>
<br/>        “小雅，你都不知道现在的女孩子有多么爱美，眉毛、眼睛、鼻子、嘴都可以整成自己喜欢的。”胡婷笑着说。
<br/>
<br/>        “呵呵，现在只要有钱什么都可以做。”宁雅笑着说。
<br/>
<br/>        “小雅，不行了你就开个美人院吧。”胡婷鼓动着宁雅。
<br/>
<br/>        “简单的美容院没有太大的意思，不如整个大了。到时候开了连锁，从国外请专家回来。”宁雅沉思着说。
<br/>
<br/>        “嘿嘿，小雅，你这就计划上了？不亏是我家小雅。”胡婷笑的花枝乱颤“到时候我去给你当管家吧，我天天闷在家里都闷死了，狸猫又不让我出去工作。我和你混他就不说什么了。”
<br/>
<br/>        “行，那就这样说定了。”宁雅大气的说。
<br/>
<br/>        “等明年公司上市和耗材推向市场，我就提出辞职，好好的做一番事情。”宁雅充满着向往。
<br/>
<br/>        “那就这样说定了，我这一段时间也没有事情可干，我先考察考察市场，看看前景到底如何，顺便看看国外的专家。”胡婷头脑也是很清晰的。
<br/>
<br/>        “行，那这一段时间就辛苦了你，我暂时还不能离开公司，目前还没有人能接受我的工作，我最近培养一个人留给陈子昂，走了我也算是心安。”宁雅对那个公司也是充满了感情，虽然才两年的时间。
<br/>
<br/>        “嘿，小雅，你管天管地还管给人家擦，钩子。”胡婷笑着打趣宁雅。
<br/>
<br/>        “唉，你好歹也是高等学府出来的人，能不能有点素质？还钩子呢，踢你两脚。”宁雅也笑的不行了。
<br/>
<br/>        “那就擦脸蛋，擦脸蛋行了吧？哎呦，妈也，等美容医院正的开起来了，我这脸蛋就值钱了，我们还真要管人家的脸蛋了。”胡婷嘻嘻哈哈。
<br/>
<br/>        “爱美之心人皆有之，每个人都喜欢自己美美哒，想一想，从你这里走出去的都是精致而美丽的人儿，想一想我心里都充满了骄傲。不行，我真的是要去做一番事业。”宁雅已经忍不住了。
<br/>
<br/>        “哎呦，说的你好伟大，好像你为人类做了好大的贡献一样。”胡婷瘪瘪嘴。
<br/>
<br/>        “你别说，每个时代的发展都会催生一种行业，行业不分贵贱，只要符合社会的发展就可以。存在必然有它的合理性。”宁雅发表着她的观点。
<br/>
<br/>        “你说的都对，那就动手干啊，还愣着干嘛？”胡婷看着宁雅。
<br/>
<br/>        “干什么？”
<br/>
<br/>        “回家啊，你不回家了吗？光在这里发表长篇大论有什么用，脚踏实地才有意思。”胡婷拉着宁雅找了照镜子。嗯，美美哒！
<br/>
<br/>        宁雅回到了家里，思虑着今天和胡婷讨论的问题，她上网查了一下数据，和她预计的差不多，应该是可以做的。
<br/>
<br/>        不过网上的数据不一定准确，她还是要仔细考察一番。
<br/>
<br/>        拿起手机，下意识的看了看信息，没有凌辰的任何消息。
<br/>
<br/>        这都好多天了没有消息，最后的消息停留在他去贵州的的那一天。
<br/>
<br/>        宁雅拨打着凌辰的电话：“对不起，你拨打的电话不在服务区。”
<br/>
<br/>        一个电话不再服务区，难道两个电话都不在服务区？
<br/>
<br/>        宁雅继续拨打着凌辰的电话，电话接通了：
<br/>
<br/>        “你好。”一个脆生生的女声。
<br/>
<br/>        “你好，我找凌辰。”宁雅迟疑着。
<br/>
<br/>        “他在书房和我爸爸说话。”那个女声回答。
<br/>
<br/>        “你是？”宁雅问。
<br/>
<br/>        “这么晚了，能接听他电话的，你说我是......”电话里的人故意拖长了声音。
<br/>
<br/>        “不好意思，打扰了。”宁雅迅速的挂了电话，凌辰从来都没有说过他有女朋友啊。
<br/>
<br/>        都有女朋友了还天天撩她？这几天不发信息是因为和女朋友在一起？
<br/>
<br/>        宁雅的心里发酸发苦，这么多年，她一直等，等着他，他的情话说的那么溜原来都是骗人的。每一次的相亲，她都找着千方百计地办法推脱，只是等来了这样的结果。
